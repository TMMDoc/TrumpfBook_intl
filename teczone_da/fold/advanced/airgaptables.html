Brugertilpassede luftspaltetabeller

Brugertilpassede luftspaltetabeller

Luftspaltetabellen til en drejebukkemaskine kortlægger en bestemt pladetykkelse til den standardluftspalte, der anvendes under bearbejdningen. Dette er indbygget i hver af de forskellige drejebukkemaskiner, men kan tilsidesættes ved at levere en brugertilpasset luftspaltetabel i form af en JSON-fil. Denne JSON-fil er betegnet air-gap-tables.json og er placeret i C:\ProgramData\Trumpf{APPNAME}\Data\System.

Nedenfor er der en typisk air-gap-tables.json-fil:

 {
  "version":1  "airGapTables": [
    {
      "machine": "112002,112003", "material": "DC01"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
     {
      "machine": "112003", "material": "*"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
    {
      "machine": "*", "material": "Alluminio"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    }
  ]
}

  • Filen indeholder en samling luftspalteværdier. Hver tabel har en maskine og materialetast, der fungerer som vælgere.

  • Disse taster er enten indstillet til * (stjerne) for at angive et jokertegn (matcher alt), eller de er en kommasepareret liste over maskin- eller materialeværdier.

  • I eksemplet ovenfor er der tre tabeller. Den første bruges til maskinerne 112001 eller 112003, kun for materialet DC01. Den anden bruges til alle materialer, tik maskinen 112003, og den tredje bruges til alle maskiner, til materialet Alluminio.Hvis flere tabeller eventuelt matcher delens maskin-/materialekombination, anvendes den første matchende tabel. Så for maskinen 112003, materiale DC01, bruges den første tabel (selvom den anden også passer).

  • Hver tabel består af et simpelt udvalg af tykkelsestaster med de tilsvarende luftspalteværdier. For eksempel bør den første indtastning ovenfor fortolkes som: Hvis tykkelsen er mindre end eller lig med 0,75 mm, brug en spalte på 2,5 mm. Tykkelsestasterne skal være i stigende rækkefølge. Hvis ingen af tabellerne i denne fil matcher den aktuelle dels materiale- og tykkelsekombination, vil standard luftspaltetabel for maskinen blive brugt.

Hvis der foretages ændringer i filen air-gap-tables.json, skal TecZone Fold genstartes, før disse ændringer træder i kraft.
Hvis JSON-filen er fejlbehæftet, vil der blive vist en fejl, når man forsøger at bearbejde en drejebukkemaskinedel, men TecZone Fold vil derefter vende tilbage til at bruge standard luftspaltetabellen for maskinen, og vil fortsætte.
