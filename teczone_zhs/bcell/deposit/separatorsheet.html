添加分隔板

某些堆放模式可以在零件层之间使用 分隔板 以使堆放更加稳定。编辑堆放时选择 Add separator 复选框会添加另一个存放分隔板的托盘。

分隔板面板

您可以单击模拟中的分隔板以调出 上面显示的 Separator 面板。

  • Name 选择器可用于选择不同的分隔板。您可以 在此选择器中选择 Import…​ 链接。

  • Pick (Z,X) 部分用于设置拾取托盘上分隔板的位置和 方向。在上图中， 分隔板已旋转 90°，并移至托盘的一侧。

  • Drop (Z,X) 部分用于设置堆放堆中分隔板的位置和 方向。

  • Gripper (Z,X) 部分用于设置 分隔板上夹持器的位置和方向。这可用于 将夹持器移离分隔板上的任何槽或孔。 （上图显示夹持器相对于默认方向 旋转了 135°）。

  • Simulate 滑块可用于仅模拟分隔板循环， 以便可以快速测试您在上述设置中所做的更改。

  • Stack 链接用于切换到堆放 编辑，Pallet 链接用于编辑 放置分隔板的托盘的位置和方向。

自定义分隔板

您还可以从 File 菜单导入自定义分隔板：

如果正常工作，将显示如下导入消息（否则，将 显示适当的错误消息）。定义自定义分隔板的 DXF 或 GEO 文件格式 在 此页 上进行了说明。

重量检查

分隔板对夹持器来说可能太重。在这种情况下，您可以在 导航器面板中看到错误指示器，指向错误会显示有关 过载的更多详细信息。

吸力检查

如果吸盘位于分隔板上的孔顶部，则会检测到 吸力损失，并在弯曲导航器中 D1 列的 Gripper 行中显示为错误。
