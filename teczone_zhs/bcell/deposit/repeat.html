重复和顺序

重复网格

上面显示的 D1 和 D2 选项卡用于指定堆垛第一个零件的方向。然后，重复选项卡 中的设置用于将其复制到行 (Z)、列 (X) 和层 (R) 的网格中。旁边的图像显示了一个堆 垛，其中许多设置已从默认值更改。

  • *网格*设置用于设置堆垛中的列数和行数。

  • Layers 设置指定每堆中有多少零件堆叠在一起。

  • Gap (Z, X) 设置用于指定连续列或连续行之间的间隙。如果将其设置为 0，则行或列彼此完全接触。

  • Stack shift 设置用于将所有零件的完整堆栈从托盘中心移开。

堆垛顺序

顺序选项卡用于控制零件堆放的顺序。此顺序选项卡上的设置如旁边所示。在本例中， 有两个堆垛方向 D1 和 D2，并且相应地有两组顺序设置（这意味着您可以从一个角开始 D1 层，而 D2 层可能从不同的角开始）。

  • Stack 设置有两个选项：By Layer 或 ui:4738[By Pile]。在此示例中，按层堆垛将同时构建两个相邻的堆 - 这意味着连续的堆垛将在 堆 1、堆 2、堆 1、堆 2 等中进行。因此，两个堆将一起升高。当您按堆堆垛时，左 侧堆的所有零件都先堆放，然后我们才开始在右侧堆上放置任何零件。

  • Start 角和 Path 设置在有多行或多列 时使用，并决定堆垛从哪个角开始，以及堆垛然后是逐行进行还是逐列进行。这在使用偏心 夹具时特别有用，因为不同的起始角可能会导致夹具与已堆放的前一个零件之间发生碰撞。

  • 图像显示了一个示例，我们从右后角开始，然后继续进行 X-First 堆放（数字 1、2、3、4 显示此层内的堆垛顺序）。如果我们从底部前左角（默认）开始此堆垛， 则在堆放此层中的第二个零件时，夹具将与前一个零件发生碰撞（因为它从零件边界偏心延 伸出来）。

  • 模拟设置（数字旋转器和滑块）可用于滚动浏览堆垛，以查看它们在托盘上发生的顺 序。在调整起始角和路径设置时，这特别有用。在上面的图像中，我们看到的是第 17 个零 件（共 20 个）的堆垛。如果我们将滑块向右移动（或调整模拟输入框中的值），您可 以查看夹具在第 18、19 和 20 次堆垛的位置。
