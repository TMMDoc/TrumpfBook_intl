编辑改抓

重新抓取后，夹具的最终位置仍可使用 Gripper 面板 进行调整。然后，此位置将保留以便用于即将进行的弯曲过程， 或直到执行新的抓取为止。

编辑重新抓取位置使用 Gripper 面板完成。

访问 Gripper 面板：

  • 直接单击机器人夹具。

  • 从改抓面板导航菜单中单击 Gripper 选项。

夹具面板

使用夹具面板，可以修改机器人夹具的位置和方向。 此外，可以配置夹具类型和每个吸盘点（杯）之间的间距。

  • 使用 Name 字段选择夹具类型。

  • 选中 Filtered 复选框以仅查看适用于 给定弯曲操作的夹具。

  • 使用 Position 和 Angle 字段更改零件上方夹具的位置和方向。

    以下是可用的夹具 Actions…​：

  • 单击 Configure 选项以更改单个夹具的位置。这将打开一个 带有名称的新夹具面板。

    • Arm

    • Slide

    • Stretch

    • Symmetry

    • Presets

      • Small

      • Large

      • Wide

      • Tall

    • Import…​

  • 使用 Set Grip Plane 选择不同的平面进行抓取。 选择此选项时，鼠标指针将变为蓝色加号。在周围悬停 以查看显示不同平面方向的加号，并选择所需的平面。 下面的图像显示了给定零件的可能抓取平面：

    夹具将吸附到所选平面，并相应地更新夹具位置和方向。

吸盘面板

典型的吸盘面板显示在旁边。

访问吸盘面板：

  • 从夹具面板单击 Suction 导航菜单。

  • 直接单击机器人夹具中的任何吸盘。

使用 Group # 选项在所有可用吸盘中选择特定的吸盘。 使用 State 选项将操作状态更改为 on、off 或 removed。 从可用列表中选择吸盘的 Type。选中 Change All 复选框以一次性更换所有吸盘。
