Bend deductions

Bend deductions

Bend deductions 用于计算钣金零件的展开长度[1]。准确生产折弯后具有正确尺寸的零件需要此值。

Bend deductions 考虑了折弯过程中发生的材料拉伸和压缩。通过使用 bend deductions，TecZone Bend 可以调整平面图案长度，以确保最终折弯的零件符合指定的尺寸。

Bend deductions 可能因材料类型、厚度、折弯半径和折弯方法等因素而异。

当将 3D 零件导入 TecZone Bend 时，软件会根据零件的几何形状和材料属性自动计算 bend deductions。如果需要，您还可以手动调整 bend deductions 以微调平面图案长度。

Bend deduction 设置

可以在 Bend Deductions 数据库中设置自定义 bend deductions。要访问，请单击 database 图标并选择 ui:EGTag.CustomBendDB[Bend Deductions]。

您可以 Add 添加新的 bend deduction 场景，或选择现有场景进行 Edit 编辑或 Delete 删除。您还可以 Copy 复制现有场景以创 建具有相似设置的新场景。

单击 Add 以创建新的 bend deduction 场景。选择并输入以下参数：

  • Material：选择 bend deductions 适用的材料。

  • Thickness：设置 bend deductions 适用的板材厚度。

  • Punch radius：设置 bend deductions 适用的上模具半径。

  • V-width type：选择使用的 V 形宽度测量类型。

  • Die V-width：设置 bend deductions 适用的下模具宽度。

  • Die radius：设置 bend deductions 适用的下模具半径。

  • Die angle：设置 bend deductions 适用的下模具角度。

单击 OK 保存新场景。

此外，选择一个 bend deduction 场景并单击 …​ 选项以查看更多选项：

  • Graph：显示折弯角度与 bend deduction 之间关系的图表。此可视 化表示有助于了解 bend deductions 如何随不同的折弯角度而变化。将光标移动到图表的 蓝色 和 红色 线上以查看特定值。

  • Import：允许您从外部 CSV 文件导入 bend deduction 数据。

  • Export：使您能够将当前 bend deduction 数据导出到 CSV 文件以进行备份或共享。

  • 3rd party：提供从第三方软件或来源导入 bend deduction 数据的选项。

Bend deduction 类型

TecZone Bend 支持三种类型的 bend deductions：Air Bending、Folding 和 Coining。

每种类型都有自己的参数和计算集，可根据使用的折弯方法准确确定 bend deductions。

Air Bending

Air bending 是一种常见的折弯方法，其中使用冲头和冲模折弯钣金而不完全关闭冲模。此 方法允许更大的灵活性，适用于各种材料和厚度。

要为 air bending 添加 bend deductions，请从 Bend Deductions 数据库中选择合适的场 景。单击 Add 以创建新的 air bending deduction 场景并输入 ui:965[Bend Angle]。

当您输入折弯角度时，相应的 Deduction (Edge)、Deduction (Tangent) 和 Inner Radius 值将被计算并显示。

单击 OK 保存新的 air bending deduction。

Folding

Folding 或 Hemming 是一种折弯方法，其中使用完全闭合的冲头和冲模折弯钣金，从而产 生尖锐的折弯。此方法通常用于较薄的材料，需要精确控制 bend deductions。

要为 folding 添加 bend deductions，请从 Bend Deductions 数据库中选择合适的场景。 单击 Add 以创建新的 folding deduction 场景并选择 Hemming method。

单击 OK 保存新的 folding deduction。

Coining

Coining 是一种折弯方法，其中钣金在高压下被压在冲头和冲模之间，从而产生精确和尖锐 的折弯。此方法通常用于较厚的材料，需要准确的 bend deductions。

要为 coining 添加 bend deductions，请从 Bend Deductions 数据库中选择合适的场景。 单击 Add 以创建新的 coining deduction 场景。

单击 OK 保存新的 coining deduction。

1. 展开长度是折弯前平板金属的总 长度。这是平面图案的实际长度，该图案被切割（使用激光切割机）并成型（使用折弯机） 以创建最终零件。
