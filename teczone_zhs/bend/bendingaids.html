折弯辅助装置

折弯辅助装置是折弯机的机器组件，用于帮助折弯较厚的板材。 它们旨在在折弯过程中提供额外的支撑和控制，确保准确且一致的结果。

折弯机可以根据折弯操作的具体要求，安装多个折弯辅助装置或单个折弯辅助装置。

折弯辅助装置仅适用于TruBend 5000和8000系列。

配置折弯辅助装置

折弯辅助装置可以在 Database / Machines / Settings 中配置。

当导入零件并选择了带有折弯辅助装置的机器时， TecZone Bend会根据零件的要求自动配置折弯辅助装置。

使用折弯辅助装置

一旦生成可行的折弯解决方案，可以进一步调整折弯辅助装置以优化折弯过程。

直接点击TecZone Bend中的折弯辅助装置以打开 Bend Aid 面板。 折弯辅助装置面板中的数字表示折弯顺序。

  • Index - 表示当前选定的折弯辅助装置。

  • Position (Z,R) - 用于调整折弯辅助装置沿Z轴和R轴的位置。

  • Enabled - 选中此复选框以激活折弯辅助装置。

  • Auto-Place - 用于根据零件的几何形状自动定位折弯辅助装置。

Speed:

  • Down - 用于调整折弯操作后折弯辅助装置的缩回速度。

  • Set All Bends - 用于将当前速度设置应用于顺序中的所有折弯。

Advanced:

  • Method - 用于定义折弯辅助装置的操作模式。选项包括：

    • Default - 将折弯辅助装置设置重置为默认值。

    • Return at UDP - 配置折弯辅助装置在每次折弯后返回到用户定义的位置。

    • Tilt product - 使折弯辅助装置在折弯过程中倾斜零件以实现更好的对齐。

    • Static angle - 允许在操作期间为折弯辅助装置设置固定角度。

  • Stop angle - 用于设置折弯辅助装置在操作期间停止的角度。

折弯辅助装置停放位置

折弯辅助装置在不使用时可以停放在指定位置。这有助于 优化工作空间并确保操作期间的安全。这还可以 防止与其他机器组件发生碰撞。

折弯辅助装置可以停放在以下位置：

  1. Default - 折弯辅助装置可以停放到工作台长度。

  2. Small - 折弯辅助装置可以放置在超出工作台长度700毫米的位置。

  3. Large - 折弯辅助装置可以放置在超出工作台长度1200毫米的位置。

对于 串联 机器，有四个折弯辅助装置可用。对于左串联 机器，只允许左侧停放。对于右串联机器，只允许 右侧停放。

停放位置选项仅显示于TruBend 8000系列 和VarioPress机器。

对于B36串联机器，停放位置 small 不适用。

创建自定义折弯辅助装置

为了在TecZone Bend中创建自定义折弯辅助装置，您首先需要创建一个 ini 文件来定义折弯辅助装置的参数，以及折弯辅助装置组件（滑架、工作台和板）的 mesh 文件。

示例baid.ini文件
[BAData]
; The user defined name of the BendAid. Only used for custom bend aids
Name=BATest
; The range of V-widths the bend aid can support
DieV=16,250
; The range of die-heights the bend aid can support
DieHeight=-30,250
; Maximum angle, in degrees
MaxAngle=55
; Lift capacity, in kg
Payload=300
; Maximum lift speed, in degrees/second
Speed=45
; When the bending aid is at X=0, this is the bound of the support table in top view
Table=-159,-1032,159,-245.6
; The X-Span of the bending aid (when positioned at 0)
XExtent=-213,213
; Tells whether the user can change the forward speed
CanChangeForwardSpeed=False
; Tells whether safety stop angle is possible in the machine
CanStopAtReturn=True
; Can the bending aid movement be varied using different methods?
AdvancedControl=True
; What type of existing bending aid can this be mapped to? Used only by custom bend aids
EType=TB8000_2000N_B36

[Carriage]
Model=Carriage.mesh
Stroke=-1000,1000
Color=White
Shift=0,0,100

[Table]
Model=Table.Mesh
Stroke=50,200
Color=White

[Plate]
Model=Plate.mesh
Stroke=0,56
Color=#4A4A4A

将ini文件和相应的mesh文件放在一个文件夹中，然后可以 将其导入TecZone Bend。

导入自定义折弯辅助装置

导入自定义折弯辅助装置：

  1. 导航到 File / Import / Bending Aids…​ 并选择已创建的 ini文件（确保相应的mesh文件在同一文件夹中）。

  2. 导入的折弯辅助装置将在机器设置中的折弯辅助装置 选择下拉菜单中可用。

当需要为特定折弯任务定制折弯辅助装置或使用默认TecZone Bend数据库中未包含的专用折弯 辅助装置时，此选项很有用。
