导入/导出

在本节中，我们将介绍Import settings的配置。

单击主页上的Settings图标。

Import settings

在本节中，我们将介绍Import settings的配置。单击图Settings标。单击Import/Export并导航到Import settings。

Units for DXF files - 在此处设置为毫米或英寸。

Stitch together lines/arcs closer than this - 设置此值（0 > 1mm）。当导入的零件具有比额定值更近的线/圆弧时，软件会在导入时自动拼接。

Maximum thickness for sheet-metal part - 要识别大型板材零件，可能需要提高板材识别阈值。(10>40mm) 默认情况下自动设置为 25 毫米或一英寸，具体取决于所使用的单位。

Point entities - 根据所选的选项将确定点如何导入。Import all - 将导入并显示所有的点。Skip points on polylines - 这会跳过折线上检测到的所有点。Skip all - 这会在导入时跳过所有点，并且不会显示任何点。

Ignore layers in DXF/DWG files - DXF 和 DWG 图纸通常在不同的图层上创建。通过此设置，软件会忽略这些图层并将所有对象移动到默认图层上。

Explode blocks in 2D drawing - 在导入期间拆分分组

Convert white entities to black - 打开此设置可在导入期间将白色对象转换为黑色。

Darken colors during DXF import - 打开此设置可在导入期间使彩色对象变暗。

Remove duplicate segments - 打开此设置可删除导入时在零件中找到的任何重复的几何形状。

DXF settings

在本节中，我们将介绍DXF settings的配置。点击 Settings 图标。单击 Import/Export 并导航到 DXF settings。

Angles in DXF are interior angles - 激活此设置可以将 DXF 中的角作为张角处理。

Export settings

在本节中，我们将介绍Export settings的配置。单击Settings图标。单击Import/Export并导航到Export settings。

No POLYLINE objects in DXF output - 通常，闭合轮廓在导出 DXF 时被输出为折线。某些 CAD 系统无法处理此输出。通过这个设置，软件输出带有线和圆弧的 DXF。这些文件在任何地方都可读，但创建的文件较大，并且线和圆弧之间的连接丢失

Output bend-info when saving DXF files - 打开此设置可导出将输出折弯参数的 DXF。

Bend-info in Starmatik format - 打开此开关可输出 Starmatik 格式的折弯信息。这是文本实体正好位于每条线（将成为折弯线）中间的位置。

Convert black to gray on output - 导出2D维数据时，DXF 文件中的对象默认情况下输出为黑色。为了能够更好的识别 CAD 程序中的物体，使用此设置对象将以灰色输出。

Start MetaCAM when PDG files are exported - 打开此设置，用于将自动配置以在 MetaCAM 中打开的 PDG 文件

Format for flat pattern - 导出平面图案时，可以将其设置为 GEO、DXF 或 PDG 文件格式。

Spline conversion

在本节中，我们将介绍 Spline conversion 设置的配置。单击Settings图标。点击 Import/Export 然后导航到 Spline conversion设置。

Convert splines on import - 在此设置样条曲线转换是否关闭或设置为“将样条曲线转换为线或圆弧”。在任何一种情况下，每个样条曲线都将转换为包含线段或圆弧段的单个折线对象。

Node-count computation - 计算生成的线或圆弧的数量，使用两种机制之一：节距或偏差。

Length of each line or arc segment - 如果在节点计数计算中选择节距，在此处设置每个圆弧或线段的长度，以使用此步长分解样条曲线。

Maximum deviation during approximation - 如果在节点计数计算中选择偏差，在此处设置原始平滑样条曲线与线或圆弧求近似值之间允许的最大偏差。折线构造为尽可能少的段，同时仍将最大误差保持在此限制内。

Assembly Explode

在本节中，我们将介绍Assembly Explode设置的配置。单击Settings图标。点击 Import/Export 然后导航到 Assembly Explode设置

Skip sheet-metal parts with no bends - 分解组件时，打开此开关将仅显示带有折弯线的板材零件。关闭将显示组件中的所有零件。

Components (nuts/bolts etc) to display - 使用此设置来选择在分解包含其他部件的组件时要显示的选项。

None - 分解后仅显示板材零件，不显示其他螺母/螺栓。

Unidentified - 仅显示软件中待识别的部件。

All - 显示所有部件。

Layer mapping

在本节中，我们将介绍 Layer mapping的配置。点击 Settings 图标。点击Import/Export 并导航到Layer mapping设置。在本节中，正在导入到软件中的零件上所使用的图层可以自动映射到它的功能（使用）。

示例：

如果导入的零件具有 MARK 图层，则可以将其设置为自动使用软件中的标记图层。

Layer name - 这是图层的名称，如果使用此名称导入零件，它将使用“使用”面板中设置的功能。

Use - 这是图层的功能。可用的不同选项包括：

Standard - 这是用于CAM的标准图层。

Auxiliary - 辅助图层，不适用于 CAM。Mark - 该图层中的任何实体都将被标记，而不是被切割。

Approach marker - 指示激光进刀位置的点实体。

Sequence marker - 指示轮廓顺序的文本标记。

Forming center - 成形加工中心标记（点或小 L）。

Forming foot print - 成形加工的轮廓（足迹）。

Evaporate - 该图层将用于区分薄膜燃烧。

Dot marking - 该图层将用于 QR 码。

Info - 这是一个仅提供信息的图层。
