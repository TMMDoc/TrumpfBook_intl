編輯改抓

重新抓取後，夾具的最終位置仍可使用 Gripper 面板進行調整。然後，此位置將保留以 便用於即將進行的彎曲過程，或直到執行新的抓取為止。

編輯重新抓取位置使用 Gripper 面板完成。

存取 Gripper 面板：

  • 直接點擊機器人夾具。

  • 從改抓面板導航選單中點擊 Gripper 選項。

夾具面板

使用夾具面板，可以修改機器人夾具的位置和方向。此外，可以配置夾具類型和每個吸盤點 （杯）之間的間距。

  • 使用 Name 字段選擇夾具類型。

  • 選中 Filtered 核取方塊以僅查看適用於 給定彎曲操作的夾具。

  • 使用 Position 和 Angle 字段更改零件上方夾具的位置和方向。

    以下是可用的夾具 Actions…​：

  • 點擊 Configure 選項以更改單個夾具的位置。這將打開一個 帶有名稱的新夾具面板。

    • Arm

    • Slide

    • Stretch

    • Symmetry

    • Presets

      • Small

      • Large

      • Wide

      • Tall

    • Import…​

  • 使用 Set Grip Plane 選擇不同的平面進行抓取。 選擇此選項時，滑鼠指標將變為藍色加號。在周圍懸停 以查看顯示不同平面方向的加號，並選擇所需的平面。 下面的圖像顯示了給定零件的可能抓取平面：

    夾具將吸附到所選平面，並相應地更新夾具位置和方向。

吸盤面板

典型的吸盤面板顯示在旁邊。

存取吸盤面板：

  • 從夾具面板點擊 Suction 導航選單。

  • 直接點擊機器人夾具中的任何吸盤。

使用 Group # 選項在所有可用吸盤中選擇特定的吸盤。 使用 State 選項將操作狀態更改為 on、off 或 removed。 從可用列表中選擇吸盤的 Type。選中 Change All 核取方塊以一次性更換所有吸盤。
