RG-工位面板

在 TecZone 中，重新抓取工位稱為 RG-Stations。 RG-Station 面板用於配置彎曲單元機的重新抓取工位。

存取 RG-Stations 面板：

  • 從 Pickup 面板、Bend 面板、Regrip 面板或 Deposit 面板點擊 RG-Stations 連結。

  • 或者直接點擊重新抓取工位。

根據彎曲操作和工位的位置， RG-Stations 面板會在面板內顯示字母指示器（P、R1、#1、D 等）。

Pickup 面板中的 Flip Part 核取方塊用於相對於目前位置 將零件上下顛倒放置。有時這對於避免向下的法蘭與重新抓取工位手臂之間的碰撞很有用。 當您翻轉零件時，TecZone 將立即重新計算新的機器人路徑。

當不使用重新抓取工位時，面板看起來很簡單，因為零件位置無法控制 （它取決於零件在沖頭和模具之間的夾緊位置）。

  • Stations 用於選擇任何一個工位或 停用或啟用兩個工位。

  • 使用 Positions 設定調整機床導軌上的 工位位置。在下面顯示的圖像中，左側的工位一開始沒有放置在零件下方。

    在更改 Positions 選項中的值後，兩個工位都將定位， 使 RG-工位的吸盤政置在零件下方。

Suction 部分中的設定用於 配置吸力手臂。

  • 使用 Type 選擇器更改吸盤類型。

  • 使用 Arm 選擇器選擇任何一個或所有吸力手臂。 選擇後，每個手臂通過其標籤（A、B、C 等）進行標識。

  • 使用 Angle 設定旋轉手臂（以 5° 為步長）。

  • 使用 Grid 設定伸出/縮回手臂（以 10 毫米 為步長）。

您也可以通過單獨拖動滑塊來互動式地更改角度和網格。將滑鼠懸停在吸力手臂上以查看角度和網格滑塊。

當選擇 All 手臂時，無法互動式地更改角度和網格。

  • 將吸力手臂的操作*狀態*選擇為 off、on 或 removed。

  • 完成吸力設定後，您可以使用 Copy Right 選項將相同的設定用於 RG-工位的另一側。

  • 如果此零件有多個重新抓取操作，則會顯示 Prev 和 Next 連結，用於在它們之間導航。

  • Regrip R1 連結用於編輯零件在重新抓取工位上的 靜止位置。

當您重新配置重新抓取工位時，TecZone 會實時檢查與零件、機器人、夾具的碰撞，並立 即更新狀態指示器。
