路徑點

概述

BendMaster 的每次移動都包含 waypoints， 這些路徑點會依次通過。移動的所有路徑點由 TecZone 決定， 但大多數可以調整。與彎曲過程中的追蹤直接相關的點無法更改。 以下列出了用於控制路徑點的各種設定：

  1. 更改路徑點的位置

  2. 插入新路徑點或移除現有路徑點

  3. 調整運動屬性，如速度和插值

  4. 在所需位置新增新路徑點

要存取 Waypoints 面板：

  • 直接點擊機器人軌道

  • 從拾取面板、彎曲面板、重新夾持面板或放置面板點擊 Waypoints 連結

  • 點擊任何 RG-Stations 面板中的路徑點連結

Waypoints 面板

旁邊顯示了一個典型的路徑點面板：

  • Phase 滑桿顯示路徑點的逐步移動。每個 階段反映特定的機器人操作，如拾取、彎曲、重新夾持和放置。

  • 使用 Display 選項檢視：

    • grip point

    • wrist point

    • robot axes。

  • 顯示並可以編輯手腕或夾持器的位置和角度。如果 選擇了機器人軸，則顯示並可以編輯機器人軸值。

  • 黃色軌跡顯示從上一步到此步以及從此 步到下一步的路徑（見下圖）。3 個小三角形表示 機器人（手腕或夾持器）的局部 X 和 Y 座標。

  • Simulate 滑桿用於將模擬從 上一階段移動到下一階段。如果這在中間，那就是我們正在編輯的 實際階段。

  • Interpolate 設定用於在 線性和軸向插值之間切換。

  • Speed 和 Rounding 設定用於 編輯此步驟的速度和精度設定。

  • Regrip stations 欄位用於沿其軸更改重新夾持站的位置。

  • Insert 選項用於透過輸入新值來創建新路徑點。 新增的路徑點可以透過 Remove 選項移除。

  • 使用 Prev 和 Next 按鈕轉到 下一步或上一步（或者您也可以使用左右箭頭鍵）。 點擊 Next 按鈕將顯示階段中的每一步。 在階段結束時，將顯示下一個對應的路徑點面板。

路徑點的屬性

根據過程的狀態，每個路徑點都會獲得一組屬性， 這些屬性與插值類型、速度和移動精度有關。 如有需要，可以更改這些屬性。

這些設定在路徑點面板的進階部分中控制。

修改 waypoints

如有需要，可以更改路徑點的位置，例如，以防止越過軸。

在與碰撞避免相關的其他方面中，可以 消除不需要的路徑點或新增新路徑點。

  • 當更改現有路徑點位置時，TecZone 會顯示警告訊息。 點擊 OK 繼續更改路徑點。

  • 點擊 Insert 按鈕將新增新路徑點。

  • 新增的路徑點在階段名稱中以 +1、+2 等表示，如下所示：

模擬 waypoints

預設情況下，simulate 滑桿位於中間位置。 使用此滑桿可以無縫模擬機器人從一個路徑點到另一個路徑點的移動。
