編輯模具裝置

當您點擊模具裝置時，會顯示折彎裝置面板，由此可以編輯折彎裝置的各種設置，並執行多個操作。

折彎裝置面板

折彎裝置面板看起來像旁邊的圖像。可用的確切設定和操作會有所不同，具體取決於您點擊沖模、凹模還是適配器裝置。此外，某些設定可能可用或不適用，具體取決於機台功能。

  • 點擊Change Tool按鈕，可使用不同的模具替換用於所選裝置的模具。（有關用於選擇替換模具的_模具選擇器_的詳情，請參閱以下章節）。

  • Length 輸入用於調整工站的長度。當您在此處輸入新值時，TecZone Bend將再次使用一組適當的_段_來組合工站，以嘗試盡可能匹配要求的長度。您可以透過查看檢查顯示在折彎裝置上的段邊界線，來目檢使用的段。

  • Position 輸入用於設定裝置的_左邊緣_的位置，沿着機台的工作台或壓板。位置也可以透過向左或向右_拖動_裝載進行調整。（請參閱以下有關拖動裝置的章節）。

  • Ear 選擇器（僅顯示用於沖模裝置）可用於強制TecZone Bend使用左側和/或右側ear[1]組合中的件）。

  • Flip Tool 用於翻轉模具（從前到後）。當您編輯使用_非對稱_模具（如鵝頸模具）的裝置時顯示。翻轉操作足夠智慧，可以計算出：是否需要翻轉其他支架和模具以保持一致性。例如，在下圖中，翻轉OW_Z4沖模（用於Z折彎）也會同時翻轉凹模、支架甚至零件，以維持過程的一致性：

  • Add Adapter 用於添加_擴展器_以增加沖模或凹模的高度。當您添加適配器時，適配器會被選中並顯示適配器的面板（這允許您更改實際使用的適配器或移除它）。

  • Add Gap 用於在模具裝置中添加間隙，以允許法蘭在沒有碰撞的情況下通過（請參閱下面關於添加間隙的部分，了解更多詳情）。

  • Taller tool 將用來自同一系列的更高模具替換所選的模具。

  • 點擊Remove以移除模具裝置。在使用此工站的折彎的折彎導航器中顯示_缺少沖模_或_缺少凹模_錯誤，這可以透過使用命令列上的*添加*來修復，以添加新裝置。

  • ACB Sensor 選擇器用於瀏覽可用於此零件的不同ACB探片對。選擇取決於板材厚度，以及正在使用的沖模。

  • 如果勾選Inventory check核取方塊，則根據可用的件_庫存_檢查所有的段（可以點擊Inventor連結進行編輯）。如果使用不在庫存中的段，將在其上繪製一個特殊的三葉草符號，如您在下圖中所見——靠近沖模中心的40mm和45mm件：

當您使用任何這些設置和操作編輯模具裝置時， TecZone Bend將立即驗證所有折彎操作的狀態，執行各種檢查，如碰撞、模具可用性等。折彎導航器狀態立即實時更新，使您可以輕鬆實驗各種設定，並立即和準確獲得反饋。

替換模具視窗

當您點擊Change Tool按鈕時，替換模具視窗出現：

此視窗顯示所有可能的替換模具。

  • 使用左側的層次結構來縮小選擇範圍 - 例如，您可以選擇只顯示鵝頸模具，讓選擇更輕鬆。

  • 使用頂部的排序選擇器，可按名稱、高度、半徑或其他排序條件排序（確切的排序條件集取決於您是否更換沖模、凹模或適配器）。

  • 您可以在搜尋框中輸入模具名稱（或簡稱），以快速縮小清單範圍。也可以輸入部分模具名稱 - 例如使用OW200，將匹配OW200、OW200/S和OW200/K模具。

  • 使用縮放滑塊調整模具圖像的大小。目前使用的模具採用藍色填充和粗輪廓。淺藍色十字陰影表示此零件使用的_其他模具_。

  • 關閉篩選核取方塊，即可不受限制地查看所有模具（無論是否合適）。在這種視圖下，不合適的模具是灰色的，將滑鼠移動到其中一個模具上會顯示它_為什麼_無法選擇：

當您將滑鼠移動到模具圖像上時， TecZone Bend將_立即_使用新選擇的模具重新計算所選的模具裝置，您可以實時看到正在考慮的模具是否會導致任何碰撞問題（如有碰撞，折彎導航器中的相應折彎將立即以紅色亮起）。這種實時預覽讓您非常輕鬆地實驗使用各種模具選擇，直到找到一個有效的模具為止。如果您點擊其中一個模具，系統會作出選擇並應用新模具到零件。反之，如果您按下Esc，您所做的預覽選擇將被恢復，原始模具保持不變。

導航與選擇

折彎裝置面板在_導航_和_選擇_子面板中顯示各種按鈕。

  • 使用Setup導航按鈕向上移動以編輯整個折彎設定。詳情請參閱設定編輯器頁面。

  • 使用Prev和Next按鈕循環編輯此設定中的不同模具裝置。

  • 使用Segments導航按鈕向下移動一個級別並編輯組成此裝置的_單個段_。關於編輯模具裝置段的以下章節提供了更多相關資訊。

  • 使用Tool Station選擇器選擇組成這個_工站_的所有模具裝置。這將選擇與所選裝置對齊的所有沖模、凹模和支架，因此它們可以一起編輯或移動。

  • 使用Same Tool選擇器選擇使用相同的模具的其他模具裝置。這通常在使用_更換模具_操作之前很有用；使用指定模具選擇所有工站可擴大可用的替換模具選擇範圍。[2]

編輯多個裝置

如果您使用Shift+單擊所有裝置來選擇多個裝置，則可以將它們一起編輯。只有所有裝置通用的設置和操作才可供編輯。只有當它們對所有裝置都相同時，才顯示用於編輯的字段，例如Length 或Position。如果您有多個工站，則宜在執行更換模具操作之前選擇所有沖模或所有凹模。在這種情況下，可用的替換模具選擇範圍更廣泛（因為沒有不同高度的沖模和凹模引起碰撞的危險）。

拖動折彎模具裝置

Position 輸入可用於設置裝置的精確位置。有時，將折彎裝置拖動到新位置會更容易。步驟如下：

  • 點擊折彎裝置以選擇它（使用Shift+點擊選擇多個裝置）

  • 點擊並開始向左/向右拖動以移動所選裝置。如上圖所示，當您將滑鼠移動到所選裝置上時，會顯示箭頭，表示您可以拖動所選裝置。當您拖動裝置時，卡入顯示圖素可讓您可以輕鬆將裝置對準其他現有裝置。如果您_拾取_一個靠近其左側邊緣的裝置，則所有裝置的左側邊緣將被用於卡入。如果您在中心拾取一個裝置，那麼中心線被用於卡入，以此類推。

進階操作

以下是使用折彎裝置編輯器面板的一些更高級的操作。

編輯模具裝置段

編輯折彎模具裝置時，點擊段導航按鈕將打開_模具段_面板，看起來像旁邊的圖像。

  • Segment 選擇器用於將所選模具段替換為更長或更短的件，或不同類型的件。

  • < Shift 和Shift >按鈕用於在組合中將所選段向左或向右移動。這不會改變組合物的整體長度，但對於向左或向右移動止擋感應器件很有用，例如用以避免孔。

  • Add 和Remove按鈕用於添加新段或將所選段從組合中移除。

  • Prev 和Next導航按鈕用於循環編輯折彎裝置中的不同段。如上圖所示，正在編輯的模具段以藍色突出顯示。

  • Mount 導航按鈕用於向上移動一個級別，並編輯整個折彎裝置，而不是單個段。

在裝置中添加間隙

有時在模具裝置上增加一個狹窄的_間隙_是有用的，通常是為了讓法蘭通過而不會發生碰撞。若要執行此操作，請點擊 Add Gap 按鈕（如果模具裝置足夠長，則會顯示該按鈕）。在面板中打開一個小區域，帶有一組相互關聯的輸入來設置左邊距、右邊距和實際間隙。由於這三者的總和必須加到模具裝置長度上，編輯其中兩個將自動設置第三個。下圖顯示正在進行的操作。因為一個法蘭與沖模碰撞，我們調整左/右邊距，直到建議的間隙與法蘭和沖模相交的位置對齊（您可以看到建議的間隙顯示為折彎裝置上兩條橙色線）。

使用雙V適配器

可以使用雙V適配器將兩個凹模彼此相鄰安裝。要手動執行此操作，請使用Add Adapter按鈕添加適配器到凹模上，然後使用Change Tool按鈕來更改適配器到雙V適配器。現在，可以將第二個凹模添加到另一個雙V適配器的槽，使用命令列中的添加命令來選擇凹模裝置。當您將凹模安裝在雙V支架的一個槽中時，您可以透過點擊顯示的DV Shift按鈕將其移動到其他槽：

1. 這些有時也被稱為_肩膀_件。
2. 例如，如果您正在僅一個沖模站執行更換模具操作且有多個沖模站，那麼您將被限制在僅選擇_相同高度_的其他沖模模具，以避免因間隙不均勻而導致沖模凹模碰撞。
