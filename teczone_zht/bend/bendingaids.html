折彎輔助裝置

折彎輔助裝置

折彎輔助裝置是折彎機的機器組件，用於幫助折彎較厚的板材。 它們旨在在折彎過程中提供額外的支撐和控制，確保準確且一致的結果。

折彎機可以根據折彎操作的具體要求，安裝多個折彎輔助裝置或單個折彎輔助裝置。

折彎輔助裝置僅適用於TruBend 5000和8000系列。

配置折彎輔助裝置

折彎輔助裝置可以在 Database / Machines / Settings 中配置。

當匯入零件並選擇了帶有折彎輔助裝置的機器時， TecZone Bend會根據零件的要求自動配置折彎輔助裝置。

使用折彎輔助裝置

一旦生成可行的折彎解決方案，可以進一步調整折彎輔助裝置以優化折彎過程。

直接點擊TecZone Bend中的折彎輔助裝置以打開 Bend Aid 面板。 折彎輔助裝置面板中的數字表示折彎順序。

  • Index - 表示目前選定的折彎輔助裝置。

  • Position (Z,R) - 用於調整折彎輔助裝置沿Z軸和R軸的位置。

  • Enabled - 選取此核取方塊以啟動折彎輔助裝置。

  • Auto-Place - 用於根據零件的幾何形狀自動定位折彎輔助裝置。

Speed:

  • Down - 用於調整折彎操作後折彎輔助裝置的縮回速度。

  • Set All Bends - 用於將目前速度設定套用於順序中的所有折彎。

Advanced:

  • Method - 用於定義折彎輔助裝置的操作模式。選項包括：

    • Default - 將折彎輔助裝置設定重置為預設值。

    • Return at UDP - 配置折彎輔助裝置在每次折彎後返回到使用者定義的位置。

    • Tilt product - 使折彎輔助裝置在折彎過程中傾斜零件以實現更好的對齊。

    • Static angle - 允許在操作期間為折彎輔助裝置設定固定角度。

  • Stop angle - 用於設定折彎輔助裝置在操作期間停止的角度。

折彎輔助裝置停放位置

折彎輔助裝置在不使用時可以停放在指定位置。這有助於 優化工作空間並確保操作期間的安全。這還可以 防止與其他機器組件發生碰撞。

折彎輔助裝置可以停放在以下位置：

  1. Default - 折彎輔助裝置可以停放到工作台長度。

  2. Small - 折彎輔助裝置可以放置在超出工作台長度700毫米的位置。

  3. Large - 折彎輔助裝置可以放置在超出工作台長度1200毫米的位置。

對於 串聯 機器，有四個折彎輔助裝置可用。對於左串聯 機器，只允許左側停放。對於右串聯機器，只允許 右側停放。

停放位置選項僅顯示於TruBend 8000系列 和VarioPress機器。

對於B36串聯機器，停放位置 small 不適用。

創建自訂折彎輔助裝置

為了在TecZone Bend中創建自訂折彎輔助裝置，您首先需要創建一個 ini 檔案來定義折彎輔助裝置的參數，以及折彎輔助裝置組件（滑架、工作台和板）的 mesh 檔案。

範例baid.ini檔案
[BAData]
; The user defined name of the BendAid. Only used for custom bend aids
Name=BATest
; The range of V-widths the bend aid can support
DieV=16,250
; The range of die-heights the bend aid can support
DieHeight=-30,250
; Maximum angle, in degrees
MaxAngle=55
; Lift capacity, in kg
Payload=300
; Maximum lift speed, in degrees/second
Speed=45
; When the bending aid is at X=0, this is the bound of the support table in top view
Table=-159,-1032,159,-245.6
; The X-Span of the bending aid (when positioned at 0)
XExtent=-213,213
; Tells whether the user can change the forward speed
CanChangeForwardSpeed=False
; Tells whether safety stop angle is possible in the machine
CanStopAtReturn=True
; Can the bending aid movement be varied using different methods?
AdvancedControl=True
; What type of existing bending aid can this be mapped to? Used only by custom bend aids
EType=TB8000_2000N_B36

[Carriage]
Model=Carriage.mesh
Stroke=-1000,1000
Color=White
Shift=0,0,100

[Table]
Model=Table.Mesh
Stroke=50,200
Color=White

[Plate]
Model=Plate.mesh
Stroke=0,56
Color=#4A4A4A

將ini檔案和相應的mesh檔案放在一個資料夾中，然後可以 將其匯入TecZone Bend。

匯入自訂折彎輔助裝置

匯入自訂折彎輔助裝置：

  1. 導覽至 File / Import / Bending Aids…​ 並選擇已創建的 ini檔案（確保相應的mesh檔案在同一資料夾中）。

  2. 匯入的折彎輔助裝置將在機器設定中的折彎輔助裝置 選擇下拉式選單中可用。

當需要為特定折彎任務自訂折彎輔助裝置或使用預設 TecZone Bend資料庫中未包含的專用折彎輔助裝置時，此選項很有用。
