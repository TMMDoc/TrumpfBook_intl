Καρτέλα Τεμάχια

Η καρτέλα Εξαρτήματα χρησιμοποιείται για την προσθήκη τεμαχίων για κατανομή και για τη ρύθμιση παραμέτρων όπως η προς κατανομή ποσότητα, οι αποδεκτές γωνίες περιστροφής κ.λπ.

Κουμπιά ενεργειών

Αυτά είναι τα κουμπιά κατά μήκος της αριστερής πλευράς του πίνακα τεμαχίων που χρησιμοποιούνται για την προσθήκη ή την αφαίρεση τεμαχίων στην/από την εργασία και για την εκτέλεση ορισμένων λειτουργιών σε όλα τα τεμάχια:

  • Χρησιμοποιήστε το Add για να προσθέσετε ένα τεμάχιο για κατανομή. Αυτό μπορεί να είναι ένα ήδη εξοπλισμένο αρχείο .fx ή οποιοδήποτε αρχείο 2D ή 3D που TecZone Laser μπορεί να εισαχθεί και να μετατραπεί σε τεμάχιο μεταλλικής πλάκας. Αυτά τα αρχεία εξοπλίζονται στη συνέχεια όπως επεξηγείται στην ενότητα Ροή εργασιών τεμαχίου.

  • Το κουμπί Add Assembly χρησιμοποιείται για την προσθήκη ενός πλήρους συγκροτήματος 3D. Το TecZone Laser επιλέγει τα τεμάχια μεταλλικής πλάκας από το συγκρότημα και τα προσθέτει στην εργασία κατανομής. Οι ποσότητες για τα επιμέρους στοιχεία του συγκροτήματος ρυθμίζονται ώστε να δημιουργηθεί μία ποσότητα για το πλήρες συγκρότημα.

  • Το κουμπί Add Layout χρησιμοποιείται για την εισαγωγή ενός αρχείου DXF που περιέχει μια πλήρως κατανεμημένη μεταλλική πλάκα και την αποσυνθέτει σε μια εργασία κατανομής.

  • Το κουμπί Import CSV χρησιμοποιείται για τη φόρτωση ενός ορισμού εργασίας κατανομής από ένα αρχείο CSV (τιμές διαχωρισμένες με κόμμα). Το αρχείο θα πρέπει να έχει συγκεκριμένη μορφή με 8 στήλες, όπως επεξηγείται στην παρακάτω συμβουλή εργαλείου:

  • Το Save Parts χρησιμοποιείται για την αποθήκευση των επιλεγμένων τεμαχίων ως αρχεία .FX. Όταν αρχίζετε την εργασία κατανομής με αρχεία γεωμετρίας (όπως DXF, GEO, δεδομένα 3D), αυτά εξοπλίζονται αυτόματα όταν προστίθενται στη λίστα τεμαχίων. Αυτό το κουμπί μπορεί να χρησιμοποιηθεί για την αποθήκευση των εξοπλισμένων εκδόσεων αυτών των τεμαχίων, ώστε να μπορούν επαναχρησιμοποιούνται σε άλλες εργασίες κατανομής.

  • Το Re-Tool All χρησιμοποιείται για να υπολογίσει εκ νέου τον εξοπλισμό για όλα τα τεμάχια της εργασίας. Αφού προβείτε σε ορισμένες τροποποιήσεις στις ρυθμίσεις μηχανήματος ή στις ρυθμίσεις για τη συγκεκριμένη εργασία κατανομής, είναι χρήσιμο να διασφαλίζετε ότι όλα τα τεμάχια χρησιμοποιούν τις νέες ρυθμίσεις.

Στήλες πίνακα τεμαχίων

Εδώ παρατίθενται ορισμένες από τος στήλες στον πίνακα τεμαχίων:

  • Η Τεμάχιο είναι η απαιτούμενη ποσότητα του τεμαχίου. Μπορείτε να καταχωρίσετε έναν απλό αριθμό εδώ ή ένα εύρος όπως 20 - 30. Αυτό συνεπάγεται ότι θα κατανεμηθούν τουλάχιστον 20 αντίγραφα αυτού του τεμαχίου, αλλά όχι περισσότερα από 30. Το TecZone Laser θα έχει τη δυνατότητα να κατανέμει οποιαδήποτε ποσότητα μεταξύ αυτών των δύο τιμών προκειμένου να μεγιστοποιηθεί η αποδοτικότητα κατανομής, αλλά δεν θα δρομολογήσει ποτέ μια καινούργια μεταλλική πλάκα για την κατανομή τεμαχίων πέραν της ελάχιστης ποσότητας.

  • Τα τεμάχια κατανέμονται με αύξουσα σειρά Priority, εάν έχουν οριστεί διαφορετικές τιμές προτεραιότητας για διάφορα τεμάχια. Ωστόσο, η Προτεραιότητα = 0 αντιμετωπίζεται ως ειδική περίπτωση - αυτά τα τεμάχια έχουν την κατώτατη προτεραιότητα.

  • Το Part Grain Pref χρησιμοποιείται για τη ρύθμιση της κατεύθυνσης κόκκων του τεμαχίου αυτή εάν είναι σημαντική. Εάν έχει ρυθμιστεί σε μια τιμή διαφορετική από None, αυτή θα περιορίσει τις γωνίες υπό τις οποίες μπορεί να περιστραφεί το τεμάχιο από τη μηχανή κατανομής. (Για να λειτουργήσει σωστά αυτό, πρέπει επίσης να καθορίσετε την κατεύθυνση των κόκκων των μεταλλικών πλακών στην καρτέλα Μεταλλικές πλάκες).

  • Η τιμή Group είναι μια υπόδειξη προς τη μηχανή κατανομής - το TecZone Laser επιχειρεί να διατηρήσει μαζί τα τεμάχια με τον ίδιο αριθμό ομάδας στην ίδια μεταλλική πλάκα, όπου αυτό είναι δυνατό.

  • Εάν ο διακόπτης Twinline είναι ενεργοποιημένος για ένα τεμάχιο, η μηχανή κατανομής θα επιχειρήσει να χρησιμοποιήσει κοπή κοινής γραμμής για αυτό το τεμάχιο (το τεμάχιο διαθέτει κοινή γραμμή μόνο με άλλα αντίγραφα του ιδίου). Εδώ παρατίθεται ένα παράδειγμα στο οποίο ένα σχεδόν ορθογώνιο τεμάχιο χρησιμοποιεί κοπή κοινής γραμμής τόσο στην κατακόρυφη όσο και στην οριζόντια κατεύθυνση:
