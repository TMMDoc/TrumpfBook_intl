Καρτέλα Διατάξεις

Η καρτέλα Layouts προβάλλει τις κατανεμημένες διατάξεις και παρέχει ορισμένες επιλογές για να τις επεξεργαστείτε.

Κάνοντας κλικ σε μία από τις διατάξεις, αυτή προβάλλεται και σάς παρέχεται η δυνατότητα να επεξεργαστείτε τη διάταξη διαδραστικά. Τα κουμπιά κατά μήκος του αριστερού άκρου της καρτέλας Layouts σάς παρέχουν τη δυνατότητα να εκτελείτε ορισμένες διαδικασίες με ολόκληρη τη διάταξη.

  • Το Add Blank χρησιμοποιείται για τη δημιουργία μιας κενής διάταξης στην οποία μπορείτε στη συνέχεια να διεξάγετε χειροκίνητη κατανομή, ενώ το Delete διαγράφει την τρέχουσα επιλεγμένη διάταξη.

  • Τα κουμπιά Move Up και Move Down χρησιμοποιούνται για την αναδιάταξη της σειράς των διατάξεων εντός της λίστας.

  • Το κουμπί Do Nesting χρησιμοποιείται για την εκ νέου κατανομή τυχόν μη κατανεμημένων τεμαχίων.

  • Το Save All χρησιμοποιείται για την αποθήκευση όλων των διατάξεων ως ξεχωριστά αρχεία .fxlyt, ενώ το Export DXFs χρησιμοποιείται για την εξαγωγή των εικόνων των πλήρως κατανεμημένων μεταλλικών πλακών ως αρχεία DXF που μπορείτε να ανοίξετε με οποιοδήποτε πρόγραμμα προβολής ή επεξεργασίας που μπορεί να λειτουργήσει με αρχεία DXF. (Κάθε τεμάχιο της διάταξης καθίσταται μια οντότητα κατανομής μπλοκ στο αρχείο DXF).

  • Το κουμπί Create NC χρησιμοποιείται για τη δημιουργία προγραμμάτων NC για όλες τις διατάξεις, καθώς και ένα φύλλο ρυθμίσεων διάταξης που παρέχει λεπτομέρειες σχετικές με όλα τα τεμάχια που χρησιμοποιούνται στην εργασία και όλα τα φύλλα διάταξης που προκύπτουν.

Εδώ παρατίθεται μια ενότητα μιας αναφοράς διάταξης που δημιουργείται με την εντολή Create NC.
