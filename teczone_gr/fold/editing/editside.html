Επεξεργασία πλευράς/τμήματος

Υπάρχουν μερικές ρυθμίσεις και λειτουργίες που συνδέονται με μια ολόκληρη πλευρά κάμψης (σε αντίθεση με μια διαδικασία μονής κάμψης). Αυτές είναι προσβάσιμες με άνοιγμα του

πίνακα πλευράς:

  • Ανοίξτε έναν πίνακα κάμψης (κάνοντας διπλό κλικ σε έναν αριθμό στον πλοηγό δίπλωσης). Στη συνέχεια, κάνετε κλικ στο κουμπί Πλευρά για να ανοίξετε τον πίνακα πλευράς για την πλευρά που περιλαμβάνει αυτήν την κάμψη.

  • Κρατήστε πατημένο το Ctrl και κάνετε κλικ σε έναν αριθμό στον πλοηγό δίπλωσης για να ανοίξετε τον πίνακα πλευράς για εκείνη την πλευρά.

Ο πίνακας πλευράς είναι όπως εκείνος που παρουσιάζεται παραδίπλα (το πραγματικό σύνολο επιλογών που είναι διαθέσιμες στο αναδυόμενο μενού Επεξεργασία αλληλουχίας μπορεί να διαφέρει, ανάλογα με το τεμάχιο).

  • Κάνετε κλικ στο κουμπί Flip Side για να κατοπτρίσετε ολόκληρη την πλευρά. Αυτό θα έχει ως αποτέλεσμα τον άμεσο επανυπολογισμό των λαμών, των συγκρατητήρων και του χειρισμού του τεμαχίου. Σε ένα αυτόματο μηχάνημα, αυτό αντικαθίσταται από το Flip Part button[1], καθώς το μηχάνημα δεν μπορεί να κατοπτρίσει μόνο μία πλευρά.

  • Κάνετε κλικ στο κουμπί Edit Sequence για να εμφανιστεί ένα υπομενού που προβάλλει τους πιθανούς συνδυασμούς αλληλουχίας που θα μπορούσατε να επιλέξετε με αυτήν την πλευρά. Στην παραπάνω εικόνα, εργαζόμαστε με ένα τεμάχιο που έχει μια αλληλουχία πλευράς όπως C1 C2 L1 L2 και έχουμε επιλέξει την πλευρά L1. Οι πιθανές επιλογές αλληλουχίας επεξηγούνται παρακάτω:

  • Η επιλογή << L1+L2 νωρίτερα μεταθέτει τις πλευρές L1 και L2 πριν από τις πλευρές C1 και C2. Έτσι, η νέα αλληλουχία θα είναι L1 L2 C1 C2.

  • Η επιλογή << L1 νωρίτερα μεταθέτει την πλευρά L1 κατά ένα βήμα νωρίτερα, με αποτέλεσμα η νέα αλληλουχία να είναι C1 L1 C2 L2. Αν και αυτό μπορεί να μην είναι τόσο συνηθισμένο, θα μπορούσε να είναι χρήσιμο σε ορισμένες καταστάσεις.

  • Η επιλογή L1 αργότερα >> μεταθέτει την πλευρά L2 αργότερα με αποτέλεσμα μια αλληλουχία C1 C2 L2 L1 (βέβαια, οι πλευρές μετονομάζονται άμεσα κατόπιν αυτού με αποτέλεσμα, η αλληλουχία να εξακολουθεί να είναι C1 C2 L1 L2 κατά τη δημιουργία του κώδικα).

Το σύνολο των διαθέσιμων επιλογών σε αυτό το μενού εξαρτάται από την τρέχουσα αλληλουχία και την πλευρά στην οποία επιλέγετε να κάνετε την αναδιάταξη.

  • Κάνετε κλικ στα κουμπιά Next ή Prev για να πλοηγηθείτε μεταξύ των διαφορετικών πλευρών σε αυτό το τεμάχιο.

  • Κάνετε κλικ στο κουμπί Bends για να μετακινηθείτε προς τα κάτω από τις πλευρές στο σύνολο των κάμψεων για αυτήν την πλευρά (αυτό είναι το αντίστροφο του Side στον πίνακα Κάμψη που σας επιτρέπει να περιηγηθείτε από μια κάμψη προς την πλευρά που περιέχει την κάμψη).

1. Αυτό έχει πολύ πιο περιορισμένη χρησιμότητα από το κουμπί Κατόπτριση πλευράς για ένα χειροκίνητο μηχάνημα και υπάρχει μόνο για τη διαχείριση ορισμένων σπάνιων καταστάσεων.
