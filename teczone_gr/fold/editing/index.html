Επεξεργασία δεδομένων δίπλωσης

Υπάρχουν διάφοροι πίνακες που χρησιμοποιούνται για την επεξεργασία των ρυθμίσεων μιας διαδικασίας κάμψης, των ρυθμίσεων λαμών, των ρυθμίσεων συγκρατητήρα κ.λπ. Οι περισσότεροι από αυτούς τους πίνακες είναι προσβάσιμοι απλώς με κλικ στο κατάλληλο στοιχείο στην οθόνη 3D - για παράδειγμα, για να επεξεργαστείτε τις ρυθμίσεις του συγκρατητήρα, κάνετε απλώς κλικ στο συγκρατητήρα.

  • Επεξεργασία μιας διαδικασίας κάμψης

  • Επεξεργασία πλευράς/τμήματος

  • Αλλαγή της αλληλουχίας

  • Επεξεργασία των συγκρατητήρων

  • Επεξεργασία των αναστολέων

  • Επεξεργασία των λαμών

Οι ακόλουθες διαδικασίες είναι επίσης διαθέσιμες για αυτόματες διπλωτικές μηχανές πάνελ με άξονα ρότορα (όπως οι 7030 και 7020):

  • Επεξεργασία της φόρτωσης

  • Επεξεργασία της απόρριψης
