Προσαρμοσμένοι πίνακες διακένου

Ο πίνακας διακένου για μια διπλωτική μηχανή πάνελ ρυθμίζει χαρτογραφεί ένα συγκεκριμένο πάχος μεταλλικής πλάκας στο βασικό διάκενο που χρησιμοποιείται κατά την επεξεργασία. Αυτό είναι ενσωματωμένο σε καθεμιά από τις διάφορες διπλωτικές μηχανές πάνελ, αλλά μπορεί να παρακαμφθεί με την παροχή ενός προσαρμοσμένου πίνακα διακένου υπό μορφή ενός αρχείου JSON. Αυτό το αρχείο JSON φέρει το όνομα air-gap-tables.json και βρίσκεται στη διαδρομή C:\ProgramData\Trumpf{APPNAME}\Data\System.

Παρακάτω παρατίθεται ένα χαρακτηριστικό αρχείο air-gap-tables.json:

 {
  "version":1  "airGapTables": [
    {
      "machine": "112002,112003", "material": "DC01"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
     {
      "machine": "112003", "material": "*"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
    {
      "machine": "*", "material": "Alluminio"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    }
  ]
}

  • Το αρχείο περιέχει μια συλλογή τιμών διακένου. Κάθε τραπέζι έχει ένα μηχάνημα και έναν κωδικό υλικού που χρησιμοποιούνται ως επιλογείς.

  • Αυτοί οι κωδικοί ρυθμίζονται σε '*' (αστερίσκος) που επισημαίνει έναν χαρακτήρα αναπλήρωσης (ταυτίζεται με οτιδήποτε), ή είναι μια λίστα διαχωρισμένων με κόμμα τιμών μηχανήματος ή υλικού.

  • Στο παραπάνω παράδειγμα, υπάρχουν τρεις πίνακες. Ο πρώτος χρησιμοποιείται για τα μηχανήματα 112001 ή 112003, αποκλειστικά για το υλικό DC01. Ο δεύτερος χρησιμοποιείται για όλα τα υλικά, για το μηχάνημα 112003 και ο τρίτος χρησιμοποιείται για όλα τα μηχανήματα, για το υλικό αλουμίνιο. Εάν πολλοί πίνακες θα μπορούσαν να ταιριάζουν με το συνδυασμό μηχανήματος/υλικού του τεμαχίου, τότε χρησιμοποιείται ο πρώτο πίνακας που ταιριάζει. Έτσι, για το μηχάνημα 112003 και το υλικό DC01 χρησιμοποιείται ο πρώτος πίνακας (παρ' όλο που ο δεύτερος ταιριάζει επίσης).

  • Κάθε πίνακας περιλαμβάνει μια απλή σειρά κωδικών πάχους με τις αντίστοιχες τιμές διακένου. Για παράδειγμα, η πρώτη καταχώριση παραπάνω θα πρέπει να ερμηνευτεί ως εξής: εάν το πάχος είναι μικρότερο ή ίσο με 0,75 mm, χρησιμοποιήστε το διάκενο 2,5 mm. Οι κωδικοί πάχους πρέπει να είναι σε αύξουσα σειρά. Εάν κανένας από τους πίνακες αυτού του αρχείου δεν ταιριάζει με το συνδυασμό υλικού και πάχους του τρέχοντος τεμαχίου, τότε θα χρησιμοποιηθεί ο προεπιλεγμένος πίνακας διακένου για το μηχάνημα.

Εάν τροποποιήσετε το αρχείο air-gap-tables.json, πρέπει να επανεκκινήσετε το TecZone Fold προτού τεθούν σε ισχύ αυτές οι τροποποιήσεις.
Εάν το αρχείο JSON δεν είναι διαμορφωμένο σωστά, προβάλλεται ένα σφάλμα όταν επιχειρήσετε να επεξεργαστείτε ένα τεμάχιο διπλωτικής μηχανής πάνελ, αλλά το TecZone Fold θα επιστρέψει τότε στη χρήση του προεπιλεγμένου πίνακα διακένου για το μηχάνημα και θα συνεχίσει.
