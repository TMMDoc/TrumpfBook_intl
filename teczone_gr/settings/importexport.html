Εισαγωγή/Εξαγωγή

Σε αυτήν την ενότητα, θα καλύψουμε τη διαμόρφωση του Ρυθμίσεις εισαγωγής.

Κάνετε κλικ στο εικονίδιο *Ρυθμίσεις * από την αρχική σελίδα.

Ρυθμίσεις εισαγωγής

Σε αυτήν την ενότητα, θα καλύψουμε τη διαμόρφωση του Ρυθμίσεις εισαγωγής. Κάνετε κλικ στο εικονίδιο Ρυθμίσεις. Κάνετε κλικ στο Εισαγωγή/Εξαγωγή και μεταβείτε στο Ρυθμίσεις εισαγωγής.

Μονάδες για αρχεία DXF - Ορίστε χιλιοστά ή ίντσες εδώ.

Μέγιστη απόσταση μεταξύ στοιχείων ελέγχου που δεν έχουν+ - Ορίστε αυτήν την τιμή (0>1 mm). Κατά την εισαγωγή ενός τεμαχίου που περιλαμβάνει γραμμές/τόξα πιο κοντά από την επιλεγμένη τιμή, το λογισμικό θα συρράψει αυτόματα αυτά τα στοιχεία κατά την εισαγωγή.

Μέγιστο πάχος ελάσματος - Για να αναγνωριστεί ένα μεγάλο τεμάχιο μεταλλικής πλάκας, θα πρέπει να αυξηθεί το όριο αναγνώρισης μεταλλικής πλάκας. (10>40 mm) Αυτή η επιλογή ρυθμίζεται αυτόματα ως προεπιλογή σε 25 mm ή μία ίντσα, ανάλογα με τη μονάδα που χρησιμοποιείται.

Μονάδες σημείων - Ανάλογα με την επιλογή που έχετε κάνει, αυτή θα καθορίσει τον τρόπο εισαγωγής των σημείων.

Εισαγωγή όλων - Όλα τα σημεία θα εισαχθούν και θα εμφανιστούν.

Παράλειψη σημείου σε πολυγραμμές - Αυτή η επιλογή παρακάμπτει όλα τα σημεία που ανιχνεύονται στις πολλαπλές γραμμές.

Παράλειψη όλων - Αυτή η επιλογή παρακάμπτει όλα τα σημεία κατά την εισαγωγή, και δεν προβάλλεται κανένα.

Αγνόηση επιπέδων σε αρχεία DXF/DWG - Τα σχέδια DXF και DWG δημιουργούνται συνήθως σε διαφορετικά επίπεδα. Με αυτήν τη ρύθμιση, το λογισμικό αγνοεί αυτά τα επίπεδα και μετακινεί όλα τα αντικείμενα στο προεπιλεγμένο επίπεδο.

Λύσιμο των μπλοκ στο σχέδιο 2D - Χωρίζει τις ομαδοποιήσεις κατά την εισαγωγή

Μετατροπή λευκών αντικειμένων σε μαύρα - Ενεργοποιήστε αυτήν τη ρύθμιση. για τη μετατροπή λευκών αντικειμένων σε μαύρα αντικείμενα κατά την εισαγωγή.

Πιο σκούρα χρώματα κατά την εισαγωγή DXF- Ενεργοποιήστε αυτήν τη ρύθμιση για να προβάλλετε τα έγχρωμα αντικείμενα με πιο σκούρο χρώμα κατά την εισαγωγή.

Κατάργηση διπλών τμημάτων - Ενεργοποιήστε αυτήν τη ρύθμιση. για να αφαιρέστε τυχόν διπλότυπη γεωμετρία που βρίσκεται στο τεμάχιο κατά την εισαγωγή.

Ρυθμίσεις DXF

Σε αυτήν την ενότητα, θα καλύψουμε τη διαμόρφωση του*Ρυθμίσεις DXF*. Κάνετε κλικ στο εικονίδιο Ρυθμίσεις. Κάνετε κλικ στο Εισαγωγή/Εξαγωγή και μεταβείτε στο Ρυθμίσεις DXF.

Γωνίες στο αρχείο dxf είναι γωνίες ανοίγματος. - Ενεργοποιήστε αυτήν τη ρύθμιση για τις γωνίες σε ένα DXF για χειρισμό ως γωνίες ανοίγματος.

Ρυθμίσεις εξαγωγής

Σε αυτήν την ενότητα, θα καλύψουμε τη διαμόρφωση του*Ρυθμίσεις εξαγωγής*. Κάνετε κλικ στο εικονίδιο Ρυθμίσεις. Κάνετε κλικ στο Εισαγωγή/Εξαγωγή και μεταβείτε στο Ρυθμίσεις εξαγωγής.

Κανένα αντικείμενο POLYLINE στην έξοδο DXF - Συνήθως, τα κλειστά περιγράμματα εξάγονται ως πολλαπλές γραμμές κατά την εξαγωγή DXF. Ορισμένα συστήματα CAD δεν μπορούν να επεξεργαστούν αυτό το αποτέλεσμα. Με αυτήν τη ρύθμιση, το λογισμικό εξάγει το DXF με γραμμές και τόξα. Αυτά τα αρχεία μπορούν να αναγνωστούν παντού, αλλά τα αρχεία που δημιουργούνται είναι μεγαλύτερα και οι συνδέσεις μεταξύ των γραμμών και των τόξων χάνονται

Έξοδος πληροφοριών κάμψης κατά την εξαγωγή αρχείων DXF. - Ενεργοποιήστε αυτήν τη ρύθμιση για να εξάγεται ένα εξαγόμενο DXF με πληροφορίες κάμψης.

Πληροφορίες Bend σε μορφή Starmatik - Ενεργοποιήστε αυτόν τον διακόπτη για να εξάγετε τις πληροφορίες κάμψης σε μορφή Starmatik. Εδώ τοποθετείται μια οντότητα κειμένου με απόλυτη ακρίβεια στο μέσο κάθε γραμμής που πρόκειται να είναι γραμμή κάμψης.

Μετατροπή μαύρου σε γκρι κατά την έξοδο - Κατά την εξαγωγή δεδομένων 2D, τα αντικείμενα στο αρχείο DXF εξάγονται με μαύρο χρώμα από προεπιλογή. Για να να είστε σε θέση να αναγνωρίζετε καλύτερα τα αντικείμενα στα προγράμματα CAD, τα αντικείμενα εξάγονται σε γκρι χρώμα με αυτήν τη ρύθμιση.

Εκκίνηση MetaCAM με την εξαγωγή αρχείων PDG. - Ενεργοποιήστε αυτήν τη ρύθμιση για να διαμορφώνεται αυτόματα ένα αρχείο PDG ώστε να ανοίγει στο MetaCAM

Μορφή εξόδου της εκτύλιξης - Κατά την εξαγωγή ενός επίπεδου μοτίβου, αυτό μπορεί να ρυθμιστεί σε μορφή αρχείου Geo, DXF ή PDG.

Μετατροπή spline

Σε αυτή την ενότητα, θα καλύψουμε τη διαμόρφωση των ρυθμίσεων Μετατροπή spline. Κάνετε κλικ στο εικονίδιο Ρυθμίσεις. Κάνετε κλικ στο Εισαγωγή/Εξαγωγή και μεταβείτε στις ρυθμίσεις Μετατροπή spline.

Μετατροπή spline κατά την εισαγωγή - Ρυθμίστε εδώ αν η μετατροπή σφηνών θα είναι απενεργοποιημένη ή ρυθμίστε για να μετατρέπονται οι σφήνες σε γραμμές ή τόξα. Σε σε κάθε περίπτωση, κάθε σφήνα θα μετατραπεί σε ένα ενιαίο αντικείμενο πολλαπλών γραμμών που περιλαμβάνει ευθύγραμμα τμήματα ή τμήματα τόξου.

Υπολογισμός σημείων στήριξης - Ο αριθμός των γραμμών ή των τόξων που δημιουργούνται υπολογίζεται χρησιμοποιώντας έναν από τους δύο μηχανισμούς: βήμα ή απόκλιση.

Μήκος ανά τμήμα γραμμών ή τόξου - Εάν επιλεγεί βήμα στον υπολογισμό αριθμού κόμβων, ορίστε εδώ το μήκος κάθε τμήματος τόξου ή γραμμής για τη διάσπαση της σφήνας χρησιμοποιώντας αυτό το μήκος βήματος.

Μέγιστη απόκλιση κατά την προσέγγιση - Εάν επιλεγεί απόκλιση στον υπολογισμό αριθμός κόμβων, ορίστε εδώ τη μέγιστη απόκλιση που επιτρέπεται μεταξύ της αρχικής ομαλής σφήνας και της κατά προσέγγιση γραμμής ή του τόξου. Η πολλαπλή γραμμή κατασκευάζεται με όσο το δυνατόν λιγότερα τμήματα, διατηρώντας παράλληλα το μέγιστο σφάλμα εντός αυτού του ορίου.

Κατάργηση μονάδας

Σε αυτήν την ενότητα, θα καλύψουμε τη διαμόρφωση των ρυθμίσεων Κατάργηση μονάδας. Κάνετε κλικ στο εικονίδιο Ρυθμίσεις. Κάνετε κλικ στο Εισαγωγή/Εξαγωγή και μεταβείτε στις ρυθμίσεις Κατάργηση μονάδας

Παράκαμψη τμημάτων κοπής - Κατά την ανάπτυξη ενός συγκροτήματος, με την ενεργοποίηση αυτού του διακόπτη προβάλλονται μόνο τα τεμάχια μεταλλικής πλάκας με γραμμές κάμψης. Με την απενεργοποίηση θα προβάλλονται όλα τα τεμάχια του συγκροτήματος.

Εμφάνιση μερών (παξιμάδια πρεσαρίσματος, πείροι, παραμορφώσεις διατρητών …) - Χρησιμοποιήστε αυτήν τη ρύθμιση για να επιλέξετε ποιες επιλογές θα εμφανίζονται κατά την ανάπτυξη ενός συγκροτήματος που περιέχει άλλα στοιχεία.

Off - Εμφανίζονται μόνο τα τεμάχια μεταλλικής πλάκας μετά την ανάπτυξη, ενώ δεν προβάλλονται άλλα παξιμάδια/μπουλόνια.

άγνωστο - Εμφανίζονται μόνο τα στοιχεία που δεν έχουν ακόμη αναγνωριστεί στο λογισμικό.

All - Εμφανίζονται όλα τα στοιχεία.

Αντιστοίχιση στρώσης

Σε αυτή την ενότητα, θα καλύψουμε τη διαμόρφωση του Αντιστοίχιση στρώσης. Κάνετε κλικ στο εικονίδιο Ρυθμίσεις. Κάνετε κλικ στο Εισαγωγή/Εξαγωγή και περιηγηθείτε στις ρυθμίσεις Αντιστοίχιση στρώσης.

Σε αυτήν την ενότητα, τα επίπεδα που χρησιμοποιούνται σε τεμάχια τα οποία εισάγονται στο λογισμικό, μπορούν να χαρτογραφηθούν αυτόματα για αυτήν τη λειτουργία (χρήση).

Παράδειγμα

Εάν ένα εισαγόμενο τεμάχιο διαθέτει ένα επίπεδο ΣΗΜΑΝΣΗΣ, αυτό μπορεί να ρυθμιστεί ώστε να χρησιμοποιεί αυτόματα το επίπεδο Σήμανση στο λογισμικό.

Όνομα στρώσης - Αυτό είναι το όνομα του επιπέδου το οποίο, εάν ένα τεμάχιο εισαχθεί με αυτό, θα χρησιμοποιήσει τη λειτουργία που έχει οριστεί στον πίνακα 'Χρήση'.

χρήση - Αυτή είναι η λειτουργία του επιπέδου. Οι διάφορες διαθέσιμες επιλογές είναι:

Στάνταρ - Αυτό είναι το βασικό επίπεδο που πρέπει να χρησιμοποιείται για το CAM.

Βοηθητικά μέσα - Ένα βοηθητικό επίπεδο, που δεν πρέπει να χρησιμοποιείται για CAM.

Σήμανση - Οποιεσδήποτε οντότητες σε αυτό το επισημαίνονται, δεν κόβονται.

Σήμανση προσέγγισης - Σημειακές οντότητες που σηματοδοτούν τη θέση προσέγγισης με λέιζερ.

Σειριακή σήμανση - Δείκτες κειμένου που σηματοδοτούν την σειρά της αλληλουχίας περιγραμμάτων.

Κέντρο σχηματοποίησης - Κεντρική σήμανση για τη διαμόρφωση (σημείο ή μικρό L).

Σχηματοποίηση αποτυπώματος πέλματος - Το περίγραμμα (αποτύπωμα) μιας διαμόρφωσης.

Εξάτμιση - Αυτό το επίπεδο θα χρησιμοποιηθεί για τη διάκριση της καύσης της μεμβράνης.

Σημειακή σήμανση - Αυτό το επίπεδο θα χρησιμοποιηθεί για κωδικούς QR.

Πληροφορίες - Αυτό είναι απλώς ένα επίπεδο πληροφοριών.
