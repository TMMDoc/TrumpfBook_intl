Χειρισμός μεταλλικής πλάκας

Με βάση την προτίμηση του χρήστη, οι παράμετροι για το χειρισμό μιας μεταλλικής πλάκας μπορούν να τροποποιηθούν στο Sheet handling.

Μεταβείτε στο Application Settings → Cut CAM → Layout → Sheet handling.

Loading

Sheet loading method - Ανάλογα με το μηχάνημα και τον πρόσθετο εξοπλισμό, αυτή η επιλογή προορίζεται για τη ρύθμιση του τρόπου φόρτωσης μιας μεταλλική στο μηχάνημα. Οι επιλογές περιλαμβάνουν τις εξής:

  • χειροκίνητα

  • σύστημα εναλλαγής παλετών

  • διάφορες επιλογές για το LiftMaster

Loading corner - Αυτή η επιλογή χρησιμοποιείται για τη ρύθμιση της γωνίας της μεταλλικής πλάκας που θα φορτωθεί πρώτα στο μηχάνημα όταν χρησιμοποιείτε ένα LiftMaster. Οι επιλογές περιλαμβάνουν τις εξής:

  • Κάτω αριστερά

  • Κάτω δεξιά

  • Επάνω δεξιά

  • Επάνω αριστερά

Measurement

Measure sheet position - Αυτή η επιλογή χρησιμοποιείται για τις ρυθμίσεις όταν η μεταλλική πλάκα πρόκειται να μετρηθεί. Οι επιλογές είναι οι εξής:

  • χωρίς μέτρηση

  • με ρύθμιση ύψους

Measuring corner - Όταν η θέση μέτρησης της μεταλλικής πλάκας έχει οριστεί σε 'With height regulation', αυτή η ρύθμιση χρησιμοποιείται για να ρυθμιστεί από πού θα μετρηθεί η μεταλλική πλάκα. Οι επιλογές περιλαμβάνουν τις εξής:

  • Κάτω αριστερά

  • Κάτω δεξιά

  • Επάνω δεξιά

  • Επάνω αριστερά

Sheet loading position

X & Y - Αυτές οι ρυθμίσεις χρησιμοποιούνται για τη ρύθμιση της θέσης φόρτωσης της μεταλλικής πλάκας στον άξονα X και στον άξονα Υ του μηχανήματος. Αυτή μπορεί να είναι μεταξύ 0 > 10000 χιλιοστών.

Sheet loading tolerance

Max. sheet loading error in X - Μέγιστο σφάλμα κατά τη φόρτωση της μεταλλικής πλάκας στο τραπέζι κατά μήκος του άξονα Χ.

Max. sheet loading error in Y - Μέγιστο σφάλμα κατά τη φόρτωση της μεταλλικής πλάκας στο τραπέζι κατά μήκος του άξονα Y.

Max. sheet loading angle error - Μέγιστο σφάλμα γωνίας κατά τη φόρτωση της μεταλλικής πλάκας στο τραπέζι.
