Κάμψη CAM

Κάμψη CAM

Back-gauges

Min.thickness to always pull-back (mm) - Αυτή η ρύθμιση σάς παρέχει τη δυνατότητα να καταχωρίσετε μια τιμή σε mm αυτόματης απόσυρσης των πίσω αναστολέων όταν επιτυγχάνεται το καταχωρισμένο πάχος της μεταλλικής πλάκας.

Clamping strategy - Αυτό χρησιμοποιείται για τη ρύθμιση του τρόπου με τον οποίο οι πίσω αναστολείς πρέπει να σταματούν στο τεμάχιο επεξεργασίας.

Είναι διαθέσιμες οι παρακάτω επιλογές:

Default - Οι πίσω αναστολείς σταματούν για κάθε κάμψη με τον πλέον ενδεδειγμένο τρόπο σύμφωνα με το λογισμικό.

Stop + Stop - Οι ευθείες επιφάνειες και των δύο πίσω αναστολέων σταματούν στο τεμάχιο επεξεργασίας.

Clamp + Stop - Ένας πίσω αναστολέας σφίγγει το το τεμάχιο επεξεργασίας και ο άλλος πίσω αναστολέας το σταματά με την ευθεία επιφάνειά του.

Angle Measurement

Το Angle measurement προορίζεται αποκλειστικά για μηχανήματα Trumpf. Η χρήση των συστημάτων μέτρησης γωνιών της σειράς αυτόματα ελεγχόμενης κάμψης συνεπάγεται ότι ακόμη και το πρώτο τεμάχιο στη σειρά θα είναι άψογο. Οι αισθητήρες προσδιορίζουν την πραγματική γωνία μαζί με την ελαστική επαναφορά και καθοδηγούν τη μπάρα πρεσαρίσματος για την εξασφάλιση της επιθυμητής γωνίας. Σε αυτήν την ενότητα μπορούν να καταχωριστούν ρυθμίσεις ACB στο μηχάνημα.

Angle Measurement Systems

Laser - Ενεργοποιήστε αυτήν την επιλογή εάν το μηχάνημα διαθέτει λέιζερ ACB. Το λέιζερ ACB είναι το σημείο όπου ένα λέιζερ προβάλλει μια γραμμή στη μεταλλική πλάκα και μια κάμερα ανιχνεύει τη γωνία.

Wireless - Ενεργοποιήστε αυτήν την επιλογή εάν το μηχάνημα χρησιμοποιεί σύστημα ACB Wireless. Το σύστημα ACB Wireless είναι δίσκοι αισθητήρων ενσωματωμένοι στο επάνω εργαλείο.

Classic - Ενεργοποιήστε αυτήν την επιλογή εάν το μηχάνημα χρησιμοποιεί ACB Classic.

Min. bend length to use 2 angle measures - Αυτή η επιλογή είναι το ελάχιστο μήκος της δίπλωσης για τη λήψη δύο μετρήσεων ώστε να διασφαλιστεί η σωστή γωνία κατά τη δίπλωση. Αυτό μπορεί να είναι μεταξύ 50 > 4000 χιλιοστών.

Min. bend length to use 3 angle measures -Αυτή η επιλογή είναι το ελάχιστο μήκος της δίπλωσης για τη λήψη τριών μετρήσεων για να διασφαλιστεί η σωστή γωνία κατά τη δίπλωσης. Αυτό μπορεί να είναι μεταξύ 50 > 4000 χιλιοστών.

ACB laser sensor park position - Αυτή είναι η θέση ηρεμίας του αισθητήρα λέιζερ.

ACB Quality limit - warning - Αυτό είναι το όριο που θα δημιουργήσει μια προειδοποίηση κατά τη μέτρηση του τεμαχίου, όπως ένας συντελεστής σφάλματος. Εάν είναι στο συγκεκριμένο εύρος, τότε θα καταχωριστεί μια προειδοποίηση.

ACB Quality limit - error - Αυτό είναι το όριο που θα δημιουργήσει ένα σφάλμα κατά τη μέτρηση του τεμαχίου, όπως ένας συντελεστής σφάλματος. Θα καταχωρίσει ένα σφάλμα εάν βρίσκεται στο συγκεκριμένο εύρος.

Geometry Adaptation Strategy

Εάν υποβληθεί σε επεξεργασία ένα αρχείο 2D ή 3D στο λογισμικό, ελέγχονται οι ακτίνες κάμψης και η μείωση κάμψης. Εκτελούνται διάφορες στρατηγικές επεξεργασίας ανάλογα με τις στρατηγικές που έχουν επιλεγεί για την τροποποίηση του τεμαχίου επεξεργασίας στις ρυθμίσεις εύκολου προγραμματισμού. Εάν η απόφαση έχει οριστεί σε διαδραστική, τότε θα εμφανιστεί ο διάλογος Adapt Geometry.

When starting with 2D flat

Ask each time - Κατά την πλοήγηση από ένα τεμάχιο 2D σε κάμψη, με τη χρήση αυτής της ρύθμισης προβάλλεται αυτόματα ο διάλογος Adapt Geometry. ΅Αυτός παρέχει επιλογές διατήρησης του μεγέθους του επίπεδου τεμαχίου, διατήρησης των διαστάσεων 3D ή μη τροποποίησης κανενός στοιχείου, καθιστώντας αυτή τη λειτουργία εξαιρετικά διαδραστική και καθοδηγούμενη από το χρήστη.

Keep 3D dimensions - Αυτή η επιλογή αλλάζει την έκπτυξη που στη συνέχεια πρέπει να αποθηκευτεί χειροκίνητα. Αυτή η στρατηγική χρησιμοποιείται εάν το ακατέργαστο τεμάχιο δεν έχει κοπεί ακόμη.

Keep 2D dimensions - Αυτή η επιλογή συνεπάγεται ότι η ακτίνα και η μείωση κάμψης προσαρμόζονται την υφιστάμενη έκπτυξη. Η γραμμή κάμψης παραμένει στην ίδια θέση. Έτσι αλλάζει το μήκος πλευράς του τεμαχίου. Η έκπτυξη δεν αλλάζει. Αυτή η στρατηγική χρησιμοποιείται εάν το ακατέργαστο τεμάχιο έχει ήδη κοπεί στο σωστό μέγεθος.

No adjustments - Η χρήση αυτής της επιλογής συνεπάγεται ότι ακτίνα και η μείωση της κάμψης παραμένουν αμετάβλητες. Το τεμάχιο προβάλλεται με τις σωστές διαστάσεις σε 3D, αλλά οι προβαλλόμενες ακτίνες κάμψης δεν αντιστοιχούν στις πραγματικές ακτίνες κάμψης. Η διάσταση κάθε κάμψης του τεμαχίου επεξεργασίας μπορεί να αλλάξει.

When starting with 3D model

Ask each time - Κατά την πλοήγηση από ένα τεμάχιο 3D σε κάμψη, με τη χρήση αυτής της ρύθμισης προβάλλεται αυτόματα ο διάλογος Adapt Geometry. ΅Αυτός παρέχει επιλογές διατήρησης του μεγέθους του επίπεδου τεμαχίου, διατήρησης των διαστάσεων 3D ή μη τροποποίησης κανενός στοιχείου, καθιστώντας αυτή τη λειτουργία εξαιρετικά διαδραστική και καθοδηγούμενη από το χρήστη.

Η διάσταση κάθε κάμψης του τεμαχίου επεξεργασίας μπορεί να αλλάξει.

Bend deduction calculation - Αυτή η επιλογή καθιστά δυνατή την αναγκαστική χρήση της μεθόδου συντελεστή K. Αυτή η ρύθμιση μπορεί να οριστεί στις Προεπιλογές εφαρμογής ή μπορεί να προσαρμοστεί για κάθε μηχάνημα ξεχωριστά.

Για μηχανήματα που δεν είναι της εταιρείας Trumpf, οι συντελεστές K χρησιμοποιούνται πάντοτε. Οι υπολογισμοί κάμψης Trumpf είναι διαθέσιμοι μόνο για καμπτικές πρέσες Trumpf.

BG-Left variant - Ανάλογα με την παραλλαγή BG που χρησιμοποιείται, το πρόγραμμα NC θα τροποποιηθεί για να αντιπροσωπεύει αυτό.

BG-Right variant - Ανάλογα με την παραλλαγή BG που χρησιμοποιείται, το πρόγραμμα NC θα τροποποιηθεί για να αντιπροσωπεύει αυτό.

Other Settings

Hemming pre-bend angle - Αυτή η επιλογή μπορεί να επεξεργαστεί τη προκαταρκτικής κάμψης της δίπλωσης μεταξύ 10° και 60°. Η προεπιλογή είναι 40° δεδομένου ότι αυτή είναι το συνιστώμενη γωνία προκαταρκτικής κάμψης της δίπλωσης για τους περισσότερους κατασκευαστές. Έχει ρυθμιστεί στις 30° για όλα τα μηχανήματα Trumpf, σε όλες τις διάφορες εκδόσεις.

Bend-guard mode override - Ρύθμιση της αυτόματης λειτουργίας υπολογισμού (η προεπιλογή). Η κάμψη θα υπολογίζει μια λειτουργία προστασίας κάμψης για κάθε κάμψη. Ωστόσο, εσείς μπορείτε να εξαναγκάσετε τη λειτουργία προστασίας κάμψης-σε μια από τις επιλογές για να χρησιμοποιηθεί μόνο αυτή, ανεξάρτητα από το αν είναι σκόπιμο ή όχι.

Μπορείτε επίσης να παρακάμψετε αυτήν τη ρύθμιση για κάθε μηχάνημα, ρυθμίζοντάς την στις Προεπιλογές μηχανήματος, αντί για Application Settings.

Machine table position - Αυτή η ρύθμιση έχει τρεις επιλογές για τη θέση του πίνακα:

  1. Καταργήθηκε

  2. Κάτω

  3. Επάνω

Tool inventory check - Η ενεργοποίηση αυτής της επιλογής θα προβάλλει τμήματα εργαλείων που μπορεί να χρειαστούν για την κάμψη του τεμαχίου επεξεργασίας. Θα προβληθούν ανεπαρκείς ενδείξεις συλλογής.

Avoid tool overload if possible - Η ενεργοποίηση αυτής της επιλογής θα διασφαλίσει την αποφυγή της υπερφόρτωσης του εργαλείου. Ακόμα και αν ένα εργαλείο με υψηλότερη προτεραιότητα θα εκτελούσε εργασία στο τεμάχιο, ένα εργαλείο χαμηλότερης προτεραιότητας θα χρησιμοποιηθεί αντ' αυτού για να αποφύγετε τα προβλήματα υπερφόρτωσης.

Ignore tool hints from GEO/DXF files - Η ενεργοποίηση αυτής της επιλογής θα αγνοήσει τυχόν πληροφορίες που βρίσκονται σε ένα αρχείο DXF/GEO σχετικά με το ποιος εξοπλισμός πρέπει να χρησιμοποιηθεί.
