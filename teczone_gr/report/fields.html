Πεδία, εικόνες και πίνακες

Κάνοντας κλικ στις πηγές δεδομένων, τις ανοίγετε και προβάλλετε πρόσθετα επίπεδα με επιλέξιμα με κλικ κουμπιά. Αυτά μπορούν να χρησιμοποιηθούν για την προσθήκη πεδίων, εικόνων και πινάκων στην αναφορά.

Προσθήκη πεδίου

Ένα πεδίο είναι μια απλή προβολή αριθμητικής τιμής ή κειμένου από την αναφορά. Για παράδειγμα, κάνοντας κλικ στο Τεμάχιο > Πληροφορίες > Όνομα, μπορείτε να εισαγάγετε ένα πεδίο στην αναφορά που προβάλλει το υλικό του τεμαχίου:

Όταν το κάνετε αυτό, το TecZone Bend θα προσθέσει ουσιαστικά δύο στοιχεία στην αναφορά - ένα πεδίο στο οποίο προβάλλεται η τιμή και μια ετικέτα στα αριστερά αυτού του πεδίου. Μπορείτε να επεξεργαστείτε τι ρυθμίσεις για την ετικέτα ή για το πεδίο κάνοντας απλώς κλικ σε αυτό.

Επεξεργασία ετικέτας

Όταν κάνετε κλικ στην ετικέτα που μόλις προστέθηκε, θα δείτε ένα πλαίσιο που σάς παρέχει τη δυνατότητα να επεξεργαστείτε διάφορες ρυθμίσεις για αυτήν την ετικέτα. Η ετικέτα επισημαίνεται επίσης και προστίθενται λαβές αλλαγής μεγέθους γύρω από την ετικέτα, παρέχοντάς σας τη δυνατότητα να αλλάξετε το μέγεθός της σύροντας οποιαδήποτε από αυτές. Μπορείτε επίσης να κάνετε κλικ δεξιά στην ετικέτα και να τη σύρετε για να αλλάξετε τη θέση της. Όταν σύρετε για να μετακινήσετε ή να αλλάξετε το μέγεθος ενός στοιχείου, εμφανίζονται αυτόματα διάφορες ενδείξεις κουμπώματος για να διευκολύνουν την ευθυγράμμιση αυτής της ετικέτας με ακρίβεια με άλλα στοιχεία που υπάρχουν ήδη στην αναφορά.

Ο πίνακας επεξεργασίας για την ετικέτα (και για τα περισσότερα άλλα στοιχεία της αναφοράς) περιλαμβάνει ένα μεγάλο αριθμό ρυθμίσεων για τον συντονισμό κάθε πτυχής της οπτικής οθόνης. Μπορείτε να δείτε αυτά τα στοιχεία ανοίγοντας τις διάφορες επεκτάσιμες ομάδες:

Η παραπάνω εικόνα παρουσιάζει έναν επιλογέα χρωμάτων που χρησιμοποιείται για τον ορισμό του χρώματος του κειμένου για την ετικέτα.

Εισαγωγή εικόνας

Μια εικόνα σε μια αναφορά σηματοδοτείται από ένα εικονίδιο κάμερας (και όχι από το εικονίδιο μολυβιού που χρησιμοποιείται για απλά πεδία). Για παράδειγμα, κάνοντας κλικ στην επιλογή Τεμάχιο > Μοντέλο > Εικόνα, μπορείτε να εισαγάγετε μια εικόνα του μοντέλου 3D του τεμαχίου:

Η παραπάνω εικόνα παρουσιάζει μια εικόνα που εισάγεται και στη συνέχεια υποβάλλεται σε επεξεργασία με κλικ σε αυτήν. Ο πίνακας για την εικόνα παρουσιάζει ορισμένες ρυθμίσεις που χρησιμοποιούνται για τον έλεγχο της εικόνας. Οι ακριβείς ρυθμίσεις ενδέχεται να αλλάξουν, ανάλογα με την πραγματική εικόνα που έχει εισαχθεί. Σε αυτήν την περίπτωση, η ρύθμιση DPI χρησιμοποιείται για τον έλεγχο της ανάλυσης απόδοσης της εικόνας (με υψηλότερες τιμές εξασφαλίζεται βελτίωση της εμφάνισης, αλλά θα αυξάνεται το μέγεθος της τελικής αναφοράς). Η ρύθμιση Χωρίς 3D PDF χρησιμοποιείται για την απενεργοποίηση της δημιουργίας αρχείων 3D PDF files[1].

Εάν εισαγάγετε μια εικόνα ενός σχεδίου 2D (επίπεδη), οι επιλογές εικόνας είναι διαφορετικές, όπως φαίνεται στην παρακάτω εικόνα:

Οι ρυθμίσεις εδώ ελέγχουν το πλάτος γραμμής που χρησιμοποιείται για περιγράμματα και για διαστάσεις. Υπάρχουν ρυθμίσεις για την εμφάνιση των αριθμών αλληλουχίας κάμψης και για τον έλεγχο του εάν συμπληρώνεται το κλειστό περίγραμμα του τεμαχίου.

Εισαγωγή πίνακα

Μπορείτε να εισαγάγετε έναν πίνακα σε μια αναφορά, επιλέγοντας ένα από τα στοιχεία που εμφανίζει ένα εικονίδιο πίνακα. Για παράδειγμα, εάν ο πίνακας Τεμάχιο > Κάμψη > Κάμψεις χρησιμοποιείται για την προβολή ορισμένων πληροφοριών για όλες τις διαδικασίες κάμψης σε ένα τεμάχιο:

Η ενότητα Επιλογή στηλών χρησιμοποιείται για την επιλογή των στηλών που εμφανίζονται στον πίνακα (από όλες τις στήλες που είναι διαθέσιμες για αυτά τα δεδομένα πίνακα). Μπορείτε να επεξεργαστείτε περισσότερα χαρακτηριστικά κάθε στήλης κάνοντας κλικ στο σύνδεσμο πλοήγησης Στήλη στο κάτω μέρος της οθόνης του πίνακα:

Έτσι εμφανίζεται ο Επεξεργαστής στήλης, όπου μπορούν να υποβληθούν σε επεξεργασία τα χαρακτηριστικά της στήλης, όπως παρουσιάζεται παρακάτω:

Το πεδίο Τίτλος χρησιμοποιείται για τον ορισμό του εμφανιζόμενου τίτλου για τη στήλη, ενώ η το πεδίο Έκφραση είναι τα εσωτερικά δεδομένα TecZone Bend που εμφανίζονται σε αυτήν τη στήλη. Μπορείτε να χρησιμοποιήσετε τα κουμπιά Shift για να μετακινήσετε αυτήν τη στήλη προς τα αριστερά ή προς τα δεξιά στη γραμμή και να χρησιμοποιήσετε τα κουμπιά Εισαγωγή και Διαγραφή για να εισαγάγετε νέες στήλες ή για να διαγράψετε την επιλεγμένη στήλη.

1. Το TecZone Bend μπορεί να διαμορφώσει ορισμένες από τις εικόνες 3D ως ενσωματωμένες εικόνες 3D PDF. Αυτές μπορούν να μεγεθυνθούν, να περιστραφούν ή να μετακινηθούν με χρήση ορισμένων προγραμμάτων προβολής αρχείων PDF. Ωστόσο, αυτές δεν υποστηρίζεται ευρέως σε όλα τα προγράμματα προβολής αρχείων PDF και ορισμένα από αυτά ενδέχεται να τις εμφανίσουν ως κενά πλαίσια
