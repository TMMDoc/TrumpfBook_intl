Διαχωριστικό φύλλο DXF

Τα προσαρμοσμένα διαχωριστικά φύλλα μπορούν να εισαχθούν από αρχεία DXF ή GEO. Αυτά τα φύλλα τοποθετούνται συνήθως μεταξύ στρωμάτων ενός τεμαχίου που εναποτίθεται σε ένα κελί κάμψης και αυτή η σελίδα επεξηγεί πώς μπορείτε να ρυθμίσετε και να χρησιμοποιήσετε ένα διαχωριστικό φύλλο σε μια επικάθιση. Το DXF πρέπει να σχεδιαστεί σε mm, και θα πρέπει να περιλαμβάνει τα εξής:

  • Ένα μεμονωμένο εξωτερικό περίγραμμα (κλειστό πολύγραμμο) που καθορίζει το εξωτερικό σχήμα του διαχωριστικού.

  • Μηδέν ή περισσότερα εσωτερικά περιγράμματα που αντιπροσωπεύουν τις οπές ή τις εγκοπές στο διαχωριστικό φύλλο.

  • Ορισμένες καταχωρίσεις κειμένου της μορφής Key = Value που καθορίζουν ορισμένες απαραίτητες ιδιότητες του διαχωριστικού φύλλου.

Η παραπάνω εικόνα παρουσιάζει ένα χαρακτηριστικό διαχωριστικό φύλλο DXF. Η ακριβής θέση του εξωτερικού περιγράμματος δεν είναι κρίσιμη - TecZone Bend θα ορίσει αυτόματα τη θέση αναφοράς του διαχωριστικού φύλλου στο κέντρο του σχεδίου.

Αυτά τα πλήκτρα χρησιμοποιούνται για τον ορισμό των σημαντικών μεταδεδομένων που αφορούν στο διαχωριστικό φύλλο:

  • Η ρύθμιση Όνομα καθορίζει το όνομα του φύλλου (όπως εμφανίζεται στη διεπαφή χρήστη TecZone Bend). Εάν υπάρχει ήδη ένα φύλλο με το ίδιο όνομα στη βάση δεδομένων διαχωριστικών φύλλων, αυτό θα αντικατασταθεί.

  • Το πάχος είναι το πάχος κάθε φύλλου σε mm. Αυτή είναι μια σημαντική τιμή, δεδομένου ότι χρησιμοποιείται για τη ρύθμιση της σωστής απόστασης των τεμαχίων στη στοίβα απόθεσης.

  • Η ρύθμιση βάρους σε kg χρησιμοποιείται για να ελεγχθεί ότι το στοιχείο αρπάγης δεν είναι υπερφορτωμένο όταν προσπαθείτε να παραλάβετε το διαχωριστικό φύλλο.

  • Η μαγνητική ρύθμιση σηματοδοτεί αν το διαχωριστικό φύλλο είναι κατασκευασμένο από σιδηρούχο υλικό και χρησιμοποιείται για να ελέγξει εάν μπορεί να παραληφθεί από ένα μαγνητικό στοιχείο αρπάγης. Αυτό μπορεί να παραλειφθεί και η προεπιλεγμένη τιμή είναι 0 (μη μαγνητική) που είναι η συνήθης για τα περισσότερα διαχωριστικά φύλλα.
