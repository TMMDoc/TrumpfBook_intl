Dobrar CAM

Batentes traseiros

Retorno automático a partir da espessura da chapa … mm - Esta configuração permite inserir um valor em mm de retirada automática dos encostos traseiro quando a espessura da chapa inserida é atingida.

Estratégia de encosto traseiro - Isso é usado para definir como os encostos traseiros devem parar na peça de trabalho.

As seguintes opções de seleção estão disponíveis:

Padrão - Os encostos traseiros param para cada dobra do modo considerado melhor pelo software.

2 x batentes ger. - As superfícies retas de ambos os encostos traseiros param contra a peça de trabalho.

Aperto+parada - Um encosto traseiro prende a peça de trabalho e o outro encosto traseiro a para com sua superfície reta.

Medição do ângulo

Medição do ângulo é apenas para máquinas Trumpf. Usar o sistema de medição de ângulo da família Automatically Controlled Bending significa que até mesmo a primeira peça de uma série sai perfeita. Os sensores determinam o ângulo atual juntos com o encosto traseiro e guie a barra de pressão para atingir o ângulo desejado. Nesta seção, podem ser inseridas as configurações para ACB na máquina.

Sistema de medição do ângulo

Laser - Marque esta opção se a máquina tiver um ACB laser.O ACB laser é onde um laser projeta uma linha na chapa e uma câmera detecta o ângulo.

Sim fios - Marque esta opção se a máquina usar ACB Wireless. ACB Wireless são discos sensores integrados na ferramenta superior.

Clássico - Marque esta opção se a máquina usar o ACB Classic.

2 medições do ângulo a partir do comprimento da dobra … mm - Esta opção é o comprimento mínimo da dobra para fazer duas medições para garantir o ângulo correto durante a dobra. Isso pode ser entre 50 > 4000 milímetros.

3 medições do ângulo a partir do comprimento da dobra … mm -Esta opção é o comprimento mínimo da dobra para fazer três medições para garantir o ângulo correto durante a dobra. Isso pode ser entre 50 > 4000 milímetros.

Laser ACB em posição de estacionamento (requer reinicialização!) - Esta é a posição de repouso do Sensor Laser.

Limite de qualidade ACB - Aviso - Este é o limite que gerará um aviso ao medir a peça, como um fator de correção. Se for no intervalo específico, então ele vai registrar um aviso.

Limite de qualidade ACB - Erro - Este é o limite que vai gerar um erro ao medir a peça, como um fator de correção. Ele vai registrar um erro se estiver no intervalo específico.

Estratégias de processamento para a adaptação da peça de trabalho

Se um arquivo 2D ou 3D for editado no software, os raios de dobra e a dedução de dobra são verificados. Diversas estratégias de processamento são executadas dependendo de quais estratégias foram selecionados para modificar a peça de trabalho nas configurações do Easy Programming. Se a decisão foi definida como interativa, será mostrado o diálogo Adaptar peça de trabalho.

Importar desenrolamento

Decisão interativa - Ao navegar de uma peça 2D para dobrar, usar esta configuração exibirá automaticamente o diálogo Adaptar peça de trabalho. Isso dará opções de retenção do tamanho plano, mantendo as dimensões 3D ou não alterando nada, tornando esse recurso muito interativo e conduzido pelo usuário.

Manter o modelo 3D - Esta opção altera o desdobramento, que então deve ser salvo manualmente. Esta estratégia é usada se o espaço em branco ainda não foi cortado.

Manter o desenrolamento - Esta opção significará que o raio de dobra e a dedução de dobra são ajustados ao desdobramento existente; a linha de dobra permanece no mesmo lugar. Isso mudará o comprimento lateral da peça. O desdobramento não é alterado. Esta estratégia é usada se o espaço em branco já tiver sido cortado ao tamanho.

Não executar nenhum ajuste - Usar esta opção significará que o raio de dobra e a dedução de dobra permanecem inalterados. A peça é exibida com as dimensões corretas em 3D, mas os raios de dobra mostrados não correspondem aos raios de dobra reais. A dimensão de cada dobra da peça de trabalho pode mudar.

Importar modelo 3D

Decisão interativa - Ao navegar de uma peça 3D para dobrar, usar esta configuração exibirá automaticamente o diálogo Adaptar peça de trabalho. Isso dará opções de retenção do tamanho plano, mantendo as dimensões 3D ou não alterando nada, tornando esse recurso muito interativo e conduzido pelo usuário.

Manter o modelo 3D - Esta opção altera o desdobramento, que então deve ser salvo manualmente. Esta estratégia é usada se o espaço em branco ainda não foi cortado.

Manter o desenrolamento - Esta opção significará que o raio de dobra e a dedução de dobra são ajustados ao desdobramento existente; a linha de dobra permanece no mesmo lugar. Isso mudará o comprimento lateral da peça. Esta estratégia é usada se o espaço em branco já tiver sido cortado ao tamanho.

Não executar nenhum ajuste - Usar esta opção significará que o raio de dobra e a dedução de dobra permanecem inalterados. A peça é exibida com as dimensões corretas em 3D, mas os raios de dobra mostrados não correspondem aos raios de dobra reais. A dimensão de cada dobra da peça de trabalho pode mudar.

Valor de redução - Esta opção possibilita forçar o método do Fator K a ser usado. Esta configuração pode ser definida nos Padrões da Aplicação ou pode ser ajustada individualmente para cada máquina.

Para máquinas não Trumpf, os fatores K são usados sempre; os cálculos de dobre Trumpf estão disponíveis somente para prensas dobradeiras Trumpf.

Versão de encosto traseiro esquerdo - Dependendo da versão BG utilizada, o programa CN será modificado para representar isso.

Versão de encosto traseiro direito - Dependendo da versão BG utilizada, o programa CN será modificado para representar isso. == Outros ajustes **Ângulo de pré-dobra ao dobrar - Esta opção pode editar o ângulo de pré-dobra do vinco entre 10° a 60°. O padrão é 40°, já que esse é o ângulo de pré-dobra de vincagem recomendado para a maioria dos fabricantes. Está definido para 30° para todas as máquinas Trumpf, em todas as diversas edições.

Pré-ajuste do modo BendGuard - Ao configurar isso para cálculo automático (o padrão), Bend calculará um modo de proteção de dobra dobra. No entanto, você pode forçar o modo de proteção de dobra para uma das opções para ele usar somente aquela, independentemente de ser apropriado ou não.

Você também pode substituir essa configuração para cada máquina, configurando-a nos Padrões da Máquina, em vez de Ajustes da aplicação.

Posição da mesa da máquina - Esta configuração tem três opções para a posição da mesa:

  1. Removida

  2. Para baixo

  3. Para cima

Controle de quantidade de peças - Ativar esta opção mostrará os segmentos da ferramenta que podem ser necessários para que a peça de trabalho seja dobrada. Marcadores de inventário insuficientes serão exibidos.

Se possível, evitar sobrecarga da ferramenta - Ativar esta opção assegurará que seja evitada uma sobrecarga da ferramenta. Mesmo que uma ferramenta com um maior prioridade funcionasse na peça, uma ferramenta de prioridade mais baixa será usada para evitar problemas de sobrecarga.

Ignorar as ferramentas no arquivo geo/dxf - Ativar esta opção ignorará qualquer informação encontrada em um arquivo DXF/GEO em relação ao qual ferramental deve ser usado.
