Contorno

O painel Contorno é usado para operações em todos os cortes a laser em um contorno específico. Por exemplo, você pode adicionar microjoints para dividir um corte a laser em dois, ou remover as juntas para combinar dois cortes a laser consecutivos no mesmo contorno em um.

Para acessar este painel, clique em um corte a laser para abrir o painel Corte a laser e, em seguida, clique no botão de navegação Contorno perto da parte inferior para abrir o painel de contorno.

  • Cancelar: Excluir todos os cortes a laser neste contorno.

  • Copiar ferramental: Copiar os cortes a laser neste contorno para outros contornos semelhantes.

  • Microjoint: As configurações nesta seção são usadas para adicionar ou remover microjoints no contorno.

    • Adicionar: Clique aqui para adicionar mais microjoints ao contorno. Cada vez que uma microjoint é adicionada, as juntas existentes são deslocadas para fazer todos os cortes a laser aproximadamente iguais em comprimento. A imagem abaixo mostra os resultados após cliques sucessivos do botão Adicionar. (Observe que entre a primeira e a segunda opção, o número de cortes a laser não é aumentado, mas uma microjoint foi adicionada no final do corte).

    • Reduzir: Reduzir o número de microjoints combinando dois cortes a laser.

    • Remover tudo: Remover todas as microjoints neste contorno (resultando em um único corte a laser cortando todo o contorno).

    • Scrap: Esta seção aparece se estivermos inspecionando um furo (contorno interno), e é usada para adicionar cortes de sucata para cortar a apara criada pelo furo, ou para remover os cortes de sucata, se tiverem sido adicionados anteriormente.

Clicar em Microjoints

Clicar no botão Clicar em Microjoints muda este painel para um modo especial onde você pode dividir cortes a laser ou mesclar dois cortes a laser consecutivos interativamente.

  • Deslocamento de canto: Digite um valor aqui para criar uma divisão a uma distância precisa de um canto. À medida que você se move em direção a este ponto, um indicador de encaixe aparece para ajudar a posicionar a divisão.

  • Microjoint: Se estiver ligado, uma microjoint é criada no ponto de divisão cada vez que um novo corte a laser é criado. Aqui está um exemplo de uma microjoint sendo adicionada (já que estamos em um canto, um indicador de encaixe junção de canto é exibido).

Você pode mover o mouse sobre uma microjoint entre dois cortes e o indicador de encaixe muda para merge. Clicar agora removerá a junta e mesclará os dois cortes a laser adjacentes em um único.
