Aba Peças

A guia Peças é usada para adicionar peças para intercalação e para configurar parâmetros como a quantidade a intercalar, ângulos de rotação aceitáveis, etc.

Botões de Ação

Estes são os botões ao longo do lado esquerdo do painel de peças, usados para adicionar ou remover peças para a tarefa e executar algumas operações em todas as peças:

  • Use Adicionar…​ para adicionar uma peça para intercalação. Isso pode ser um arquivo .fx já equipado ou qualquer arquivo 2D ou 3D que TecZone Laser possa importar e converter em uma peça de chapa. Esses arquivos serão então _equipados conforme explicado na seção Fluxo de trabalho da peça.

  • O botão Adicionar conjunto…​ é usado para adicionar uma montagem 3D completa. TecZone Laser selecionará as peças de chapa da montagem e adiciona-as ao trabalho de intercalação. As quantidades para os componentes individuais do conjunto são configuradas para produzir uma quantidade do conjunto completo.

  • O botão Adicionar layout…​ é usado para importar um DXF contendo uma planilha totalmente aninhada e decompô-la novamente em um trabalho de intercalação.

  • O botão Importar CSV…​ é usado para carregar a definição de um trabalho de intercalação de um arquivo CSV (valores separados por vírgulas). O arquivo deve ter um formato específico com 8 colunas, conforme explicado na dica abaixo:

  • Salvar peça(s)…​ é usado para salvar as peças selecionadas como arquivos .FX. Quando você inicia a tarefa de intercalação com arquivos de geometria (como DXF, Geo, dados 3D), estes são ferramentados automaticamente quando adicionados à lista de peças. Este botão pode ser usado para salvar as versões com ferramentadas dessas peças para que elas possam ser reutilizadas em outros trabalhos de intercalação.

  • O Referramentar tudo é usado para recalcular o ferramental para todas as peças no trabalho. É útil depois de fazer algumas alterações nas configurações da máquina, ou nas configurações para este trabalho de intercalação específico, para certificar-se de que todas as peças estejam usando as novas configurações.

Colunas da tabela de peças

Aqui estão algumas das colunas da tabela de peças:

  • Peça é a quantidade necessária da peça. Você pode inserir um número simples aqui, ou um intervalo como 20 - 30. Isso significa que pelo menos 20 cópias desta peça serão intercaladas, mas não mais do que 30. O TecZone Laser liberará para intercalar qualquer quantidade entre esses dois valores para maximizar a eficiência de intercalação, mas nunca iniciará uma nova chapa apenas para intercalar peças além da quantidade mínima.

  • As peças são intercaladas em ordem crescente de Prioridade, se diferentes valores de prioridade forem definidos para diferentes peças. No entanto, Prioridade = 0 é tratada como um caso especial: essas peças têm a menor prioridade.

  • Sentido de laminação preferencial da peça. é usado para configurar o sentido de laminação da peça, se isso for importante. Se estiver definido para um valor diferente de Nenhum, que restringirá os ângulos nos quais a peça pode ser girada pelo mecanismo de intercalação. (Para que isso funcione corretamente, você também deve especificar a direção do grão das chapas na aba Chapas).

  • O valor Agrupar é uma dica para o mecanismo de nesting - TecZone Laser tentará para manter as peças com o mesmo valor de grupo juntas na mesma chapa, sempre que possível.

  • Se o interruptor TwinLine estiver ligado para uma peça, o mecanismo de intercalação tentará usar o corte de linha comum para esta peça (a peça é apenas de linha comum com outras cópias de si mesma).
