Bend deductions

Bend deductions

Bend deductions são usadas para calcular o comprimento desenvolvido[1] de uma peça de chapa metálica. É necessário para produzir com precisão peças com as dimensões corretas após a dobra.

Bend deductions levam em conta o alongamento e a compressão do material que ocorrem durante o processo de dobra. Ao usar bend deductions, o TecZone Bend pode ajustar o comprimento do padrão plano para garantir que a peça dobrada final atenda às dimensões especificadas.

As bend deductions podem variar com base em fatores como tipo de material, espessura, raio de dobra e método de dobra.

Quando uma peça 3D é importada para o TecZone Bend, o software calcula automaticamente as bend deductions com base na geometria da peça e nas propriedades do material. Você também pode ajustar manualmente as bend deductions, se necessário, para ajustar o comprimento do padrão plano.

Configurações de bend deduction

Bend deductions personalizadas podem ser definidas no banco de dados Bend Deductions. Para acessar, clique no ícone database e selecione Bend Deductions.

Você pode Add adicionar um novo cenário de bend deduction ou selecionar um cenário existente para Edit editar ou Delete excluir. Você também pode Copy copiar um cenário existente para criar um novo com configurações semelhantes.

Clique em Add para criar um novo cenário de bend deduction. Selecione e digite os seguintes parâmetros:

  • Material: Selecione o material ao qual as bend deductions se aplicam.

  • Thickness: Defina a espessura da chapa à qual as bend deductions se aplicam.

  • Punch radius: Defina o raio da ferramenta superior ao qual as bend deductions se aplicam.

  • V-width type: Escolha o tipo de medição de largura V usado.

  • Die V-width: Defina a largura da ferramenta inferior à qual as bend deductions se aplicam.

  • Die radius: Defina o raio da ferramenta inferior ao qual as bend deductions se aplicam.

  • Die angle: Defina o ângulo da ferramenta inferior ao qual as bend deductions se aplicam.

Clique em OK para salvar o novo cenário.

Além disso, selecione um cenário de bend deduction e clique na opção …​ para ver mais opções:

  • Graph: Exibe um gráfico mostrando a relação entre o ângulo de dobra e a bend deduction. Esta representação visual ajuda a entender como as bend deductions mudam com diferentes ângulos de dobra. Mova o cursor sobre as linhas azul e vermelha do gráfico para ver valores específicos.

  • Import: Permite importar dados de bend deduction de um arquivo CSV externo.

  • Export: Permite exportar os dados atuais de bend deduction para um arquivo CSV para fins de backup ou compartilhamento.

  • 3rd party: Fornece opções para importar dados de bend deduction de software ou fontes de terceiros.

Tipos de bend deduction

TecZone Bend suporta três tipos de bend deductions: Air Bending, Folding e Coining.

Cada tipo tem seu próprio conjunto de parâmetros e cálculos para determinar com precisão as bend deductions com base no método de dobra usado.

Air Bending

Air bending é um método de dobra comum em que a chapa metálica é dobrada usando um punção e uma matriz sem fechar completamente a matriz. Este método permite mais flexibilidade e é adequado para uma ampla gama de materiais e espessuras.

Para adicionar bend deductions para air bending, selecione um cenário adequado do banco de dados Bend Deductions. Clique em Add para criar um novo cenário de air bending deduction e digite o Bend Angle.

Ao inserir o ângulo de dobra, os valores correspondentes de ui:962[Deduction (Edge)], Deduction (Tangent) e Inner Radius serão calculados e exibidos.

Clique em OK para salvar a nova air bending deduction.

Folding

Folding ou Hemming é um método de dobra em que a chapa metálica é dobrada usando um punção e uma matriz que fecha completamente, criando uma dobra acentuada. Este método é frequentemente usado para materiais mais finos e requer controle preciso de bend deductions.

Para adicionar bend deductions para folding, selecione um cenário adequado do banco de dados Bend Deductions. Clique em Add para criar um novo cenário de folding deduction e selecione o Hemming method.

Clique em OK para salvar a nova folding deduction.

Coining

Coining é um método de dobra em que a chapa metálica é prensada entre um punção e uma matriz com alta pressão, resultando em uma dobra precisa e acentuada. Este método é normalmente usado para materiais mais espessos e requer bend deductions precisas.

Para adicionar bend deductions para coining, selecione um cenário adequado do banco de dados Bend Deductions. Clique em Add para criar um novo cenário de coining deduction.

Clique em OK para salvar a nova coining deduction.

1. O comprimento desenvolvido é o comprimento total da chapa metálica plana antes da dobra. Este é o comprimento real do padrão plano que é cortado (usando cortador a laser) e formado (usando prensa dobradeira) para criar a peça final.
