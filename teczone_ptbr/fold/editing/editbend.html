Editar uma dobra

Algumas configurações básicas de uma operação de dobra podem ser editadas abrindo o painel dobra. Para abrir o painel de dobra, use o navegador de dobras:

  • Clique em um dos números no navegador de dobras para alternar para uma dobra.

  • Clique novamente no mesmo número para abrir o painel de dobra para essa dobra.

O painel de dobra pode ser visto ao lado. Ele tem algumas configurações e operações para trabalhar na dobra.

  • A caixa de entrada Folga é usada para ajustar a lacuna para dobra (esta é a lacuna entre a lâmina e as ferramentas superiores de fixação). À medida que a lacuna é ajustada a partir do seu valor padrão, o navegador de dobras exibirá um aviso de lacuna fora do padrão. A verificação de colisão também será feita imediatamente com a lacuna atualizada e você pode ajustar a lacuna para evitar colisões. A força de dobra também será calculada, e se você definir a lacuna tão baixa que a força de dobra máxima é excedida, o navegador de dobras exibirá isso como um erro.

  • A configuração P86 é usada para controlar a aplicação de pressão de pisamento adicional durante o ciclo de dobra.

  • Use o botão Saltar a dobra para ignorar esta dobra; esta dobra não será processada na máquina de dobra oscilante e precisa ser processada mais tarde (possivelmente por uma prensa dobradeira).

  • O botão Nova página é usado para dividir um lado em dois lados separados; isso só aparece se você não estiver editando a primeira dobra de um lado. Clicar neste botão divide o lado nesta dobra e inicia um novo lado. Existem vários motivos pelos quais você pode querer dividir um lado:

    • O lado pode precisar ser dividido em dois, e a peça pode precisar ser espelhada entre os dois lados antes que o processamento possa ser concluído (apenas em máquinas manuais).

    • Algumas dobras de outro lado podem precisar ser concluídas antes que as dobras restantes deste lado podem ser concluídas. Nesse caso, você pode dividir o lado, e sequenciar novamente os lados para que as dobras do outro lado sejam inseridas no meio.

  • Os botões Continuar e Voltar são usados para navegar entre as diferentes dobras na peça. (Você também pode usar o PageUp e as teclas PageDown para isso).

  • O botão Página é usado para navegar a partir desta operação de dobra para o lado de contenção. Clicar neste botão exibe o painel Bend Side (descrito abaixo) que pode ser usado para executar algumas operações em todo o lado de dobra (ou seção).

Se esta dobra tiver sido ignorada, essas configurações não serão exibidas; há apenas um botão Não saltar a dobra para trazer a dobra de volta ao processamento.
