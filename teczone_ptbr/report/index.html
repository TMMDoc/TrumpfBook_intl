Relatórios

Além de gerar programas CN, TecZone Bend também pode gerar vários relatórios para uma peça, dependendo dos módulos instalados. Ao trabalhar com a tecnologia de dobra, por exemplo, é útil gerar uma chapa de configuração que lista as ferramentas a serem montadas e fornece detalhes sobre a sequência de dobra.

O Designer de Relatórios em TecZone Bend ajuda você a projetar o conteúdo e o estilo de tais relatórios. Depois de criar um relatório, você pode exibi-lo em PDF ou HTML.[1] formatos. Veja como o designer de relatórios fica quando está aberto:

Usando um relatório

Você pode abrir um relatório para edição escolhendo o menu Arquivo/Abrir e escolhendo um arquivo de modelo de relatório (são arquivos com a extensão .ftrep). Você pode encontrar alguns modelos de relatório de amostra na pasta Amostras da sua instalação TecZone Bend.

Antes de abrir um modelo de relatório para trabalhar, você deve primeiro ter uma peça aberta, e você deve ter processado a peça até a conclusão (você deve gerar os dados apropriados da tecnologia de Dobrar/Cortar/Virar). Isso ocorre porque o designer de relatórios trabalha com dados ao vivo; ele mostra dados reais da peça à medida que você cria o relatório, para que você possa ver imediatamente como será o resultado final impresso.

Você também pode começar com um relatório em branco escolhendo o menu Arquivo/Novo/Modelo de Relatório. Você pode então preencher o relatório adicionando vários itens de relatório (campos de texto, imagens, etiquetas, etc.) e, em seguida, salvá-lo.

Quando você tem um modelo de relatório aberto, você pode usar os comandos Arquivo/Exportar para PDF ou o comando Arquivar/Exportar para HTML para renderizar o relatório para o formato PDF ou para o formato HTML. O relatório é impresso usando os dados da peça que está atualmente aberta e ativa. Você também pode alternar o relatório entre a visualização de dados ao vivo (o padrão) e uma exibição de design que exibe as expressões subjacentes que são usadas para preencher os diversos campos no relatório.

Relatórios impressos automaticamente:

Você pode configurar TecZone Bend para imprimir automaticamente um relatório de dobra sempre que uma solução de dobra é gerada (assim como pode gerar um programa CN automaticamente quando a solução é calculada). Use a página de configurações de saídas do módulo de tecnologia de dobra para configurar isso.

No exemplo acima, a geração automática de relatórios está ativada. O modelo de relatório está definido como lib:Report/bend.ftrep, que é o relatório padrão modelo para o CAM de dobra (instalado como parte de cada instalação TecZone Bend). Se você criar um modelo de relatório diferente usando o designer de relatórios, poderá especificar este modelo aqui.

As configurações também permitem controlar a pasta para a qual o relatório é gerado e o formato (HTML ou PDF) em que o relatório é escrito.

1. Os arquivos HTML gerados pelo gravador de relatórios TecZone Bend são independentes.Eles incluem todas as imagens necessárias, chapas de estilo e outros recursos incorporados dentro do arquivo HTML. Assim, eles podem ser facilmente transportados ou arquivados sem quaisquer dependências externas.
