Campos, Imagens e Tabelas

Clicar nas fontes de dados as abre e exibe níveis adicionais de botões clicáveis. Estes podem ser usados para adicionar campos, imagens e tabelas ao relatório.

Adicionar um campo

Um campo é uma exibição numérica ou de texto simples do relatório. Por exemplo, clicando em Peça > Info > Nome, você pode inserir um campo no relatório que exibe o material da peça:

Quando você fizer isso, TecZone Bend adicionará dois itens ao relatório - um campo exibindo o valor e um rótulo à esquerda desse campo. Você pode editar as configurações para o rótulo, ou para o campo, apenas clicando nele.

Editando um rótulo

Ao clicar no rótulo que acabou de ser adicionado, você verá um painel que permite que você edite várias configurações para esse rótulo. O rótulo também é destacado e são adicionadas alças de dimensionamento ao redor do rótulo, permitindo que você o redimensione arrastando qualquer uma delas. Você também pode clicar com o botão direito no rótulo e arrastá-lo para reposicioná-lo. Quando você está arrastando para mover ou redimensionar um item, diversos indicadores de encaixe são exibidos automaticamente para facilitar o alinhamento deste rótulo com precisão com outros itens já no relatório.

O painel de edição para o rótulo (e para a maioria dos outros itens de relatório) contém um grande número de configurações para ajustar todos os aspectos da exibição visual. Você pode ver isso abrindo os diversos grupos expansíveis:

A imagem acima à direita mostra um seletor de cores em uso para definir a cor do texto para o rótulo.

Inserindo uma imagem

Uma imagem em um relatório é indicada por um ícone de câmera (em vez do ícone de lápis usado para campos simples). Por exemplo, clicando em Peça > Modelo > Imagem, você pode inserir uma imagem do modelo 3D da peça:

A imagem acima mostra uma figura inserida e, em seguida, sendo editada clicando nela. O painel da imagem mostra algumas configurações usadas para controlar a imagem. As configurações exatas podem mudar, dependendo da imagem real que foi inserida. Nesse caso, a configuração DPI é usada para controlar a resolução de renderização da imagem (valores mais altos ficarão mais agradáveis, mas farão com que o tamanho final do relatório se torne maior). A configuração No PDF 3D é usada para desativar a geração de _PDF 3D_files.[1]

Se você inserir uma imagem de um desenho 2D (plano), as opções de imagem são diferentes, como visto na imagem abaixo:

As configurações aqui controlam a largura da linha usada para contornos e para dimensões. Existem configurações para exibir os números de sequência de dobra e para controlar se o contorno fechado da peça está preenchido.

Inserindo uma tabela

Você pode inserir uma tabela em um relatório, escolhendo um dos itens que exibe um ícone de tabela. Por exemplo, se você usar a tabela Part > Bend > Bends para exibir algumas informações sobre todas as operações de dobra em uma peça:

A seção Escolher colunas é usada para selecionar as colunas que são exibidas na tabela (dentre todas as colunas disponíveis para estes dados tabulares). Você pode editar mais atributos de cada coluna clicando no link de navegação Coluna na parte inferior do painel da mesa:

Isso abre o editor de colunas, onde os atributos da coluna podem ser editados, como visto abaixo:

O campo Título é usado para definir o título exibido para a coluna, enquanto o campo Expressão são os dados internos TecZone Bend exibidos nesta coluna. Você pode usar os botões Shift para deslocar esta coluna para a esquerda ou para a direita na ordenação, e use os botões Inserir e Excluir para inserir novas colunas ou para excluir a coluna selecionada.

1. TecZone Bend pode renderizar algumas de suas imagens 3D como imagens PDF 3D incorporadas; estes podem ser ampliados, girados ou deslocados usando alguns visualizadores de PDF. No entanto, estes não são amplamente suportados em todos os visualizadores de PDF, e alguns deles podem exibi-los como caixas em branco.
