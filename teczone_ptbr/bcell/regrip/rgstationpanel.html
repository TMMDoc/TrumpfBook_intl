Painel de estações RG

As Estações de Remanipulação são referidas como RG-Stations no TecZone. O painel RG-Station é usado para configurar as estações de remanipulação de uma máquina de célula de dobra.

Para acessar o painel RG-Stations:

  • Clique no link RG-Stations no painel Pickup, Bend, Regrip ou Deposit.

  • Como alternativa, clique diretamente em uma estação de remanipulação.

Dependendo da operação de dobra e das posições das estações, o painel RG-Stations exibe um indicador alfabético (P, R1, #1, D, etc.) dentro do painel.

A caixa de seleção Flip Part no painel Pickup é usada para colocar a peça de cabeça para baixo em relação à posição atual. Às vezes isso é útil para evitar colisões entre flanges voltados para baixo e os braços da estação de remanipulação. O TecZone recalculará imediatamente um novo caminho do robô quando você virar a peça.

Quando uma estação de remanipulação não é usada, o painel parece simples, pois a posição da peça não pode ser controlada (depende da posição de fixação da peça entre punção e matriz).

  • O Stations é usado para selecionar uma das estações ou para desativar ou ativar ambas as estações.

  • Use as configurações Positions para ajustar as posições das estações no trilho da mesa da máquina. Na imagem mostrada abaixo, à esquerda, as estações não estão posicionadas sob a peça no início.

    Após alterar os valores na opção Positions, ambas as estações são posicionadas de modo que as ventosas da RG-Station sejam colocadas sob a peça.

As configurações na seção Suction são usadas para configurar os braços de sucção.

  • Use o seletor Type para alterar o tipo de ventosa.

  • Use o seletor Arm para selecionar um ou todos os braços de sucção. Quando selecionado, cada braço é identificado por seu rótulo (A, B, C, etc.).

  • Use a configuração Angle para girar o braço (em incrementos de 5°).

  • Use a configuração Grid para estender/retrair o braço (em incrementos de 10 mm).

Você também pode alterar interativamente o ângulo e a grade arrastando os controles deslizantes individualmente. Passe o mouse sobre um braço de sucção para ver os controles deslizantes de ângulo e grade.

Quando All braços estão selecionados, não é possível alterar o ângulo e a grade interativamente.

  • Selecione o estado de operação do braço de sucção para off, on ou removed.

  • Após concluir a configuração de sucção, você pode usar a opção Copy Right para usar a mesma configuração para o outro lado da RG-Station.

  • Os links Prev e Next são exibidos se houver várias operações de remanipulação nesta peça e são usados para navegar entre elas.

  • O link Regrip R1 é usado para editar a posição de repouso da peça nas estações de remanipulação.

À medida que você reconfigura a estação de remanipulação, o TecZone verifica colisões com a peça, o robô e a garra em tempo real e atualiza os indicadores de status imediatamente.
