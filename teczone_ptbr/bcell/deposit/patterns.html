Tipos de padrão de depósito

Tipos de padrão de depósito

Simple Grid

O padrão Simple Grid é o mais comumente usado e simplesmente empilha peças em uma grade retangular no palete.

Quando usamos a posição Simple Grid, apenas uma posição de depósito D1 está disponível por padrão.

Weave

O padrão Weave é útil com peças longas e finas. As peças nas camadas D2 são colocadas a 90° das peças na camada D1 como mostrado ao lado.

  • A configuração Weave controla o número de peças empilhadas lado a lado em cada camada do entrelaçamento.

  • O Gap é o espaço entre cada uma dessas peças dentro do entrelaçamento

  • As configurações de Grid na aba Repeat podem então ser usadas para repetir todo o empilhamento entrelaçado várias vezes no palete. Neste exemplo, o empilhamento entrelaçado é repetido duas vezes (2 colunas, 1 linha).

  • A configuração Gap (Z,X) na aba Repeat é usada quando o entrelaçamento é repetido e especifica o passo entre as camadas entrelaçadas.

Alternating

O padrão Alternating é o mais flexível. Neste padrão, você pode controlar a orientação, inversão e deslocamento relativo de cada camada alternada.

O padrão alternado fornece um empilhamento mais estável e também compacto de peças.

Quando usamos o padrão Alternating, as posições de depósito D1 e D2 junto com Regrip R1 estão disponíveis por padrão.

Ajuste os valores de Delta shift em D1, D2 e adicione Regrip para travar as peças umas com as outras.

Alternating Batch

O padrão Alternating Batch é semelhante ao padrão alternado, mas com a possibilidade de empilhar componentes em lotes e alternar camadas.

Batch shift é usado para mover peças de e para os eixos Z e X dentro de um lote.

Batch count é usado para definir o número de componentes por lote por depósito D1, que pode variar de 1 a 50. No exemplo abaixo, está definido como 5.

Delta shift é usado para mover um depósito D1 inteiro consistindo de lotes de e para os eixos Z e X.

Use a opção layers na aba Repeat grid (R) para definir o número de lotes por depósito. No exemplo abaixo, está definido como 2.

Drop to Basket

Este padrão é usado para soltar peças pequenas em uma cesta de coleta, usando a garra mecânica.

Conveyor

O padrão Conveyor é usado para depositar peças em uma esteira de peças. Esta opção é essencial quando peças longas e finas são depositadas na esteira.

Quando a garra de mandíbula mecânica é usada para peças de tamanho médio, todos os outros tipos de padrão serão desabilitados para seleção, exceto para o padrão de esteira.

Inclined

O padrão Inclined é usado para empilhar peças verticalmente contra uma parede (ou um palete que tem uma parede lateral).

Este padrão nem sempre está disponível. A peça deve ser grande o suficiente para que isso seja viável. Para peças pequenas, o robô não consegue alcançar baixo o suficiente para empilhar a peça contra a parede.

Inclined pair

Este tipo de padrão é muito semelhante ao padrão Alternating, exceto que as peças são depositadas verticalmente como um par.

Um Regrip adicional é introduzido em D1 ou D2 para formar um par.

Auto-pitch determina o ângulo de inclinação do empilhamento automaticamente.

A configuração Pitch é usada para ajustar o ângulo de inclinação do empilhamento e os valores de Delta shift são usados para mover a peça nos eixos X e R para fazer um empilhamento mais compacto.

Manual

O padrão Manual é usado para colocar cada camada de peças em uma orientação ligeiramente diferente.

  • Use o botão Add para adicionar uma nova peça no topo da camada superior.

  • Use os botões de navegação ao lado da aba Dn para ir para a camada anterior ou seguinte.

  • As configurações Delta shift, Rotate e Pitch podem ser usadas para ajustar a peça de modo que ela se encaixe melhor na peça abaixo. Você também pode usar o Delta shift e clicar em Auto-pitch para pedir ao TecZone Bend para calcular um ângulo de inclinação ideal.

  • Use o botão Remove para remover a peça superior.

  • Depois que um empilhamento manual é criado, você pode usar as configurações na aba Repeat (R) para repetir o mesmo empilhamento no palete. A imagem acima mostra o mesmo empilhamento repetido para 2 colunas, 1 linha com um espaço de 250 mm entre as colunas.

  • As configurações na aba Sequence (S) podem ser usadas para controlar se as peças são empilhadas by layer ou by pile. Se você usar o empilhamento pilha por pilha, todas as peças na primeira pilha são finalizadas antes de começarmos na segunda pilha.
