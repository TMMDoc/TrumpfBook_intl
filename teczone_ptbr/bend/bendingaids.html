Bending Aids

Bending Aids

Bending Aids são componentes da máquina prensa-dobradeira que ajudam a dobrar chapas metálicas mais espessas. Eles são projetados para fornecer suporte e controle adicionais durante o processo de dobra, garantindo resultados precisos e consistentes.

Uma prensa-dobradeira pode ter vários bending aids ou um único bending aid instalado, dependendo dos requisitos específicos da operação de dobra.

Bending aids são aplicáveis apenas para séries TruBend 5000 e 8000.

Configurar Bending Aids

Bending Aids podem ser configurados em Database / Machines / Settings.

Quando uma peça é importada e uma máquina com bending aids é selecionada, TecZone Bend configura automaticamente os bending aids com base nos requisitos da peça.

Usar Bending Aids

Uma vez gerada uma solução de dobra viável, os bending aids podem ser ajustados ainda mais para otimizar o processo de dobra.

Clique diretamente no bending aid no TecZone Bend para abrir o painel Bend Aid. O número no painel bend aid representa a sequência de dobra.

  • Index - Indica o bending aid atualmente selecionado.

  • Position (Z,R) - Usado para ajustar a posição do bending aid ao longo dos eixos Z e R.

  • Enabled - Marque esta caixa de seleção para ativar o bending aid.

  • Auto-Place - Usado para posicionar automaticamente o bending aid com base na geometria da peça.

Speed:

  • Down - Usado para ajustar a velocidade de retração do bending aid após a operação de dobra.

  • Set All Bends - Usado para aplicar a configuração de velocidade atual a todas as dobras na sequência.

Advanced:

  • Method - Usado para definir o modo operacional do bending aid. As opções incluem:

    • Default - Redefine as configurações do bending aid para valores padrão.

    • Return at UDP - Configura o bending aid para retornar a uma posição definida pelo usuário após cada dobra.

    • Tilt product - Permite que o bending aid incline a peça durante o processo de dobra para melhor alinhamento.

    • Static angle - Permite definir um ângulo fixo para o bending aid durante a operação.

  • Stop angle - Usado para definir o ângulo no qual o bending aid irá parar durante a operação.

Posição de estacionamento de Bending Aids

Bending Aids podem ser estacionados em uma posição designada quando não estão em uso. Isso ajuda a otimizar o espaço de trabalho e garante segurança durante a operação. Isso também evita colisões com outros componentes da máquina.

Bending Aids podem ser estacionados nas seguintes posições:

  1. Default - Bending aids podem ser estacionados até o comprimento da mesa.

  2. Small - Bending aids podem ser colocados 700 mm além do comprimento da mesa.

  3. Large - Bending aids podem ser colocados 1200 mm além do comprimento da mesa.

Para máquinas tandem, há quatro bending aids disponíveis. Para máquinas tandem esquerdas, apenas o estacionamento do lado esquerdo é permitido. Para máquinas tandem direitas, apenas o estacionamento do lado direito é permitido.

A opção de posição de estacionamento será exibida apenas para séries TruBend 8000 e máquinas VarioPress.

Para máquinas tandem B36, a posição de estacionamento small não é aplicável.

Criar bending aids personalizados

Para criar bending aids personalizados no TecZone Bend, primeiro você precisa criar um arquivo ini que define os parâmetros do bending aid e arquivos mesh para os componentes do bending aid (Carriage, Table e Plate).

Sample baid.ini file
[BAData]
; The user defined name of the BendAid. Only used for custom bend aids
Name=BATest
; The range of V-widths the bend aid can support
DieV=16,250
; The range of die-heights the bend aid can support
DieHeight=-30,250
; Maximum angle, in degrees
MaxAngle=55
; Lift capacity, in kg
Payload=300
; Maximum lift speed, in degrees/second
Speed=45
; When the bending aid is at X=0, this is the bound of the support table in top view
Table=-159,-1032,159,-245.6
; The X-Span of the bending aid (when positioned at 0)
XExtent=-213,213
; Tells whether the user can change the forward speed
CanChangeForwardSpeed=False
; Tells whether safety stop angle is possible in the machine
CanStopAtReturn=True
; Can the bending aid movement be varied using different methods?
AdvancedControl=True
; What type of existing bending aid can this be mapped to? Used only by custom bend aids
EType=TB8000_2000N_B36

[Carriage]
Model=Carriage.mesh
Stroke=-1000,1000
Color=White
Shift=0,0,100

[Table]
Model=Table.Mesh
Stroke=50,200
Color=White

[Plate]
Model=Plate.mesh
Stroke=0,56
Color=#4A4A4A

Coloque o arquivo ini e os arquivos mesh correspondentes em uma pasta que pode então ser importada para o TecZone Bend.

Importar bending aids personalizados

Para importar bending aids personalizados:

  1. Navegue até File / Import / Bending Aids…​ e selecione o arquivo ini já criado (certifique-se de que os arquivos mesh correspondentes estejam na mesma pasta).

  2. Os bending aids importados estarão então disponíveis no menu suspenso de seleção de bending aid nas configurações da máquina.

Esta opção é útil quando um bending aid precisa ser personalizado para tarefas de dobra específicas ou ao usar bending aids especializados não incluídos no banco de dados padrão do TecZone Bend.
