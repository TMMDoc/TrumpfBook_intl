Medição de ângulo

Sistemas de Medição do ângulo como ACB™™, ACBLaser ou LCB™ podem ser usados para melhorar a precisão do ângulo durante a dobra. Esses sistemas medem o ângulo real alcançado durante a dobra e a resiliência da chapa e usam esta informação para ajustar o ponto morto inferior da barra dinamicamente para atingir o ângulo-alvo de dobra.

Sistemas de disco e laser

Os sistemas de medição de ângulo são baseados em discos sensores montados nos punções, ou em sistemas ópticos usando um raio laser projetado na peça de trabalho. TecZone Bend suporta ambos, e quando você configura uma máquina, existem algumas opções que você pode ligar, dependendo das opções da máquina que você instalou.

A imagem acima mostra um painel de configurações para a máquina Trumpf 5085 B23. Esta máquina suporta dois tipos de medição de ângulo baseada em disco (ACB-Classic e ACB-Wireless) bem como a medição baseada em laser (ACB-Laser). Esta máquina em particular é configurada com todas as três opções ativadas.

Se uma prensa-dobradeira tiver ambos os sistemas de medição de ângulo, você pode ainda querer priorizar um sobre o outro, ou você pode querer desativar a medição do ângulo para uma peça específica. Você pode fazer isso usando o painel Configurações.[1]

A seção Medição do ângulo da página Configurações/BendCAM tem uma lista de opções que permite ligar ou desligar os diversos sistemas de medição de ângulo. Você pode priorizar um sobre o outro selecionando-o e movendo-o para cima ou para baixo usando os botões de seta para cima/baixo ao lado. Neste exemplo particular, tanto a medição de ângulo baseada em disco quanto em laser estão habilitadas e priorizamos o laser primeiro.[2]

A opção Usar Velocidade ACB decide se a estratégia ACB Speed (disponível em máquinas Touchpoint) será usada como padrão.

Métodos ACB

Existem diferentes métodos ACB que são utilizáveis no TecZone Bend. A maioria destes são comuns entre a medição de ângulo baseada em disco e baseada em laser, então discutimos os métodos diferentes primeiro.

Identificar

O método Identify ou Identification ou Identify Springback é o uso canônico do sistema ACB. A prensa-dobradeira usa o sistema de medição de ângulo para medir o ângulo real sendo formado quando a peça é dobrada. Depois que a peça estiver totalmente dobrada, a máquina vai para o estado de descompressão para permitir que o metal retorne elasticamente. A resiliência também é medida usando o ACB e é compensada por uma dobra excessiva precisa. Para evitar a possibilidade de ultrapassar o ângulo-alvo durante a medição, a máquina visa uma ligeira subdobra no início e, em seguida, faz uma nova dobra subsequente para atingir o alvo. Este é o método ACB mais preciso, mas também o mais demorado.

Aprendizagem Y

No método Aprendizagem Y, o sistema ACB não é usado. O alvo Y da barra (ponto morto inferior) é simplesmente copiado de uma dobra anterior (conhecida como a Dobra de referência), que já foi medido usando o método Identificar. Esta dobra anterior deve ser idêntica em termos de ângulo, raio e uso da ferramenta.

Resiliência aprendida

No método Resiliência aprendida ou Learn Resiliência, o sistema ACB usa o ângulo de resiliência de uma dobra anterior que foi medida usando o método Identify. O sistema ACB ainda é usado para regular a dobra, mas como uma segunda medição após a descompressão não é necessária, o ciclo é mais rápido do que um ciclo completo de identificação.

Introdução da resiliência

O método Introdução da resiliência, ou Enter Springback ou Corrected ou ACB Smart usa o sistema ACB para medir o ângulo alvo, mas usa um valor de retorno que é inserido pelo usuário. Em outras palavras, o ACB simplesmente dobrará a peça para o ângulo alvo + o valor de correção inserido pelo usuário. Nenhuma segunda medição é feita após a descompressão.

Notas sobre medição de ângulo

Nomes do sistema laser

O sistema de medição de ângulo baseado em laser é conhecido por nomes diferentes, como ACB-Laser ou LCB. Todos eles são programados de forma idêntica no TecZone Bend, embora o hardware real e a implementação na máquina sejam diferentes. A maior parte da interface do usuário do TecZone Bend usa apenas a palavra ACB ou ACB-Laser, então vamos usar esse termo nessa discussão também.

Comportamento de tolerância automática

A tolerância automática do TecZone Bend atribui métodos de ACB laser a cada dobra de uma maneira ideal. Inicialmente, ela começa usando o método Identify para a primeira dobra e, em seguida, tenta usar o método Aprendizagem Y para dobras sucessivas, sempre que possível. Se já houver uma dobra com o mesmo ângulo, raio, conjunto de ferramentas e com sentido de laminação semelhante, o TecZone Bend usará o método Aprendizagem Y, em vez de fazer outra Identificação.

Algumas dobras podem ser ignoradas automaticamente porque:

  • O ângulo de dobra não está dentro da faixa de medição do sensor (muito agudo / muito obtuso).

  • Esta não é uma dobra no ar (por exemplo, cunhagem, dobra em Z, vinco ou virar)

  • Esta é uma pré-dobra para outra dobra (não precisa de um ângulo preciso)

A imagem acima mostra uma peça onde algumas dobras não têm método ACB - estas são indicadas com os ícones de círculo oco na linha ACB, e a tooltip para cada célula indica uma __ razão pela qual nenhuma medição ACB é realizada.

Obtendo uma medição válida

Quando uma medição ACB é realizada, TecZone Bend decidirá se uma, duas ou três medições são necessárias com base no comprimento da linha de dobra. As posições reais de medição são calculadas automaticamente, mas também podem ser modificadas usando o Painel ACB (discutido abaixo). Normalmente, o TecZone Bend moverá ambos os sensores ACB _agrupado_s e usará sensores dianteiros e traseiros para uma Medição do laser ACB, mas isso também pode ser editado.

O TecZone Bend pode avaliar o comprimento de traço do laser para uma Medição do laser ACB. Este é o comprimento da linha de laser projetada que é visível para a câmera. Partes da linha podem ser obstruídas pela matriz ou pelos encostos. Algumas peças podem passar por furos na chapa, ou por conformações. A avaliação leva tudo isso em consideração e pode calcular o real comprimento da linha disponível para o sistema óptico. TecZone Bend compara este comprimento disponível contra um comprimento mínimo e ideal definido pela máquina, e gera um erro apropriado ou aviso (como mostrado na seção Símbolos TecZone Bend acima).

O TecZone Bend tem alguma inteligência para mover a posição de medição para a esquerda ou direita para evitar furos, formações ou outros fatores que possam limitar o comprimento do traço. Além disso, o TecZone Bend também adicionará automaticamente uma retração aos encostos se eles entrarem no caminho de uma medição a laser. Além disso, a avaliação do comprimento do traço do laser é feita continuamente, então, se você mover as posições do sensor ACB ou ajustar os encostos, poderá ver os resultados imediatamente no navegador.

1. Tal como acontece com todas as configurações, use Ajustes da aplicação para definir isso globalmente (para todas as máquinas), use as Configurações da Máquina para definir os padrões para uma máquina específica e use Configurações das peças para alterar os padrões para esta peça em particular.
2. Ativar um sistema de medição de ângulo aqui tem efeito apenas se a máquina tiver esse tipo específico de sistema de medição de ângulo instalado como opção. Em outras palavras, você pode realmente usar isso apenas para suprimir um sistema particular de ser usado.
