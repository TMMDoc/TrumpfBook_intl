Recalcular solução

Quando você pressiona ou usa o painel de fluxo de trabalho para calcular uma solução, TecZone Bend B calcula uma solução de dobra usando um conjunto de configurações e parâmetros padrão. Você pode fazer um novo cálculo desta solução usando diferentes configurações clicando no menu Recalcular na barra de comandos. Às vezes, isso também é útil para concluir o cálculo depois de ter editado interativamente algumas ferramentas, ou a sequência.
Por exemplo:

  • Você pode alterar a sequência e pedir que TecZone Bend calcule novas estações de ferramentas enquanto mantém a sequência inalterada.

  • Você pode mudar as estações de ferramentas e pedir ao TecZone Bend para calcular uma nova sequência, ou para alinhar a peça às estações recém-criadas.

  • Você pode tentar a operação de ferramentas com um conjunto diferente de ferramentas, ou tentar usando uma configuração já criada.

A caixa de diálogo Recalcular

A caixa de diálogo Recalcular se parece com a imagem ao lado.

Substituições

Esta seção contém algumas opções que você pode usar para substituir o comportamento padrão do TecZone Bend.

  • Ative o botão Não alterar sequência para forçar o TecZone Bend a calcular uma nova solução sem alterar a sequência. Diferentes estações de ferramentas, posições de encosto ou as direções de inserção da peça serão tentadas pelo TecZone Bend, mantendo a sequência inalterada.

  • Se o interruptor Nenhuma medição do ângulo. estiver ligado, o TecZone Bend calcula uma solução que não realiza medições de ângulo e não usará nenhum sensor de medição (ferramentas ACB)

  • As ferramentas Ignorar dos arquivos GEO/DXF dizem ao TecZone Bend para ignorar as dicas de ferramentas que são frequentemente encontrados em arquivos GEO e, às vezes, em arquivos DXF.

Configurações

Esta seção define a estratégia para configurações de ferramentas; por padrão, o TecZone Bend calculará a Configuração do zero, mas você pode usar essas configurações para alterar esse comportamento.

  • Calcular nova configuração é o comportamento padrão. O TecZone Bend calculará uma configuração usando todas as ferramentas permitidas (consulte a seção Ferramentas abaixo).

  • Manter a configuração existente diz ao TecZone Bend que nenhuma alteração de configuração deve ser feita; ele tentará processar todas as dobras usando a configuração atual da peça. Se algumas dobras tiverem colisões, ou ter comprimento insuficiente da ferramenta para processamento, esses erros permanecem não corrigidos e podem ser visualizados após o processo de recálculo.

  • Se você selecionar UsarConfiguração do arquivo, será solicitado que você escolha um arquivo de configuração de dobra (bsetup). A Configuração carregada a partir deste arquivo é então usada para o cálculo; essa configuração não é modificada, nem são criadas quaisquer estações adicionais. As configurações que você carregou recentemente do arquivo são exibidas no seletor que aparece abaixo desta seção, e você pode clicar no botão + perto desse seletor para escolher um arquivo de configuração diferente.

  • A opção Usar Configuração de outra Peça está disponível apenas se houver algumas outras peças atualmente abertas, e elas foram ferramentadas para a mesma prensa dobradeira. Nesse caso, você pode pedir ao TecZone Bend para usar a mesma configuração de uma dessas outras peças; um seletor aparece, para deixar você escolher a peça que você quer.

Ferramentas

Se você escolher a opção Calcular nova configuração na seção acima, poderá usar as Ferramentas para escolher a lista de ferramentas de dobra que podem ser usadas para calcular a nova configuração. Se você escolher uma das outras opções para a Configuração, esta seção não está disponível.

  • Ferramentas preferidas para a Máquina é a opção padrão. Para cada prensa dobradeira, você pode escolher um conjunto de tools[1] como as ferramentas preferidas, e o TecZone Bend usará essas ferramentas ao equipar a peça.[2] (Se você ainda não configurou as ferramentas preferidas para uma máquina, o TecZone Bend usa todas as ferramentas no inventário como o conjunto de ferramentas preferido para a máquina).

  • Se você escolher Ferramentas no Inventário, o TecZone Bend usará todas as ferramentas disponíveis no inventário da instalação. Se você ainda não configurou o inventário de ferramentas, o TecZone Bend usa todas as ferramentas do Catálogo como seu inventário.

  • Todas as ferramentas do Catálogo é a mais ampla seleção de ferramentas; o TecZone Bend usará todas as ferramentas disponíveis no catálogo de ferramentas do fabricante para tentar uma solução. Enquanto isso pode significar que a solução pode não ser realmente viável para produção (usando o inventário que você tem disponível), ela é útil para uma exploração especulativa de quais ferramentas seriam necessárias para produzir uma peça particularmente complexa. (Além disso, esse modo também diz ao TecZone Bend para assumir um inventário infinito de peças, ou seja, o ferramental ideal é produzido supondo que você tenha um número infinito de peças de cada comprimento).

  • Você pode criar conjuntos de ferramentas personalizados e armazená-los em arquivos de lista de ferramentas (btools). Esses arquivos podem ser usados para criar a peça, quando você seleciona a opção Usar conjunto de ferramentas do Arquivo.Quando você escolhe essa opção, é solicitado um arquivo de lista de ferramentas e os arquivos de lista de ferramentas que você tem usado recentemente são exibidos para facilitar a seleção. Você pode clicar no sinal de + perto desta lista para escolher uma nova lista de ferramentas.

  • Para o controle mais direto sobre os punções e matrizes, você pode selecionar a opção Determinar ferramenta superior e inferior e, em seguida, escolha o punção e a matriz exatos que deseja usar.

Prioridade de sequência

Ao sequenciar as dobras em uma peça, existem alguns requisitos potencialmente conflitantes. Por exemplo, se TecZone Bend deve tentar minimizar o número de rotações da peça, ou tentar minimizar as dobras onde o centro de gravidade da peça fica dentro da máquina?
É bastante provável que cada operador tenha um conjunto diferente de regras de prioridade, e também é possível que peças diferentes possam exigir configurações de prioridade diferentes. Por isso, o TecZone Bend permite que você faça um recálculo com uma ordem diferente de prioridades, usando a seção Prioridade de sequência desta caixa de diálogo.

Use os botões de seta e à direita para embaralhar as regras; os itens próximos ao topo têm prioridade mais alta. Você pode desativar alguns desses itens prioritários completamente; isso diz para o TecZone Bend não considerar isso como um fator na hora de decidir a sequência. (Às vezes, desligar alguns desses pode produzir uma sequência que usa poucas estações de ferramentas).

  • Evite colisões durante a sobredobra: para obter precisão do ângulo, a maioria das dobras precisará ser levemente sobredobrada para compensar o retorno elástico. Esta regra de sequência diz ao TecZone Bend para fazer dobras em uma ordem tal que a dobra excessiva seja possível. Por exemplo, na peça abaixo, devemos fazer a dobra 1 primeiro e a dobra 2 em seguida; a outra ordem bloquearia a dobra excessiva.

  • Evitar ferramentas com baixa prioridade: Diz ao TecZone Bend para evitar o uso de ferramentas de baixa prioridade, ou ferramentas que não estão na lista preferida, na medida do possível.

  • Evitar batentes em ângulos abertos: As posições de encosto devem, idealmente, estar no mesmo plano que o eixo de dobra, e não deve haver dobras já processadas entre a linha de dobra e o ponto de encosto. Aqui está um exemplo de um encosto ruim que esta regra tenta evitar:

  • Comprimento de aba mais longo preponderante antes da linha de flexão: especialmente com peças pesadas, é útil evitar situações em que o centro de gravidade da peça está atrás da linha de ferramentas. Esta regra tenta minimizar tais situações.[3]

  • Minimizar a oscilação das peças: Inversões de peça são quando a peça tem que ser virada (sucessivas dobras estão em diferentes direções para cima/para baixo).

  • Minimizar os giros das peças: Uma rotação de peça é quando uma peça é virada para a esquerda ou para a direita no plano horizontal; isso é considerado mais fácil para o operador do que uma virada de peça; a ordenação padrão dessas duas últimas regras tenta favorecer as rotações de peça sobre inversões de peças, sempre que possível.

1. Para fazer isso, use o botão de inventário na barra de comandos, e escolha Ferramentas preferidas no menu que aparece.
2. TecZone Bend tentará usar apenas as ferramentas preferidas para a peça, mas se houver colisões ou outros problemas que não podem ser resolvidos, passará a utilizar outras ferramentas do inventário disponível.
3. TecZone Bend usa alguma discrição sobre o peso da peça e até que ponto o centro de gravidade está atrás da linha de ferramentas, para considerar isso como um problema. Tecnicamente, considera que uma peça inclina para trás se houver um momento basculante de 0,15 Nm (Newton metros) para trás.
