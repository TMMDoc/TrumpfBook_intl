Criar dados de dobra

Você pode criar Dados de Tecnologia de Dobra para uma peça de chapa usando o módulo Bend. Isso também é chamado de ferramentar a peça para dobrar. Vamos supor que você já instalou uma máquina, e que você configurou o inventário de ferramentas, e que você importou uma peça CAD (2D ou 3D):

Mudando para Dobrar CAM

Agora, a peça pode ser trabalhada para dobrar, apenas pressionando a tecla B. A peça é trabalhada com a máquina de dobra padrão, que é a prensa-dobradeira que você usou mais recentemente. Uma sequência de ferramentas, configuração de dobra, posições de encosto são todas calculadas, e você deverá ver um resultado como este:

Se não houver erros, o programa CN (e possivelmente a ficha de configuração) para a peça também são gerados e salvos na pasta de saída configurada para esta máquina. Aqui está uma seção de uma configuração de peça típica. sheet[1] (também conhecida como relatório de dobra).

Consulte também o painel Fluxo de trabalho, que pode ser acessado pressionando a tecla W. Isso fornece mais controle sobre a criação de dados de tecnologia de dobra e opções para rotear a peça através de laser CAM ou nesting.

Outras operações

Aqui está um rápido resumo de algumas das operações que você pode fazer uma vez que uma peça tenha sido ferramentada para dobrar.

  • Se você clicar na Spacebar, uma simulação de dobra será iniciada. Você também pode usar os controles de simulação no Bend Navigator para iniciar, parar ou rebobinar a simulação:

  • Se houver algum aviso ou erro, eles aparecem no Bend Navigator, e você pode visualizá-los e resolvê-los clicando nas células correspondentes:

  • Você pode editar as ferramentas de dobra (punções e matrizes), a configuração de encostos ou a configuração de medição de ângulo apenas clicando diretamente nesse objeto na visualização da simulação:

  • Você pode alterar a máquina clicando no nome da máquina na guia abaixo da peça, e selecionando uma máquina diferente:

  • Clique no ícone Ajustes ícone na barra de ferramentas à esquerda para editar configurações adicionais para cada dobra (ou para editar as configurações padrão usadas para esta máquina de dobra, ou para todo o aplicativo TecZone Bend).

  • Clique no ícone Exibição na barra de ferramentas esquerda para editar a visualização da máquina - você pode girar a exibição de componentes adicionais, como a barra de pressão, a viga de dobra, liga/desliga do sistema de suporte do encosto traseiro. Você também pode tornar vários componentes translúcidos.

Quando uma solução de dobra é criada sem erros, o programa CN da peça e uma ficha de configuração para a dobra são gerados automaticamente. Você pode usar a página de configurações Bend Outputs para configurar se o programa CN da peça e a ficha de configuração são gerados automaticamente, e também configurar o destino para essas saídas, e o formato para os relatórios. Se o código tiver sido gerado, o botão nc-code na barra de ferramentas esquerda ficará acinzentado:

Ao fazer qualquer alteração na peça, este botão fica ativado novamente, e você pode gerar novamente o programa CN clicando neste ícone novamente (ou pressionando a tecla C).

1. Você pode usar o Editor de relatórios para personalizar esta ficha de configuração.
