사용자 정의 에어갭 테이블

스위블 벤딩 머신용 에어갭 테이블은 특정 시트 두께를 가공 중에 사용되는 표준 에어갭에 매핑합니다. 이는 다양한 스위블 벤딩 머신에 내장되어 있지만, JSON 파일 형태로 사용자 정의 에어갭 테이블을 제공하면 오버라이드할 수 있습니다.이 JSON 파일은 air-gap-tables.json이라고 하며, `C:\ProgramData\Trumpf{APPNAME}\ Data\ System`에 위치합니다.

다음은 일반적인 air-gap-tables.json 파일입니다:

 {
  "version":1  "airGapTables": [
    {
      "machine": "112002,112003", "material": "DC01"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
     {
      "machine": "112003", "material": "*"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
    {
      "machine": "*", "material": "Alluminio"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    }
  ]
}

  • 파일에는 에어갭 값의 컬렉션이 포함되어 있습니다. 각 테이블에는 선택기로 역할을 하는 재료 키와 기계가 있습니다.

  • 이 키는 '*' (별표) 로 설정되어 와일드카드(모든 항목과 일치)를 나타내거나 기계 또는 재료 값의 쉼표로 구분된 목록입니다.

  • 위의 예에서는 세 개의 테이블이 있습니다. 첫 번째는 기계 112001 또는 112003, 재료 DC01에서만 사용됩니다. 두 번째는 모든 재료, 기계 112003 에 사용되고, 세 번째는 모든 기계, 재료 Alluminio에 사용됩니다.여러 개의 테이블이 파트의 기계/재료 조합과 일치하는 경우 가장 먼저 일치하는 테이블이 사용됩니다. 따라서 기계 112003의 경우 재료 DC01의 첫 번째 테이블이 사용됩니다(두 번째도 일치함에도 불구하고).

  • 각 테이블은 두께 키의 간단한 매열과 해당 에어갭 값으로 구성됩니다. 예를 들어, 위의 첫 번째 항목은 다음과 같아야 합니다: 두께가 0.75 mm 이하인 경우의 2.5mm의 간격을 사용합니다. 두께 키는 오름차순으로 정렬되어야 합니다. 이 파일 내의 어떤 테이블도 현재 파트의 재료와 두께 조합과 일치하지 않으면, 기계의 기본 에어갭 테이블이 사용됩니다.

에어갭 테이블 .json 파일을 변경한 경우 해당 변경 내용을 적용하기 전에, TecZone Fold를 다시 시작해야 합니다.
JSON 파일의 형식이 잘못된 경우, 스위블 벤딩 머신 파트를 처리하려고 할 때 오류가 표시되지만, TecZone Fold는 해당 기계의 기본 에어갭 테이블을 다시 사용하고 계속 진행합니다.
