RG-스테이션 패널

RG-스테이션 패널

재그립 스테이션은 TecZone에서 RG-Stations 로 표시됩니다. RG-Station 패널은 벤드 셀 머신의 재그립 스테이션을 구성하는 데 사용됩니다.

RG-Stations 패널에 액세스하려면:

  • Pickup 패널, Bend 패널, Regrip 패널 또는 Deposit 패널에서 RG-Stations 링크를 클릭합니다.

  • 또는 재그립 스테이션을 직접 클릭합니다.

벤드 작업 및 스테이션의 위치에 따라 RG-Stations 패널은 패널 내에 알파벳 표시(P, R1, #1, D 등)를 표시합니다.

Pickup 패널의 Flip Part 체크박스는 현재 위치에 대해 부품을 거꾸로 배치하는 데 사용됩니다. 이는 아래쪽을 향한 플랜지와 재그립 스테이션 암 사이의 충돌을 방지하는 데 유용합니다. TecZone은 부품을 뒤집을 때 즉시 새로운 로봇 경로를 재계산합니다.

재그립 스테이션을 사용하지 않을 때는 부품 위치를 제어할 수 없기 때문에(펀치와 다이 사이의 부품 클램핑 위치에 따라 다름) 패널이 단순해 보입니다.

  • *Stations*는 스테이션 중 하나를 선택하거나 두 스테이션을 모두 비활성화하거나 활성화하는 데 사용됩니다.

  • Positions 설정을 사용하여 머신 베드 레일의 스테이션 위치를 조정합니다. 아래 이미지에서 왼쪽에 표시된 것처럼 처음에는 스테이션이 부품 아래에 배치되지 않습니다.

    Positions 옵션의 값을 변경하면 RG-Station의 석션 컵이 부품 아래에 배치되도록 두 스테이션이 위치합니다.

Suction 섹션의 설정은 석션 암을 구성하는 데 사용됩니다.

  • Type 선택기를 사용하여 석션 컵 유형을 변경합니다.

  • Arm 선택기를 사용하여 석션 암 중 하나 또는 전체를 선택합니다. 선택하면 각 암은 레이블(A, B, C 등)로 식별됩니다.

  • Angle 설정을 사용하여 암을 회전합니다(5° 단위).

  • Grid 설정을 사용하여 암을 확장/수축합니다(10mm 단위).

슬라이더를 개별적으로 드래그하여 각도와 그리드를 대화식으로 변경할 수도 있습니다. 석션 암 위로 마우스를 가져가면 각도 및 그리드 슬라이더가 표시됩니다.

All 암을 선택하면 각도와 그리드를 대화식으로 변경할 수 없습니다.

  • 석션 암의 작동 *상태*를 off, on 또는 _removed_로 선택합니다.

  • 석션 설정을 완료한 후 Copy Right 옵션을 사용하여 RG-Station의 다른 쪽에 동일한 설정을 사용할 수 있습니다.

  • 이 부품에 여러 개의 재그립 작업이 있는 경우 Prev 및 Next 링크가 표시되며 이들 간에 탐색하는 데 사용됩니다.

  • Regrip R1 링크는 재그립 스테이션에서 부품의 휴지 위치를 편집하는 데 사용됩니다.

재그립 스테이션을 재구성하면 TecZone은 부품, 로봇, 그리퍼와의 충돌을 실시간으로 확인하고 상태 표시기를 즉시 업데이트합니다.
