웨이포인트

개요

BendMaster의 각 이동은 차례로 통과하는 waypoints 로 구성됩니다. 이동의 모든 웨이포인트는 TecZone에 의해 결정되지만 대부분은 조정할 수 있습니다. 벤딩 프로세스 중 추적과 직접 관련된 포인트는 변경할 수 없습니다. 웨이포인트를 제어하는 데 사용되는 다양한 설정은 아래에 나열되어 있습니다:

  1. 웨이포인트의 위치 변경

  2. 새 웨이포인트 삽입 또는 기존 웨이포인트 제거

  3. 속도 및 보간과 같은 모션 속성 조정

  4. 원하는 위치에 새 웨이포인트 추가

Waypoints 패널에 액세스하려면:

  • 로봇 레일을 직접 클릭합니다

  • Pickup 패널, Bend 패널, Regrip 패널 또는 Deposit 패널에서 Waypoints 링크를 클릭합니다.

  • RG-Stations 패널에서 waypoints 링크를 클릭합니다

Waypoints 패널

일반적인 웨이포인트 패널이 옆에 표시됩니다:

  • Phase 슬라이더는 웨이포인트의 단계별 이동을 보여줍니다. 각 페이즈는 픽업, 벤드, 리그립 및 디포짓과 같은 특정 로봇 작업을 반영합니다.

  • Display 옵션을 사용하여 다음을 봅니다:

    • grip point

    • wrist point

    • robot axes.

  • 손목 또는 그리퍼의 위치와 각도가 표시되며 편집할 수 있습니다. 로봇 축이 선택된 경우 로봇 축 값이 표시되며 편집할 수 있습니다.

  • 노란색 트레이스는 이전 단계에서 이 단계로, 그리고 이 단계에서 다음 단계로의 경로를 보여줍니다(아래 이미지 참조). 3개의 작은 삼각형은 로봇(손목 또는 그리퍼)의 로컬 X 및 Y 좌표를 나타냅니다.

  • Simulate 슬라이더는 시뮬레이션을 이전 페이즈에서 다음 페이즈로 이동하는 데 사용됩니다. 이것이 중간에 있으면 편집 중인 실제 페이즈입니다.

  • Interpolate 설정은 선형 보간과 축 지향 보간 사이를 전환하는 데 사용됩니다.

  • Speed 및 Rounding 설정은 이 단계의 속도 및 정확도 설정을 편집하는 데 사용됩니다.

  • Regrip stations 필드는 축을 따라 리그립 스테이션의 위치를 변경하는 데 사용됩니다.

  • Insert 옵션은 새 값을 입력하여 새 웨이포인트를 생성하는 데 사용됩니다. 새로 추가된 웨이포인트는 Remove 옵션으로 제거할 수 있습니다.

  • Prev 및 Next 버튼을 사용하여 다음 또는 이전 단계로 이동합니다(또는 왼쪽 및 오른쪽 화살표 키를 사용할 수도 있습니다). Next 버튼을 클릭하면 페이즈의 각 단계가 표시됩니다. 페이즈가 끝나면 다음에 해당하는 웨이포인트 패널이 표시됩니다.

웨이포인트의 속성

프로세스 상태에 따라 각 웨이포인트에는 보간 종류, 속도 및 이동 정밀도와 관련된 일련의 속성이 부여됩니다. 필요한 경우 이러한 속성을 변경할 수 있습니다.

이러한 설정은 웨이포인트 패널의 고급 섹션에서 제어됩니다.

Table 1. 보간
Interpolate 타입 설명

Linear

경로를 따라 TCP의 선형 이동

Axis

시작 위치에서 끝 위치로의 이동이 경로에 관계없이 가능한 한 빠릅니다.
TCP 선형 이동 없음.
Table 2. 속도
Speed 타입 설명

Max.

최대 속도

Fast

최대 속도의 75%

Medium

최대 속도의 50%

Reduced

최대 속도의 30%

Slow

최대 속도의 20%
Table 3. 라운딩
Rounding 타입 설명

Max.

더 짧은 경로 길이의 50%에서 라운딩 시작

Large

더 짧은 경로 길이의 35%에서 라운딩 시작

Medium

더 짧은 경로 길이의 20%에서 라운딩 시작

Reduced

더 짧은 경로 길이의 10%에서 라운딩 시작

Fine

더 짧은 경로 길이의 5%에서 라운딩 시작

Exact

오버레이 없이 보정 포인트에 정확하게 접근

  1. 웨이포인트 포인트와 엔드포인트 사이의 경로 길이

  2. 실행된 길이

  3. 시작 포인트와 웨이포인트 사이의 경로 길이

  4. Waypoint 3 (종료 포인트)

  5. Waypoint 2 (라운딩 포인트)

  6. Waypoint 1 (시작 포인트)

waypoints 표시

벤드 파트에 툴이 설정되면 각 이동 페이즈에 대해 모든 웨이포인트가 자동으로 계산됩니다. 이러한 포인트는 해당 설명과 함께 페이즈별로 시뮬레이션할 수 있습니다.

아래 이미지는 첫 번째 벤딩 작업의 웨이포인트 페이즈를 보여줍니다(측면도):

waypoints 수정

필요한 경우 웨이포인트의 위치를 변경할 수 있습니다. 예를 들어 축을 넘어가는 것을 방지하기 위해서입니다.

충돌 회피와 관련된 다른 측면 중에서 원하지 않는 웨이포인트를 제거하거나 새 웨이포인트를 추가할 수 있습니다.

  • 기존 웨이포인트 위치가 변경되면 TecZone에 경고 메시지가 표시됩니다. *OK*를 클릭하여 웨이포인트 변경을 진행합니다.

  • Insert 버튼을 클릭하면 새 웨이포인트가 추가됩니다.

  • 새로 추가된 웨이포인트는 아래와 같이 페이즈 이름에 +1, +2 등으로 표시됩니다:

waypoints 시뮬레이션

기본적으로 simulate 슬라이더는 중간 위치에 있습니다. 이 슬라이더를 사용하여 한 웨이포인트에서 다른 웨이포인트로의 로봇 이동이 원활하게 시뮬레이션됩니다.
