가져오기/내보내기

이 섹션에서는 가져오기 세팅 의 컨피규레이션을 다룹니다.

홈페이지에서 세팅 아이콘을 클릭합니다.

가져오기 세팅

이 섹션에서는 가져오기 세팅 의 컨피규레이션을 다룹니다.세팅 아이콘을 클릭합니다. 가져오기/내보내기 를 클릭하고 가져오기 세팅 으로 이동합니다.

DXF 파일을 위한 단위 - 여기서 밀리미터 또는 인치로 설정합니다.

병합되지 않은 컨투어 엘리먼트 사이의 + - 이 값(0 > 1mm)을 설정합니다. 목표값보다 더 가까운 라인/호가 있는 파트를 가져올 경우, 소프트웨어는 가져올 때 자동으로 함께 스티치됩니다.

최소 시트 두께 - 대형 판재 파트를 인식하려면 판재 인식 임계값을 높여야 합니다. (10>40mm). 사용되는 단위에 따라 기본적으로 25mm 또는 1인치로 자동 설정됩니다.

포인트 단위 - 선택한 옵션에 따라 포인트를 가져오는 방법이 결정됩니다.

모두 가져오기 - 모든 포인트를 가져와 표시합니다.

폴리라인의 포인트 건너뛰기 - 폴리라인에서 인식된 모든 포인트를 건너뜁니다.

모두 건너뛰기 - 가져오기에서 모든 포인트를 건너뛰고 아무 것도 표시되지 않습니다.

DXF-/DWG 파일에서 레이어 무시 - DXF 및 DWG 도면은 일반적으로 다른 레이어에서 생성됩니다. 이 설정을 사용하는 경우 소프트웨어는 이러한 레이어를 무시하고 모든 객체를 기본 레이어로 이동시킵니다.

블록을 2D 도면으로 분해 - 가져오기 중 그룹화를 분할합니다

백색 객체를 검정색으로 전환 - 이 설정을 켜면 가져오기 중에 흰색 객체가 검은색 객체로 변환됩니다.

DXF 가져오기 중에 더 어두운 컬러- 가져오기 중 색상이 있는 객체를 어둡게 하려면 이 설정을 켭니다.

중복된 세그먼트 제거 - 가져올 때 파트에서 발견된 중복 형상을 제거하려면 이 설정을 켭니다.

DXF 세팅

이 섹션에서는 DXF 세팅 의 컨피규레이션을 다룹니다. 이 세팅 아이콘을 클릭합니다. 가져오기/내보내기 를 클릭하고 DXF 세팅 으로 이동합니다.

dxf 파일의 각도는 개방각입니다. - DXF의 각도를 개방 각도로 처리하려면 이 설정을 활성화합니다.

내보내기 세팅

이 섹션에서는 내보내기 세팅 의 컨피규레이션을 다룹니다. 세팅 아이콘을 클릭합니다. 가져오기/내보내기 를 클릭하고 내보내기 세팅 으로 이동합니다.

DXF 출력에 POLYLINE 객체 없음 - 일반적으로 닫힌 컨투어는 DXF를 내보낼 때 폴리라인으로 출력됩니다. 일부 CAD 시스템에서는 이 출력물을 처리할 수 없습니다. 이 설정을 사용하는 경우 소프트웨어는 라인과 호가 포함된 DXF를 출력합니다. 이 파일은 어디에서나 읽을 수 있지만 생성된 파일의 크기는 더 커지고 라인과 호 사이의 연결이 끊어집니다.

DXF 파일의 내보내기 시 벤딩 정보를 출력합니다. - 벤딩 파라미터와 함께 내보낸 DXF를 출력하러면 이 설정을 켭니다.

Starmatik 형식의 벤드 정보 - Starmatik 형식으로 벤딩 파라미터를 출력하려면 이 스위치를 켭니다. 텍스트 객체가 정확히 절곡선이 되는 각 라인의 중앙에 배치되는 위치입니다.

출력에서 검정색을 회색으로 전환 - 2D 데이터를 내보낼 때, DXF 파일의 객체는 기본적으로 검정색으로 출력됩니다. CAD 프로그램에서 객체를 더 잘 인식할 수 있도록 이 설정에서 객체가 회색으로 출력됩니다.

PDG 파일의 내보내기로 MetaCAM 시작. - MetaCAM에서 PDG 파일이 자동으로 열리도록 컨피규레이션하려면 이 설정을 켭니다.

전개도의 출력 형식 - 평면 패턴을 내보낼 때 GEO, DXF 또는 PDG 파일 형식으로 설정할 수 있습니다.

스플라인 변환

이 섹션에서는 스플라인 변환 설정의 컨피규레이션을 다룹니다. 세팅 아이콘을 클릭합니다. 이 가져오기/내보내기 를 클릭하고 스플라인 변환 설정으로 이동합니다.

가져오기 중 스플라인 변환 - 스플라인 변환을 끄거나 스플라인을 라인이나 호로 변환할지 여부를 여기에서 설정합니다. 두 경우 모두, 각 스플라인은 라인 세그먼트나 호 세그먼트가 있는 단일 폴리라인 객체로 변환됩니다.

보간점 계산 - 생성된 라인 또는 호의 수는 두 가지 메커니즘, 피치 또는 편차를 사용하여 계산됩니다.

선 세그먼트 또는 원호 세그먼트당 길이 - 노드 수 계산에서 피치가 선택된 경우 여기에서 각 호 또는 라인 세그먼트의 길이를 설정하여 이 단계 길이로 스플라인을 분해합니다.

근사화 중 최대 편차 - 노드 수 계산에서 편차를 선택한 경우 여기에서 원래의 부드러운 스플라인과 라인 또는 호 근사화 사이에 허용되는 최대 편차를 설정합니다. 폴리라인은 가능한 한 적은 수의 세그먼트로 구성되지만 최대 오류는 이 한계 범위 내에서 유지됩니다.

조립품 분해

이 섹션에서는 조립품 분해 설정의 컨피규레이션을 다룹니다. 세팅 아이콘을 클릭합니다. 이 가져오기/내보내기 를 클릭하고 조립품 분해 설정으로 이동합니다.

절단 파트 무시 - 모듈 분해 시 이 스위치를 켜면 절곡선이 있는 판재 파트만 표시됩니다. 끄면 모듈의 모든 파트가 표시됩니다.

컴포넌트의 표시 (인서트 너트, 볼트, + - 이 설정으로 기타 부품이 포함된 모듈을 분해할 경우 표시할 옵션을 선택합니다.

Off - 분해 후 판재 파트만 표시되고, 다른 너트/볼트는 표시되지 않습니다.

알 수 없음 - 아직 식별되지 않은 부품만 소프트웨어에 표시됩니다.

모두 - 모든 부품이 표시됩니다.

레이어 할당

이 섹션에서는레이어 할당의 컨피규레이션을 다룹니다. 이 세팅 아이콘을 클릭합니다. 가져오기/내보내기 를 클릭하고 레이어 할당 설정으로 이동합니다.

이 섹션에서는 소프트웨어로 가져오는 파트에 사용되는 레이어를 해당 기능(사용)에 자동으로 매핑할 수 있습니다.

예:

가져온 파트에 마크(Mark) 레이어가 있는 경우 소프트웨어에서 자동으로 마크 레이어 를 사용하도록 설정할 수 있습니다.

레이어 이름 - 파트를 가져올 경우, '사용' 패널에 설정된 기능을 사용하는 레이어의 이름입니다.

사용 - 이는 레이어의 기능입니다. 사용 가능한 다양한 옵션은 다음과 같습니다:

표준 - CAM에 사용할 표준 레이어입니다.

보조도구 - CAM에 사용되지 않는 보조 레이어입니다.

마킹 - 이 레이어의 모든 엔터티는 절단 없이 마킹됩니다.

접근 마커 - 레이저 접근 위치를 나타내는 포인트 엔티티입니다.

시퀀스 마커 - 컨투어의 시퀀스 순서를 나타내는 텍스트 마커입니다.

성형부 중심 - 성형을 위한 센터 마크(포인트 또는작은 L)입니다.

풋프린트 성형 - 성형의 윤곽선(풋프린트)입니다.

증발 - 이 레이어는 필름 연소를 구별하는 데 사용됩니다.

포인트 마킹 - 이 레이어는 QR 코드에 사용됩니다.

정보 - 이 레이어는 정보 전용 레이어입니다.
