스위블 벤딩

Geometry Adaptation Strategy

2D 또는 3D 파일이 소프트웨어에서 편집되는 경우, 벤딩 반경과 단축값이 점검됩니다. Easy Programming 설정에서 공작물을 변경하기 위해 선택한 전략에 따라 다양한 처리 전략이 실행됩니다. 결정이 대화식으로 설정된 경우, Adapt Geometry 대화상자가 표시됩니다.

2D 평면으로 시작할 때:

Ask each time - 2D 파트에서 벤딩으로 이동할 때 이 설정을 사용하면 Adapt Geometry 대화 상자가 자동으로 표시됩니다. 이는 평면 크기를 유지하거나, 3D 치수를 유지하거나, 아무 것도 변경하지 않는 옵션을 제공하여 이 기능을 매우 상호작용적이고 사용자 중심적으로 만들어줍니다.

Keep 3D dimensions - 이 옵션은 전개도를 변경하므로 수동으로 저장해야 합니다. 이 전략은 블랭크가 아직 절단되지 않은 경우 사용됩니다.

Keep 2D dimensions - 이 옵션은 벤딩 반경과 단축값이 기존 전개도에 맞게 조정되므로, 벤딩선은 같은 위치에 유지됩니다. 이는 파트의 측면 길이를 변경합니다. 전개도는 변경되지 않습니다. 이 전략은 블랭크가 이미 크기에 맞게 절단된 경우에 사용됩니다.

No adjustments - 이 옵션을 사용하면 벤딩 반경과 단축값이 변경되지 않습니다. 해당 파트는 3D로 올바른 치수로 표시되지만, 표시된 벤딩 반경은 실제 벤딩 반경과 일치하지 않습니다. 각 공작물 벤딩의 치수는 변경할 수 있습니다.

3D 모델로 시작할 때:

Ask each time - 3D 파트에서 벤딩으로 이동할 때 이 설정을 사용하면 Adapt Geometry 대화상자가 자동으로 표시됩니다. 이는 평면 크기를 유지하거나, 3D 치수를 유지하거나, 아무 것도 변경하지 않는 옵션을 제공하여 이 기능을 매우 상호작용적이고 사용자 중심적으로 만들어줍니다.

Keep 3D dimensions - 이 옵션은 전개도를 변경하므로 수동으로 저장해야 합니다. 이 전략은 블랭크가 아직 절단되지 않은 경우 사용됩니다.

Keep 2D dimensions - 이 옵션은 벤딩 반경과 단축값이 기존 전개도에 맞게 조정되므로, 벤딩선은 같은 위치에 유지됩니다. 이는 파트의 측면 길이를 변경합니다. 이 전략은 블랭크가 이미 크기에 맞게 절단된 경우에 사용됩니다.

No adjustments - 이 옵션을 사용하면 벤딩 반경과 단축값이 변경되지 않습니다. 해당 파트는 3D로 올바른 치수로 표시되지만, 표시된 벤딩 반경은 실제 벤딩 반경과 일치하지 않습니다. 각 공작물 벤딩의 치수는 변경할 수 있습니다.

Machine options

KB40 version - 기계에 설치된 KB40 로더가 있는 경우 이 옵션을 활성화합니다. 이를 통해 KB40 팔레트를 생성하고 변경하는 기능이 활성화됩니다.

KB40 frame - KB40 frame 의 타입(STD 또는 XL)

Place part at center of the KB40 pallet -KB40 팔레트의 중앙에 파트를 배치하려면 이 옵션을 활성화합니다.

Additional pad - loading head - 추가 중앙 진공 패드의 경우 이 옵션을 활성화합니다.

Modular central vacuum - loading head - 모듈식 중앙 진공 패드의 경우 이 옵션을 활성화합니다(각 모듈에 대해 개별적으로 진공을 켜고 끌 수 있음)

KB44 part-flipper installed - 터닝 스테이션(KB44)이 설치된 경우 이 옵션을 활성화합니다.
