Bending Aids

Bending Aids는 두꺼운 판재를 벤딩하는 데 도움이 되는 프레스 브레이크 기계 구성요소입니다. 이는 벤딩 프로세스 중에 추가 지원 및 제어를 제공하여 정확하고 일관된 결과를 보장하도록 설계되었습니다.

프레스 브레이크는 벤딩 작업의 특정 요구 사항에 따라 여러 개의 bending aids 또는 단일 bending aid를 설치할 수 있습니다.

Bending aids는 TruBend 5000 및 8000 시리즈에만 적용됩니다.

Bending Aids 구성

Bending Aids는 Database / Machines / Settings 에서 구성할 수 있습니다.

파트를 가져오고 bending aids 가 있는 기계를 선택하면 TecZone Bend 가 파트의 요구 사항에 따라 bending aids를 자동으로 구성합니다.

Bending Aids 사용

실행 가능한 벤딩 솔루션이 생성되면 bending aids를 추가로 조정하여 벤딩 프로세스를 최적화할 수 있습니다.

TecZone Bend에서 bending aid를 직접 클릭하여 Bend Aid 패널을 엽니다. Bend aid 패널의 번호는 벤딩 순서를 나타냅니다.

  • Index - 현재 선택된 bending aid를 나타냅니다.

  • Position (Z,R) - Z 및 R 축을 따라 bending aid의 위치를 조정하는 데 사용됩니다.

  • Enabled - 이 체크박스를 선택하여 bending aid를 활성화합니다.

  • Auto-Place - 파트의 형상을 기반으로 bending aid를 자동으로 배치하는 데 사용됩니다.

Speed:

  • Down - 벤딩 작업 후 bending aid의 후퇴 속도를 조정하는 데 사용됩니다.

  • Set All Bends - 현재 속도 설정을 시퀀스의 모든 벤딩부에 적용하는 데 사용됩니다.

Advanced:

  • Method - bending aid의 작동 모드를 정의하는 데 사용됩니다. 옵션은 다음과 같습니다:

    • Default - bending aid 설정을 기본값으로 재설정합니다.

    • Return at UDP - 각 벤딩 후에 사용자 정의 위치로 돌아가도록 bending aid를 구성합니다.

    • Tilt product - 더 나은 정렬을 위해 벤딩 프로세스 중에 bending aid가 파트를 기울일 수 있도록 합니다.

    • Static angle - 작동 중 bending aid에 대해 고정 각도를 설정할 수 있습니다.

  • Stop angle - 작동 중 bending aid가 정지할 각도를 설정하는 데 사용됩니다.

Bending Aids 파킹 위치

Bending Aids는 사용하지 않을 때 지정된 위치에 파킹할 수 있습니다. 이는 작업 공간을 최적화하고 작동 중 안전을 보장하는 데 도움이 됩니다. 또한 다른 기계 구성요소와의 충돌을 방지합니다.

Bending Aids는 다음 위치에 파킹할 수 있습니다:

  1. Default - Bending aids를 테이블 길이까지 파킹할 수 있습니다.

  2. Small - Bending aids를 테이블 길이 너머 700mm까지 배치할 수 있습니다.

  3. Large - Bending aids를 테이블 길이 너머 1200mm까지 배치할 수 있습니다.

탠덤 기계의 경우 4개의 bending aids를 사용할 수 있습니다. 왼쪽 탠덤 기계의 경우 왼쪽 파킹만 허용됩니다. 오른쪽 탠덤 기계의 경우 오른쪽 파킹만 허용됩니다.

파킹 위치 옵션은 TruBend 8000 시리즈 및 VarioPress 기계에만 표시됩니다.

B36 탠덤 기계의 경우 small 파킹 위치는 적용되지 않습니다.

사용자 정의 bending aids 생성

TecZone Bend에서 사용자 정의 bending aids를 생성하려면 먼저 bending aid의 매개변수를 정의하는 ini 파일과 bending aid 구성요소(Carriage, Table 및 Plate)에 대한 mesh 파일을 생성해야 합니다.

Sample baid.ini file
[BAData]
; The user defined name of the BendAid. Only used for custom bend aids
Name=BATest
; The range of V-widths the bend aid can support
DieV=16,250
; The range of die-heights the bend aid can support
DieHeight=-30,250
; Maximum angle, in degrees
MaxAngle=55
; Lift capacity, in kg
Payload=300
; Maximum lift speed, in degrees/second
Speed=45
; When the bending aid is at X=0, this is the bound of the support table in top view
Table=-159,-1032,159,-245.6
; The X-Span of the bending aid (when positioned at 0)
XExtent=-213,213
; Tells whether the user can change the forward speed
CanChangeForwardSpeed=False
; Tells whether safety stop angle is possible in the machine
CanStopAtReturn=True
; Can the bending aid movement be varied using different methods?
AdvancedControl=True
; What type of existing bending aid can this be mapped to? Used only by custom bend aids
EType=TB8000_2000N_B36

[Carriage]
Model=Carriage.mesh
Stroke=-1000,1000
Color=White
Shift=0,0,100

[Table]
Model=Table.Mesh
Stroke=50,200
Color=White

[Plate]
Model=Plate.mesh
Stroke=0,56
Color=#4A4A4A

ini 파일과 해당 mesh 파일을 폴더에 배치하면 TecZone Bend로 가져올 수 있습니다.

사용자 정의 bending aids 가져오기

사용자 정의 bending aids를 가져오려면:

  1. File / Import / Bending Aids…​ 로 이동하여 이미 생성된 ini 파일을 선택합니다(해당 mesh 파일이 같은 폴더에 있는지 확인).

  2. 가져온 bending aids는 기계 설정의 bending aid 선택 드롭다운에서 사용할 수 있습니다.

이 옵션은 특정 벤딩 작업에 맞게 bending aid를 사용자 정의해야 하거나 기본 TecZone Bend 데이터베이스에 포함되지 않은 특수 bending aids를 사용할 때 유용합니다.
