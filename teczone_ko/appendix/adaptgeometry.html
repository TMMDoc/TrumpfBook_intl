형상 조정 기능

이 문서는 형상 조정 기능에서 사용할 수 있는 세 가지 다른 프로세스 옵션을 설명합니다:

형상 조정 기능은 입력 부품이 실제 프로세스 조건과 일치하지 않을 때 호출됩니다. 부품에는 다음 문제 중 하나 또는 둘 다 존재할 수 있습니다:

  • 벤드의 내부 반경이 너무 작을 수 있습니다. 각 프로세스(패널 벤딩 또는 프레스 브레이크 에어 벤딩 등)에는 해당 프로세스로 가능한 최소 내부 반경이 있습니다. 부품의 설계 반경이 이보다 작으면 부품을 설계대로 제조할 수 없으며 반경을 변경해야 합니다.

  • 벤드 라인의 K-팩터가 잘못되었을 수 있습니다. 프로세스 기계(패널 벤더 대 프레스 브레이크)와 프로세스 도구 또는 조건(펀치/다이/폴딩 블레이드/에어 갭)이 모두 결합되어 벤드의 실제 K-팩터를 결정합니다. 벤드는 다른 K 팩터로 설계되었을 수 있습니다.[1]

잘못된 반경과 잘못된 K-팩터를 모두 가지고 있는 간단한 2-D 부품을 예로 들어 형상 조정 기능이 이 부품을 어떻게 수정하는지 살펴보겠습니다.

Figure 1. 2D 부품

위에 입력 2D 부품이 표시되어 있습니다 - 이것은 정확히 중간에 하나의 벤드 라인이 있는 간단한 직사각형입니다. 부품 폭은 96.75mm입니다. 재료 두께는 2.5mm입니다.

벤드 라인은 내부 반경 0.5와 K-팩터 0.5로 _설계_되었습니다. 따라서 평면 폭(중립선의 길이)은 다음과 같습니다:

각도 * (내부반경 + K팩터 * 두께) = PI/2 * (0.5 + 0.5 * 2.5) = 2.75 mm

이것은 90도 벤드이므로 벤드 공제는 간단히 다음과 같습니다:

2 * 두께 - 평면폭 = 2 * 3 - 2.75 = 3.25 mm

아무것도 변경하지 않음

형상 조정 처리 없이 접었을 때, 다음 치수의 부품이 생성됩니다:

Figure 2. 아무것도 변경하지 않음

이것은 또한 아무것도 변경하지 않음 옵션으로 얻는 정확한 결과입니다. K-팩터도 반경도 변경되지 않았으므로 2D 도면의 설계 의도와 정확히 일치하는 3D 부품이 생성됩니다. 이것을 제조용으로 설정하면 결과적으로 제조된 부품은 실제로 이러한 치수와 일치하지 않습니다. 주로 기계에서 생성된 벤드가 설계된 벤드와 반경 및 유효 K-팩터 모두에서 다르기 때문입니다. 이 경우 제조 후 50x50보다 약간 작은 부품이 생성됩니다.

평면 크기 유지

이 옵션을 선택하면 TecZone는 평면의 크기를 변경하지 않지만 올바른 벤드 반경과 K-팩터로 벤드 라인에 주석을 답니다. 올바른 벤드 반경이 2.5mm이고 K-팩터가 0.46mm라고 가정해 봅시다. 이제 부품은 먼저 다음과 같이 내부적으로 변경됩니다:

Figure 3. 평면 크기 유지

전체 부품 크기는 변경되지 않고(여전히 96.75mm), 벤드 라인의 위치는 변경되지 않았습니다. 그러나 벤드는 이제 반경 2.5와 K-팩터 0.46이므로 벤드의 평면 폭이 5.73으로 증가했습니다. 이것은 이제 접혀서 다음과 같은 결과가 됩니다:

Figure 4. 평면 크기 유지2

보시다시피 부품의 2D 치수는 변경되지 않았지만 3-D 치수는 50.51 x 50.51로 변경되었습니다(초기 설계 치수 50 x 50에서).

3-D 치수 유지

이 옵션을 선택하면 부품의 3D 치수는 변경되지 않지만 벤드 반경과 K-팩터가 변경되므로 전개된 부품이 변경됩니다. 2-D 평판으로 시작할 때 (이 예에서처럼) TecZone는 다음 단계를 따릅니다.

  • 초기 설계 의도의 반경과 K-팩터를 사용하여 모델을 3D로 접습니다. 이렇게 하면 그림 2의 모델과 동일하게 보이는 3D 모델이 생성됩니다.

  • 이제 반경을 목표 반경 2.5mm로 변경하고 K-팩터를 목표 0.46으로 변경합니다. 이렇게 하면 아래와 같은 3D 모델이 생성됩니다:

Figure 5. 3D 유지

  • 그런 다음 이 모델을 전개하여(벤드에 대해 K-팩터 0.46 사용) 다음 평면이 생성됩니다:

Figure 6. 3D 유지2

이 옵션을 사용하면 3D 치수가 설계 치수 50x50으로 유지됩니다. 전개된 부품 폭은 96.75에서 95.73mm로 변경되었습니다.

요약

  • 평면이 아직 생산되지 않은 경우 일반적으로 최상의 옵션은 *3-D 치수 유지*입니다

  • 평면이 이미 생산되어 수정할 수 없는 경우 최상의 옵션은 *평면 크기 유지*입니다

이 두 옵션 모두 TecZone 창에 표시된 3D 데이터와 최종 제조 결과가 일치하는 부품을 생성합니다. 반대로 아무것도 변경하지 않음 옵션은 실제 벤드 반경과 K-팩터가 설계에서 올바르게 모델링되지 않았기 때문에 TecZone에 표시된 3D 데이터와 일치하지 않는 최종 제조 부품을 생성합니다.

1. 이것은 일반적으로 2-D 부품에만 해당하는 문제입니다. 3-D 부품은 각 벤드와 연관된 벤드 반경을 가지지만 아직 K-팩터를 정의하지 않습니다. K-팩터가 설정되어야 하는 것은 전개 프로세스 중이며, 따라서 평면으로 전개될 때 각 벤드의 '벤딩 여유’를 결정합니다.
