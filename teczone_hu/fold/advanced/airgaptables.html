Egyedi légrés táblázatok

A forgató hajlítógép légrés táblázatában egy adott lemezvastagság a megmunkálás során használt standard légréshez van hozzárendelve. Ez be van építve az egyes forgató hajlítógépkbe, de felülbírálható egy JSON fájl formájában megadott egyedi légrés táblázattal. Ez a JSON fájl az air-gap-tables.json nevet viseli és a C:\ProgramData\Trumpf{APPNAME}\Data\System mappában található.

Az alábbiakban egy tipikus air-gap-tables.json (légrés táblázatok) fájl látható:

 {
  "version":1  "airGapTables": [
    {
      "machine": "112002,112003", "material": "DC01"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
     {
      "machine": "112003", "material": "*"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
    {
      "machine": "*", "material": "Alluminio"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    }
  ]
}

  • A fájl légrésértékek kollekcióját tartalmazza. Minden táblázatban van egy gép- és anyagazonosító, amelyek szelektorként szolgálnak.

  • Ezek az azonosító kódok * (csillag) jelöléssel vannak ellátva, ami helyettesítő karakterként szolgál (mindenhez illeszkedik) vagy gépek, illetve anyagértékek vesszővel elválasztott listáját tartalmazzák.

  • A fenti példában három táblázat található. Az első a 112001 vagy 112003 jelű gépekhez használatos, kizárólag a DC01 anyaghoz. A második az összes anyaghoz használható a 112003 jelű gépen, a harmadik pedig minden géphez, az alumínium anyaghoz. Ha több táblázat is megfelel az alkatrész gép/anyag kombinációjának, akkor az első megfelelő táblázatot használja a rendszer. Tehát a 112003 jelű géphez és a DC01 anyaghoz az első táblázatot használja (még akkor is, ha a második is megfelel).

  • Minden táblázat egyszerű vastagság-azonosítókból álló adattömb, a megfelelő légrés-értékekkel. Például a fenti első bejegyzést úgy kell értelmezni, hogy ha a vastagság kisebb vagy egyenlő 0,75 mm-rel, akkor 2,5 mm-es légrést kell használni. A vastagság azonosítóknak növekvő sorrendben kell szerepelniük. Ha a fájlban található táblázatok egyike sem felel meg az aktuális alkatrész anyag- és vastagság-kombinációjának, akkor a gép alapértelmezett légréstáblázata kerül alkalmazásra.

csaló Ha bármilyen változtatást hajt végre az air-gap-tables.json fájlban, akkor újra kell indítania a TecZone Fold programot, hogy a változtatások érvénybe lépjenek.
Ha a JSON fájl hibás, hibaüzenet jelenik meg, amikor megpróbál megmunkálni egy alkatrészt a forgató hajlítógépen, a TecZone Fold azonban visszalép a gép alapértelmezett légrés-táblázatához és folytatja a munkát.
