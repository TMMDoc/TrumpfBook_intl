Özel hava boşluğu tabloları

Özel hava boşluğu tabloları

Bir panel bükme makinesine yönelik hava boşluğu tablosu, belirli bir sac kalınlığını işleme sırasında kullanılacak olan standart hava boşluğuna eşler. Bu, farklı panel bükme makinelerinin her birinde yerleşik olarak mevcuttur; ancak JSON dosyası formatındaki özel bir hava boşluğu tablosu aracılığıyla geçersiz kılınabilir. Bu JSON dosyası air-gap-tables.json adıyla şu hedeftedir: C:\ProgramData\Trumpf{APPNAME}\Data\System.

Aşağıda tipik bir air-gap-tables.json dosyası sağlanmıştır:

 {
  "version":1  "airGapTables": [
    {
      "machine": "112002,112003", "material": "DC01"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
     {
      "machine": "112003", "material": "*"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
    {
      "machine": "*", "material": "Alluminio"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    }
  ]
}

  • Dosya, hava boşluğu değerlerinin bir koleksiyonunu içerir. Her tablo, seçici işlevi gören bir makine ve malzeme anahtarına sahiptir.

  • Bu anahtarlar, joker karakteri (her şeyle eşleşebilir) belirtmek üzere * (yıldız işareti) olarak ayarlanır veya makine veya malzeme değerlerinin virgülle ayrılmış olan bir listesidir.

  • Yukarıdaki örnekte üç tablo sağlanmıştır. Birincisi, 112001 veya 112003 numaralı makinelerde, yalnızca DC01 malzemesi için kullanılır. İkincisi tüm malzemeler için, 112003 numaralı makine için kullanılır ve üçüncüsü de tüm makinelerde, Alluminio malzemesi için kullanılır. Eğer bir parçanın makine/malzeme kombinasyonu birden fazla tablo ile eşleşiyorsa, ilk eşleşen tabla kullanılır. Bundan dolayı, 112003 numaralı makinede DC01 malzemesi için ilk tablo kullanılmıştır (ikincisi de eşleşse bile).

  • Her bir tablo, karşılık gelen hava boşluğu değerleriyle basit bir kalınlık anahtarları dizisinden oluşur. Örnek olarak, yukarıdaki birinci giriş şu şekilde yorumlanmalıdır: Eğer kalınlık 0,75 mm’den az veya buna eşitse, 2,5 mm’lik bir boşluk kullanılmalıdır. Kalınlık anahtarları, artan sırada olmalıdır. Bu dosyadaki tabloların hiçbiri mevcut parçanın malzeme ve kalınlık kombinasyonuyla eşleşmiyorsa, makine için varsayılan hava boşluğu tablosu kullanılır.

Eğer air-gap-tables.json dosyasında bir değişiklik yaparsanız, değişikliklerin etkili olması için TecZone Fold yeniden başlatılmalıdır.
JSON dosyası hatalı biçimlendirilmişse, bir panel bükme makinesi parçasını işlemeye çalıştığınızda bir hata mesajı gösterilir; ancak TecZone Fold bu durumda makine için varsayılan hava boşluğu tablosunu kullanmaya geri döner ve işlemeyi sürdürür.
