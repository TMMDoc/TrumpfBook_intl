Laser CAM

Laser CAM

A sheet metal flat can be cut out from a sheet using a 2D laser cutter, before it is sent for subsequent processing on a press brake or a panel bender. The Cut module of TecZone is used to program 2D laser cutting machines. Typically, a number of parts (often of different types) are cut out of a single sheet. This process involves two stages:

  • Tooling up a part for laser cutting.

  • Nesting several such parts on a sheet.

The Part Workflow section discusses the first step, while the Job Workflow section discusses the second.

Getting Started

Here are some basic resources to get you started with TecZone Laser:

  • All modules of TecZone Laser share a common user-interaction model called the Clarity interface.

  • You can install laser machines from the Machine Database.

  • The starting point for all the CAM modules is CAD data, and you can get those CAD files by importing 2D or 3D data, or by drawing it using the Sketch tools built into TecZone.
