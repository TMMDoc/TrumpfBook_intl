Import Assembly

  • Click Workflow icon and select icon.

  • Cancel assembly or use Shortcut key X. Assembly breaks up into individual bending parts for the respective bending parts.

After an assembly has dissolved, a message is displayed indicating the total number of parts found in the assembly as well as the number of different parts.

If a part is selected, the number of parts is displayed.
