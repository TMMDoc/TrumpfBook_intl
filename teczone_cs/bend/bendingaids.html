Pomůcky pro ohýbání

Pomůcky pro ohýbání

Pomůcky pro ohýbán í jsou komponenty ohraňovacího lisu, které pomáhají ohýbat tlustší plechové materiály. Jsou navrženy tak, aby poskytovaly další podporu a kontrolu během procesu ohýbání, zajišťující přesné a konzistentní výsledky.

Ohraňovací lis může mít nainstalováno více pomůcek pro ohýbání nebo jednu pomůcku pro ohýbání, v závislosti na specifických požadavcích ohýbací operace.

Pomůcky pro ohýbání jsou použitelné pouze pro série TruBend 5000 a 8000.

Konfigurace pomůcek pro ohýbání

Pomůcky pro ohýbání lze konfigurovat v Database / Machines / Settings.

Když je importován díl a vybrán stroj s pomůckami pro ohýbání, TecZone Bend automaticky konfiguruje pomůcky pro ohýbání na základě požadavků dílu.

Použití pomůcek pro ohýbání

Jakmile je vytvořeno proveditelné ohýbací řešení, pomůcky pro ohýbání mohou být dále upraveny k optimalizaci procesu ohýbání.

Klikněte přímo na pomůcku pro ohýbání v TecZone Bend, abyste otevřeli panel Bend Aid. Číslo v panelu pomůcky pro ohýbání představuje sekvenci ohýbání.

  • Index - Označuje aktuálně vybranou pomůcku pro ohýbání.

  • Position (Z,R) - Používá se k úpravě pozice pomůcky pro ohýbání podél os Z a R.

  • Enabled - Zaškrtněte toto políčko pro aktivaci pomůcky pro ohýbání.

  • Auto-Place - Používá se k automatickému umístění pomůcky pro ohýbání na základě geometrie dílu.

Speed:

  • Down - Používá se k úpravě rychlosti zpětného pohybu pomůcky pro ohýbání po operaci ohybu.

  • Set All Bends - Používá se k aplikaci aktuálního nastavení rychlosti na všechny ohyby v sekvenci.

Advanced:

  • Method - Používá se k definování provozního režimu pomůcky pro ohýbání. Možnosti zahrnují:

    • Default - Obnoví nastavení pomůcky pro ohýbání na výchozí hodnoty.

    • Return at UDP - Konfiguruje pomůcku pro ohýbání, aby se vrátila na uživatelem definovanou pozici po každém ohybu.

    • Tilt product - Umožňuje pomůcce pro ohýbání naklonit díl během procesu ohýbání pro lepší zarovnání.

    • Static angle - Umožňuje nastavit pevný úhel pro pomůcku pro ohýbání během operace.

  • Stop angle - Používá se k nastavení úhlu, při kterém se pomůcka pro ohýbání zastaví během operace.

Parkovací pozice pomůcek pro ohýbání

Pomůcky pro ohýbání mohou být zaparkováné v určené pozici, když nejsou používány. To pomáhá optimalizovat pracovní prostor a zajišťuje bezpečnost během operace. To také zabraňuje kolizím s jinými komponenty stroje.

Pomůcky pro ohýbání mohou být zaparkováné v následujících pozicích:

  1. Default - Pomůcky pro ohýbání mohou být zaparkováné až po délku stolu.

  2. Small - Pomůcky pro ohýbání mohou být umístěny 700 mm za délkou stolu.

  3. Large - Pomůcky pro ohýbání mohou být umístěny 1200 mm za délkou stolu.

Pro tandemové stroje jsou k dispozici čtyři pomůcky pro ohýbání. Pro levé tandemové stroje je povoleno pouze parkování na levé straně. Pro pravé tandemové stroje je povoleno pouze parkování na pravé straně.

Možnost parkovací pozice bude zobrazena pouze pro série TruBend 8000 a stroje VarioPress.

Pro tandemové stroje B36 není parkovací pozice small použitelná.

Vytváření vlastních pomůcek pro ohýbání

Pro vytvoření vlastních pomůcek pro ohýbání v TecZone Bend musíte nejprve vytvořit soubor ini, který definuje parametry pomůcky pro ohýbání a soubory mesh pro komponenty pomůcky pro ohýbání (Carriage, Table a Plate).

Ukázkový soubor baid.ini
[BAData]
; The user defined name of the BendAid. Only used for custom bend aids
Name=BATest
; The range of V-widths the bend aid can support
DieV=16,250
; The range of die-heights the bend aid can support
DieHeight=-30,250
; Maximum angle, in degrees
MaxAngle=55
; Lift capacity, in kg
Payload=300
; Maximum lift speed, in degrees/second
Speed=45
; When the bending aid is at X=0, this is the bound of the support table in top view
Table=-159,-1032,159,-245.6
; The X-Span of the bending aid (when positioned at 0)
XExtent=-213,213
; Tells whether the user can change the forward speed
CanChangeForwardSpeed=False
; Tells whether safety stop angle is possible in the machine
CanStopAtReturn=True
; Can the bending aid movement be varied using different methods?
AdvancedControl=True
; What type of existing bending aid can this be mapped to? Used only by custom bend aids
EType=TB8000_2000N_B36

[Carriage]
Model=Carriage.mesh
Stroke=-1000,1000
Color=White
Shift=0,0,100

[Table]
Model=Table.Mesh
Stroke=50,200
Color=White

[Plate]
Model=Plate.mesh
Stroke=0,56
Color=#4A4A4A

Umístěte soubor ini a odpovídající soubory mesh do složky, kterou pak lze importovat do TecZone Bend.

Import vlastních pomůcek pro ohýbání

Import vlastních pomůcek pro ohýbání:

  1. Přejděte na File / Import / Bending Aids…​ a vyberte již vytvořený soubor ini (ujistěte se, že odpovídající soubory mesh jsou ve stejné složce).

  2. Importované pomůcky pro ohýbání budou poté k dispozici v rozevíracím seznamu výběru pomůcek pro ohýbání v nastavení stroje.

Tato možnost je užitečná, když je třeba přizpůsobit pomůcku pro ohýbání pro specifické ohýbací úkoly nebo při použití specializovaných pomůcek pro ohýbání, které nejsou zahrnuty ve výchozí databázi TecZone Bend.
