Vlastní tabulky vzduchových mezer

Tabulka vzduchových mezer pro stroj s výkyvným ohýbáním přiřazuje konkrétní tloušťku plechu ke standardní vzduchové mezeře používané během zpracování. Tato funkce je zabudována do všech strojů s výkyvným ohýbáním, ale lze ji přepsat zadáním vlastní tabulky vzduchových mezer ve formě souboru JSON. Tento soubor JSON se nazývá air-gap-tables.json a je umístěn v C:\ProgramData\Trumpf{APPNAME}\Data\System.

Níže je uveden typický soubor air-gap-tables.json:

 {
  "version":1  "airGapTables": [
    {
      "machine": "112002,112003", "material": "DC01"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
     {
      "machine": "112003", "material": "*"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
    {
      "machine": "*", "material": "Alluminio"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    }
  ]
}

  • Soubor obsahuje kolekci hodnot vzduchové mezery. Každá tabulka má klíč stroje a materiálu, které fungují jako voliče.

  • Tyto klíče jsou buď nastaveny na * (hvězdička) pro označení zástupného znaku (odpovídá všemu), nebo jsou to čárkami oddělené seznamy hodnot strojů nebo materiálů.

  • Ve výše uvedeném příkladu jsou tři tabulky. První se používá pro stroje 112001 nebo 112003, pouze pro materiál DC01. Druhá se používá pro všechny materiály, pro stroj 112003, a třetí se používá pro všechny stroje, pro materiál Alluminio. Pokud může kombinaci stroje/materiálu daného dílu odpovídat více tabulek, použije se první odpovídající tabulka. Pro stroj 112003 a materiál DC01 se tedy použije první tabulka (i když druhá tabulka také odpovídá).

  • Každá tabulka se skládá z jednoduchého pole klíčů tloušťky s odpovídajícími hodnotami vzduchové mezery. Například první záznam výše by měl být interpretován takto: pokud je tloušťka menší nebo rovná 0,75 mm, použijte mezeru 2,5 mm. Tloušťkové klíče musí být seřazeny vzestupně. Pokud žádná z tabulek v tomto souboru neodpovídá kombinaci materiálu a tloušťky aktuálního dílu, použije se výchozí tabulka vzduchových mezer pro daný stroj.

Pokud provedete jakékoli změny v souboru air-gap-tables.json, musíte TecZone Fold restartovat, aby se tyto změny projevily.
Pokud je soubor JSON nesprávně formátován, při pokusu o zpracování dílu strojem s výkyvným ohýbáním se zobrazí chyba, ale poté se TecZone Fold vrátí k použití výchozí tabulky vzduchových mezer pro stroj a bude pokračovat.
