Report

Report

Oltre a generare programmi NC, TecZone Bend può anche generare vari report per un pezzo, a seconda dei moduli installati. Quando si lavora con la tecnologia di piegatura, ad esempio, è utile generare una scheda di setup che elenca gli utensili da montare e fornisce dettagli sulla sequenza di piegatura.

Il Report Designer in TecZone Bend aiuta a progettare il contenuto e lo stile di tali report. Una volta progettato un report, è possibile emetterlo in formato PDF o HTML[1] formati. Ecco come appare il Report Designer quando è aperto:

Utilizzo di un report

È possibile aprire un report per la modifica scegliendo il menu File/Apri e selezionando un file modello di report (si tratta di file con estensione .ftrep). È possibile trovare alcuni modelli di report di esempio nella cartella Esempi dell’installazione TecZone Bend.

Prima di poter aprire un modello di report con cui lavorare, innanzitutto aprire il pezzo e lavorarlo fino al completamento (si dovrebbero generare gli opportuni dati della tecnologia di piegatura/taglio/risvolto). Questo perché il Report Designer funziona con i dati in tempo reale; mostra i dati effettivi del pezzo mentre si progetta il report, in modo da poter vedere immediatamente come sarà il risultato finale stampato.

È anche possibile iniziare con un report vuoto scegliendo il menu File/Nuovo/Modello di report. È quindi possibile popolare il report aggiungendo vari elementi del report (campi di testo, immagini, etichette, ecc.) e quindi salvarlo.

Quando un modello di report è aperto, è possibile utilizzare i comandi File/Esporta in PDF o File/Esporta in HTML per eseguire il rendering del report in formato PDF o HTML. Il report viene stampato utilizzando i dati del pezzo attualmente aperto e attivo. È inoltre possibile passare dalla visualizzazione dei dati in tempo reale (impostazione predefinita) alla vista di progettazione che mostra le espressioni sottostanti utilizzate per popolare i vari campi del report.

Report stampati automaticamente:

È possibile fare in modo che TecZone Bend stampi automaticamente un report piega ogni volta che una soluzione di piegatura viene generata (proprio come può generare automaticamente un programma NC quando la soluzione viene calcolata). Utilizzare la pagina delle impostazioni Output dal modulo della tecnologia di piegatura per la configurazione.

Nell’esempio precedente, è attivata la generazione automatica dei report. Il modello di report è impostato su lib:Report/bend.ftrep, che è il modello di report predefinito per CAM piega (viene installato come parte di ogni installazione TecZone Bend). Se si crea un modello di report diverso utilizzando il Report Designer, è possibile specificare il modello qui.

Le impostazioni consentono inoltre di controllare la cartella in cui viene generato il report e il formato (HTML o PDF) in cui è scritto il report.

1. I file HTML generati dall’autore del report TecZone Bend sono indipendenti. Includono tutte le immagini necessarie, i fogli di stile e altre risorse incorporate all’interno del file HTML. Pertanto, possono essere facilmente trasportati o archiviati senza dipendenze esterne.
