Tabelle traferro personalizzate

La tabella traferro per una piegatrice a grembiale mappa un particolare spessore della lamiera al traferro standard utilizzato durante la lavorazione. È integrata in ciascuna delle varie piegatrici a grembiale, ma può essere sovrascritta fornendo una tabella traferro personalizzata sotto forma di file JSON. Questo file JSON è chiamato air-gap-tables.json e viene inserito in C:\ProgramData\Trumpf{APPNAME}\Data\System.

Di seguito è riportato un tipico file air-gap-tables.json:

 {
  "version":1  "airGapTables": [
    {
      "machine": "112002,112003", "material": "DC01"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
     {
      "machine": "112003", "material": "*"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
    {
      "machine": "*", "material": "Alluminio"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    }
  ]
}

  • Il file contiene una raccolta di valori di traferro. Ogni tabella ha un tasto Macchina e Materiale che fungono da selettori.

  • Questi tasti sono impostati su * (asterisco) per indicare un carattere jolly (corrisponde a tutto), oppure sono un elenco separato da virgole dei valori della macchina o del materiale.

  • Nell’esempio precedente, ci sono tre tabelle. La prima viene utilizzata per le macchine 112001 o 112003, solo per il materiale DC01. La seconda viene utilizzata per tutti i materiali, per la macchina 112003, e la terza viene utilizzata per tutte le macchine, per il materiale alluminio. Se più tabelle potrebbero corrispondere alla combinazione macchina/materiale del pezzo, viene utilizzata la prima tabella che corrisponde. Quindi, per la macchina 112003, materiale DC01, viene utilizzata la prima tabella (anche se anche la seconda corrisponde).

  • Ogni tabella è costituita da una semplice serie di tasti di spessore, con i corrispondenti valori di traferro. Ad esempio, la prima voce di cui sopra deve essere interpretata come: se lo spessore è inferiore o uguale a 0,75 mm, utilizzare un traferro di 2,5 mm. I tasti di spessore devono essere in ordine crescente. Se nessuna delle tabelle all’interno di questo file corrisponde alla combinazione materiale/spessore dell’attuale spessore, verrà utilizzata la tabella traferro predefinita per la macchina.

Se si apportano modifiche al file air-gap-tables.json, è necessario riavviare TecZone Fold prima che queste modifiche abbiano effetto.
Se il file JSON non è ben formato, verrà visualizzato un errore quando si tenta di lavorare un pezzo della piegatrice a grembiale, ma poi TecZone Fold tornerà a utilizzare la tabella traferro predefinita per la macchina e continuerà a farlo.
