Ausili di piegatura

Ausili di piegatura

Gli ausili di piegatura sono componenti della pressa piegatrice che aiutano a piegare lamiere più spesse. Sono progettati per fornire supporto e controllo aggiuntivi durante il processo di piegatura, garantendo risultati accurati e costanti.

Una pressa piegatrice può avere più ausili di piegatura o un singolo ausilio di piegatura installato, a seconda dei requisiti specifici dell’operazione di piegatura.

Gli ausili di piegatura sono applicabili solo per le serie TruBend 5000 e 8000.

Configurazione degli ausili di piegatura

Gli ausili di piegatura possono essere configurati in Database / Machines / Settings.

Quando viene importata una parte e viene selezionata una macchina con ausili di piegatura, TecZone Bend configura automaticamente gli ausili di piegatura in base ai requisiti della parte.

Utilizzo degli ausili di piegatura

Una volta generata una soluzione di piegatura fattibile, gli ausili di piegatura possono essere ulteriormente regolati per ottimizzare il processo di piegatura.

Fare clic direttamente sull’ausilio di piegatura in TecZone Bend per aprire il pannello Bend Aid. Il numero nel pannello dell’ausilio di piegatura rappresenta la sequenza di piegatura.

  • Index - Indica l’ausilio di piegatura attualmente selezionato.

  • Position (Z,R) - Utilizzato per regolare la posizione dell’ausilio di piegatura lungo gli assi Z e R.

  • Enabled - Selezionare questa casella di controllo per attivare l’ausilio di piegatura.

  • Auto-Place - Utilizzato per posizionare automaticamente l’ausilio di piegatura in base alla geometria della parte.

Speed:

  • Down - Utilizzato per regolare la velocità di retrazione dell’ausilio di piegatura dopo l’operazione di piegatura.

  • Set All Bends - Utilizzato per applicare l’impostazione di velocità corrente a tutte le pieghe nella sequenza.

Advanced:

  • Method - Utilizzato per definire la modalità operativa dell’ausilio di piegatura. Le opzioni includono:

    • Default - Ripristina le impostazioni dell’ausilio di piegatura ai valori predefiniti.

    • Return at UDP - Configura l’ausilio di piegatura per tornare a una posizione definita dall’utente dopo ogni piega.

    • Tilt product - Abilita l’ausilio di piegatura a inclinare la parte durante il processo di piegatura per un migliore allineamento.

    • Static angle - Consente di impostare un angolo fisso per l’ausilio di piegatura durante il funzionamento.

  • Stop angle - Utilizzato per impostare l’angolo al quale l’ausilio di piegatura si fermerà durante il funzionamento.

Posizione di parcheggio degli ausili di piegatura

Gli ausili di piegatura possono essere parcheggiati in una posizione designata quando non sono in uso. Questo aiuta a ottimizzare lo spazio di lavoro e garantisce la sicurezza durante il funzionamento. Questo inoltre previene collisioni con altri componenti della macchina.

Gli ausili di piegatura possono essere parcheggiati nelle seguenti posizioni:

  1. Default - Gli ausili di piegatura possono essere parcheggiati fino alla lunghezza del tavolo.

  2. Small - Gli ausili di piegatura possono essere posizionati 700 mm oltre la lunghezza del tavolo.

  3. Large - Gli ausili di piegatura possono essere posizionati 1200 mm oltre la lunghezza del tavolo.

Per le macchine tandem, sono disponibili quattro ausili di piegatura. Per le macchine tandem sinistre, è consentito solo il parcheggio sul lato sinistro. Per le macchine tandem destre, è consentito solo il parcheggio sul lato destro.

L’opzione di posizione di parcheggio verrà visualizzata solo per le serie TruBend 8000 e le macchine VarioPress.

Per le macchine B36 tandem, la posizione di parcheggio small non è applicabile.

Creazione di ausili di piegatura personalizzati

Per creare ausili di piegatura personalizzati in TecZone Bend, è necessario prima creare un file ini che defin isce i parametri dell’ausilio di piegatura e file mesh per i componenti dell’ausilio di piegatura (Carriage, Table e Plate).

Esempio di file baid.ini
[BAData]
; Il nome definito dall'utente dell'ausilio di piegatura. Utilizzato solo per ausili di piegatura personalizzati
Name=BATest
; L'intervallo di larghezze V che l'ausilio di piegatura può supportare
DieV=16,250
; L'intervallo di altezze delle matrici che l'ausilio di piegatura può supportare
DieHeight=-30,250
; Angolo massimo, in gradi
MaxAngle=55
; Capacità di sollevamento, in kg
Payload=300
; Velocità massima di sollevamento, in gradi/secondo
Speed=45
; Quando l'ausilio di piegatura è a X=0, questo è il limite del tavolo di supporto in vista dall'alto
Table=-159,-1032,159,-245.6
; L'estensione X dell'ausilio di piegatura (quando posizionato a 0)
XExtent=-213,213
; Indica se l'utente può modificare la velocità in avanti
CanChangeForwardSpeed=False
; Indica se l'angolo di arresto di sicurezza è possibile nella macchina
CanStopAtReturn=True
; Il movimento dell'ausilio di piegatura può essere variato utilizzando diversi metodi?
AdvancedControl=True
; A quale tipo di ausilio di piegatura esistente può essere mappato? Utilizzato solo da ausili di piegatura personalizzati
EType=TB8000_2000N_B36

[Carriage]
Model=Carriage.mesh
Stroke=-1000,1000
Color=White
Shift=0,0,100

[Table]
Model=Table.Mesh
Stroke=50,200
Color=White

[Plate]
Model=Plate.mesh
Stroke=0,56
Color=#4A4A4A

Posizionare il file ini e i file mesh corrispondenti in una cartella che può quindi essere importata in TecZone Bend.

Importazione di ausili di piegatura personalizzati

Per importare ausili di piegatura personalizzati:

  1. Navigare su File / Import / Bending Aids…​ e selezionare il file ini già creato (assicurarsi che i file mesh corrispondenti siano nella stessa cartella).

  2. Gli ausili di piegatura importati saranno quindi disponibili nel menu a discesa di selezione dell’ausilio di piegatura nelle impostazioni della macchina.

Questa opzione è utile quando un ausilio di piegatura deve essere personalizzato per specifiche attività di piegatura o quando si utilizzano ausili di piegatura specializzati non inclusi nel database predefinito di TecZone Bend.
