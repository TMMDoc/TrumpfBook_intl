Uscite di piega

Impostazioni piano di attrezzaggio

Modello per il piano di attrezzaggio - È la posizione e il nome del modello da utilizzare per la comparsa dei report.

Percorso di archiviazione del piano di attrezzaggio - Impostare come posizione dove salvare tutti i file dei report.

Formato di emissione del piano di attrezzaggio - Può essere impostato come HTML o PDF. Nota: affinché il PDF utilizzi il contenuto multimediale e 3D, occorre installare Adobe Acrobat Reader.

Crea piano di attrezzaggio con programma di piegatura - Attivare affinché un report venga generato automaticamente quando si passa a una piega ed è stata trovata una soluzione. Se la soluzione rileva errori o avvertimenti, il report non verrà generato.

Inserire il piano di attrezzaggio nel programma NC, se possibile - Attivare questa funzione per allegare report al programma NC.

Impostazioni codice

Percorso di archiviazione del file NC - Impostare come posizione dove salvare tutti i file di programma NC.

Postprocessore per il file NC - È la posizione e il nome di un file batch o di un utensile di altro produttore a cui fare riferimento quando si genera il programma NC.

Crea file NC con programma di piegatura - Attivare affinché il programma NC venga generato automaticamente quando si passa alla modalità Piegatura a matrice (a condizione che venga generata una soluzione valida senza errori).

Inserisci grafica nel codice NC - È un’opzione B23 che permette di confrontare la grafica BNC-SAT con il pezzo effettivo. La grafica BNC-Sat viene visualizzata su un monitor. Un’immagine del pezzo effettivo viene visualizzata sullo stesso monitor. L’utente può ora spostare, ruotare o girare il pezzo fino a quando la grafica BNC e le immagini del pezzo effettivo si sovrappongono. Questo aiuta l’utente a vedere se il pezzo è posizionato correttamente per la piegatura.

Semplifica grafica (rimuovi piccoli fori) - Per impostazione predefinita, è attivato (e la grafica BNC è semplificata). Può però essere disattivato; se disattivato, tutti i fori vengono emessi in BNC.

Codice di blocco (senza nuovo calcolo da parte di HMI) - Attivare questa opzione per bloccare il codice. Ciò fa sì che il controller non modifichi il codice né esegua ricalcoli.

Formato di emissione dello sviluppo - Può essere impostato come GEO, DXF, o PDG per l’opzione di esportazione dei file predefinita quando si salva automaticamente un design piatto. Quando viene trovata una soluzione per un pezzo, il modello piatto può automaticamente essere esportato e salvato nella destinazione del modello piatto. Tuttavia, se si trovano errori o avvertimenti, il design piatto dovrà essere salvato manualmente.

Percorso di archiviazione dello sviluppo - Impostare come posizione dove salvare tutti i modelli piatti.

Generazione file FX insieme a programma NC - Attivare questa opzione affinché un file FX venga salvato nella posizione scelta durante la generazione del programma NC.

Percorso di archiviazione del file FX - Impostare come posizione per il salvataggio di tutti i file FX.
