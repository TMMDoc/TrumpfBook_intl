อุปกรณ์ช่วยการพับ (Bending Aids)

อุปกรณ์ช่วยการพับ (Bending Aids)

อุปกรณ์ช่วยการพับเป็นส่วนประกอบของเครื่องพับโลหะที่ช่วยในการพับแผ่นโลหะที่หนาขึ้น ออกแบบมาเพื่อให้การสนับสนุนและการควบคุมเพิ่มเติมระหว่าง กระบวนการพับ เพื่อให้แน่ใจว่าผลลัพธ์มีความแม่นยำและสม่ำเสมอ

เครื่องพับโลหะสามารถติดตั้งอุปกรณ์ช่วยการพับหลายตัวหรือตัวเดียว ขึ้นอยู่กับความต้องการเฉพาะของการดำเนินการพับ

อุปกรณ์ช่วยการพับใช้ได้เฉพาะกับ TruBend 5000 และ 8000 series เท่านั้น

การกำหนดค่าอุปกรณ์ช่วยการพับ

สามารถกำหนดค่าอุปกรณ์ช่วยการพับได้ใน Database / Machines / Settings (ฐานข้อมูล / เครื่องจักร / การตั้งค่า)

เมื่อนำเข้าชิ้นงานและเลือกเครื่องที่มีอุปกรณ์ช่วยการพับ TecZone Bend จะกำหนดค่าอุปกรณ์ช่วยการพับโดยอัตโนมัติตามความต้องการ ของชิ้นงาน

การใช้อุปกรณ์ช่วยการพับ

เมื่อสร้างโซลูชันการพับที่เป็นไปได้แล้ว สามารถปรับแต่งอุปกรณ์ช่วยการพับเพิ่มเติม เพื่อเพิ่มประสิทธิภาพกระบวนการพับ

คลิกโดยตรงที่อุปกรณ์ช่วยการพับใน TecZone Bend เพื่อเปิดแผง Bend Aid (อุปกรณ์ช่วยการพับ) ตัวเลขในแผงอุปกรณ์ช่วยการพับแสดงถึงลำดับการพับ

  • Index - แสดงอุปกรณ์ช่วยการพับที่เลือกในปัจจุบัน

  • Position (Z,R) - ใช้สำหรับปรับตำแหน่งของอุปกรณ์ช่วยการพับตามแกน Z และ R

  • Enabled (เปิดใช้งาน) - เลือกช่องนี้เพื่อเปิดใช้งานอุปกรณ์ช่วยการพับ

  • Auto-Place (วางอัตโนมัติ) - ใช้สำหรับวางตำแหน่งอุปกรณ์ช่วยการพับโดยอัตโนมัติตามรูปทรงเรขาคณิตของชิ้นงาน

Speed (ความเร็ว):

  • Down (ลง) - ใช้สำหรับปรับความเร็วการถอยกลับของอุปกรณ์ช่วยการพับหลังการดำเนินการพับ

  • Set All Bends (ตั้งค่าทุกการพับ) - ใช้สำหรับนำการตั้งค่าความเร็วปัจจุบันไปใช้กับทุกการพับในลำดับ

Advanced (ขั้นสูง):

  • Method (วิธีการ) - ใช้สำหรับกำหนดโหมดการทำงานของอุปกรณ์ช่วยการพับ ตัวเลือกประกอบด้วย:

    • Default (ค่าเริ่มต้น) - รีเซ็ตการตั้งค่าอุปกรณ์ช่วยการพับเป็นค่าเริ่มต้น

    • Return at UDP (กลับที่ UDP) - กำหนดค่าอุปกรณ์ช่วยการพับให้กลับไปยังตำแหน่งที่ผู้ใช้กำหนดหลังการพับแต่ละครั้ง

    • Tilt product (เอียงชิ้นงาน) - ทำให้อุปกรณ์ช่วยการพับสามารถเอียงชิ้นงานระหว่างกระบวนการพับเพื่อการจัดวางที่ดีขึ้น

    • Static angle (มุมคงที่) - อนุญาตให้ตั้งมุมคงที่สำหรับอุปกรณ์ช่วยการพับระหว่างการทำงาน

  • Stop angle (มุมหยุด) - ใช้สำหรับตั้งมุมที่อุปกรณ์ช่วยการพับจะหยุดระหว่างการทำงาน

ตำแหน่งจอดของอุปกรณ์ช่วยการพับ

อุปกรณ์ช่วยการพับสามารถจอดไว้ในตำแหน่งที่กำหนดเมื่อไม่ได้ใช้งาน สิ่งนี้ช่วย เพิ่มประสิทธิภาพพื้นที่การทำงานและรับรองความปลอดภัยระหว่างการทำงาน นอกจากนี้ยัง ป้องกันการชนกับส่วนประกอบเครื่องจักรอื่นๆ

อุปกรณ์ช่วยการพับสามารถจอดได้ในตำแหน่งต่อไปนี้:

  1. Default (ค่าเริ่มต้น) - อุปกรณ์ช่วยการพับสามารถจอดได้จนถึงความยาวโต๊ะ

  2. Small (เล็ก) - อุปกรณ์ช่วยการพับสามารถวางได้ 700 มม. เกินความยาวโต๊ะ

  3. Large (ใหญ่) - อุปกรณ์ช่วยการพับสามารถวางได้ 1200 มม. เกินความยาวโต๊ะ

สำหรับเครื่อง tandem (แบบคู่) มีอุปกรณ์ช่วยการพับสี่ตัว สำหรับเครื่อง tandem ด้านซ้าย อนุญาตให้จอดด้านซ้ายเท่านั้น สำหรับเครื่อง tandem ด้านขวา อนุญาตให้จอดด้านขวาเท่านั้น

ตัวเลือกตำแหน่งจอดจะแสดงเฉพาะสำหรับ TruBend 8000 series และเครื่อง VarioPress เท่านั้น

สำหรับเครื่อง B36 tandem ตำแหน่งจอด small (เล็ก) ไม่สามารถใช้ได้

การสร้างอุปกรณ์ช่วยการพับแบบกำหนดเอง

เพื่อสร้างอุปกรณ์ช่วยการพับแบบกำหนดเองใน TecZone Bend คุณต้องสร้าง ไฟล์ ini ที่กำหนดพารามิเตอร์ของอุปกรณ์ช่วยการพับและไฟล์ mesh สำหรับ ส่วนประกอบของอุปกรณ์ช่วยการพับ (Carriage, Table และ Plate)

ตัวอย่างไฟล์ baid.ini
[BAData]
; The user defined name of the BendAid. Only used for custom bend aids
Name=BATest
; The range of V-widths the bend aid can support
DieV=16,250
; The range of die-heights the bend aid can support
DieHeight=-30,250
; Maximum angle, in degrees
MaxAngle=55
; Lift capacity, in kg
Payload=300
; Maximum lift speed, in degrees/second
Speed=45
; When the bending aid is at X=0, this is the bound of the support table in top view
Table=-159,-1032,159,-245.6
; The X-Span of the bending aid (when positioned at 0)
XExtent=-213,213
; Tells whether the user can change the forward speed
CanChangeForwardSpeed=False
; Tells whether safety stop angle is possible in the machine
CanStopAtReturn=True
; Can the bending aid movement be varied using different methods?
AdvancedControl=True
; What type of existing bending aid can this be mapped to? Used only by custom bend aids
EType=TB8000_2000N_B36

[Carriage]
Model=Carriage.mesh
Stroke=-1000,1000
Color=White
Shift=0,0,100

[Table]
Model=Table.Mesh
Stroke=50,200
Color=White

[Plate]
Model=Plate.mesh
Stroke=0,56
Color=#4A4A4A

วางไฟล์ ini และไฟล์ mesh ที่เกี่ยวข้องในโฟลเดอร์ซึ่งสามารถ นำเข้าสู่ TecZone Bend ได้

การนำเข้าอุปกรณ์ช่วยการพับแบบกำหนดเอง

เพื่อนำเข้าอุปกรณ์ช่วยการพับแบบกำหนดเอง:

  1. ไปที่ File / Import / Bending Aids…​ (ไฟล์ / นำเข้า / อุปกรณ์ช่วยการพับ…​) และเลือก ไฟล์ ini ที่สร้างไว้แล้ว (ตรวจสอบให้แน่ใจว่าไฟล์ mesh ที่เกี่ยวข้องอยู่ในโฟลเดอร์เดียวกัน)

  2. อุปกรณ์ช่วยการพับที่นำเข้าจะพร้อมใช้งานใน รายการเลือกอุปกรณ์ช่วยการพับในการตั้งค่าเครื่องจักร

ตัวเลือกนี้มีประโยชน์เมื่ออุปกรณ์ช่วยการพับจำเป็นต้องปรับแต่งสำหรับงานการพับเฉพาะ หรือเมื่อใช้อุปกรณ์ช่วยการพับเฉพาะทางที่ไม่ได้รวมอยู่ในฐานข้อมูล TecZone Bend มาตรฐาน
