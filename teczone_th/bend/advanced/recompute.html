Recompute solution

เมื่อคุณกด B หรือใช้แผง Workflow เพื่อคำนวณวิธีแก้ TecZone Bend จะคำนวณวิธีการพับโดยใช้ชุดการตั้งค่าและพารามิเตอร์เริ่มต้น คุณสามารถคำนวณวิธีนี้ใหม่โดยใช้การตั้งค่าอื่นๆ ได้ โดยคลิกที่เมนู Recompute ในแถบคำสั่ง ซึ่งบางครั้ง มีประโยชน์เมื่อคุณต้องการคำนวณต่อหลังจากแก้ไข การจัดเรียงเครื่องมือหรือลำดับ
ตัวอย่างเช่น:

  • คุณสามารถเปลี่ยนลำดับและให้ TecZone Bend คำนวณตำแหน่งเครื่องมือใหม่ โดยคงลำดับเดิมไว้

  • คุณสามารถเปลี่ยนตำแหน่งเครื่องมือและให้ TecZone Bend คำนวณลำดับใหม่ หรือจัดตำแหน่งชิ้นงานให้ตรงกับตำแหน่งเครื่องมือที่สร้างขึ้นใหม่่

  • คุณสามารถลองทำงานเครื่องมือด้วยชุดเครื่องมืออื่น หรือใช้การตั้งค่าที่สร้างไว้แล้ว

The Recompute dialog

Overrides

ส่วนนี้มีสวิตช์บางตัวที่คุณสามารถใช้เพื่อแทนที่การทำงานปกติของ TecZone Bend

  • เปิดสวิตช์ Don’t change sequence switch เพื่อบังคับให้ TecZone Bend คำนวณ คำตอบใหม่โดยไม่เปลี่ยนลำดับ ตำแหน่งเครื่องมือ เกจวัดที่แตกต่างกัน หรือทิศทางการใส่ชิ้นงานจะถูกลองใช้โดย TecZone Bend โดยคงลำดับเดิมไว้

  • หากเปิดสวิตช์ ไม่มีการวัดค่ามุม TecZone Bend จะคำนวณคำตอบ ที่ไม่ทำการวัดมุมใดๆ และจะไม่ใช้เซนเซอร์วัดเกจใดๆ (เครื่องมือ ACB)

  • ตัวเลือก Ignore tools from GEO/DXF files จะสั่งให้ TecZone Bend เพิกเฉยต่อคำแนะนำเครื่องมือ ที่มักพบในไฟล์ GEO และบางครั้งในไฟล์ DXF

Setups

ส่วนนี้ใช้กำหนดกลยุทธ์การตั้งค่าเครื่องมือ โดยปกติ TecZone Bend จะคำนวณ Setup ใหม่ตั้งแต่ต้น แต่คุณสามารถใช้การตั้งค่าเหล่านี้เพื่อเปลี่ยนพฤติกรรมได้

  • Compute new Setup เป็นการทำงานเริ่มต้น TecZone Bend จะคำนวณการตั้งค่าโดยใช้เครื่องมือทั้งหมดที่อนุญาตให้ใช้ (ดูส่วน Tools ด้านล่าง)

  • Keep existing Setup จะสั่งให้ TecZone Bend ไม่เปลี่ยนแปลงการตั้งค่าใดๆ และพยายาม ประมวลผลการพับทั้งหมดโดยใช้การตั้งค่าปัจจุบันของชิ้นงาน หากการพับบางจุดมีการชนกัน หรือมีความยาวเครื่องมือไม่เพียงพอสำหรับการประมวลผล ข้อผิดพลาดเหล่านั้นจะยังไม่ถูกแก้ไข และสามารถดูได้หลังจากขั้นตอนการ Recompute

  • หากคุณเลือก Use การตั้งค่าจากไฟล์ ระบบจะขอให้คุณเลือกไฟล์การตั้งค่าการพับ (bsetup) การตั้งค่าที่โหลดจากไฟล์นี้จะถูกใช้เพื่อคำนวณ โดยการตั้งค่านั้นจะไม่ถูกแก้ไข และจะไม่มีการสร้างตำแหน่งเครื่องมือเพิ่มเติม การตั้งค่าที่คุณเพิ่งโหลดจากไฟล์จะแสดง ในตัวเลือกด้านล่าง และคุณสามารถคลิกปุ่ม + ใกล้ตัวเลือกนั้นเพื่อเลือกไฟล์การตั้งค่าอื่น

  • ตัวเลือก Use Setup from another Part ใช้ได้ต่อเมื่อมีชิ้นงานอื่นที่เปิดอยู่ และได้ทำการตั้งค่าเครื่องมือสำหรับเครื่องพับโลหะเดียวกันแล้ว ในกรณีนี้ คุณสามารถให้ TecZone Bend ใช้การตั้งค่าเดียวกับหนึ่งในชิ้นงานอื่น โดยมีตัวเลือกให้คุณเลือกชิ้นงานที่ต้องการ

Tools

หากคุณเลือก Compute new Setup ในส่วนด้านบน คุณสามารถใช้ส่วน Tools เพื่อเลือกชุดเครื่องมือดัดที่อาจใช้ในการคำนวณการตั้งค่าใหม่ได้ หากคุณเลือก ตัวเลือกอื่นสำหรับ Setup ส่วนนี้จะไม่สามารถใช้งานได้

  • Preferred tools for Machine เป็นตัวเลือกเริ่มต้น สำหรับเครื่องพับโลหะแต่ละเครื่อง คุณสามารถเลือก ชุด tools[1] เป็น_เครื่องมือที่ต้องการ_ และ TecZone Bend จะใช้เครื่องมือดังกล่าวเมื่อทำการตั้งค่าเครื่องมือให้กับชิ้นงาน[2] (หากคุณยังไม่ได้ตั้งค่าเครื่องมือที่ต้องการสำหรับเครื่องจักร TecZone Bend จะใช้เครื่องมือทั้งหมดในรายการเครื่องมือเป็นชุดเครื่องมือที่ต้องการสำหรับเครื่องจักร)

  • หากคุณเลือก Tools in Inventory TecZone Bend จะใช้เครื่องมือทั้งหมดที่มีอยู่ใน รายการเครื่องมือของการติดตั้งนั้น หากคุณยังไม่ได้ตั้งค่ารายการเครื่องมือ TecZone Bend จะใช้เครื่องมือทั้งหมดใน Catalog เป็นรายการเครื่องมือของคุณ

  • All Tools in Catalog เป็นการเลือกเครื่องมือที่กว้างที่สุด TecZone Bend จะใช้ เครื่องมือทุกชิ้นที่มีอยู่ในแค็ตตาล็อกของผู้ผลิตเพื่อหาวิธีแก้ปัญหา แม้ว่า อาจหมายความว่าวิธีแก้ปัญหานั้นอาจไม่สามารถทำได้จริงในการผลิต (โดยใช้ เครื่องมือที่คุณมีอยู่) แต่ก็มีประโยชน์สำหรับการสำรวจว่า เครื่องมือใดบ้างที่จำเป็นในการผลิตชิ้นงานที่ซับซ้อนเป็นพิเศษ (นอกจากนี้ โหมดนี้ยังสั่งให้ TecZone Bend สมมติว่ามีชิ้นส่วนในสต็อกไม่จำกัดจำนวน หมายความว่า การจัดชุดเครื่องมือที่เหมาะสมที่สุดจะถูกสร้างขึ้นโดยสมมติว่ามีชิ้นส่วนแต่ละความยาว จำนวนไม่จำกัด)

  • คุณสามารถสร้างชุดเครื่องมือแบบกำหนดเองและบันทึกไว้ในไฟล์รายการเครื่องมือ (btools) ไฟล์เหล่านี้ สามารถใช้ในการตั้งค่าเครื่องมือให้กับชิ้นงานเมื่อคุณเลือกตัวเลือก Use tool-set from File เมื่อคุณเลือกตัวเลือกนี้ ระบบจะให้คุณเลือกไฟล์รายการเครื่องมือ และไฟล์รายการเครื่องมือ ที่คุณใช้ล่าสุดจะแสดงให้เลือกได้ง่าย คุณสามารถคลิกเครื่องหมาย + ใกล้รายการนี้ เพื่อเลือกชุดรายการเครื่องมือใหม่

  • สำหรับการควบคุมหัวเจาะและดายโดยตรงมากที่สุด คุณสามารถเลือกตัวเลือก กำหนดเครื่องมือส่วนบนและส่วนล่าง รวมถึงเลือกหัวเจาะและดายที่ต้องการใช้ได้เอง

ลำดับความสำคัญของลำดับการพับ

เมื่อกำหนดลำดับการพับในชิ้นงาน อาจมีความต้องการบางอย่างที่ขัดแย้งกัน ตัวอย่างเช่น ควร TecZone Bend พยายามลด number of part rotations จำนวนครั้งในการหมุนชิ้นงานหรือควร ลดการพับที่ _ ตำแหน่งจุดศูนย์ถ่วงของชิ้นงานอยู่ภายในเครื่อง_
มีความเป็นไปได้สูงที่ผู้ปฏิบัติงานแต่ละคนจะมีชุดกฎความสำคัญที่แตกต่างกัน และอาจเป็นไปได้ว่าชิ้นงานต่างชนิดกันต้องใช้การตั้งค่าความสำคัญที่แตกต่างกันด้วย ดังนั้น TecZone Bend จึง อนุญาตให้คุณทำการคำนวณใหม่โดยใช้การจัดลำดับความสำคัญแบบอื่น โดยใช้้ ส่วน Sequence Priority ของหน้าต่างนี้

ใช้ปุ่มลูกศร และ เพื่อสลับกฎลำดับ รายการที่อยู่ใกล้ด้านบนจะมีความสำคัญสูงกว่า คุณสามารถ ปิดกฎความสำคัญบางข้อได้ทั้งหมด ซึ่งจะสั่งให้ TecZone Bend ไม่พิจารณา กฎนั้นเมื่อตัดสินใจลำดับ (บางครั้ง การปิดกฎบางข้อ อาจทำให้ได้ลำดับที่ใช้สถานีเครื่องมือน้อยลง)

  • Avoid collisions during overbend: เพื่อให้ได้ความแม่นยำของมุม การพับส่วนใหญ่จำเป็นต้อง_พับเกิน_เล็กน้อยเพื่อชดเชยการคืนตัวของวัสดุ กฎลำดับนี้สั่งให้ TecZone Bend ทำการพับตามลำดับที่สามารถพับเกินได้ ตัวอย่างเช่น ในชิ้นงานด้านล่าง เราควรพับส่วนที่ 1 ก่อน แล้วพับส่วนที่ 2 ต่อไป การทำลำดับตรงข้าม จะทำให้ไม่สามารถพับเกินได้

  • หลีกเลี่ยงเครื่องมือที่มีลำดับความสำคัญต่ำ: สั่ง TecZone Bend ให้หลีกเลี่ยงการใช้เครื่องมือที่มีความสำคัญต่ำ หรือเครื่องมือที่ไม่อยู่ในรายการเครื่องมือที่ต้องการเท่าที่ทำได้

  • หลีกเลี่ยงมุมเปิดชนกัน: ตำแหน่งการวัดควรอยู่ในระนาบเดียวกับ แกนการพับและไม่ควรมีการพับที่ประมวลผลแล้วอยู่ระหว่าง เส้นพับกับจุดวัด ตัวอย่างการวางตำแหน่งวัดที่ไม่ดี ที่กฎนี้พยายามหลีกเลี่ยง:

  • ความยาวของขาที่ยาวเกินออกมาด้านหน้าของเส้นโค้ง: โดยเฉพาะกับชิ้นงานที่มีน้ำหนักมาก ควรหลีกเลี่ยง สถานการณ์ที่จุดศูนย์ถ่วงของชิ้นงานอยู่หลังแนวเครื่องมือ กฎนี้ พยายามลดสถานการณ์ดังกล่าว[3]

  • ลดการใช้ชิ้นส่วนลงให้น่อยที่สุด: Part flips คือเมื่อชิ้นงานต้องกลับด้าน (การพับต่อเนื่อง อยู่ในทิศขึ้น/ลงที่ต่างกัน)

  • ลดการหมุนชิ้นส่วนลงให้น้อยที่สุด: Part rotation คือเมื่อชิ้นงานหมุนไปทางซ้ายหรือขวา ในระนาบแนวนอน ซึ่งถือว่าง่ายกว่าสำหรับผู้ปฏิบัติงานมากกว่า การกลับด้านชิ้นงาน ลำดับเริ่มต้นของสองกฎนี้จะพยายามให้ความสำคัญกับการหมุนชิ้นงาน มากกว่าการกลับด้านเมื่อทำได้

1. ที่ต้องการได้ โดยกดปุ่ม inventory ในแถบคำสั่ง และเลือก Preferred Tools จากเมนู
2. TecZone Bend จะ พยายามใช้เฉพาะเครื่องมือที่ต้องการสำหรับชิ้นงาน แต่ถ้ามีการชนกัน หรือปัญหาอื่นที่แก้ไขไม่ได้ โดยจะเริ่มใช้เครื่องมืออื่นจาก รายการเครื่องมือที่มีอยู่
3. TecZone Bend ใช้วิจารณญาณ เกี่ยวกับน้ำหนักชิ้นงานและระยะที่จุดศูนย์ถ่วงอยู่หลังแนวเครื่องมือ เพื่อพิจารณาว่าเป็นปัญหาหรือไม่ ในทางเทคนิค จะถือว่าชิ้นงานมีแนวโน้มล้มไปด้านหลัง หากมีค่าโมเมนต์การล้ม 0.15 Nm ไปด้านหลัง
