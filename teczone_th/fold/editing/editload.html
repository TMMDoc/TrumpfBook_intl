การแก้ไขการโหลด

การโหลดสำหรับเครื่องดัดแผงอัตโนมัติ (เช่น TBC-7030 หรือ TBC-7020) สามารถแก้ไขได้โดยการคลิกสองครั้งที่เซลล์ Loading ใน navigator นี่ คือเซลล์ที่มีป้ายกำกับว่า L การคลิกครั้งแรกจะเลือกการโหลดและการคลิกครั้งที่สอง จะเปิดแผงโหลด

7030 (Front-Loading)

แผง Loading สำหรับเครื่อง front-loading 7030 มีลักษณะดังนี้:

  • การตั้งค่า Centering angle ระบุการวางแนวของชิ้นงาน ระหว่างการจัดกึ่งกลาง โปรดทราบว่านี่คือด้านที่หันเข้าสู่เครื่อง และจะตรงข้ามกับด้านที่อ้างอิงเกจด้านหน้า ในตัวอย่างนี้ ด้าน C2 หันเข้าสู่เครื่อง (และด้าน C1 ที่ +90° ถูกใช้สำหรับการอ้างอิงเกจ) การตั้งค่านี้จะแสดงรายการการวางแนวที่เหมาะสม สำหรับการจัดกึ่งกลางเริ่มต้น

  • ตัวเลือก Part Pickup ช่วยให้คุณเลือกได้ว่าชิ้นงานจะถูกหยิบขึ้น โดยใช้ถ้วยดูดสุญญากาศหรือใช้คีมจับด้านหลัง (จะแสดง เฉพาะเมื่อมีทั้งสองตัวเลือก) ชิ้นงานบางชิ้นอาจเล็กเกินไปที่จะหยิบด้วย แขนจับแบบดูดและบางชิ้นอาจใหญ่เกินไปที่จะหยิบด้วย rear pliers

  • แถบเลื่อน Simulate ที่ด้านล่างสามารถใช้จำลองขั้นตอนต่างๆ ของกระบวนการจัดกึ่งกลางเบื้องต้น

ส่วน Handling

ส่วนนี้ใช้สำหรับกำหนดตำแหน่งการหมุนและตัวเลือกการจัดการอื่นๆ ระหว่างรอบการจัดกึ่งกลาง

  • การตั้งค่า Rotor (Z, X) ควบคุมตำแหน่งของ rotor (ตำแหน่งของ ศูนย์กลางการหมุน) ซึ่งวัดสัมพันธ์กับจุดกึ่งกลางของชิ้นงานในระหว่าง การจัดกึ่งกลาง

  • คลิก Propagate เพื่อใช้ศูนย์กลางการหมุนเดียวกันในการประมวลผล ทุกด้าน โดยทั่วไป TecZone Fold จะพยายามปรับตำแหน่งศูนย์กลางการหมุน เพื่อลดจำนวนการหมุนซ้ำที่ต้องใช้ อย่างไรก็ตาม หากคุณปรับตำแหน่งการหมุนด้วยตนเอง โดยใช้การตั้งค่าด้านบน คุณสามารถใช้ปุ่มนี้เพื่อทำการปรับแต่งเพิ่มเติมได้

  • การตั้งค่า Speed % ควบคุมความเร็วในการหมุน (เป็นเปอร์เซ็นต์ ของความเร็วสูงสุดที่เป็นไปได้)[1] === ส่วน Gauging ส่วนนี้ช่วยให้คุณสามารถวนดูตำแหน่งเกจต่างๆ ได้

  • การตั้งค่า Auto-Place สามารถใช้เพื่อสลับระหว่างตำแหน่งเกจ ที่ต่างกัน โดยขึ้นอยู่กับรูปทรงของชิ้นงาน อาจมีตำแหน่งหลายจุด ที่ขอบชิ้นงานซึ่งสามารถทำการวัดด้วยเกจแบบมุมหรือแบบเว้าได้ การคลิก ปุ่มนี้ซ้ำๆ จะวนผ่านตัวเลือกทั้งหมดนี้ (เมื่อคุณทำเช่นนี้ คุณจะเห็นตำแหน่งเกจจริงได้รับการอัปเดตในส่วน Fine Position)

  • การตั้งค่า Slack ใช้เพื่อเพิ่มช่องว่างเพิ่มเติม (ในทิศทาง Z) ระหว่างเกจ ซึ่งสามารถใช้เพื่อช่วยการวัดด้วยเกจ ในกรณีที่ชิ้นงานไม่ได้ถูกตัดแต่งตามขนาดพอดี

ส่วน Fine Position

ส่วนนี้ช่วยให้คุณแก้ไขตำแหน่งของเกจด้วยตนเอง โดยทั่วไป ไม่จำเป็นต้องทำ ซึ่งส่วนใหญ่แล้วการเลือกตำแหน่งเกจเริ่มต้น จะเหมาะสมที่สุด และในสถานการณ์อื่น ๆ เพียงคลิกไม่กี่ครั้งที่ปุ่ม Auto Place ที่กล่าวถึงข้างต้นมักจะทำให้เกจถูกจัดวางอย่างถูกต้อง การตั้งค่า Fine Position จะใช้เฉพาะในกรณีที่ TecZone Fold หาตำแหน่งเกจไม่ได้หรือในกรณีที่ชิ้นงานมีรูปทรงซับซ้อน ผิดปกติเท่านั้น

  • การตั้งค่า Left (Z, X) ควบคุมตำแหน่งของเกจด้านซ้าย ตำแหน่ง Z วัดจากเส้นกึ่งกลาง ในขณะที่ตำแหน่ง X วัดจาก ขอบของชิ้นงานในทิศทาง X เมื่อคุณปรับการตั้งค่านี้ เกจจะเคลื่อน และคุณอาจพบการชนกันหรือเกิดข้อผิดพลาด gauge-not-engaged

  • การตั้งค่า Right (Z, X) ควบคุมตำแหน่งของเกจด้านขวา

  • ตัวบ่งชี้ Trimmed ในหน้าจอจะแสดงสลับระหว่าง Trimmed, Untrimmed หรือ Diagonal เพื่อระบุประเภทของการวัดที่ใช้งานอยู่

7030 + KB40 (Front / Rear Loading)

สำหรับเครื่อง 7030 ที่ติดตั้ง KB40 จะมีตัวเลือกเพิ่มเติมในระหว่างกระบวนการโหลด:

  • ใช้การตั้งค่า Stack angle เพื่อเลือกการวางแนวของชิ้นงาน บนพาเลท KB40 จากนั้น คุณสามารถกำหนดโหมดการโหลด (ด้านหน้าหรือด้านหลัง) และมุมการจัดกึ่งกลางสำหรับ stack angle นั้นๆ โปรแกรม NC (XML) ที่สร้างขึ้นโดย TecZone Fold จะมีการตั้งค่า การจัดกึ่งกลางสำหรับแต่ละ stack angle ฝังอยู่ในนั้น โดยขึ้นอยู่กับ stack angle จริง ระบบจะใช้การตั้งค่าที่เหมาะสม คุณสามารถตรวจสอบมุม stack ทั้งหมดที่ถูกโปรแกรมไว้ได้โดยชี้ไปที่เซลล์ Gauging:

  • การตั้งค่า Loading mode ใช้เพื่อเลือกระหว่างโหมด_การจัดกึ่งกลางด้านหลัง_กับ การจัดกึ่งกลางด้านหน้า

  • การตั้งค่า Centering angle ใช้เพื่อเลือกการวางแนว ของชิ้นงานระหว่างการจัดกึ่งกลางเริ่มต้น หากโหมดการโหลดถูกตั้งค่าเป็น front centering มุมการจัดกึ่งกลางจะเหมือนกับ stack angle (เนื่องจากไม่ได้มีการหมุนใดๆ ก่อนที่ชิ้นงานจะถูกดันเข้ากับเกจด้านหน้า) หากโหมดการโหลดถูกตั้งค่าเป็น back centering มุมการจัดกึ่งกลางจะเป็นหนึ่งในสองแบบนี้:

    • stack angle: ชิ้นงานจะถูกจัดกึ่งกลางก่อน จากนั้นถูกดึงกลับ และหมุนไปยังการวางแนวด้านแรก

    • first side angle: ชิ้นงานจะถูกหมุนไปยังการวางแนวเดียว กับด้านแรกก่อนการจัดกึ่งกลาง

ส่วน Handling, Gauging และ Fine Position มีลักษณะคล้ายกับส่วนที่สอดคล้องกันของรุ่น 7030 (ดูส่วนที่กล่าวไว้ด้านบน) แตกต่างกันเพียงว่าอาจอ้างอิงถึงตำแหน่งของ back-gauges หรือ front-gauges (ขึ้นอยู่กับการตั้งค่า Loading mode)

7020 (Ergo-Centering)

ในเครื่องดัดแผง 7020 จะไม่มี front-gauges การจัดกึ่งกลางเริ่มต้น จะทำกับ rear gauges และแผงจะมีลักษณะดังนี้:

การตั้งค่า Centering Angle ใช้เพื่อเลือกการวางแนวของ ชิ้นงานในระหว่างการจัดกึ่งกลางเริ่มต้น ซึ่งอาจแตกต่างจากการวางแนวด้านแรกได้ ในกรณีนี้ ชิ้นงานจะถูกจัดกึ่งกลาง ดึงกลับ และหมุน ก่อนที่จะประมวลผลด้านแรก

ส่วน Ergo-Centering

ส่วนนี้ใช้เพื่อเลือกว่าฟังก์ชัน ergonomic centering จะถูกเปิดใช้งานหรือไม่ และใช้ในการโปรแกรมการจัดกึ่งกลางตามหลักสรีรศาสตร์

  • หากการตั้งค่า Ergonomic load ถูกกำหนดเป็น <No> จะไม่มีการจัดกึ่งกลางตามหลักสรีรศาสตร์ ชิ้นงานจะถูกโหลดด้วย gauges อยู่ที่กึ่งกลาง (ในตำแหน่ง การประมวลผล) หากตั้งค่าเป็น Left หรือ Right จะมีการโปรแกรมรอบการจัดกึ่งกลาง ตามหลักสรีรศาสตร์

  • ตัวเลือก Gripping plier กำหนดว่าคีมจับตัวใดจะถูกใช้สำหรับ _จับ_ชิ้นงานและดึงไปยังกึ่งกลาง (อีกคีมหนึ่ง จะใช้สำหรับอ้างอิงมุมของชิ้นงาน) โดยทั่วไป หากการจัดกึ่งกลางสรีรศาสตร์เกิดขึ้น ที่ด้านซ้าย จะใช้คีม_ด้านขวา_สำหรับจับและในทางกลับกัน

  • การตั้งค่า Z-Right (หรือ Z-Left) ควบคุมตำแหน่ง ของคีมจับที่ใช้งาน เมื่อคุณเลื่อนคีมไปตามชิ้นงาน ค่าตำแหน่ง X ของคีมจะถูกปรับโดยอัตโนมัติ เพื่อให้สามารถจับขอบด้านหน้าของชิ้นงานได้

  • สวิตช์ Regrip with both สามารถใช้เพื่อเพิ่มความมั่นคง ระหว่างการเคลื่อนย้ายชิ้นงานเข้าสู่กึ่งกลางของเครื่องจักร หากเปิดใช้งาน คีมอีกตัวก็จะถูกใช้เพื่อจับขอบของชิ้นงานในขณะที่ถูกดึง เข้าไปในเครื่อง

  • การตั้งค่า Z-Regrip (แสดงเฉพาะเมื่อเปิดสวิตช์ Regrip with both) จะกำหนดตำแหน่งของคีมที่สองเมื่อจับชิ้นงาน

ส่วน Handling

ส่วนนี้เกือบเหมือนกับ handling section ของเครื่อง 7030

  • การตั้งค่า Grip with ใช้เลือกว่าชิ้นงานจะถูกจับด้วย vacuum manipulator (X3) หรือ rotary clamps (X2/X4) ระหว่างรอบการจัดกึ่งกลาง

  • Rotor (Z, X) คือการกำหนดตำแหน่งของ rotary clamp (สัมพันธ์กับจุดกึ่งกลางของ ชิ้นงาน) เมื่อชิ้นงานถูกยึด/หมุนระหว่างรอบการจัดกึ่งกลาง หากด้านนั้น ถูกประมวลผลต่อโดยใช้ rotary clamp (ตามการตั้งค่า Grip with ที่กล่าวไว้ข้างต้น) ตำแหน่งนี้คือตำแหน่งที่คีมจับชิ้นงานในระหว่างการประมวลผลด้านนั้นด้วย

การตั้งค่า Gauging และ Fine Position มีลักษณะคล้ายกับ เครื่อง 7030 (ดูในส่วนที่กล่าวไว้ด้านบน) แต่แตกต่างตรงที่ตอนนี้ควบคุมตำแหน่งของ back-gauges แทนที่จะเป็น front-gauges

1. ความเร็วสูงสุดของการหมุนจะถูกคำนวณจาก โมเมนต์ความเฉื่อยของชิ้นงาน
