การแก้ไขด้าน/เซกชัน

มีการตั้งค่าและการทำงานบางอย่างที่เกี่ยวข้องกับทั้ง_ด้าน_การดัด (ไม่ใช่แค่การดัดเพียง_จุดเดียว_) ซึ่งสามารถเข้าถึงได้โดยการเปิด

Side Panel:

  • เปิดแผง Bend (โดยการดับเบิลคลิกที่หมายเลขใน Fold Navigator) จากนั้น คลิกที่ปุ่ม Side เพื่อเปิด Side panel ของด้าน ที่มีการดัดนั้น

  • กดปุ่ม Ctrl ค้างไว้และคลิกที่หมายเลขใน Fold Navigator เพื่อเปิด Side panel ของด้านนั้น

Side panel จะมีลักษณะเหมือนในภาพที่แสดงด้านข้าง (ชุดของตัวเลือกจริงๆ ที่ปรากฏใน Edit Sequence popup อาจแตกต่างกัน โดยขึ้นอยู่กับชิ้นงาน)

  • คลิกปุ่ม หมุนชิ้นงาน เพื่อพลิกด้านทั้งหมดกลับด้าน ซึ่งจะทำให้ใบมีด, blank-holders และการจัดการชิ้นงานถูกคำนวณใหม่ทันที บนเครื่องอัตโนมัติ ฟังก์ชันนี้จะถูกแทนที่ด้วย หมุนชิ้นส่วน button[1]เนื่องจากเครื่องจักรไม่สามารถพลิกเพียงด้านเดียวได้

  • คลิกที่ปุ่ม แก้ไขลำดับส่วนโค้งงอ เพื่อเปิดเมนูย่อยที่แสดง _การสับ_ลำดับที่สามารถทำได้กับด้านนี้ ในภาพด้านบน เรากำลังทำงานกับชิ้นงานที่มีลำดับด้านเป็น C1 C2 L1 L2 และเราได้เลือกด้าน L1 ตัวเลือกการสับลำดับ ที่เป็นไปได้มีดังนี้:

  • << L1+L2 earlier ย้ายทั้ง L1 และ L2 ให้มาก่อน C1 และ C2 ดังนั้น ลำดับใหม่จะเป็น L1 L2 C1 C2

  • << L1 earlier ย้าย L1 ให้เร็วขึ้นหนึ่งขั้น ลำดับใหม่คือ C1 L1 C2 L2 แม้ลำดับนี้จะไม่พบบ่อยนัก แต่ก็อาจมีประโยชน์ ในบางสถานการณ์

  • L1 later >> ย้าย L1 ไปช้าลง ส่งผลให้ได้ลำดับ C1 C2 L2 L1 (โดยระบบจะรีเนมใหม่ทันที เพื่อให้ลำดับยังคงเป็น C1 C2 L1 L2 เมื่อโค้ดถูกสร้างขึ้น)

ตัวเลือกที่ปรากฏในเมนูนี้ขึ้นอยู่กับลำดับปัจจุบัน และด้านที่คุณเลือกจะจัดเรียงใหม่

  • คลิกปุ่ม ย้อนกลับ หรือ ถัดไป เพื่อเลื่อนดูด้านต่างๆ ของชิ้นงาน

  • คลิกที่ปุ่ม ส่วนโค้งงอ เพื่อเลื่อนลงจากด้าน ไปยังชุดการดัดของด้านนั้น (ตรงข้ามกับปุ่ม ด้านข้าง ใน Bend panel ที่เลื่อนขึ้น จากการดัดไปยังด้านที่ครอบการดัดนั้น)

1. ซึ่งมีประโยชน์จำกัดกว่า ปุ่ม Flip Side ที่ใช้บนเครื่องแมนนวลและมีไว้สำหรับจัดการ ในบางกรณีเท่านั้น
