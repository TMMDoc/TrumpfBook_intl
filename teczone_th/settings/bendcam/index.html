Bend CAM

แกนตัดด้านหลัง

การหดกลับโดยอัตโนมัติ หากแผงมีความหนาตั้งแต่ … มม. - การตั้งค่านี้ อนุญาตให้คุณป้อนค่า (มม.) สำหรับการถอยกลับอัตโนมัติของ backgauge เมื่อถึงค่าความหนาของแผ่นที่ป้อน

กลไกแกนตัดด้านหลัง - ใช้สำหรับตั้งค่าการหยุดของ backgauge บนชิ้นงาน

ตัวเลือกที่สามารถตั้งค่าได้ ได้แก่:

มาตรฐาน - backgauge หยุดสำหรับแต่ละการพับ ตามที่ซอฟต์แวร์เห็นว่าเหมาะสมที่สุด

แกนตัดตรง 2 x - พื้นผิวตรงของ backgauge ทั้งสองหยุดที่ชิ้นงาน

จุดยึด+จุดหยุด - backgauge ตัวหนึ่งจับยึดชิ้นงาน และอีกตัวหยุดชิ้นงานด้วยพื้นผิวตรง

การวัดค่ามุม

การวัดค่ามุม ใช้ได้เฉพาะกับเครื่อง TRUMPF เท่านั้น การใช้ระบบวัดมุมจากตระกูล Automatically Controlled Bending หมายความว่า แม้แต่ชิ้นแรกในชุดก็จะออกมาสมบูรณ์ เซ็นเซอร์จะตรวจจับมุมจริงพร้อมกับ spring back และควบคุม ram เพื่อให้ได้มุมที่ต้องการ ในส่วนนี้ สามารถป้อนการตั้งค่าสำหรับ ACB บนเครื่องจักรได้

ระบบการวัดค่ามุม

เลเซอร์ - เลือกตัวเลือกนี้หากเครื่องมี ACB Laser ACB Laser คือระบบที่เลเซอร์ยิงเส้นลงบนแผ่นโลหะ และกล้องตรวจจับ มุมนั้น

ไร้สาย - เลือกตัวเลือกนี้หากเครื่องจักรใช้ ACB Wireless ACB Wireless คือดิสก์เซ็นเซอร์ที่ติดตั้งในเครื่องมือด้านบน

คลาสสิค - เลือกตัวเลือกนี้หากเครื่องจักรใช้ ACB Classic

การวัดค่ามุม 2 จุด หากแผงมีความหนาตั้งแต่ … มม. - ตัวเลือกนี้ เป็นการกำหนดความยาวขั้นต่ำของแนวพับเพื่อทำการวัดสองครั้ง เพื่อให้มั่นใจว่า มุมที่พับถูกต้อง ซึ่งสามารถกำหนดได้ระหว่าง 50 ถึง 4000 มิลลิเมตร

การวัดค่ามุม 3 จุด หากแผงมีความหนาตั้งแต่ … มม. - ตัวเลือกนี้ เป็นการกำหนดความยาวขั้นต่ำของแนวพับเพื่อทำการวัดสามครั้ง เพื่อให้มั่นใจว่า มุมที่พับถูกต้อง ซึ่งสามารถกำหนดได้ระหว่าง 50 ถึง 4000 มิลลิเมตร

เลเซอร์ ACB ในตำแหน่งจอดรถ (ต้องรีสตาร์ท!) - ตำแหน่งพักของ Laser Sensor์

ขีดจำกัดด้านคุณภาพสำหรับ ACB - คำเตือน - ค่านี้เป็นค่าขีดจำกัด ที่จะสร้างคำเตือนเมื่อทำการวัดชิ้นงาน ซึ่งใช้เป็น fudge factor หาก ค่าที่วัดได้อยู่ในช่วงที่กำหนด ระบบจะแจ้งเตือน

ขีดจำกัดด้านคุณภาพสำหรับ ACB - ความผิดปกติ - ค่านี้เป็นค่าขีดจำกัด ที่จะสร้าง error เมื่อทำการวัดชิ้นงาน ซึ่งใช้เป็น fudge factor หากค่าที่วัดได้ อยู่ในช่วงที่กำหนด ระบบจะรายงานข้อผิดพลาด

การแก้ไขเพื่อให้เข้ากับชิ้นงาน

หากไฟล์ 2D หรือ 3D ถูกแก้ไขในซอฟต์แวร์ รัศมีการพับและการหักพับ จะถูกตรวจสอบ กลยุทธ์การประมวลผลต่างๆ จะถูกดำเนินการตามที่เลือกไว้ใน Easy Programming settings สำหรับการแก้ไขชิ้นงาน หาก การตัดสินใจถูกตั้งค่าเป็น interactive ระบบจะแสดง กล่องโต้ตอบ พอดีกับชิ้นงาน

นำเข้าผลลัพธ์

การตัดสินใจพร้อมดำเนินงาน - เมื่อเปลี่ยนจากชิ้นงาน 2D ไปเป็นการพับ หากเปิดใช้งานตัวเลือกนี้ ระบบจะแสดงกล่องโต้ตอบ พอดีกับชิ้นงาน โดยอัตโนมัติ ซึ่งจะให้ทางเลือก เช่น เก็บขนาดแผ่น, เก็บขนาด 3D หรือไม่เปลี่ยนแปลงใดๆ ทำให้ฟีเจอร์นี้ใช้งานได้ง่ายและ มีปฏิสัมพันธ์สูง

ประมวลผลโมเดล 3D - ตัวเลือกนี้เปลี่ยนการคลี่แผ่น ซึ่งจะต้องบันทึกด้วยตนเอง กลยุทธ์นี้ใช้หากแผ่นโลหะ ยังไม่ถูกตัดตามขนาด

ประมวลผลลัพธ์ - ตัวเลือกนี้หมายความว่า รัศมีการพับและการหักพับจะถูกปรับตามการคลี่แผ่นที่มีอยู่ และแนวพับยังคงอยู่ในตำแหน่งเดิม ซึ่งจะเปลี่ยนความยาวด้านข้างของ ชิ้นงาน การคลี่แผ่นจะไม่ถูกเปลี่ยนแปลง กลยุทธ์นี้ใช้หากแผ่นโลหะ ถูกตัดตามขนาดแล้ว

ไม่มีการปรับค่าใดๆ - การใช้ตัวเลือกนี้หมายความว่า รัศมีการพับและการหักพับยังคงไม่เปลี่ยนแปลง ชิ้นงานจะแสดงผลด้วย มิติที่ถูกต้องใน 3D แต่รัศมีการพับที่แสดงอาจไม่สอดคล้องกับ รัศมีจริง มิติของการพับแต่ละชิ้นงานสามารถเปลี่ยนแปลงได้

นำเข้าโมเดล 3D

การตัดสินใจพร้อมดำเนินงาน - เมื่อเปลี่ยนจากชิ้นงาน 3D ไปเป็นการพับ หากเปิดใช้งานตัวเลือกนี้ ระบบจะแสดงกล่องโต้ตอบ พอดีกับชิ้นงาน โดยอัตโนมัติ ซึ่งจะให้ทางเลือก เช่น เก็บขนาดแผ่น, เก็บขนาด 3D หรือไม่เปลี่ยนแปลงใดๆ ทำให้ฟีเจอร์นี้ใช้งานได้ง่ายและ มีปฏิสัมพันธ์สูง

ประมวลผลโมเดล 3D - ตัวเลือกนี้เปลี่ยนการคลี่แผ่น ซึ่งจะต้องบันทึกด้วยตนเอง กลยุทธ์นี้ใช้หากแผ่นโลหะ ยังไม่ถูกตัดตามขนาด

ประมวลผลลัพธ์ - ตัวเลือกนี้หมายความว่า รัศมีการพับและการหักพับจะถูกปรับตามการคลี่แผ่นที่มีอยู่ และแนวพับยังคงอยู่ในตำแหน่งเดิม ซึ่งจะเปลี่ยนความยาวด้านข้างของ ชิ้นงาน กลยุทธ์นี้ใช้หากแผ่นโลหะถูกตัดตามขนาดแล้ว

ค่าการย่อให้สั้นลง - ตัวเลือกนี้ทำให้สามารถ บังคับใช้ วิธี K-Factor ได้ การตั้งค่านี้สามารถกำหนดใน Application Defaults หรือปรับแต่งสำหรับเครื่องจักรแต่ละเครื่องได้

สำหรับเครื่องที่ไม่ใช่ TRUMPF ค่า K-Factor จะถูกใช้เสมอ แต่การคำนวณการพับของ TRUMPF มีให้เฉพาะเครื่อง เครื่องพับโลหะของ TRUMPF เท่านั้น

รุ่น HA ด้านซ้าย - ขึ้นอยู่กับรุ่น BG ที่ใช้ โปรแกรม NC จะถูกแก้ไขเพื่อให้สอดคล้องกับสิ่งนี้

รุ่น HA ด้านขวา - ขึ้นอยู่กับรุ่น BG ที่ใช้ โปรแกรม NC จะถูกแก้ไขเพื่อให้สอดคล้องกับสิ่งนี้

การตั้งค่าเพิ่มเติม

มุมส่วนโค้งงอเมื่อพับ - ตัวเลือกนี้สามารถแก้ไข มุมพับขอบก่อนพับระหว่าง 10° ถึง 60° ค่าเริ่มต้นคือ 40° เนื่องจากเป็น มุมพับขอบก่อนพับที่ผู้ผลิตส่วนใหญ่แนะนำ สำหรับเครื่อง TRUMPF ทุกเครื่อง ในทุกรุ่น ค่าเริ่มต้นถูกตั้งไว้ที่ 30°

โหมด Bendguard ที่ตั้งไว้ล่วงหน้า - หากตั้งค่าเป็น auto-compute (ค่าเริ่มต้น) ระบบจะคำนวณโหมด bend-guard สำหรับแต่ละการพับโดยอัตโนมัติ อย่างไรก็ตาม คุณสามารถบังคับโหมด bend-guard ให้ใช้ตัวเลือกใดตัวเลือกหนึ่งได้ โดยไม่สนใจว่าเหมาะสมหรือไม่

คุณยังสามารถ override การตั้งค่านี้สำหรับแต่ละเครื่อง โดยตั้งค่าใน Machine Defaults แทนที่จะเป็น การตั้งค่าการใช้งาน

ตำแหน่งโต๊ะเครื่องจักร - ตัวเลือกนี้มีสามค่าสำหรับ ตำแหน่งของตาราง:

  1. นำออก

  2. ลง

  3. ขึ้น

เช็คปริมาณ - การเปิดใช้งานตัวเลือกนี้จะแสดงส่วนของเครื่องมือ ที่อาจจำเป็นสำหรับการพับชิ้นงาน ตัวบ่งชี้สต็อกที่ไม่เพียงพอ จะปรากฏขึ้น

หลีกเลี่ยงการโอเวอร์โหลดเครื่องมือถ้าเป็นไปได้ - การเปิดใช้งาน ตัวเลือกนี้จะช่วยป้องกันไม่ให้เครื่องมือเกิดการใช้งานเกินกำลัง แม้ว่าเครื่องมือที่มีลำดับความสำคัญสูงกว่า จะสามารถทำงานได้ แต่เครื่องมือที่มีลำดับความสำคัญต่ำกว่าจะถูกใช้แทนเพื่อหลีกเลี่ยง ปัญหาการใช้งานเกินกำลัง

ไม่รวมเครื่องมือจากไฟล์ geo/dxf - การเปิดใช้งาน ตัวเลือกนี้จะเพิกเฉยต่อข้อมูลใดๆ ที่พบในไฟล์ DXF/GEO เกี่ยวกับ เครื่องมือที่จะถูกใช้งาน
