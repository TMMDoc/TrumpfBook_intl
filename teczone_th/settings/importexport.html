Import/Export

ในส่วนนี้จะครอบคลุมการตั้งค่าของ Import settings

คลิกที่ไอคอน Settings จากหน้าแรก

Import settings

ในส่วนนี้จะครอบคลุมการตั้งค่าของ Import settings คลิกที่ไอคอน Settings คลิก Import/Export และนำทางไปยัง Import settings

Units for DXF files - ตั้งค่าเป็นมิลลิเมตรหรือนิ้ว

Stitch together lines/arcs closer than this - ตั้งค่า ค่านี้ (≥0–1 มม.) เมื่อมีการนำเข้าชิ้นงานที่มีเส้น/ส่วนโค้งใกล้กว่าค่าที่กำหนด ซอฟต์แวร์จะทำการเชื่อมต่อโดยอัตโนมัติขณะนำเข้า

Maximum thickness for sheet-metal part - เพื่อให้สามารถ จดจำชิ้นงานแผ่นโลหะขนาดใหญ่ได้ จะต้องเพิ่มค่า threshold สำหรับการจดจำ แผ่นโลหะ (10>40 มม.) โดยค่าเริ่มต้นจะถูกตั้งไว้ที่ 25 มม. หรือ 1 นิ้ว โดยขึ้นอยู่กับหน่วยที่เลือกใช้

Point entities - ขึ้นอยู่กับตัวเลือกที่ตั้ง จะกำหนดว่าข้อมูลจุดจะถูกนำเข้าอย่างไร

Import all - นำเข้าจุดทั้งหมด และแสดงผล

Skip points on polylines - ข้าม จุดทั้งหมดที่ตรวจพบใน polylines

Skip all - ข้ามจุดทั้งหมดในการนำเข้า และจะไม่มีการแสดงผลใดๆ

Ignore layers in DXF/DWG files - แบบร่าง DXF และ DWG มักถูกสร้างในหลาย layer ด้วยการตั้งค่านี้ ซอฟต์แวร์จะละเว้น layer เหล่านี้และย้ายวัตถุทั้งหมดไปยัง layer เริ่มต้น

Explode blocks in 2D drawing - แยกกลุ่มระหว่าง การนำเข้า

Convert white entities to black - เปิดใช้งานตัวเลือกนี้ เพื่อแปลงวัตถุสีขาวให้เป็นวัตถุดำขณะนำเข้า

Darken colors during DXF import- เปิดใช้งานตัวเลือกนี้ เพื่อทำให้วัตถุที่มีสีเข้มขึ้นเมื่อทำการนำเข้า

Remove duplicate segments - เปิดใช้งานตัวเลือกนี้ เพื่อลบ geometry ที่ซ้ำกันในชิ้นงานเมื่อทำการนำเข้า

DXF settings

ในส่วนนี้จะครอบคลุมการตั้งค่าของ DXF settings คลิกที่ ไอคอน Settings คลิก Import/Export และนำทาง ไปยัง DXF settings

Angles in DXF are interior angles - เปิดใช้งาน การตั้งค่านี้เพื่อให้มุมในไฟล์ DXF ถูกจัดการเป็นมุมเปิด

Export settings

ในส่วนนี้จะครอบคลุมการตั้งค่าของ Export settings คลิกที่ ไอคอน Settings คลิก Import/Export และ นำทางไปยัง Export settings

No POLYLINE objects in DXF output - โดยทั่วไป contour แบบปิด จะถูกส่งออกเป็น polylines เมื่อทำการ export DXF CAD บางระบบไม่สามารถ ประมวลผลผลลัพธ์นี้ได้ ด้วยการตั้งค่านี้ ซอฟต์แวร์จะส่งออก DXF พร้อมเส้นตรง และส่วนโค้ง ไฟล์นี้สามารถอ่านได้ทุกที่ แต่จะมีขนาดใหญ่กว่า และการเชื่อมต่อระหว่างเส้นและส่วนโค้งจะสูญหาย

Output bend-info when saving DXF files - เปิดใช้งานการตั้งค่านี้ เพื่อให้ไฟล์ DXF ที่ส่งออกมีข้อมูลการพับ

Bend-info in Starmatik format - ปิดใช้งาน switch นี้ เพื่อส่งออกข้อมูลการพับในรูปแบบ Starmatik ตำแหน่งนี้คือจุดที่เอนทิตีข้อความ จะถูกวางไว้ตรงกลางของเส้นแต่ละเส้นที่เป็นเส้นพับ

Convert black to gray on output - เมื่อส่งออกข้อมูล 2D วัตถุในไฟล์ DXF จะถูกส่งออกเป็นสีดำโดยค่าเริ่มต้น เพื่อให้ โปรแกรม CAD สามารถจดจำวัตถุได้ดีขึ้น วัตถุจะถูกส่งออก เป็นสีเทาด้วยการตั้งค่านี้

Start MetaCAM when PDG files are exported - เปิดใช้งาน การตั้งค่านี้สำหรับไฟล์ PDG เพื่อกำหนดค่าโดยอัตโนมัติให้เปิดใน MetaCAM

Format for flat pattern - เมื่อทำการส่งออกรูปแบบแบน สามารถกำหนดให้ออกเป็นไฟล์ GEO, DXF หรือ PDG ได้

Spline conversion

ในส่วนนี้จะครอบคลุมการตั้งค่าของ การตั้งค่า Spline conversion คลิกที่ไอคอน Settings คลิกที่ Import/Export และนำทางไปยังการตั้งค่า Spline conversion

Convert splines on import - ตั้งค่าที่นี่ว่าจะปิดการแปลง spline หรือจะแปลง spline ให้เป็นเส้นตรงหรือส่วนโค้ง ไม่ว่าจะเลือกแบบใด spline แต่ละเส้นจะถูกแปลงเป็น polyline เดียวที่ประกอบด้วย segment แบบเส้นตรงหรือโค้ง

Node-count computation - จำนวนของเส้นหรือส่วนโค้ง ที่จะถูกสร้างขึ้นถูกคำนวณจากหนึ่งในสองวิธี: pitch หรือ deviation

Length of each line or arc segment - หากเลือก Pitch ในการคำนวณแบบ Node-Count ให้ตั้งค่าความยาวของเส้นโค้งหรือเส้นตรงแต่ละ segment เพื่อแบ่ง spline ตามความยาวที่กำหนด

Maximum deviation during approximation - หาก เลือก Deviation ในการคำนวณแบบ Node-Count ให้ตั้งค่า deviation สูงสุดที่อนุญาตระหว่าง spline ดั้งเดิมกับเส้นตรง หรือเส้นโค้งที่ประมาณค่า polyline จะถูกสร้างขึ้นด้วยจำนวน segment ที่น้อยที่สุด ขณะเดียวกันยังรักษาค่าความผิดพลาดสูงสุดให้อยู่ในขอบเขตที่กำหนด

Assembly Explode

ในส่วนนี้จะครอบคลุมการกำหนดค่าของการตั้งค่า Assembly Explode คลิกที่ไอคอน Settings คลิกที่ Import/Export และนำทางไปยังการตั้งค่า Assembly Explode

Skip sheet-metal parts with no bends - เมื่อทำการ explode assembly หากเปิดใช้งานตัวเลือกนี้ จะแสดงเฉพาะชิ้นส่วนแผ่นโลหะที่มีเส้นพับ การปิดใช้งานจะแสดงชิ้นส่วนทั้งหมดใน assembly

Components (nuts/bolts etc) to display - ใช้การตั้งค่านี้ เพื่อเลือกตัวเลือกที่จะแสดงเมื่อทำการ explode assembly ที่มีชิ้นส่วนอื่นรวมอยู่ด้วย

None - แสดงเฉพาะชิ้นส่วนแผ่นโลหะหลังจาก explode และจะไม่แสดง nuts/bolts อื่นๆ

Unidentified - แสดงเฉพาะชิ้นส่วน ที่ซอฟต์แวร์ยังไม่สามารถระบุได้

All - แสดงชิ้นส่วนทั้งหมด

Layer mapping

ในส่วนนี้จะครอบคลุมการตั้งค่าของ Layer mapping คลิกที่ ไอคอน Settings คลิกที่ Import/Export และนำทางไปยัง การตั้งค่า Layer mapping

ในส่วนนี้ layer ที่ใช้ในชิ้นงานที่ถูกนำเข้าใน ซอฟต์แวร์สามารถถูก map โดยอัตโนมัติไปยังการใช้งานที่เกี่ยวข้อง

ตัวอย่าง:

หากชิ้นงานที่นำเข้ามี MARK layer จะสามารถตั้งค่า ให้ใช้ Mark layer ในซอฟต์แวร์โดยอัตโนมัติได้

Layer name - ชื่อนี้คือชื่อ layer ที่หากชิ้นงาน ถูกนำเข้ามาพร้อมกับ layer นี้ จะใช้ functionality ที่ตั้งค่าไว้ในแผง ‘Use’

Use - การทำงานของ layer นี้ ตัวเลือกที่แตกต่างกันที่มีให้ ได้แก่:

Standard - Layer มาตรฐานที่ใช้สำหรับ CAM

Auxiliary - Layer เสริมที่ไม่ใช้สำหรับ CAM

Mark - เอนทิตีใดๆ ใน layer นี้จะถูก mark แต่ไม่ถูกตัด

Approach marker - เอนทิตีจุดที่ระบุ ตำแหน่งเข้าใกล้ของเลเซอร์

Sequence marker - ข้อความที่ใช้เป็นเครื่องหมายระบุ ลำดับของเส้นรอบรูป

Forming center - จุดกึ่งกลางสำหรับการขึ้นรูป (point หรือ L เล็ก)

Forming foot print - เส้นรอบนอก (footprint) ของ การขึ้นรูป

Evaporate - Layer นี้จะใช้สำหรับระบุ การเผาฟิล์ม

Dot marking - Layer นี้จะใช้สำหรับ QR codes

Info - Layer ที่ใช้เพื่อข้อมูลเท่านั้น
