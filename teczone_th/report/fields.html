ฟิลด์ รูปภาพ และตาราง

การคลิกที่แหล่งข้อมูลจะเปิดและแสดงปุ่มคลิก เพิ่มเติม ซึ่งสามารถใช้ในการเพิ่ม fields, pictures และ tables ลงในรายงานได้

การเพิ่มฟิลด์

ฟิลด์คือการแสดงค่าข้อความหรือตัวเลขจากรายงาน ตัวอย่างเช่น เมื่อคลิก Part > Info > Name คุณสามารถแทรกฟิลด์ในรายงานเพื่อแสดง วัสดุของชิ้นงานได้:

เมื่อทำเช่นนี้ TecZone Bend จะเพิ่ม_สอง_รายการเข้าไปในรายงาน - Field ที่แสดงค่า และ Label ที่อยู่ด้านซ้ายของฟิลด์นั้น คุณสามารถแก้ไข การตั้งค่าสำหรับ label หรือฟิลด์ได้ เพียงคลิก

การแก้ไข Label

เมื่อคุณคลิกที่ label ที่เพิ่งถูกเพิ่ม ระบบจะแสดงแผง สำหรับแก้ไขการตั้งค่าต่างๆ ของ label label จะถูกเน้น และมี sizing handles แสดงรอบๆ เพื่อให้คุณสามารถปรับขนาดได้ โดยการลาก คุณยังสามารถคลิกขวาที่ label และลากเพื่อย้ายตำแหน่งใหม่ ได้ เมื่อคุณลากเพื่อย้ายหรือปรับขนาด ระบบจะแสดง snap indicators โดยอัตโนมัติ เพื่อช่วยให้จัดตำแหน่ง label ให้ตรงกับองค์ประกอบอื่นๆ ในรายงาน

แผงแก้ไขสำหรับ label (และสำหรับองค์ประกอบรายงานอื่นๆ ส่วนใหญ่) จะมี ตัวเลือกจำนวนมาก เพื่อปรับแต่งการแสดงผล คุณสามารถ ดูตัวเลือกเหล่านี้ได้โดยการเปิดกลุ่มที่ขยายออกมา:

ภาพด้านบนแสดง color picker ที่ใช้สำหรับเลือกสีข้อความ ของ label

การแทรกรูปภาพ

*รูปภาพ*ในรายงานจะถูกระบุด้วยไอคอน_กล้อง_ (แทนที่ไอคอน_ดินสอ_ ที่ใช้สำหรับฟิลด์ทั่วไป) ตัวอย่างเช่น เมื่อคลิก Part > Model > Picture คุณสามารถแทรกรูปภาพของโมเดล 3D ของชิ้นงานได้:

ภาพด้านบนแสดงให้เห็นรูปภาพที่ถูกแทรกเข้าไปแล้วและกำลังถูกแก้ไขโดยการคลิก แผงสำหรับรูปภาพจะแสดงการตั้งค่าที่ใช้ควบคุมรูปภาพ การตั้งค่า นี้อาจเปลี่ยนไปตามรูปภาพที่ถูกเพิ่ม ในกรณีนี้ การตั้งค่า DPI ใช้ควบคุมความละเอียดการเรนเดอร์ของ รูปภาพ (ค่าที่สูงกว่าจะสวยกว่า แต่จะทำให้รายงาน มีขนาดใหญ่ขึ้น) ตัวเลือก No 3D PDF ใช้เพื่อปิดการสร้างไฟล์ 3D PDF [1]

หากคุณแทรกรูปภาพของแบบร่าง 2D (แบบแบน) ตัวเลือกของรูปภาพจะแตกต่างออกไป ดังที่เห็นในภาพด้านล่าง:

[Picture 2](img/picture2.png)

การตั้งค่าที่นี่ใช้สำหรับควบคุมความกว้างของเส้นที่ใช้สำหรับเส้นขอบและสำหรับมิติ ยังมีการตั้งค่าเพื่อแสดงหมายเลขลำดับการพับ และควบคุมว่า รูปร่างปิดของชิ้นงานจะถูก filled หรือไม่

การแทรกตาราง

คุณสามารถแทรก*ตาราง*ลงในรายงาน โดยเลือกจากไอคอนที่แสดง สัญลักษณ์_ตาราง_ ตัวอย่างเช่น Part > Bend > Bends table ใช้สำหรับแสดง ข้อมูลทั้งหมดเกี่ยวกับการพับในชิ้นงาน:

Pick Columns ใช้สำหรับเลือกคอลัมน์ที่จะถูกแสดงใน ตาราง (จากคอลัมน์ทั้งหมดที่มีให้เลือกในข้อมูลนี้) คุณสามารถแก้ไข แอตทริบิวต์เพิ่มเติมของคอลัมน์ได้ โดยคลิกที่ลิงก์ Column ด้านล่าง ของแผงตาราง:

Column editor จะปรากฏขึ้น เพื่อให้แก้ไขคุณสมบัติ ของคอลัมน์ได้:

ฟิลด์ Title ใช้สำหรับกำหนดชื่อที่แสดงของคอลัมน์ ส่วนฟิลด์ Expression คือข้อมูล TecZone Bend ภายในที่จะแสดงในคอลัมน์นั้น คุณสามารถใช้ปุ่ม Shift เพื่อย้ายคอลัมน์ไปทางซ้ายหรือขวา และใช้ปุ่ม Insert และ Delete เพื่อเพิ่มหรือลบคอลัมน์

1. TecZone Bend สามารถเรนเดอร์เป็นภาพที่ฝังอยู่ใน 3D PDF และซูม หมุน หรือเลื่อนได้ใน PDF viewer บางตัว ออย่างไรก็ตาม ไม่ใช่ทุก PDF viewer ที่รองรับและบางตัวอาจแสดงเป็น กล่องว่าง
