Anpassade luftspalttabeller

Anpassade luftspalttabeller

Luftspalttabellen för en svängbockningsmaskin mappar en viss plåttjocklek till den standardluftspalt som används under bearbetningen. Denna är inbyggd i alla olika svängbockningsmaskiner, men kan åsidosättas genom att ange en anpassad luftspalttabell i form av en JSON-fil.Denna JSON-fil heter air-gap-tables.json och placeras i C:\ProgramData\Trumpf{APPNAME}\Data\System.Nedan visas en typisk air-gap-tables.json-fil:

 {
  "version":1  "airGapTables": [
    {
      "machine": "112002,112003", "material": "DC01"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
     {
      "machine": "112003", "material": "*"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
    {
      "machine": "*", "material": "Alluminio"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    }
  ]
}

  • Filen innehåller en samling luftspaltsvärden. Varje tabell har en maskin- och materialnyckel som fungerar som väljare.

  • Dessa nycklar är antingen inställda på * (asterisk) för att ange ett jokertecken (matchar allt), eller så är de en kommaseparerad lista med maskin- eller materialvärden.

  • I exemplet ovan finns det tre tabeller. Den första används för maskinerna 112001 eller 112003, endast för materialet DC01. Den andra används för alla material, för maskinen 112003, och den tredje används för alla maskiner, för materialet Alluminio.Om flera tabeller kan matcha delens maskin-/materialkombination används den första matchande tabellen. För maskin 112003 och material DC01 används alltså den första tabellen (även om den andra också matchar).

  • Varje tabell består av en enkel matris med tjockleksnycklar och motsvarande luftspaltsvärden. Till exempel ska den första posten ovan tolkas som: om tjockleken är mindre än eller lika med 0,75 mm, använd ett mellanrum på 2,5 mm. Tjockleksnycklarna måste vara i stigande ordning. Om ingen av tabellerna i den här filen matchar den aktuella delens material- och tjocklekskombination, används standardtabellen för luftgap för maskinen.

Om du gör några ändringar i filen air-gap-tables.json måste du starta om TecZone Fold innan ändringarna träder i kraft.
Om JSON-filen är felaktigt formaterad visas ett felmeddelande när du försöker bearbeta en svängbockningsdel, men TecZone Fold faller då tillbaka till att använda standardtabellen för luftspalt för maskinen och fortsätter.
