Bockningshjälpmedel

Bockningshjälpmedel

Bockningshjälpmedel är komponenter i kantpressmaskiner som hjälper till att bocka tjockare plåtmetaller. De är utformade för att ge ytterligare stöd och kontroll under bockningsprocessen, vilket säkerställer exakta och konsekventa resultat.

En kantpress kan ha flera bockningshjälpmedel eller ett enda bockningshjälpmedel installerat, beroende på de specifika kraven för bockningsoperationen.

Bockningshjälpmedel är endast tillämpliga för TruBend 5000 och 8000-serien.

Konfigurera bockningshjälpmedel

Bockningshjälpmedel kan konfigureras i Database / Machines / Settings (Databas / Maskiner / Inställningar).

När en del importeras och en maskin med bockningshjälpmedel väljs, konfigurerar TecZone Bend automatiskt bockningshjälpmedlen baserat på delens krav.

Använda bockningshjälpmedel

När en genomförbar bockningslösning har genererats kan bockningshjälpmedlen ytterligare justeras för att optimera bockningsprocessen.

Klicka direkt på bockningshjälpmedlet i TecZone Bend för att öppna panelen Bend Aid (Bockningshjälpmedel). Numret i bockningshjälpmedelspanelen representerar bockningssekvensen.

  • Index - Anger det för närvarande valda bockningshjälpmedlet.

  • Position (Z,R) - Används för att justera positionen för bockningshjälpmedlet längs Z- och R-axlarna.

  • Enabled (Aktiverad) - Markera denna kryssruta för att aktivera bockningshjälpmedlet.

  • Auto-Place (Automatisk placering) - Används för att automatiskt positionera bockningshjälpmedlet baserat på delens geometri.

Speed (Hastighet):

  • Down (Ned) - Används för att justera indragningshastigheten för bockningshjälpmedlet efter bockningsoperationen.

  • Set All Bends (Ställ in alla bockningar) - Används för att tillämpa den aktuella hastighetsinställningen på alla bockningar i sekvensen.

Advanced (Avancerat):

  • Method (Metod) - Används för att definiera driftläget för bockningshjälpmedlet. Alternativen inkluderar:

    • Default (Standard) - Återställer bockningshjälpmedlens inställningar till standardvärden.

    • Return at UDP (Återgå vid UDP) - Konfigurerar bockningshjälpmedlet att återgå till en användardefinierad position efter varje bockning.

    • Tilt product (Luta produkt) - Gör det möjligt för bockningshjälpmedlet att luta delen under bockningsprocessen för bättre justering.

    • Static angle (Statisk vinkel) - Tillåter inställning av en fast vinkel för bockningshjälpmedlet under drift.

  • Stop angle (Stoppvinkel) - Används för att ställa in vinkeln vid vilken bockningshjälpmedlet kommer att stanna under drift.

Parkeringsposition för bockningshjälpmedel

Bockningshjälpmedel kan parkeras i en utsedd position när de inte används. Detta hjälper till att optimera arbetsytan och säkerställer säkerhet under drift. Detta förhindrar också kollisioner med andra maskinkomponenter.

Bockningshjälpmedel kan parkeras i följande positioner:

  1. Default (Standard) - Bockningshjälpmedel kan parkeras till bordslängden.

  2. Small (Liten) - Bockningshjälpmedel kan placeras 700 mm bortom bordslängden.

  3. Large (Stor) - Bockningshjälpmedel kan placeras 1200 mm bortom bordslängden.

För tandem-maskiner finns fyra bockningshjälpmedel tillgängliga. För vänster tandem- maskiner är endast parkering på vänster sida tillåten. För höger tandem-maskiner är endast parkering på höger sida tillåten.

Alternativet parkeringsposition visas endast för TruBend 8000-serien och VarioPress-maskiner.

För B36 tandem-maskiner är parkeringspositionen small (liten) inte tillämplig.

Skapa anpassade bockningshjälpmedel

För att skapa anpassade bockningshjälpmedel i TecZone Bend måste du först skapa en ini-fil som definierar bockningshjälpmedlets parametrar och mesh-filer för bockningshjälpmedelskomponenterna (Carriage, Table och Plate).

Exempel på baid.ini-fil
[BAData]
; The user defined name of the BendAid. Only used for custom bend aids
Name=BATest
; The range of V-widths the bend aid can support
DieV=16,250
; The range of die-heights the bend aid can support
DieHeight=-30,250
; Maximum angle, in degrees
MaxAngle=55
; Lift capacity, in kg
Payload=300
; Maximum lift speed, in degrees/second
Speed=45
; When the bending aid is at X=0, this is the bound of the support table in top view
Table=-159,-1032,159,-245.6
; The X-Span of the bending aid (when positioned at 0)
XExtent=-213,213
; Tells whether the user can change the forward speed
CanChangeForwardSpeed=False
; Tells whether safety stop angle is possible in the machine
CanStopAtReturn=True
; Can the bending aid movement be varied using different methods?
AdvancedControl=True
; What type of existing bending aid can this be mapped to? Used only by custom bend aids
EType=TB8000_2000N_B36

[Carriage]
Model=Carriage.mesh
Stroke=-1000,1000
Color=White
Shift=0,0,100

[Table]
Model=Table.Mesh
Stroke=50,200
Color=White

[Plate]
Model=Plate.mesh
Stroke=0,56
Color=#4A4A4A

Placera ini-filen och motsvarande mesh-filer i en mapp som sedan kan importeras till TecZone Bend.

Importera anpassade bockningshjälpmedel

För att importera anpassade bockningshjälpmedel:

  1. Navigera till File / Import / Bending Aids…​ (Fil / Importera / Bockningshjälpmedel…​) och välj den redan skapade ini-filen (se till att motsvarande mesh-filer finns i samma mapp).

  2. De importerade bockningshjälpmedlen kommer sedan att vara tillgängliga i bockningshjälpmedelsvalslistan i maskininställningarna.

Detta alternativ är användbart när ett bockningshjälpmedel behöver anpassas för specifika bockningsuppgifter eller vid användning av specialiserade bockningshjälpmedel som inte ingår i standard- TecZone Bend-databasen.
