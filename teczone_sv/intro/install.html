Installera programvaran

Installera programvaran

Förberedelser inför installationen

Systemkrav

  • Ett av följande operativsystem. Observera att Windows 7 kräver att Service Pack 1 är installerat, annars fungerar inte TecZone Bend på Windows 7.

    • Windows 7 SP1, 64-bitars (Windows 7 SP1 stöds med Extended Security Updates installerat)

    • Windows 8.1, 64-bitars

    • Windows 10, 64-bitars

    • Windows 11, 64-bitars

  • 8 GB RAM (16 GB rekommenderas)

  • 1024 x 768 skärmupplösning (1920 x 1080 rekommenderas)

  • Grafikkort med stöd för OpenGL 4.0 (diskret GPU från Nvidia rekommenderas)

  • 3-knappsmus eller styrkula

Nödvändig programvara

Följande programvara krävs och installeras automatiskt av TecZone Bend-installationsprogrammet om den inte finns:

  • Visual C++ 2005 SP1 Redistributable (x86)

  • Visual C++ 2010 SP1 Redistributable (x86)

  • Visual C++ 2013 Redistributable (x64) – endast på 64-bitarsplattformar

  • Visual C++ 2019 Redistributable (x64) – endast på 64-bitarsplattformar

  • Microsoft .Net Framework 4.7.2

  • Microsoft .Net 6.0.1

Installera programvaran

Kör TecZone BendSetup.exe (det exakta filnamnet kan variera och kan innehålla revisionsnummer eller datum). Om de nödvändiga förutsättningarna (t.ex. .Net Framework) inte redan är installerade kan installationen ta upp till 15 minuter. Om de nödvändiga förutsättningarna redan är installerade bör installationen ta cirka 1–2 minuter.Följ instruktionerna för att installera TecZone Bend. Installationsprocessen placerar en genväg till TecZone Bend-applikationen på skrivbordet och i startmenyn. Du kan också starta applikationen genom att köra programmet TecZone Bend.exe direkt.

Köra TecZone Bend

När du kör TecZone Bend för första gången visas ett språkval där du kan välja vilket språk som ska användas för TecZone Bend. (Beroende på din installation kan det hända att du ser en annan uppsättning språk här.)Det språk du väljer här används för TecZone Bend och kan senare ändras genom att klicka på Språk i kommandoraden. (Denna ikon visas när du startar TecZone Bend och när du inte har några delar öppna).
