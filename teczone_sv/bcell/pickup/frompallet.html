Upplockning från palett

När vakuumsug eller magnetiskt gripdon används plockas ämnen (platta delar) upp från en palett. Följande parametrar påverkar denna process:

  • Palettens position i maskincellen.

  • Position och orientering av delstapeln på paletten.

  • Position och orientering av suggripdonet på delen.

  • Sugkoppskonfigurationen på gripdonet (vilka koppar är på/av och vilken typ av koppar som är monterade i varje uttag).

  • De områden på delen som avbildas av kameran på roboten och används som referenser för kalibrering av delens position.Panelerna som används för att redigera alla dessa inställningar visas nedan – de är alla sammankopplade med navigeringslänkar uppåt/nedåt som leder till andra paneler i en logisk sekvens:

Som bilden ovan visar kan dessa paneler också nås enkelt genom att klicka på olika objekt i simuleringen:

  • För att öppna panelen Palett klickar du på en palett.

  • För att redigera delstapeln på en palett (panelen Upplockning) klickar du på de ämnen som ligger på paletten.

  • För att redigera gripdonets position på ämnet (panelen Gripdon) klickar du på gripdonet.

  • För att redigera sugkoppskonfigurationen på ett gripdon (panelen Sug) klickar du på en av sugkopparna.

  • För att redigera _avbildnings_positionerna (som används av systemet för finregistrering) klickar du på kameran.

Palettpanel

Panelen Pallet används för att välja en palett och placera den i cellen. Du öppnar panelen genom att klicka på en palett. Flux placerar också simuleringens tidslinje så att roboten befinner sig vid den punkt där delen ska plockas upp från paletten.

  • Använd alternativet Name för att välja en annan palett. Normalt visas endast paletter som kan användas för detta ämne, men du kan avmarkera kryssrutan Filtered så visas alla tillgängliga paletter.

  • Om du håller muspekaren över ett namn i namnlistan visas en kort sammanfattning av den paletten tillsammans med en miniatyrbild:

  • Använd Position för att placera paletten i Z och X (i cellkoordinater) och använd Angle för att rotera paletten. När du flyttar eller roterar paletten följer delstapeln på paletten och gripdonet/roboten med.

  • Använd knapparna Next och Prev för att navigera till andra paletter i cellen, till exempel paletten med detaljplacering.

  • Använd navigeringsknappen Pickup för att redigera positionen för delstapeln på paletten.

  • Använd knappen Save Config under avsnittet Advanced för att spara denna cellkonfiguration (inklusive alla paletter) som standard för denna maskin.

Panel för upplockning

Panelen Upplockning används för att redigera positionen för delstapeln på paletten. Du kan öppna denna panel direkt genom att klicka på stapeln med ämnen som ligger på paletten.(Du kan också komma åt den via länken Pickup i panelen Palett).

  • Använd Position för att placera stapeln på paletten; dessa koordinater anger mitten av delen i Z och X, relativt palettens hörn, och finns i palettens lokala koordinatsystem.

  • Använd ingången Angle för att rotera delen på paletten.

  • Använd Flip part för att vända delen. Observera att detta vanligtvis innebär att en extra _omgreppnings_operation måste utföras innan den första bockningen kan bearbetas (detta läggs till automatiskt av TecZone Bend).

  • Använd länken Camera för att växla till panelen Kamera, där du kan ställa in bildigenkänningsfasen för upplockningen.

  • Länken Waypoints öppnar redigeraren Vägpunkter, där du kan finjustera robotens bana under upplockningen.

  • Länken RG-Stations öppnar panelen Omgreppningsstation, där du kan konfigurera omgreppningsstationens position under upplockningen.När delen flyttas runt på paletten förblir gripdonet fastsittande på delen och roboten följer rörelserna.

Panel för gripdon

Panelen Gripdon används för att välja ett annat gripdon eller för att ändra positionen och orienteringen för gripdonets upplockning av delen.

  • Använd alternativet Name för att välja ett annat gripdon. Normalt visas endast gripdon som är lämpliga för denna del (baserat på gripdonets storlek och nyttolast), men du kan avmarkera kryssrutan Filtered så visas alla gripdon.

  • Om du håller muspekaren över ett namn i gripdonslistan visas en sammanfattning av det gripdonet tillsammans med en miniatyrbild:

  • Använd Position för att flytta gripdonets mittpunkt i förhållande till delens mittpunkt och använd Angle för att rotera gripdonet i förhållande till delens orientering.

  • Använd länken Suction för att växla till finredigering av gripdonet (välja olika sugkoppar, slå på/av koppar).

  • Knappen Set Grip Plane kan användas för att placera gripdonet på ett annat plan. Normalt används det största planet i modellen för att placera gripdonet. Om du vill ändra detta klickar du på den här knappen. Klicka sedan på det plan där gripdonet ska placeras:

  • Knappen Use Jaw Gripper används för att växla denna del till att använda delmataren och käftgripdonet (mekaniskt gripdon). Alla steg i bockningscykeln från upplockning till nedläggning beräknas om med hjälp av käftgripdonet gripper[1].

Varningar om sug

Om gripdonet flyttas så att några av sugkopparna ligger utanför plåten eller placeras över hål i delen markeras kopparna och ett fel visas i kolumnen Upplockning i navigatorn, som visas i bilden nedan:

Åtgärdsmeny

Knappen Åtgärder används för att öppna en meny med några användbara åtgärder för gripdonet:

  • Auto-place: Shift: Försöker flytta gripdonet över delen så att alla sugkoppar ligger över delen och inte över några hål (om möjligt).

  • Turn off leaking cups: Stänger av alla sugkoppar som ligger över hål eller utanför delens gränser part[2].

  • Turn on all cups: Slår på alla sugkoppar som är avstängda.

  • Save Config: Om du konfigurerar gripdonet genom att stänga av eller ta bort några sugkoppar eller genom att ändra armlängder eller vinklar (för MultiGrippers som kan ändra form) kan du spara den ändrade konfigurationen av gripdonet med ett nytt namn för att enkelt kunna återanvända den.

  • Export Gripper: Spara det aktuella gripdonet som en .fxbgrip-fil som kan importeras till en annan installation av TecZone Bend. Detta är användbart om du har importerat ett anpassat gripdon och behöver dela det med andra installationer.

Sugpanel

Panelen Sug används för att konfigurera sugkoppslayouten för ett gripdon. Du kan öppna den här panelen genom att klicka direkt på en sugkopp eller genom att välja länken Sug i panelen för gripdon.

  • Använd alternativet Cup # för att välja en viss sugkopp i gripdonet, eller använd knapparna Next och Prev för att navigera mellan sugkopparna. Den valda sugkoppen markeras med blått och kan redigeras.

  • För varje sugkopp kan du ställa in State på På, Av eller Borttagen. Se beskrivningen nedan för mer information om detta.

  • Använd panelen Type för att byta till en annan typ av sugkopp. Vanligtvis byter man ut alla sugkoppar i ett gripdon till den nya typen, men du kan också blanda och matcha sugkoppar genom att stänga av knappen Change All och sedan byta sugkoppar. (Observera att när du gör detta kommer valet av sugkoppar att begränsas, eftersom alla sugkoppar som är monterade på en gripdonram måste ha samma arbetshöjd.) Bilden ovan visar två av sugkopparna ersatta med mindre (SAXM50 istället för standard SAXM80).

  • Använd knappen Reset för att återställa gripdonet till sitt ursprungliga läge – alla sugkoppar är påslagna och återställs till den standardtyp som är definierad i gripdonet.Standardläget för en sugkopp är På, vilket innebär att sugkoppen är ansluten till vakuumledningen och hjälper till vid lyft.Om en sugkopp ligger över ett hål i delen kan du ändra läget till Av, vilket innebär att det inte finns något vakuum. (Detta minskar gripdonets lyftkraft och ändrar lyftcentrum, vilket Flux tar hänsyn till vid kapacitetskontroller av gripdonet.) Observera att koppen fortfarande är monterad i ramen och ingår i kollisionskontroller. Flux visar avstängda sugkoppar som trådar, vilket du kan se med två av sugkopparna i bilden ovan.Slutligen kan du ställa in en sugkopp på Borttagen, vilket innebär att koppen har tagits bort från ramen i den faktiska maskinen. Det finns ingen lyftkraft från denna kopp och den orsakar ingen kollision. Detta är ibland bra om en kopp faller över ett format område eller orsakar en kollision med dynan eller maskinbordet under drift.

Kamerapanel

Upplockningen kräver att en eller flera bilder tas av kameran och att ett bildbehandlingssystem använder dessa bilder för att uppskatta exakt var delen befinner sig på paletten. Klicka på kameran som är monterad på roboten eller välj navigeringsknappen Kamera i panelen Upplockning för att öppna kamerapanelen. Flux positionerar också simuleringen så att roboten är i den position som krävs för att ta bilden:

  • Använd listan Index (eller knapparna Next och Prev) för att bläddra igenom de olika bilderna för finregistrering som används för denna del. När du gör detta visar Flux en orange kontur som visar bildigenkänningsområdet på delen (se bilden ovan).

  • Använd inmatningarna i Position för att flytta detta område i Z och X för att bättre inkludera vissa intressanta egenskaper som kan förbättra igenkänningsnoggrannheten (hörn, små hål, hörnklipp).

  • Använd knappen Add för att lägga till en ytterligare igenkänningsbild (upp till 4) och använd knappen Remove för att ta bort den aktuella igenkänningsbilden. Under simuleringen av upplockningen av delen visar Flux hur roboten rör sig till vart och ett av dessa igenkänningsområden med kameran nedåt och pausar för att ta bilden. Om Use backlight table är aktiverad överförs delen till ett belyst bord innan kameran används för att avbilda den. Detta ökar kontrasten och är lämpligt för delar med hög reflektionsgrad. TecZone Bend lägger till bakgrundsbelysning och placerar den automatiskt nära paletten för upplockning, men du kan sedan konfigurera positionen genom att klicka på den:

1. Det här alternativet är endast tillgängligt för delar som sträcker sig över arbetsstorleksområdena för sugkoppsgripdon och käftgripdon.
2. Observera att detta kan leda till att gripdonets sugkraft minskar under den punkt där det kan plocka upp delen på ett tillförlitligt sätt. Detta indikeras av ett fel i kolumnen Upplockning i navigatorn.
