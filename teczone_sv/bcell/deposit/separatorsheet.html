Lägg till separatorark

Vissa avsättningsmönster kan använda separatorark mellan lagren av delar för att göra avsättningen mer stabil. Att markera kryssrutan Add separator när du redigerar en avsättning lägger till en annan pall som innehåller separatorarken.

Separatorarkpanel

Du kan klicka på separatorarket i simuleringen för att visa panelen Separator som visas ovan.

  • Väljaren Name kan användas för att välja ett annat separatorark. Du kan välja länken Import…​ i denna väljare.

  • Avsnittet Pick (Z,X) används för att ställa in positionen och orienteringen för separatorarket på upphämtningspallen. I bilden ovan har separatorarket roterats 90° och förskjutits till ena sidan av pallen.

  • Avsnittet Drop (Z,X) används för att ställa in positionen och orienteringen för separatorarket på avsättningsstapeln.

  • Avsnittet Gripper (Z,X) används för att ställa in positionen och orienteringen för griparen på separatorarket. Detta kan användas för att flytta griparen bort från eventuella spår eller hål i separatorarket. (Bilden ovan visar griparen roterad 135° från standard orientering).

  • Skjutreglaget Simulate kan användas för att simulera endast separatorarkcykeln så att ändringar du gör i inställningarna ovan snabbt kan testas.

  • Länken Stack används för att växla till redigering av avsättningen och länken Pallet används för att redigera positionen och orienteringen för pallen som separatorarket placeras på.

Anpassade separatorark

Du kan också importera ett anpassat separatorark från menyn File:

Om detta fungerar korrekt visas ett importmeddelande som detta (annars visas ett lämpligt felmeddelande). Formatet för DXF- eller GEO-filen för att definiera ett anpassat separatorark förklaras på denna sida.

Viktkontroll

Separatorarket kan vara för tungt för griparen. I så fall kan du se en felindikator i navigatörpanelen, och genom att peka på felet visas mer detaljer om överbelastningen.

Sugkontroll

Om sugkopparna ligger ovanpå ett hål på separatorarket upptäcks förlusten av sug och visas som ett fel i raden Gripper i kolumnen D1 iböjnavigat ören.
