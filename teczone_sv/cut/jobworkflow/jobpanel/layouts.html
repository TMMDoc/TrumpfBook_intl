Fliken Layouter

Fliken Layouts visar de inbäddade layouterna och innehåller några alternativ för att redigera dem.

Om du klickar på en av layouterna visas den och du kan redigera layouten interaktivt. Knapparna längs vänsterkanten av fliken Layouts gör det möjligt att utföra vissa åtgärder på hela layouten.

  • Add Blank används för att skapa en tom layout där du sedan kan utföra manuell nesting, medan Delete tar bort den aktuella layouten.

  • Knapparna Move Up och Move Down används för att ändra ordningen på layouterna i listan.

  • Knappen Do Nesting används för att göra om nestingen av alla delar som inte är nestade.

  • Save All används för att spara alla layouter som separata .fxlyt-filer, medan Export DXFs används för att exportera bilderna av de fullständigt nestade plåtarna som DXF-filer, som kan öppnas med valfri visare eller redigerare som kan hantera DXF-filer. (Varje del i layouten blir en blockinsatt enhet i DXF-filen.)

  • Knappen Create NC används för att generera NC-program för alla layouter, samt en layoutinställningsplåt som innehåller detaljer om alla delar som används i jobbet och alla resulterande layoutplåtar.Här är ett avsnitt från en layoutrapport som genereras av kommandot Create NC.
