De software installeren

Voorbereiden op installatie

Systeemvereisten

  • Een van de volgende besturingssystemen. Windows 7 vereist dat Service Pack 1 wordt geïnstalleerd, anders kan TecZone Bend niet werken op Windows 7.

    • Windows 7 SP1, 64-bits (Windows 7 SP1 wordt ondersteund met uitgebreide beveiligingsupdates geïnstalleerd)

    • Windows 8.1, 64-bits

    • Windows 10, 64-bit

    • Windows 11, 64-bit

  • 8 GB RAM (16 GB aanbevolen)

  • 1024 x 768 schermresolutie (1920 x 1080 aanbevolen)

  • Grafische kaart met OpenGL 4.0-ondersteuning (Nvidia discrete GPU aanbevolen)

  • 3-knops muis of trackball

Vereiste software

De volgende software is vereist en zal automatisch worden geïnstalleerd door het TecZone Bend-installatieprogramma, als deze niet wordt gevonden:

  • Visual C++ 2005 SP1 Redistributable (x86)

  • Visual C++ 2010 SP1 Redistributable (x86)

  • Visual C++ 2013 Redistributable (x64) – alleen op 64-bit platforms

  • Visual C++ 2019 Redistributable (x64) – alleen op 64-bit platforms

  • Microsoft .Net Framework 4.7.2

  • Microsoft .Net 6.0.1

De software installeren

Voer TecZone BendSetup.exe uit (de exacte bestandsnaam kan verschillen en kan het revisienummer of de -datum bevatten). Als de vereisten (zoals het .Net Framework) nog niet zijn geïnstalleerd, kan de installatie tot 15 minuten in beslag nemen. Indien de vereisten al geïnstalleerd zijn, duurt de installatie ongeveer 1 of 2 minuten.Volg de instructies om TecZone Bend te installeren. Dit installatieproces plaatst een snelkoppeling naar de TecZone Bend-applicatie op uw bureaublad en in het startmenu. Je kunt de applicatie ook starten door het programma TecZone Bend.exe direct uit te voeren.

TecZone Bend uitvoeren

Wanneer u TecZone Bend voor de eerste keer uitvoert, verschijnt er een taalselectie en kunt u de taal selecteren die wordt gebruikt voor TecZone Bend. (Afhankelijk van uw installatie kunt u hier een andere set talen zien).De taal die u hier selecteert wordt gebruikt voor TecZone Bend, en dit kan vervolgens worden gewijzigd door op Taal in de opdrachtbalk te klikken. (Dit pictogram verschijnt wanneer u TecZone Bend opstart, en wanneer u geen stukken open heeft gehad).
