Aangepaste luchtspleettabellen

Aangepaste luchtspleettabellen

De luchtspleettabel voor een zwenkbuigmachine brengt een bepaalde plaatdikte in kaart voor de standaard luchtspleet die wordt gebruikt tijdens de bewerking. Dit is ingebouwd in elk van de verschillende zwenkbuigmachines, maar kan worden overschreven door het verstrekken van een aangepaste luchtspleettabel in de vorm van een json-bestand.Dit json-bestand heet air-gap-tables.json en wordt geplaatst in C:\ProgramData\Trumpf{APPNAME}\Data\System.Hieronder staat een typisch air-gap-tables.json bestand:

 {
  "version":1  "airGapTables": [
    {
      "machine": "112002,112003", "material": "DC01"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
     {
      "machine": "112003", "material": "*"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
    {
      "machine": "*", "material": "Alluminio"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    }
  ]
}

  • Het bestand bevat een verzameling luchtspleetwaarden. Elke tabel heeft een machine- en materiaaltoets die als keuzeschakelaars fungeren.

  • Deze toetsen zijn ingesteld op * (asterisk) om een jokerteken aan te geven (komt overeen met alles), of ze zijn een door komma’s gescheiden lijst van machine- of materiaalwaarden.

  • In bovenstaand voorbeeld zijn er drie tabellen. De eerste wordt gebruikt voor machines 112001 of 112003, alleen voor het materiaal DC01. De tweede wordt gebruikt voor alle materialen, voor de machine 112003, en de derde wordt gebruikt voor alle machines, voor het materiaal Alluminio.Als meerdere tabellen overeenkomen met de machine/materiaalcombinatie van het stuk, zal de eerste overeenkomende tabel wordt gebruikt. Dus, voor de machine 112003, materiaal DC01 wordt de eerste tabel gebruikt (ook al komt de tweede ook overeen).

  • Elke tabel bestaat uit een eenvoudige reeks diktetoetsen, met de bijbehorende luchtspleetwaarden. Het eerste item hierboven moet bijvoorbeeld worden geïnterpreteerd als: als de dikte kleiner is dan of gelijk is aan 0,75 mm, gebruik dan een spleet van 2,5 mm. De diktetoetsen moeten in oplopende volgorde staan. Als geen van de tabellen in dit bestand overeenkomt met de materiaal- en diktecombinatie, wordt de standaard luchtspleettabel voor de machine gebruikt.

Als u wijzigingen aanbrengt in het bestand air-gap-tables.json, moet u TecZone Fold opnieuw starten voordat deze wijzigingen van kracht worden.
Als het json-bestand slecht is gevormd, wordt er een fout weergegeven wanneer u probeert een stuk voor zwenkbuigmachine verwerken, maar valt TecZone Fold dan terug op het gebruik van de standaard luchtspleettabel voor de machine, en gaat door.
