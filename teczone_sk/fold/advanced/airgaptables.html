Tabuľky vlastných vzduchových medzier

Tabuľka medzier pre stroj s výkyvným ohýbaním mapuje konkrétnu hrúbku plechu voči štandardnej medzere použitej počas obrábania Táto tabuľka sa nachádza vo všetkých strojoch s výkyvným ohýbaním, dá sa ale prepísať vlastnou tabuľkou vzduchových medzier vo forme súboru JSON. Tento súbor JSON sa nazýva air-gap-tables.json, nachádza sa v priečinku C:\ProgramData\Trumpf{APPNAME}\Data\System.

Nižšie je uvedený typický súbor air-gap-tables.json:

 {
  "version":1  "airGapTables": [
    {
      "machine": "112002,112003", "material": "DC01"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
     {
      "machine": "112003", "material": "*"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
    {
      "machine": "*", "material": "Alluminio"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    }
  ]
}

  • Súbor obsahuje zbierku hodnôt vzduchových medzier. Každá tabuľka má kľúč stroja a materiálu, ktoré slúžia ako selektory.

  • Tieto kľúče sú buď nastavené na * (hviezdička) označujúcu náhradný znak (zodpovedá všetkému), alebo sú to čiarkami oddelené zoznamy hodnôt stroja alebo materiálu.

  • Vo vyššie uvedenom príklade sú tri tabuľky. Prvá sa používa pre stroje 112001 alebo 112003, iba pre materiál DC01. Druhá sa používa pre všetky materiály, pre stroj 112003, tretia sa používa pre všetky stroje, pre materiál Alluminio. Ak kombinácii stroj/materiál dielca vyhovuje viacero tabuliek, použije sa prvá zodpovedajúca tabuľka. Takže pre stroj 112003, materiál DC01, sa použije prvá tabuľka (aj keď vyhovuje aj tá druhá).

  • Každá tabuľka pozostáva z jednoduchého poľa kľúčov hrúbky s príslušnými hodnotami vzduchovej medzery. Napríklad, prvý vyššie uvedený záznam by sa mal interpretovať takto: ak je hrúbka menšia alebo rovná 0,75 mm, použite medzeru 2,5 mm. Kľúče pre hrúbku musia byť zoradené vzostupne. Ak žiadna z tabuliek v tomto súbore nezodpovedá kombinácii materiálu a hrúbky aktuálneho dielca, použije sa predvolená tabuľka medzier pre stroj.

Ak vykonáte akékoľvek zmeny v súbore air-gap-tables.json, musíte reštartovať TecZone Fold, aby sa tieto zmeny prejavili.
Ak je súbor JSON nesprávne naformátovaný, pri pokuse o obrábanie dielca v stroj s výkyvným ohýbaním sa zobrazí chyba, TecZone Fold sa ale potom vráti k použitiu predvolenej tabuľky medzier pre stroj a bude pokračovať.
