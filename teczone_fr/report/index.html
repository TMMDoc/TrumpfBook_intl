Rapports

Rapports

En plus de générer des programmes CN, TecZone Bend peut également générer divers rapports pour une pièce, en fonction des modules installés. Lorsque vous travaillez avec la technologie de pliage, par exemple, il est utile de générer une fiche de configuration qui répertorie les outils à monter, et fournit des détails sur la phase de pliage.

Le Concepteur de rapports de TecZone Bend vous aide à concevoir le contenu et le style de ces rapports. Une fois que vous avez conçu un rapport, vous pouvez l’exporter au format PDF ou HTML.[1] Voici à quoi ressemble le concepteur de rapports lorsqu’il est ouvert :

Utilisation d’un rapport

Vous pouvez ouvrir un rapport pour le modifier en choisissant le menu Fichier/Ouvrir et en choisissant un fichier de modèle de rapport (il s’agit de fichiers avec l’extension .ftrep). Vous trouverez des exemples de modèles de rapport dans le dossier Modèles de votre installation TecZone Bend.

Avant de pouvoir ouvrir un modèle de rapport pour travailler, vous devez d’abord ouvrir une pièce et l’avoir usinée jusqu’à la fin (vous devez générer les données de technologie appropriées pour le pliage/la découpe/la réalisation de plis écrasés). En effet, le concepteur de rapports travaille avec des données en temps réel ; il affiche les données réelles de la pièce au fur et à mesure que vous concevez le rapport, ce qui vous permet de voir immédiatement à quoi ressemblera le résultat final imprimé.

Vous pouvez également commencer avec un rapport vierge en choisissant le menu Fichier/Nouveau/Modèle de rapport. Vous pouvez ensuite remplir le rapport en ajoutant divers éléments de rapport (champs de texte, images, labels, etc.), puis l’enregistrer.

Lorsque vous avez un modèle de rapport ouvert, vous pouvez utiliser les commandes Fichier/Exporter au format PDF ou Fichier/Exporter au format HTML pour convertir le rapport au format PDF ou HTML.Le rapport est imprimé à partir des données de la pièce actuellement ouverte et active. Vous pouvez également basculer entre l'affichage des données en temps réel (par défaut) et un affichage de conception qui affiche les expressions sous-jacentes utilisées pour remplir les différents champs du rapport.

Rapports imprimés automatiquement:

Vous pouvez configurer TecZone Bend pour qu’il imprime automatiquement un rapport de pliage chaque fois qu’un programme de pliage est généré (tout comme il peut générer automatiquement un programme CN lorsque le programme est calculé). Utilisez la page des réglages de sortie du module de technologie de pliage pour le configurer.

Dans l’exemple ci-dessus, la génération automatique de rapports est activée. Le modèle de rapport est défini sur lib :Report/bend.ftrep, qui est le modèle de rapport _par défaut _ pour Bend CAM (il est installé lors de chaque installation TecZone Bend). Si vous créez un modèle de rapport différent à l’aide du concepteur de rapports, vous pouvez spécifier ce modèle ici.

Les réglages vous permettent également de contrôler le dossier dans lequel le rapport est généré et le format (HTML ou PDF) dans lequel le rapport est enregistré.

1. Les fichiers HTML générés par le rédacteur de rapport TecZone Bend sont autonomes.Ils comprennent toutes les images, feuilles de style et autres ressources nécessaires intégrées directement dans le fichier HTML. Ainsi, ils peuvent être facilement transportés ou archivés sans aucune dépendance externe.
