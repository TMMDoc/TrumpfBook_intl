Champs, images et tableaux

Champs, images et tableaux

Cliquez sur les sources de données pour les ouvrir et afficher des niveaux supplémentaires de boutons cliquables. Ils peuvent être utilisés pour ajouter des champs, des images et des tableaux au rapport.

Ajout d’un champ

Un champ est un simple affichage numérique ou textuel du rapport. Par exemple, en cliquant sur sur Pièce > Info > Nom, vous pouvez insérer un champ dans le rapport qui affiche le matériau de la pièce :

Lorsque vous effectuez cette opération TecZone Bend ajoutera deux éléments au rapport : un champ affichant la valeur et un label à gauche de ce champ. Vous pouvez modifier les réglages du label ou du champ, en cliquant simplement dessus.

Modification d’un label

Lorsque vous cliquez sur le label qui vient d’être ajouté, un panneau s’affiche et vous permet de modifier divers réglages pour ce label. Le label est également mis en surbrillance et des prises de redimensionnement sont ajoutées autour du label, ce qui vous permet de le redimensionner en faisant glisser l’une d’entre elles. Vous pouvez également cliquer avec le bouton droit sur le label et le faire glisser pour le repositionner. Lorsque vous faites glisser un élément pour le déplacer ou le redimensionner, divers indicateurs d’encliquetage s’affichent automatiquement pour vous aider à aligner précisément ce label avec les autres éléments déjà présents dans le rapport.

Le panneau d’édition de label (et de la plupart des autres éléments du rapport) contient un grand nombre de réglages permettant de régler tous les aspects de l’affichage visuel. Vous pouvez les voir en ouvrant les différents groupes qui peuvent être développés :

L’image ci-dessus à droite montre un sélecteur de couleurs utilisé pour définir la couleur du texte du label.

Insertion d’une image

Une image dans un rapport est indiquée par une icône représentant un appareil photo (à la place de l’icône représentant un crayon utilisée pour les champs simples). Par exemple, en cliquant sur Pièce > Modèle > Image, vous pouvez insérer une image du modèle 3D de la pièce :

L’image ci-dessus montre une image insérée puis modifiée en cliquant dessus.Le panneau de l’image affiche certains réglages utilisés pour contrôler l’image. Les réglages exacts peuvent changer, en fonction de l’image réelle qui a été insérée.Dans ce cas, le réglage DPI sert à contrôler la résolution de rendu de l’image (des valeurs plus élevées donneront un résultat plus esthétique, mais augmenteront la taille finale du rapport). Le réglage Pas de PDF 3D permet de désactiver la génération de fichiers PDF 3D.footnote[TecZone Bend peut afficher certaines de ses images 3D sous forme d’images PDF 3D intégrées ; celles-ci peuvent être agrandies, pivotées ou déplacées à l’aide de certains lecteurs PDF. Cependant, ces formats ne sont pas largement pris en charge par tous les lecteurs PDF, et certains d’entre eux peuvent les afficher sous forme de cases vides.].

Si vous insérez une image d’un dessin en 2D (plat), les options d’image sont différentes, comme le montre l’image ci-dessous :

Les réglages ici contrôlent la largeur de ligne utilisée pour les contours et pour les dimensions. Il existe des réglages permettant d’afficher les numéros de séquences de pliage et de contrôler si le contour fermé de la pièce est rempli.

Insertion d’un tableau

Vous pouvez insérer un tableau dans un rapport en sélectionnant l’un des éléments affichant une icône de tableau. Par exemple, si vous utilisez le tableau Pièce > Pliage > Pliages pour afficher des informations sur toutes les phases de pliage d’une pièce :

La section Sélection colonnes permet de sélectionner les colonnes qui s’affichent dans le tableau (parmi toutes les colonnes disponibles pour ces données tabulaires). Vous pouvez modifier d’autres attributs de chaque colonne en cliquant sur le lien de navigation Colonne situé en bas du panneau du tableau :

Cela ouvre l'Editeur de colonne, où vous pouvez modifier les attributs de la colonne, comme ci-dessous :

Le champ Titre permet de définir le titre affiché pour la colonne, tandis que le champ Expression correspond aux données internes TecZone Bend qui s’affichent dans cette colonne. Vous pouvez utiliser les boutons Shift pour déplacer cette colonne vers la gauche ou vers la droite dans l’ordre, et utiliser les boutons Insérer et Supprimer pour insérer de nouvelles colonnes ou pour supprimer la colonne sélectionnée.
