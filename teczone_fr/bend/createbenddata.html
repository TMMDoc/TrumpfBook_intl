Création de paramètres de pliage

Vous pouvez créer des données de technologie de pliage pour une pièce en tôle à l’aide du module Pliage. Ceci est également appelé préparation de la pièce pour le pliage. Supposons que vous avez déjà installé une machine, que vous avez configuré l’inventaire des outils et que vous avez importé une pièce CAO (2D ou 3D) :

Passage à Bend CAM

Maintenant, la pièce peut être préparée pour le pliage en appuyant simplement sur la touche B. La pièce est préparée avec la presse plieuse par défaut, qui est la presse plieuse que vous avez utilisée en dernier. Un enchaînement des outils, la configuration de pliage et les positions de butée sont toutes calculées, et vous devriez obtenir un résultat similaire à celui-ci :

Si aucune erreur n’est détectée, le programme CN (et éventuellement la fiche de configuration) pour la pièce sont également générés et enregistrés dans le dossier de sortie configuré pour cette machine. Voici un extrait d’une fiche de configuration de pièce type sheet[1] (également appelée rapport de pliage).

Voir également le panneau Flux de travail, accessible en appuyant sur la touche W. Cela permet un meilleur contrôle de la création des données relatives à la technologie de pliage et offre davantage d’options pour le cheminement de la pièce via le Laser CAM ou l’imbrication.

Autres opérations

Voici un bref résumé de certaines des opérations que vous pouvez effectuer une fois qu’une pièce a été préparée pour le pliage.

  • Si vous cliquez sur la Spacebar, une simulation de pliage démarre. Vous pouvez également utiliser les commandes de simulation du navigateur de pliage pour démarrer, arrêter ou revenir en arrière dans la simulation :

  • Si des avertissements ou des erreurs apparaissent, ils s’affichent dans le navigateur de pliage, et vous pouvez les consulter et les résoudre en cliquant sur les cellules correspondantes :

  • Vous pouvez modifier la préparation de pliage (poinçons et matrices), le réglage de la butée ou le réglage de mesure de l’angle en cliquant directement sur l’objet concerné dans la vue de la simulation :

  • Vous pouvez changer de machine en cliquant sur le nom de la machine dans l’onglet situé sous la pièce, puis en sélectionnant une autre machine :

  • Cliquez sur l’icône Réglages dans la barre d’outils à gauche pour modifier les réglages supplémentaires de chaque pliage (ou pour modifier les réglages par défaut utilisés pour cette presse plieuse ou pour l’ensemble de l’application TecZone Bend).

  • Cliquez sur l’icône Affichage dans la barre d’outils à gauche pour modifier l’affichage de la machine. Vous pouvez activer ou désactiver l’affichage de composants supplémentaires tels que le rail de matrice, le coulisseau de pliage ou le système de support du doigt de butée. Vous pouvez également rendre divers composants translucides.

Lorsqu’un programme de pliage est créé sans erreur, le programme CN de la pièce et une fiche de configuration pour le pliage sont générés automatiquement. Vous pouvez utiliser la page de réglages Bend Outputs pour définir si le programme CN de la pièce et la fiche de configuration sont générés automatiquement, ainsi que pour configurer la destination de ces sorties et le format des rapports. Si du code a été généré, le bouton Code CN dans la barre d’outils de gauche est grisé :

Lorsque vous apportez des modifications à la pièce, ce bouton redevient actif et vous pouvez régénérer le programme CN en cliquant à nouveau sur cette icône (ou en appuyant sur la touche C).

1. Vous pouvez utiliser l’éditeur de rapports pour personnaliser cette fiche de configuration.
