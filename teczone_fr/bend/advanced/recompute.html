Recalcul du programme

Lorsque vous appuyez sur B ou utilisez le panneau Flux de travail pour calculer un programme, TecZone Bend calcule un programme de pliage à l’aide d’un ensemble de réglages par défaut et de paramètres. Vous pouvez recalculer ce programme en utilisant différents réglages en cliquant sur le menu Recalcul dans la barre de commandes. Cela peut parfois être utile pour terminer le calcul après avoir modifié de manière interactive certaines préparations ou la séquence.
Par exemple :

  • Vous pouvez modifier la séquence et demander à TecZone Bend de calculer de nouveaux postes de préparation tout en conservant la séquence inchangée.

  • Vous pouvez modifier les postes de préparation et demander à TecZone Bend de calculer une nouvelle séquence ou d’aligner la pièce sur les stations nouvellement créées.

  • Vous pouvez essayer l’opération de préparation avec un autre jeu d’outils ou essayer d’utiliser une configuration déjà créée.

Asservissements

Cette section contient quelques réglages que vous pouvez utiliser pour remplacer le comportement par défaut de TecZone Bend.

  • Activez le commutateur Ne pas modifier la séquence pour forcer TecZone Bend à calculer un nouveau programme sans modifier la séquence. Différents postes de préparation, positions de butée ou sens d’insertion des pièces seront testées par TecZone Bend, tout en conservant la séquence inchangée.

  • Si le commutateur Pas de mesure de l’angle. est allumé, TecZone Bend calcule un programme qui n’effectue aucune mesure de l’angle et n’utilise aucun capteur de mesure (outils détecteurs)

  • La case Ignorer les outils des fichiers GEO/DXF indique à TecZone Bend d’ignorer les remarques de préparations qui se trouvent souvent dans les fichiers GEO et parfois dans les fichiers DXF.

Configurations

Cette section définit la stratégie de configuration de la préparation ; par défaut, TecZone Bend calculera la configuration à partir de zéro, mais vous pouvez utiliser ces réglages pour modifier ce comportement.

  • Calculer nouvelle configuration est le comportement par défaut - TecZone Bend calculera une configuration en utilisant tous les outils autorisés (voir la section Outils ci-dessous).

  • Conserver configuration existante indique à TecZone Bend qu’aucune modification ne doit être apportée à la configuration ; Flux tente d’usiner tous les pliages en utilisant la configuration actuelle de la pièce. Si certains pliages présentent des collisions ou si la longueur d’outil est insuffisante pour l’usinage, ces erreurs ne sont pas corrigées et peuvent être visualisées après le processus de recalcul.

  • Si vous sélectionnez Utiliser Configuration à partir du fichier, vous êtes invité à choisir un fichier de configuration de pliage (bsetup). La configuration chargée à partir de ce fichier est ensuite utilisée pour le calcul ; cette configuration n’est pas modifiée et aucune station supplémentaire n’est créée. Les configurations que vous avez récemment chargées à partir d’un fichier s’affichent dans le sélecteur qui apparaît sous cette section. Vous pouvez ensuite cliquer sur le bouton + à côté de ce sélecteur pour choisir un autre fichier de configuration.

  • L’option Utiliser configuration à partir d’une autre pièce n’est disponible que s’il existe d’autres pièces actuellement ouvertes et qu’elles ont été préparées pour la même presse plieuse. Dans ce cas, vous pouvez demander à TecZone Bend d’utiliser la même configuration que l’une de ces autres pièces ; un sélecteur apparaît pour vous permettre de choisir la pièce souhaitée.

Outils

Si vous choisissez l’option Calculer nouvelle configuration dans la section ci-dessus, vous pouvez utiliser la section Outils pour sélectionner la liste des outils de pliage pouvant être utilisés pour calculer la nouvelle configuration. Si vous choisissez l’une des autres options pour la configuration, cette section n’est alors pas disponible.

  • Outils préférés pour la machine est l’option par défaut. Pour chaque presse plieuse, vous pouvez choisir un ensemble d’outils tools[1] comme Outils préférés, et TecZone Bend utilisera ces outils lors du montage de la pièce.[2] (Si vous n’avez pas encore configuré les outils préférés pour une machine, TecZone Bend utilise tous les outils du stock magasin comme jeu d’outils préférés pour la machine).

  • Si vous choisissez Outils en stock magasin, TecZone Bend utilisera tous les outils disponibles dans le stock magasin de l’installation. Si vous n’avez pas encore configuré le stock magasin des outils, TecZone Bend utilise tous les outils du catalogue comme stock magasin.

  • L’option Tous les outils du catalogue correspond à la sélection d’outils la plus large ; TecZone Bend utilisera tous les outils disponibles dans le catalogue des outils du fabricant pour tenter de trouver un programme. Bien que cela puisse signifier que le programme n’est pas réellement viable pour la production (en utilisant le stock magasin dont vous disposez), cela reste utile pour explorer de manière spéculative les outils qui seraient nécessaires pour produire une pièce particulièrement complexe. (De plus, ce mode indique également à TecZone Bend de supposer un stock magasin infini de pièces, c’est-à-dire que la préparation optimale est produite en supposant que vous disposez d’un nombre infini de pièces de chaque longueur).

  • Vous pouvez créer des jeux d’outils personnalisés et les enregistrer dans des fichiers de liste d’outils (btools). Ces fichiers peuvent ensuite être utilisés pour le montage de la pièce, lorsque vous sélectionnez l’option Utiliser le jeu d’outil à partir du fichier. Lorsque vous sélectionnez cette option, vous êtes invité à choisir un fichier de liste d’outils, et les fichiers de liste d’outils que vous avez récemment utilisés s’affichent pour faciliter la sélection. Vous pouvez cliquer sur le signe + à côté de cette liste pour sélectionner une nouvelle liste d’outils.

  • Pour un contrôle plus direct des poinçons et des matrices, vous pouvez sélectionner l’option Prédéfinir poinçon et matrice puis choisir le poinçon et la matrice que vous souhaitez utiliser.

Priorité de séquence

Lors du séquençage des pliages d’une pièce, il existe quelques exigences potentiellement contradictoires. Par exemple, TecZone Bend doit-il essayer de minimiser le nombre de rotations de la pièce, ou doit-il essayer de minimiser les pliages lorsque le centre de gravité de la pièce se trouve à l’intérieur de la machine ?
Il est fort probable que chaque opérateur ait un ensemble de règles de priorité différent, et il est également possible que différentes pièces nécessitent des réglages de priorité différents. TecZone Bend vous permet donc de refaire le calcul avec un ordre de priorité différent, en utilisant la section

Priorité de séquence de cette fenêtre de dialogue.

Utilisez les flèches et sur la droite pour mélanger les règles ; les éléments situés en haut ont la priorité. Vous pouvez désactiver complètement certains de ces éléments prioritaires ; cela indique à TecZone Bend de ne pas tenir compte de ce facteur lors de la détermination de la séquence. (Parfois, désactiver certaines de ces options peut permettre d’obtenir une séquence qui utilise peu de stations d’outils).

  • Eviter les collisions lors du surpliage : permet d’obtenir une précision angulaire, la plupart des pliages auront besoin devront être légèrement surpliés afin de compenser le retour élastique. Cette règle de séquence indique à TecZone Bend de faire des pliages dans un ordre tel qu’un pliage excessif est possible. Par exemple, dans la pièce ci-dessous, nous devons d’abord effectuer le pliage 1, puis le pliage 2 ; l’ordre inverse empêcherait le pliage excessif.

  • Eviter les outils à priorité inférieure : indique à TecZone Bend d’éviter d’utiliser des outils de faible priorité, ou des outils qui ne figurent pas dans la liste des outils préférés.

  • Eviter une butée sur les angles ouverts : les positions de butée doivent idéalement se trouver dans le même plan que l’axe de pliage, et il ne doit pas y avoir de pliages déjà usinés entre la ligne de pliage et le point de mesure. Voici un exemple de mauvaise mesure que cette règle tente d’éviter : image::open-gauging.png[Open Gauging,align="center"width=248]

  • Longueur de côté plus longue principalement avant la ligne de pliage : en particulier avec les pièces lourdes, il est utile d’éviter les situations où le centre de gravité de la pièce se trouve derrière la ligne d’outil. Cette règle vise à minimiser de telles situations.[3]

  • Minimiser les retournements de pièces : les retournements de pièces se produisent lorsque la pièce doit être retournée (les pliages successifs s’effectuent dans des sens de rotation ascendants/descendants différents).

  • Minimiser les rotations de pièces : une rotation de pièce consiste à tourner une pièce vers la gauche ou vers la droite dans le plan horizontal ; cette opération est considérée comme plus facile pour l’opérateur qu’un retournement de pièce ; l’ordre par défaut de ces deux dernières règles tente de privilégier les rotations de pièces par rapport aux retournements, dans la mesure du possible.

1. Pour ce faire, utilisez le bouton Stock magasin dans la barre de commandes, et choisissez Outils préférés dans le menu qui apparaît.
2. TecZone Bend essaiera d’utiliser uniquement les outils préférés pour la pièce, mais s’il y a des collisions ou d’autres problèmes qui ne peuvent être résolus, il commencera à utiliser d’autres outils disponibles dans le stock magasin.
3. TecZone Bend fait preuve d’un certain discernement quant au poids de la pièce et à la distance entre le centre de gravité et la ligne d’outil pour déterminer s’il s’agit d’un problème. Techniquement, Flux considère qu’une pièce bascule vers l’arrière s’il y a un moment de basculement de 0,15 Nm (Newton-mètre) vers l’arrière.
