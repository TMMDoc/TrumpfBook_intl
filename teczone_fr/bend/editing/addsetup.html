Ajout de configurations et de stations

Vous pouvez ajouter une nouvelle configuration à une pièce de pliage ou ajouter une nouvelle station à une configuration existante. Pour ce faire, cliquez sur l’icône Ajouter dans la barre d’outils à gauche :

  • Configuration : crée une nouvelle configuration et l’affecte au pliage actuel. La configuration est initialement vide.

  • Montage : ajoute un montage de poinçon ou de matrice (ou les deux) à la configuration actuelle. Si vous avez choisi de créer à la fois des montages de poinçon et de matrice, le pliage actuel est attribué à la station nouvellement créée.

  • Configuration à partir d’un fichier : vous permet de sélectionner et d’importer une configuration à partir d’un fichier *.fxbsetup*et d’attribuer cette configuration pour le pliage actuel.

Vous pouvez créer des fichiers .fxbsetup à partir du panneau de configuration en cliquant sur le bouton Enregistrer… qui s’y trouve.
