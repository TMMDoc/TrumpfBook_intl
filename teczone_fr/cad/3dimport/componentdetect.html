Identification de composants

Les composants tels que les écrous, les vis, etc. ne nécessitent généralement aucun traitement supplémentaire dans ce logiciel. Cependant, il est utile de collecter, détecter et utiliser ces composants afin de générer une nomenclature potentielle des articles supplémentaires à acheter pour composer le sous-groupe.

Il existe des différences importantes dans la pratique quant à la manière dont même un composant standard tel qu’une vis M8 est réellement dessiné. Le logiciel utilise un système d’apprentissage qui apprend les noms des composants à partir de l’exemple. Au début, TecZone Bend ne connaît aucun composant standard et ceux-ci s’affichent en jaune (non identifiés).

Pour identifier le composant :

  1. Importez une pièce de sous-groupe.

  1. Cliquez sur l’icône Workflow et sélectionnez l’assemblage Décomposer.

  1. Tous les composants du sous-groupe sont ouverts sous forme de fichiers 3D séparés.

  1. Cliquez sur un composant jaune au niveau de sa surface pour accéder au panneau Position de préhension.

Figure 1. Panneau Surface

  1. Cliquez sur Identifier…​ pour accéder au panneau Ajouter un composant.

Figure 2. Ajouter un composant

  1. Ajustez les réglages au besoin :

    1. Afficher : tous les autres composants déjà définis sont également affichés ici. Ce composant peut être affiché à titre de référence en cliquant sur son nom. Pour revenir à l’ajout du nouveau composant, vous devez cliquer sur *NOUVEAU*en haut de la liste.

    2. Image d’aperçu : affiche une prévisualisation du composant sélectionné.

    3. Variantes : le même nom que le nom d’un composant existant peut être utilisé. Il est courant que différentes méthodes soient utilisées pour modéliser, par exemple, la même vis M8. Si plusieurs composants portant le même nom sont ajoutés, ils deviennent des variantes. Les composants disponibles en plusieurs variantes ont leur quantité indiquée après leur nom (par exemple : x4). Si l’un de ces composants est sélectionné, toutes les variantes différentes peuvent être affichées.

    4. Nom : ce nom est utilisé pour afficher le composant.

    5. Identificateur : un identifiant peut être saisi ici.

Toutes ces différentes variantes ont le même nom et le même identifiant. Ils ne sont représentés que par différents modèles 3D. Si le logiciel détecte l’un de ces modèles 3D dans un sous-groupe, ce composant sera marqué avec ce nom et cet identifiant. Il est possible que deux composants différents portant le même nom aient été regroupés de manière incorrecte. Dans une telle situation, le bouton Séparer peut être utilisé pour séparer les variantes spécifiques avec un nouveau nom.

A l’inverse, si deux composants qui avaient initialement des noms différents doivent être regroupés, il suffit de sélectionner l’un d’eux et de le renommer afin qu’il corresponde à l’autre. Les deux sont regroupés en tant que variantes du même composant.

  1. Cliquez sur le bouton Ajouter pour enregistrer le composant dans la base de données pour une utilisation ultérieure.
