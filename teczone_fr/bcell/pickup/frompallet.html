Préhension depuis une palette

Lorsqu’une ventouse ou un préhenseur magnétique est utilisé, les formats bruts (pièces plates) sont prélevés sur une palette. Ces paramètres influencent ce processus :

  • La position de la palette dans la cellule de la machine.

  • La position et l’orientation de la pile de pièces sur la palette.

  • La position et l’orientation du préhenseur à ventouses sur la pièce.

  • La configuration des ventouses du préhenseur (quelles ventouses sont activées/désactivées et quel type de ventouses est monté dans chaque embout).

  • Les zones de la pièce qui sont filmées par la caméra du robot et utilisées comme références pour calibrer la position de la pièce.

Les panneaux utilisés pour modifier tous ces réglages sont affichés ci-dessous. Ils sont tous reliés entre eux par des liens de navigation haut/bas qui mènent à d’autres panneaux dans un ordre logique :

Comme le montre l’image ci-dessus, ces panneaux sont également facilement accessibles en cliquant simplement sur divers objets de la simulation :

  • Pour ouvrir le panneau Palette, cliquez sur une palette.

  • Pour modifier la pile de pièces sur une palette (panneau Prélèvement), cliquez sur les formats bruts de la palette.

  • Pour modifier la position du préhenseur sur le format brut (panneau Préhenseur), cliquez sur le préhenseur.

  • Pour modifier la configuration des ventouses d’un préhenseur (panneau Aspiration), cliquez sur l’une des ventouses.

  • Pour modifier les positions d'imagerie (utilisées par le système d’identification précise), cliquez sur la caméra.

Panneau Palette

Le panneau Palette permet de sélectionner une palette et de la positionner dans la cellule. Vous pouvez ouvrir ce panneau en cliquant simplement sur une palette ; Flux positionnera également la chronologie de simulation de manière à ce que le robot se trouve à l’endroit où il doit prélever la pièce sur la palette.

  • Utilisez le choix Nom pour sélectionner une palette différente. Normalement, seules les palettes utilisables pour ce format brut sont répertoriées, mais vous pouvez désactiver la case à cocher Filtré pour afficher toutes les palettes disponibles.

  • En déplaçant la souris sur un nom dans la liste des noms, vous affichez un résumé rapide de cette palette, accompagné d’une vignette :

  • Utilisez les entrées Position pour positionner la palette en Z et X (en coordonnées de cellule), et utilisez l’entrée Angle pour faire pivoter la palette. Lorsque vous déplacez ou faites pivoter la palette, la pile de pièces sur la palette et le préhenseur/robot suivent tous le mouvement.

  • Utilisez les boutons Suivant et Retour pour naviguer vers les autres palettes de la cellule ; par exemple la palette avec les opérations de dépôt de pièces.

  • Utilisez le bouton de navigation Logement pour modifier la position de la pile de pièces sur la palette.

  • Utilisez le bouton Sauvegarder config. situé sous la section Avancé pour enregistrer cette configuration de cellule (y compris toutes les palettes) comme configuration par défaut pour cette machine.

Panneau Prélèvement

Le panneau Prélèvement est utilisé pour modifier la position de la pile de pièces sur la palette. Vous pouvez ouvrir ce panneau directement en cliquant sur la pile de formats bruts se trouvant sur la palette. (Vous pouvez également y accéder en utilisant le lien Logement du panneau Palette).

  • Utilisez les entrées Position pour positionner la pile sur la palette ; ces coordonnées indiquent le centre de la pièce en Z et en X, par rapport au coin de la palette et se trouvent dans l’espace de coordonnées local de la palette.

  • Utilisez l’entrée Angle pour faire pivoter la pièce dans la palette.

  • Utilisez le commutateur Tourner la pièce pour miroiter la pièce. Notez que cela signifie généralement qu’une reprise de la pièce supplémentaire sera nécessaire avant que le premier pliage puisse être usiné (celui-ci sera ajouté automatiquement par TecZone Bend).

  • Utilisez le lien Caméra…​ pour passer au panneau Caméra, où vous pouvez configurer la phase de reconnaissance d’image du prélèvement.

  • Le lien Points de calibrage…​ affiche l’éditeur Points de passage, où vous pouvez affiner la trajectoire du robot pendant le prélèvement.

  • Le lien Tab. repr. ouvre le panneau Table de reprise des pièces où vous pouvez configurer la position de la table de reprise des pièces pendant l’opération de prélèvement.

Lorsque la pièce est déplacée sur la palette, le préhenseur reste coincé avec la pièce et le robot suit ses mouvements.

Panneau Préhenseur

Le panneau Préhenseur est utilisé pour sélectionner un autre préhenseur ou pour modifier la position et l’orientation dans lesquelles le préhenseur saisit la pièce.

  • Utilisez le choix Nom pour sélectionner un préhenseur différent. Normalement, seuls les préhenseurs adaptés à cette pièce sont affichés (en fonction de la taille du préhenseur et de la charge utile), mais vous pouvez désactiver la case à cocher Filtré pour afficher tous les préhenseurs.

  • En déplaçant la souris sur un nom dans la liste des préhenseurs, vous affichez un résumé de ce préhenseur, accompagné d’une vignette :

  • Utilisez les entrées Position pour déplacer le centre du préhenseur par rapport au centre de la pièce et utilisez l’entrée Angle pour tourner le préhenseur par rapport à l’orientation de la pièce.

  • Utilisez le lien Ventouse pour passer à un réglage précis du préhenseur (sélection de différentes ventouses, activation/désactivation des ventouses).

  • Le bouton Sélectionner la surface de préhension peut être utilisé pour positionner le préhenseur sur un autre plan. Normalement, le plus grand plan du modèle est utilisé pour positionner le préhenseur. Si vous souhaitez modifier cela, cliquez sur ce bouton. Cliquez ensuite sur le plan où le préhenseur doit être positionné :

  • Le bouton Utiliser le préhenseur à pince permet de basculer cette pièce pour utiliser le distributeur de pièces et le préhenseur à mors (préhenseur mécanique). Toutes les étapes du cycle de pliage, depuis le prélèvement jusqu’au dépôt, sont recalculées à l’aide du préhenseur à morsgripper[1].

Avertissements relatifs à l’aspiration

Menu Actions

Le bouton Actions permet d’afficher un menu proposant plusieurs actions utiles sur le préhenseur :

  • Auto-Shift tente de repositionner le préhenseur au-dessus de la pièce, en veillant à ce que toutes les ventouses se trouvent à l’intérieur de la pièce et non au-dessus des ouvertures (si possible).

  • Désactivation de ventouses non étanches : désactive toutes les ventouses qui se trouvent au-dessus des ouvertures ou en dehors des limites de la pièce part[2].

  • Activation de toutes les ventouses : active toutes les ventouses qui sont désactivées.

  • Sauvegarder config. : si vous configurez le préhenseur en désactivant ou en retirant certaines ventouses, ou en modifiant la longueur ou l’angle des bras (pour les multipréhenseurs qui peuvent changer de forme), vous pouvez enregistrer la nouvelle configuration du préhenseur sous un nouveau nom afin de pouvoir la réutiliser facilement.

  • Exporter le préhenseur : enregistre le préhenseur actuel sous forme de fichier .fxbgrip qui peut être importé dans une autre installation de TecZone Bend. Ceci est utile si vous avez importé un préhenseur personnalisé et que vous souhaitez le partager avec d’autres installations.

Panneau Aspiration

Le panneau Aspiration est utilisé pour configurer la configuration des ventouses d’un préhenseur. Vous pouvez ouvrir ce panneau en cliquant directement sur une ventouse, ou en choisissant le lien Aspiration dans le panneau Préhenseur.

  • Utilisez le choix Ventouse # pour sélectionner une ventouse particulière dans le préhenseur ou utilisez les boutons Suivant et Retour pour naviguer parmi les ventouses. La ventouse sélectionnée est surlignée en bleu et peut être modifiée.

  • Pour chaque ventouse, vous pouvez régler Etat sur On, Off ou Supprimée. Consultez la description ci-dessous pour plus de détails à ce sujet.

  • Utilisez le panneau Type pour changer de type de ventouse. En général, on remplace toutes les ventouses d’un préhenseur par le nouveau type, mais vous pouvez également mélanger les ventouses en désactivant le bouton Tout modifier en remplaçant ensuite les ventouses. (Notez que lorsque vous procédez ainsi, le choix des ventouses sera limité, car toutes les ventouses montées sur un cadre de préhenseur doivent avoir la même hauteur de travail). L’image ci-dessus montre deux des ventouses remplacées par des ventouses plus petites (SAXM50, au lieu des ventouses SAXM80 utilisées par défaut).

  • Utilisez le bouton Réinitialiser pour restaurer l’état d’origine du préhenseur : toutes les ventouses sont activées et réinitialisées au type de ventouse par défaut défini dans le préhenseur.

L’état par défaut d’une ventouse est On, ce qui signifie que les ventouses sont connectées à la conduite de vide et facilitent le levage.

Si une ventouse se trouve au-dessus d’une ouverture dans la pièce, vous pouvez passer à l’état Off, ce qui signifie qu’il n’y a pas de vide. (Cela réduit la force de levage du préhenseur et modifie le centre de levage, ce dont Flux tient compte pour vérifier la capacité du préhenseur). Notez que la ventouse est toujours montée dans le cadre et participe aux contrôles anticollision. Flux affiche les ventouses désactivées sous forme de représentation filaire, comme on peut le voir avec deux des ventouses dans l’image ci-dessus.

Enfin, vous pouvez définir une ventouse comme étant Supprimée, ce qui signifie que la ventouse a été retirée du cadre de la machine réelle. Cette ventouse ne se soulève pas et ne provoque pas de collision. Cela s’avère parfois utile si une ventouse tombe sur une zone déformée ou provoque une collision avec la matrice ou la table de machine pendant le fonctionnement.

Panneau Caméra

Le processus de prélèvement nécessite la capture d’une ou plusieurs images par la caméra, puis un système de traitement d’images utilise ces images pour estimer exactement où se trouve la pièce sur la palette. Cliquez sur la caméra montée sur le robot ou sélectionnez le bouton de navigation Caméra dans le panneau Prélèvement pour ouvrir le panneau Caméra. Flux positionne également la simulation de manière à ce que le robot soit dans la pose requise pour capturer l’image :

  • Utilisez la liste Index (ou les boutons Suivant et Retour) pour faire défiler les différentes images d’identification précise utilisées pour cette pièce. Pendant que vous effectuez cette opération, Flux affiche un contour orange qui indique la zone d’identification d’image sur la pièce (voir l’image ci-dessus).

  • Utilisez les entrées Position pour repositionner cette zone en Z et en X afin de mieux intégrer certaines caractéristiques intéressantes susceptibles d’améliorer la précision de l’identification (coins, petites ouvertures, grugeages).

  • Utilisez le bouton Ajouter pour ajouter une image d’identification supplémentaire (jusqu’à 4), et utilisez le bouton Enlever pour supprimer l’image d’identification actuelle. Pendant la simulation du prélèvement de pièce, Flux montre le robot se déplaçant vers chacune de ces zones d’identification avec la caméra vers le bas et s’arrêtant pour acquérir l’image.

Si le commutateur Utiliser une station de lumière est activé, la pièce est transférée sur une table éclairée avant d’être photographiée par la caméra. Cela augmente le contraste et est utile pour les pièces très réfléchissantes. TecZone Bend ajoute automatiquement la table de rétroéclairage et la positionne près de la palette de prélèvement, mais vous pouvez ensuite configurer la position en cliquant dessus :

1. Cette option est disponible uniquement pour les pièces qui chevauchent les plages de dimensions de travail des préhenseurs à ventouses et des préhenseurs à mors
2. Notez que cela peut réduire la force d’aspiration du préhenseur au point où elle ne peut plus saisir la pièce de manière fiable. Ceci est indiqué par une erreur dans la colonne Prélèvement du navigateur.
