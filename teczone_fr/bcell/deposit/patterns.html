Types de modèles de dépôt

Types de modèles de dépôt

Grille simple

Le modèle Grille simple est le plus couramment utilisé et empile simplement les pièces dans une grille rectangulaire sur la palette.

Lorsque nous utilisons la position Grille simple, une seule position de dépôt D1 est disponible par défaut.

Tissage

Le modèle Tissage est utile avec des pièces longues et minces. Les pièces dans les couches D2 sont placées à 90° des pièces dans la couche D1 comme indiqué ci-contre.

  • Le paramètre Tissage contrôle le nombre de pièces empilées côte à côte dans chaque couche du tissage.

  • L'Écart est l’écart entre chacune de ces pièces dans le tissage

  • Les paramètres de Grille dans l’onglet Répétition peuvent ensuite être utilisés pour répéter l’ensemble de la pile de tissage plusieurs fois sur la palette. Dans cet exemple, la pile de tissage est répétée deux fois (2 colonnes, 1 rangée).

  • Le paramètre Gap (Z,X) dans l’onglet Répétition est utilisé lorsque le tissage est répété et spécifie le pas entre les couches de tissage.

Alternance

Le modèle Alternance est le plus flexible. Dans ce modèle, vous pouvez contrôler l’orientation, le retournement et le décalage relatif de chaque couche alternée.

Le modèle alterné offre un empilement des pièces plus stable et également plus compact.

Lorsque nous utilisons le modèle Alternance, les positions de dépôt D1 et D2 ainsi que la Reprise R1 sont disponibles par défaut.

Ajustez les valeurs Delta shift dans D1, D2 et ajoutez une Reprise pour verrouiller les pièces les unes avec les autres.

Alternance par lot

Le modèle Alternance par lot est similaire au modèle d’alternance, mais avec la possibilité d’empiler les composants par lots et de couches alternées.

Le Batch shift est utilisé pour déplacer les pièces vers et depuis les axes Z et X dans un lot.

Le Batch count est utilisé pour définir le nombre de composants par lot par dépôt D1, qui peut aller de 1 à 50. Dans l’exemple ci-dessous, il est défini sur 5.

Le Delta shift est utilisé pour déplacer un dépôt D1 entier composé de lots vers et depuis les axes Z et X.

Utilisez l’option couches dans l’onglet Grille de répétition (R) pour définir le nombre de lots par dépôt. Dans l’exemple ci-dessous, il est défini sur 2.

Drop to Basket

Ce modèle est utilisé pour déposer de petites pièces dans un panier de collecte, en utilisant le préhenseur mécanique.

Convoyeur

Le modèle Convoyeur est utilisé pour déposer des pièces sur un convoyeur de pièces. Cette option est essentielle lors du dépôt de pièces longues et minces sur un convoyeur.

Lorsqu’un préhenseur à mâchoires mécaniques est utilisé pour des pièces de taille moyenne, tous les autres types de modèles seront désactivés pour la sélection, à l’exception du modèle convoyeur.

Inclined

Le modèle Inclined est utilisé pour empiler les pièces verticalement contre un mur (ou une palette qui a une paroi latérale).

Ce modèle n’est pas toujours disponible. La pièce doit être suffisamment grande pour que cela soit réalisable. Pour les petites pièces, le robot ne peut pas descendre assez bas pour empiler la pièce contre le mur.

Inclined pair

Ce type de modèle est très similaire au modèle Alternance, sauf que les pièces sont déposées verticalement en paire.

Une Reprise supplémentaire est introduite soit dans le dépôt D1 soit D2 pour faire une paire.

L'Pas automatique détermine l’angle d’inclinaison de la pile automatiquement.

Le paramètre Pas est utilisé pour ajuster l’angle d’inclinaison de l’empilement et les valeurs Delta shift sont utilisées pour déplacer la pièce dans les axes X et R pour créer une pile plus compacte.

Manuel

Le modèle Manuel est utilisé pour placer chaque couche de pièces avec une orientation légèrement différente.

  • Utilisez le bouton Ajouter pour ajouter une nouvelle pièce sur la couche la plus haute.

  • Utilisez les boutons de navigation à côté de l’onglet Dn pour passer à la couche précédente ou suivante.

  • Les paramètres Delta shift, Rotation et Pas peuvent être utilisés pour ajuster la pièce afin qu’elle repose mieux sur la pièce en dessous. Vous pouvez également utiliser le Delta shift et cliquer sur Pas automatique pour demander à TecZone Bend de calculer un angle de pas optimal.

  • Utilisez le bouton Supprimer pour retirer la pièce la plus haute.

  • Après la création d’une pile manuelle, vous pouvez utiliser les paramètres de l’onglet Répétition (R) pour répéter la même pile sur toute la palette. L’image ci-dessus montre la même pile répétée pour 2 colonnes, 1 rangée avec un écart de 250 mm entre les colonnes.

  • Les paramètres de l’onglet Séquence (S) peuvent être utilisés pour contrôler si les pièces sont empilées By Layer ou By Pile. Si vous utilisez l’empilement pile par pile, toutes les pièces de la première pile sont terminées avant de commencer la deuxième pile.
