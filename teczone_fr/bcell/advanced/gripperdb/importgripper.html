Importer un préhenseur depuis DXF

Un préhenseur à ventouses pour les machines BendMaster peut être importé depuis un fichier DXF. Le fichier DXF représente la vue de dessus du préhenseur (vue de dessus signifie depuis les ventouses, PAS depuis le BendMaster). Les entités POLYLINE dans le fichier DXF fournissent la forme de la plaque du préhenseur et les entités TEXT fournissent des informations supplémentaires sur le préhenseur (comme son nom, son épaisseur et le placement des ventouses).

Les fichiers au format requis (DXF, ARV ou GEO) peuvent être importés dans la base de données de pliage.

Pour importer un préhenseur :

  1. Ouvrez la Database dans TecZone Bend.

  2. Sélectionnez l’option Bend Grippers dans la liste.

  3. Cliquez sur l’option Import…​ et sélectionnez un préhenseur en tant que fichier DXF depuis le répertoire.

Voici à quoi ressemble ce préhenseur après l’importation (montré en deux vues depuis le dessous et le dessus).

Préhenseur de base

Consultez le dessin simple illustré ci-dessous, qui définit un préhenseur et peut être enregistré en tant que fichier DXF pour être réimporté dans l'inventaire des préhenseurs.

Voici quelques points importants sur la façon dont le DXF est dessiné :

  • L’origine du dessin (0,0) correspond au centre du poignet du robot. Vous pouvez voir cela représenté par un point et un cercle dans la figure. (Le point et le cercle ne sont pas vraiment nécessaires pour l’importation du préhenseur, car le centre du préhenseur est implicitement toujours à l’origine).

  • Ce dessin contient également des entités POINT représentant chacun des emplacements des ventouses. Encore une fois, ceux-ci sont principalement pour rendre le dessin plus clair et ne sont pas réellement utilisés par l’importateur de préhenseur (les positions des ventouses sont spécifiées par des entités de texte).

  • La forme réelle de la plaque est dessinée sous forme d’entités POLYLINE fermées. Si vous avez d’autres polylignes à l’intérieur de celle extérieure, elles deviennent des trous dans la plaque.

Préhenseur à deux couches

Des préhenseurs légèrement plus complexes peuvent être modélisés en utilisant un calque BASEPLATE séparé. Créez un calque nommé BASEPLATE dans le fichier DXF, et dessinez des formes POLYLINE fermées dans ce calque. Ensuite, ce calque peut avoir des limites d’extrusion Z séparées spécifiées par la clé BASEPLATEEXTENT. Les outils nécessaires pour créer et modifier des calques sont maintenant ajoutés.

  1. Cliquez sur Related lorsqu’un dessin est en cours de modification.

  2. Sélectionnez le menu Layers pour ajouter de nouveaux calques, les renommer, définir des couleurs et modifier les styles de lignes.

  3. Dans le dessin, cliquer sur une entité vous permet de changer le calque de cette entité.

    Le calque apparaît dans le panneau des entités uniquement s’il existe plusieurs calques dans le dessin.

  4. Après avoir modifié le dessin à l’aide de ces outils, enregistrez le dessin au format DXF. Voici un DXF dessiné en utilisant un tel calque BASEPLATE (affiché en vert dans l’image).

    Pour ce préhenseur, le poignet s’étend de Z=0 à Z=23. Ensuite, la plaque de base (vert) s’étend de Z=23 à Z=40, comme indiqué par la clé BASEPLATEEXTENT. Enfin, la plaque du préhenseur réelle (tenant les ventouses) s’étend de Z=40 à Z=63, comme indiqué par la clé PLATEEXTENT.
    La plaque du préhenseur comporte également des trous hexagonaux.

Préhenseur multicouche

Les Préhenseurs multicouches sont une version mise à jour du Préhenseur à deux couches dans laquelle plus de deux couches peuvent être créées et importées en utilisant TecZone Bend. Pour utiliser un préhenseur multicouche, le DXF devra inclure le texte VERSION = 2. Un exemple de préhenseur multicouche est montré ci-dessous :

Chaque couche doit être définie avec ses étendues comme LAYER_1 = 23 : 35. Les entités dans ce calque seront ajoutées à ces étendues. De même, les ventouses sont définies sous la forme, CupNo = X pos, Y pos, [Type de ventouse, Hauteur de montage, Groupe de ventouse].

Les positions X et Y sont obligatoires.

Différentes définitions possibles pour les ventouses sont expliquées ci-dessous :

CupNo = X pos, Y pos avec le nom de la ventouse spécifié dans CUPNAME comme dans la version précédente
CupNo = X pos, Y pos, Type de ventouse, où chaque type de ventouse est défini séparément (pour les ventouses ovales, les noms varient avec l’angle)
CupNo = X pos, Y pos, Type de ventouse, Hauteur de montage
CupNo = X pos, Y pos, Type de ventouse, Hauteur de montage, Groupe de ventouse

Exemple

CUP1 = 275, 314, SAOB60x30_90grad_ged, 72, 2

Si la hauteur de montage n’est pas spécifiée, l’étendue maximale de la plaque est considérée par défaut. De même, le groupe de ventouses sera défini sur 1, s’il n’est pas défini.

Pour le moment, TecZone Bend ne prend pas en charge différentes hauteurs de montage pour les ventouses.

Préhenseur multicircuit

Vous pouvez créer un préhenseur multicircuit (un préhenseur qui a plusieurs circuits de vide indépendants). Pour un tel préhenseur, chaque ventouse se voit attribuer un numéro de groupe non nul.

Attribuez des groupes de ventouses en utilisant un format spécial pour CUP1POS, CUP2POS, etc. comme indiqué ci-dessous :

CUP1POS = 340,120 #6

CUP2POS = 420,140 #8

La valeur après le # est le numéro de groupe de ventouse pour la ventouse. Dans cet exemple, la ventouse 1 est assignée au groupe de ventouse 6, la ventouse 2 est assignée au groupe de ventouse 8, et ainsi de suite.

Pour qu’un préhenseur multicircuit soit correctement configuré, chaque ventouse doit être assignée à un numéro de groupe de ventouse de cette manière.

Le nombre maximum de ventouses pouvant être utilisées dans un préhenseur est limité à 100. Si vous importez un préhenseur avec plus de 100 ventouses, TecZone affichera un message d’erreur d’échec d’importation.
