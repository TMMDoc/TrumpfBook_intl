DetectLine

La fonction DetectLine est utile lorsque vous coupez de la tôle pré-poinçonnée. Les découpes laser doivent être alignées avec précision sur les bords préexistants de la tôle ou de la pièce, ou sur les ouvertures. Lorsque DetectLine est activé, un système de caméra mesure et détermine la position des tôles inséréessheets[1].

Configuration machine

Pour utiliser la fonction DetectLine, allumez l’interrupteur DetectLine du panneau d’implantation de la machine lors de la configuration de la machine.

Utilisation de DetectLine

En général, vous utiliserez une tôle qui comporte déjà certaines géométries de pré-poinçonnage. Ce type de tôle sera importé à partir d’un fichier DXF. Supposons que nous ayons une tôle qui présente deux ouvertures près du bord gauche. Lorsque vous utilisez cette tôle pour un agencement, vous pouvez voir que TecZone Laser a déjà détecté ces trous pré-poinçonnés et a utilisé l’un des trous et le coin de la feuille comme points de mesure.

Si vous affichez le panneau Layout en cliquant près de la limite de la tôle, vous pourrez y voir le bouton Change Measure Points. En pointant sur le bouton, vous obtenez des informations sur les points de mesure actuellement sélectionnés, qui sont également indiqués par des croix rouges sur la configuration :

En cliquant sur le bouton, vous passez dans un mode où vous pouvez cliquer sur des éléments tels que les coins de la tôle, les centres des trous pré-poinçonnés, les coins des géométries pré-poinçonnés pour déplacer les points de mesure. Il n’y a que deux points de mesure possibles, donc si vous essayez d’en ajouter plus de deux, le point le plus ancien que vous avez créé sera supprimé.

  • Les points de mesure valides sont :

    • Les angles de la tôle (un seul de ces points d’angle peut être utilisé).

    • Les centres de trous circulaires prédécoupés dont le diamètre est inférieur à 100 mm.

    • Les coins à angle droit de tous les trous polygonaux prédécoupés.

  • Le bouton Copy Measure Points permet de copier les mêmes points de mesure dans tous les agencements qui utilisent la même tôle.

Génération de code

Si cette fonction est active, TecZone Laser ajoute un tableau SHEET_MEASURE dans le code, et y fait référence à l’aide de la commande TC_SHEET_MEASURE. Cela permet à la machine de mesurer les trous, les bords ou d’autres caractéristiques utilisées pour configurer cette fonction. Les découpes laser sont ensuite alignées avec précision sur la géométrie ou les bords préexistants de la tôle.

1. Cette fonction sert également à vérifier l’alignement de la position focale du laser
