Laser CAM

Laser CAM

Une tôle plate peut être découpée à partir d’une tôle à l’aide d’une lame laser 2D, avant d’être envoyée pour un usinage ultérieur sur une presse plieuse ou un centre de pliage à plat. Le module de coupe de TecZone est utilisé pour programmer des machines de découpe laser 2D. En général, plusieurs pièces (souvent de types différents) sont découpées dans une tôle unitaire. Ce processus comporte deux étapes :

  • La préparation d’une pièce pour la découpe laser.

  • L’imbrication de plusieurs pièces de ce type sur une tôle.

La section Flux de travail de la pièce traite de la première étape, tandis que la section Flux de travail de la tâche traite de la deuxième.

Premiers pas

Voici quelques ressources de base pour vous aider à démarrer avec TecZone Laser :

  • Tous les modules de TecZone Laser partagent un modèle d’interaction opérateur commun appelé interface Clarity.

  • Vous pouvez installer des machines laser à partir de la base de données des machines.

  • Le point de départ de tous les modules CAM est constitué par les données CAO. Vous pouvez obtenir ces fichiers CAO en important des données 2D ou 3D, ou en les dessinant à l’aide des outils d’esquisse intégrés à TecZone Fold.
