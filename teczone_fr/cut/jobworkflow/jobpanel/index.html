Panneau Tâches

Lorsque vous créez une nouvelle tâche d’imbrication ou que vous ouvrez une tâche d’imbrication existante à partir d’un fichier, le panneau job s’affiche en haut de l’écran :

Le panneau Tâches a plusieurs onglets auxquels vous pouvez accéder à l’aide des raccourcis clavier indiqués dans le tableau ci-dessous.

Onglet Tâche

L’onglet job affiche certaines informations saisies dans la fenêtre de dialogue New Job et fournit des réglages supplémentaires que le moteur d’imbrication peut utiliser lors du calcul de l’imbrication.

  • Techology : vous pouvez sélectionner un autre LTT à utiliser (pour la machine, le matériau et l’épaisseur spécifiés). Les LTT les plus fréquemment utilisés sont répertoriés en premier et sont signalés par une étoile.

  • Parts distance : ce réglage permet de spécifier la distance minimale autorisée entre les pièces (également appelée largeur de pont) lors de l’imbrication.

  • Start corner : spécifie le coin de la tôle à partir duquel l’imbrication commencera.

  • Allow nesting in holes : si cette option est activée, les petites pièces seront imbriquées à l’intérieur des ouvertures des pièces plus grandes, lorsque cela est possible.

  • Spread out parts on last layout with no remainder : si cette option est activée, TecZone Laser prendra la dernière tôle de l’imbrication qui dispose d’un espace insuffisant pour être utilisée comme tôle entamée et répartira les pièces sur cette tôle afin de minimiser les problèmes de dissipation de la chaleur (puisqu’il n’y a de toute façon aucune réduction de l’efficacité de l’imbrication).

  • Avoid placing thin long parts along slats : si cette option est activée, les pièces longues et fines seront placées en travers des barres support plutôt que le long de celles-ci, afin de minimiser le risque que les pièces tombent entre les barres support.

  • Minimize number of different layouts : si cette option est activée, TecZone Laser tentera de reproduire les agencements plusieurs fois (dans les limites imposées par la quantité de pièces requise) afin de minimiser le nombre d'agencements différents créés. // AVREVIEW : Why?

Modification du matériau / de la machine

Vous pouvez cliquer sur le lien Change material/machine dans l’onglet Job pour modifier la machine et le matériau utilisés pour l’imbrication. Cela permettra de re-préparer toutes les pièces pour la nouvelle machine et le nouveau matériau, puis de refaire l’imbrication.
