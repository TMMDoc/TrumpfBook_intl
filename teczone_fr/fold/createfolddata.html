Création de données de plis écrasés

Vous pouvez créer des données de technologie de plis écrasés (préparation de centre de pliage à plat) pour une pièce en tôle à l’aide du module Pliage. Supposons que vous ayez déjà installé une machine et que vous avez importé une pièce CAO (2D ou 3D) :

Passage à Fold CAM

Maintenant, la pièce peut être préparée pour un centre de pliage à plat en appuyant simplement sur la touche F. TecZone Fold affiche un menu dans lequel vous pouvez sélectionner l’un des centres de pliage à plat installés. La pièce est ensuite préparée pour cette machine. Les séquences de pliage, la composition du serre-flan supérieur, le jaugeage et tous les outils de pliage supplémentaire ou outils serre-flans escamotables optionnels sont tous calculés, et vous devriez obtenir un résultat similaire à celui-ci : image::TooledUpPart.png[Tooled-up Part,967]

S’il n’y a pas d’erreurs, le programme CN est généré pour la pièce et enregistré dans le dossier de sortie qui a été configuré pour cette machine.

Voir également le panneau Flux de travail, accessible en appuyant sur la touche W. Cela permet un meilleur contrôle de la création des données relatives à la technologie de pliage et offre davantage d’options pour le cheminement de la pièce via le Laser CAM, l’imbrication, etc.

Autres opérations

Voici un bref résumé de certaines des opérations que vous pouvez effectuer une fois qu’une pièce a été préparée pour le pliage. Les pages suivantes de ce manuel fournissent plus de détails sur chacune de ces opérations.

  • Si vous cliquez sur Spacebar, une simulation de centre de pliage à plat démarre. Vous pouvez également utiliser les commandes de simulation du navigateur de plis écrasés pour démarrer, arrêter ou revenir en arrière dans la simulation :

  • Si des avertissements ou des erreurs apparaissent, ils s’affichent dans le navigateur de plis écrasés, et vous pouvez les consulter et les résoudre en cliquant sur les cellules correspondantes.

  • Vous pouvez modifier la préparation de type pli écrasé (serre-flan supérieur, jaugeage, outils serre-flans escamotables et outils de pliage supplémentaire, préhenseurs).

  • Vous pouvez changer de machine en cliquant sur le nom de la machine dans l’onglet situé sous la pièce, puis en sélectionnant une autre machine :

Lorsqu’un programme de pli écrasé est créé, s’il n’y a pas d’erreurs, le programme CN de la pièce est alors généré. Vous pouvez contrôler ce comportement (si le programme CN est généré automatiquement) à partir de la page Sorties de plis écrasés de la fenêtre de dialogue Réglages. Si du code a été généré, le bouton Code CN dans la barre d’outils de gauche est grisé :

Lorsque vous apportez des modifications à la pièce, ce bouton redevient actif et vous pouvez régénérer le programme CN en cliquant à nouveau sur cette icône (ou en appuyant sur la touche C).
