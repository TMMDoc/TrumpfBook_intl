Modification d'un pliage

Certains réglages de base d’une phase de pliage peuvent être modifiés en faisant apparaître le panneau de pliage. Pour faire apparaître le panneau de pliage, utilisez le navigateur de plis écrasés :

  • Cliquez sur l’un des numéros dans le navigateur de plis écrasés pour passer à un pliage.

  • Cliquez à nouveau sur le même numéro pour faire apparaître le panneau de pliage pour ce pliage.

Le panneau de pliage est visible ci-contre. Cela comprend certains réglages et opérations pour travailler sur le pliage.

  • La zone de saisie Entrefer permet de régler l’entrefer pour le pliage (il s’agit de l’espace entre la lame et les serre-flans supérieurs). Lorsque l’entrefer est réglé à partir de sa valeur par défaut, le navigateur de pli écrasé affiche un avertissement d'entrefer non standard. Le contrôle anticollision sera également effectué immédiatement avec l’entrefer mis à jour et vous pourrez régler l’entrefer pour éviter les collisions. La force de compression sera également calculée, et si vous réglez l’entrefer à une valeur trop faible, de sorte que la force de compression maximale soit dépassée, le navigateur de plis écrasés affichera cela comme une erreur.

  • Le réglage P86 permet de contrôler l’application d’une pression de maintien supplémentaire pendant le cycle de pliage.

  • Utilisez le Sauter le pliage pour ignorer ce pliage ; ce pliage ne sera pas usiné par le centre de pliage à plat et devra être usiné ultérieurement (éventuellement à l’aide d’une presse plieuse).

  • Le bouton Nouveau côté permet de diviser un bord de pli en deux bords de pli distincts ; il n’apparaît que si vous ne modifiez pas le premier pliage d’un bord de pli. Cliquez sur ce bouton pour diviser le bord de pli au niveau de ce pliage et commencer un nouveau bord de pli. Il existe plusieurs raisons pour lesquelles vous pouvez vouloir diviser un bord de pli :

    • Il peut être nécessaire de diviser le bord de pli en deux et de faire miroiter la pièce entre les deux bords de pi avant de pouvoir terminer l’usinage (uniquement sur les machines manuelles).

    • Il peut être nécessaire de réaliser certains pliages d’un autre bord de pli avant de pouvoir terminer les pliages restants de ce bord de pli. Dans ce cas, vous pouvez diviser le bord de pli et réorganiser les bords de pli afin que les pliages de l’autre côté soient insérés entre les deux.

  • Les boutons Retour et Suivant permettent de naviguer entre les différents pliages de la pièce. (Vous pouvez également utiliser les touches PageUp et PageDown.)

  • Le bouton Côté permet de remonter de cette section développée vers la page contenant cette section. Cliquez sur ce bouton pour afficher le panneau Bord de pli (décrit ci-dessous) qui permet d’effectuer certaines opérations sur l’ensemble du bord de pli (ou de la section).

Si ce pliage a été ignoré, ces réglages ne s’affichent pas ; seul un bouton Ne pas sauter le pliage permet de réintégrer le pliage dans l’usinage.
