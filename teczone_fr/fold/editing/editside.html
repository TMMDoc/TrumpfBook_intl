Modification d'un bord de pli / d'une section

Il existe quelques réglages et opérations liés à un bord de pli complet (par opposition à un pli individuel). Ceux-ci sont accessibles en ouvrant le

Panneau Bord de pli :

  • Ouvrez un panneau Pliage (en cliquant deux fois sur un nombre dans le navigateur de plis écrasés). Cliquez ensuite sur le bouton Bord de pli pour ouvrir le panneau Bord de pli pour le bord de pli qui contient ce pliage.

  • Maintenez Ctrl enfoncé et cliquez sur un numéro dans le navigateur de plis écrasés pour ouvrir le panneau Bord de pli correspondant à ce bord de pli.

Le panneau bord de pli ressemble à celui illustré ci-contre (les options disponibles dans la fenêtre contextuelle Modifier la séquence peuvent varier en fonction de la pièce).

  • Cliquez sur le bouton Retourner la pièce pour miroiter tout le bord de pli. Cela entraînera le recalcul immédiat des lames, des serre-flans supérieurs et de la manipulation des pièces. Sur une machine automatique, cette fonction est remplacée par le bouton Tourner la pièce button[1], car la machine ne peut pas miroiter qu’un seul bord de pli.

  • Cliquez sur le bouton Editer séquence de pliages pour afficher un sous-menu qui présente les permutations possibles de séquence que vous pouvez effectuer avec ce bord de pli. Dans l’image ci-dessus, nous travaillons avec une pièce qui présente une séquence de bord de pli telle que C1 C2 L1 L2 et nous avons sélectionné le bord de pli L1. Les choix possibles pour la séquence sont expliqués ci-dessous :

  • << L1+L2 plus tôt déplace L1 et L2 afin qu’ils se produisent avant C1 et C2. Ainsi, la nouvelle séquence serait L1 L2 C1 C2.

  • << L1 plus tôt déplace L1 d’une étape en arrière, de sorte que la nouvelle séquence serait C1 L1 C2 L2. Bien que cela ne soit pas très courant, cela peut s’avérer utile dans certaines situations.

  • L1 plus tard >> déplace le bord de pli L2 plus tard, ce qui donne une séquence qui serait C1 C2 L2 L1 (bien sûr, les bords de pli sont immédiatement renommés après cela, de sorte que la séquence se lit toujours C1 C2 L1 L2 lorsque le code est généré).

L’ensemble des choix disponibles dans ce menu dépend de la séquence actuelle et du bord de pli que vous sélectionnez pour la réorganisation.

  • Cliquez sur les boutons Retour ou Suivant pour naviguer entre les différents bords de pli de cette pièce.

  • Cliquez sur le bouton Pliages pour descendre depuis les bords de pli vers l’ensemble de pliages de ce bord de pli (c’est l’inverse du bouton Côté du panneau Pliage qui vous permet de remonter depuis un pliage vers le bord de pli qui contient le pliage).

1. Cette fonction est beaucoup moins utile que le bouton Miroiter bord de pli d’une machine manuelle et n’existe que pour gérer certaines situations rares.
