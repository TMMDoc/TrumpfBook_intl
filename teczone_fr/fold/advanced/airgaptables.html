Tableaux d'entrefer personnalisés

Tableaux d'entrefer personnalisés

Le tableau d’entrefer pour centre de pliage à plat permet de faire correspondre une épaisseur tôle donnée à l’entrefer standard utilisé pendant l’usinage. Cette fonction est intégrée à chacun des différents centres de pliage à plat, mais peut être désactivée en fournissant un tableau d’entrefer personnalisé sous forme de fichier JSON. Ce fichier JSON s’appelle air-gap-tables.json et se trouve dans le répertoire C:\ProgramData\Trumpf{APPNAME}\Data\System.

Vous trouverez ci-dessous un fichier air-gap-tables.json type :

 {
  "version":1  "airGapTables": [
    {
      "machine": "112002,112003", "material": "DC01"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
     {
      "machine": "112003", "material": "*"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
    {
      "machine": "*", "material": "Alluminio"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    }
  ]
}

  • Le fichier contient une série de valeurs d’entrefer. Chaque tableau comporte une clé machine et une clé matériau qui font office de sélecteurs.

  • Ces clés sont soit définies sur * (astérisque) pour indiquer un caractère générique (correspond à tout), soit une liste séparée par des virgules de valeurs de machine ou de matériel.

  • Dans l’exemple ci-dessus, il y a trois tableaux. Le premier est utilisé pour les machines 112001 ou 112003, uniquement pour le matériau DC01. Le deuxième est utilisé pour tous les matériaux, pour la machine 112003, et le troisième est utilisé pour toutes les machines, pour le matériau Alluminio.Si plusieurs tableaux correspondent à la combinaison machine/matériau de la pièce, le premier tableau correspondant est utilisé. Ainsi, pour la machine 112003, le matériau DC01, c’est le premier tableau qui est utilisé (même si le deuxième correspond également).

  • Chaque tableau est constitué d’un simple tableau de clés d’épaisseur, avec les valeurs d’entrefer correspondantes. Par exemple, la première entrée ci-dessus doit être interprétée comme suit : si l’épaisseur est inférieure ou égale à 0,75 mm, utilisez une fente de 2,5 mm. Les clés d’épaisseur doivent être classées par ordre croissant. Si aucune des tableaux de ce fichier ne correspond à la combinaison matériau/épaisseur de la pièce actuelle, le tableau d’entrefer par défaut de la machine sera utilisé.

Si vous apportez des modifications au fichier air-gap-tables.json, vous devez redémarrer TecZone Fold pour que ces modifications prennent effet.
Si le fichier JSON est mal formaté, une erreur s’affiche lorsque vous essayez d’usiner une pièce de centre de pliage à plat, mais TecZone Fold revient alors au tableau d’entrefer par défaut de la machine et continue.
