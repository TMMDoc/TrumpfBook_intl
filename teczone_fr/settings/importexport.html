Importation/exportation

Importation/exportation

Dans cette section, nous aborderons la configuration des Import settings.

Cliquez sur l’icône Settings sur la page d’accueil.

Import settings

Dans cette section, nous aborderons la configuration des Import settings. Cliquez sur l’icône Settings. Cliquez sur Import/Export et accédez aux Import settings.

Units for DXF files - Réglez ici en millimètres ou en pouces.

Stitch together lines/arcs closer than this - Réglez cette valeur (0>1mm). Lors de l’importation d’une pièce comportant des lignes/arcs plus proches que la valeur définie, le logiciel les assemblera automatiquement lors de l’importation.

Maximum thickness for sheet-metal part - Pour reconnaître une grande pièce en tôle, il faudrait augmenter la valeur seuil de reconnaissance de la tôle. (10>40mm) Cette valeur est automatiquement définie par défaut sur 25 mm ou un pouce, selon l’unité utilisée.

Point entities - L’option sélectionnée déterminera la manière dont les points seront importés.

Import all - Tous les points seront importés et affichés.

Skip points on polylines - Cela ignore tous les points détectés sur les polylignes.

Skip all - Cela ignore tous les points à l’importation, et aucun n’est affiché.

Ignore layers in DXF/DWG files - Les dessins DXF et DWG sont généralement créés sur différents calques. Avec ce réglage, le logiciel ignore ces calques et déplace tous les objets sur le calque par défaut.

Explode blocks in 2D drawing - Divise les groupes lors de l’importation

Convert white entities to black - Activez ce réglage pour convertir des objets blancs en objets noirs pendant l’importation.

Darken colors during DXF import- Activez ce réglage pour assombrir les objets colorés pendant l’importation.

Remove duplicate segments - Activez ce réglage pour supprimer toute géométrie en double trouvée dans la pièce lors de l’importation.

DXF settings

Dans cette section, nous aborderons la configuration des DXF settings. Cliquez sur l’icône Settings. Cliquez sur Import/Export et accédez aux DXF settings.

Angles in DXF are interior angles - Activez ce réglage pour que les angles d’un fichier DXF soient usinés comme des angles d’ouverture.

Export settings

Dans cette section, nous aborderons la configuration des Export settings. Cliquez sur l’icône Settings. Cliquez sur Import/Export et accédez aux Export settings.

No POLYLINE objects in DXF output - En général, les contours fermés sont exportés sous forme de polylignes lors de l’exportation de fichiers DXF. Certains systèmes CAO ne peuvent pas exécuter cette sortie. Avec ce réglage, le logiciel génère le fichier DXF avec des lignes et des arcs. Ces fichiers sont lisibles partout, mais les fichiers créés sont plus volumineux et les connexions entre les lignes et les arcs sont perdues

Output bend-info when saving DXF files - Activez ce réglage pour qu’un fichier DXF exporté soit généré avec des paramètres de pliage.

Bend-info in Starmatik format - Activez ce commutateur pour afficher les paramètres de pliage au format Starmatik. C’est ici qu’une entité de texte est positionnée exactement au milieu de chaque ligne qui doit être une ligne de pliage.

Convert black to gray on output - Lors de l’exportation de données 2D, les objets du fichier DXF sont affichés en noir par défaut. Afin de pouvoir mieux reconnaître les objets dans les programmes de CAO, ceux-ci sont affichés en gris avec ce réglage.

Start MetaCAM when PDG files are exported - Activez ce réglage pour qu’un fichier PDG soit automatiquement configuré pour s’ouvrir dans MetaCAM

Format for flat pattern - Lors de l’exportation d’un modèle plat, vous pouvez choisir le format de fichier GEO, DXF ou PDG.

Spline conversion

Dans cette section, nous aborderons la configuration des réglages Spline conversion. Cliquez sur l’icône Settings. Cliquez sur Import/Export et accédez aux réglages Spline conversion.

Convert splines on import - Définissez ici si la conversion des splines est désactivée ou si les splines doivent être converties en lignes ou en arcs. Dans les deux cas, chaque spline sera convertie en un seul objet polyligne contenant des segments de ligne ou des segments d’arc.

Node-count computation - Le nombre de lignes ou d’arcs générés est calculé à l’aide de l’un des deux mécanismes suivants : pas ou déviation.

Length of each line or arc segment - Si Pas est sélectionné dans le calcul du nombre de nœuds, définissez ici la longueur de chaque segment de ligne ou d’arc afin de décomposer la spline en utilisant cette longueur de pas.

Maximum deviation during approximation - Si Déviation est sélectionné dans le calcul du nombre de nœuds, définissez ici la déviation maximale autorisée entre la spline lisse d’origine et l’approximation en ligne ou arc. La polyligne est construite avec le moins de segments possible, tout en maintenant l’erreur maximale dans cette limite.

Assembly Explode

Dans cette section, nous aborderons la configuration des réglages Assembly Explode. Cliquez sur l’icône Settings. Cliquez sur Import/Export et accédez aux réglages Assembly Explode

Skip sheet-metal parts with no bends - Lors de l’éclatement d’un sous-groupe, l’activation de ce commutateur affiche uniquement les pièces en tôle avec des lignes de pliage. La désactivation affichera toutes les pièces du sous-groupe.

Components (nuts/bolts etc) to display - Utilisez ce réglage pour sélectionner les options à afficher lors de l’éclatement d’un sous-groupe contenant d’autres composants.

None - Seules les pièces en tôle sont affichées après l’éclatement, les autres écrous/boulons ne le sont pas.

Unidentified - Seuls les composants qui n’ont pas encore été détectés dans le logiciel sont affichés.

All - Tous les composants sont affichés.

Layer mapping

Dans cette section, nous aborderons la configuration du Layer mapping. Cliquez sur l’icône Settings. Cliquez sur Import/Export et accédez aux réglages Layer mapping.

Dans cette section, les calques utilisés sur les pièces importées dans le logiciel peuvent être automatiquement mappées à leur fonctionnalité (utilisation).

Exemple

Si une pièce importée comporte un calque MARK, celle-ci peut être configurée pour utiliser automatiquement le calque Mark dans le logiciel.

Layer name - Il s’agit du nom du calque qui, si une pièce est importée avec celui-ci, utilisera les fonctionnalités définies dans le panneau « Utilisation ».

Use - Il s’agit de la fonctionnalité du calque. Les différentes options disponibles sont :

Standard - Il s’agit du calque standard à utiliser pour la CAM.

Auxiliary - Un calque auxiliaire, à ne pas utiliser pour la CAM.

Mark - Toutes les entités de ce calque seront marquées, non coupées.

Approach marker - Entités de point indiquant la position d’approche du laser.

Sequence marker - Indicateurs de texte indiquant l’ordre de séquence des contours.

Forming center - Marquage central pour la déformation (point ou petit L).

Forming foot print - Contour (empreinte) d’une déformation.

Evaporate - Calque utilisé pour distinguer la combustion du film de protection.

Dot marking - Calque utilisé pour les codes QR.

Info - Calque d’informations uniquement.
