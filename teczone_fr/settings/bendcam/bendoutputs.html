Sorties de pliage

Réglages plan de travail

Template pour le plan de réglage - Il s’agit de l’emplacement et du nom du modèle à utiliser pour l’affichage des rapports.

Emplacement d’enregistrement du plan de réglage - Définissez cela comme emplacement de sauvegarde de tous les fichiers de rapport.

Format de sortie du plan de réglage - Cela peut être défini au format HTML ou PDF. Remarque : pour pouvoir utiliser le contenu multimédia et 3D du fichier PDF, Adobe Acrobat Reader doit être installé.

Créer un plan de réglage avec programme de pliage - Activez cette option pour générer automatiquement un rapport lorsque vous passez en mode pliage et qu’un programme a été trouvé. Si le programme détecte des erreurs ou des avertissements, le rapport ne sera pas généré.

Coller le plan de travail dans le programme CN si possible - Activez cette fonction pour joindre des rapports au programme CN.

Réglages code

Emplacement d’enregistrement du fichier CN - Définissez cela comme emplacement de sauvegarde de tous les fichiers de programme CN.

Postprocesseur pour le fichier CN - Il s’agit de l’emplacement et du nom d’un fichier Batch ou d’un outil d’autres fabricants auquel se référer lors de la génération du programme CN.

Créer un fichier CN avec programme de pliage - Activez cette option pour que le programme CN soit généré automatiquement lors du passage en mode Pliage en vé (à condition qu’un programme valide soit généré sans erreur).

Ecrire un graphique dans le fichier CN - Il s’agit d’une option B23 qui vous permet de comparer les graphiques BNC-SAT avec la pièce réelle. Le graphique BNC-Sat s’affiche sur un moniteur. Une image de la pièce réelle s’affiche sur le même moniteur. L’opérateur peut désormais déplacer, faire pivoter ou tourner la pièce jusqu’à ce que les graphiques BNC et les images de la pièce réelle se superposent. Cela aide l’opérateur à voir si la pièce est correctement positionnée pour le pliage.

Simplifier graphique (effacer petits trous) - Par défaut, cette option est activée (et les graphiques BNC sont simplifiés). Cependant, cette option peut être désactivée. Si elle est désactivée, toutes les ouvertures sont affichées dans le BNC.

Code de verrouillage (pas de nouveau calcul par l’IHM) - Activez cette option pour verrouiller le code. Cela garantit que le contrôleur ne modifie pas le code et n’effectue aucun nouveau calcul.

Format de sortie du développement - Cet élément peut être défini comme GEO, DXF ou PDG pour l’option d’exportation de fichier par défaut lors de l’enregistrement automatique d’un d’un modèle plat. Lorsqu’un programme est trouvé pour une pièce, le modèle plat peut être automatiquement exporté et enregistré dans le dossier de destination des modèles plats. Toutefois, si des erreurs ou des avertissements sont détectés, le modèle plat devra être enregistré manuellement.

Chemin de sortie du développement - Définissez cela comme emplacement de sauvegarde de tous les modèles plats.

Générer un fichier FX avec le programme CN - Activez cette option pour enregistrer un fichier FX à l’emplacement choisi lors de la génération du programme CN.

Chemin de sortie du fichier FX - Définissez cela comme emplacement de sauvegarde pour tous les fichiers FX.
