Secuencia

Laser Sequence

Laser Processing Sequence - Utilícela para definir la secuencia de mecanizado por láser. Resulta útil si tiene alguna penetración o marcado que tiene que mecanizarse primero.

Standard - No se obliga a ninguna secuencia específica entre el marcado y el corte.

Marcar y luego cortar - Primero realiza todo el marcado y después el corte.

Penetrar y luego cortar - Divide todo el equipamiento láser en fase de trabajo de penetración y después fase de trabajo de corte. Primero se efectúan todas las penetraciones, seguidas de las fases de trabajo de corte/marcado.

Mark → Pierce → Cut - Igual que lo anterior, pero primero se procesan todos los marcados.

Pierce → Mark → Cut - Igual que la tercera opción, pero entre cortar y marcar se efectúa el marcado.

Laser Sequence - Este desplegable da opciones para escoger la secuencia láser.

Partwise - Si se selecciona esta opción, la secuenciación se realiza por piezas. En cuanto se inician las fases de trabajo en una pieza, no se procederá con otra pieza hasta que finalicen todas las fases de trabajo sobre esa pieza. No hace falta que la secuencia sea la misma dentro de cada pieza en todas las instancias del layout. En función de la procedencia del cabezal de la máquina y de su destino siguiente, el orden dentro de una pieza puede variar de una instancia a otra de la misma pieza.

Partwise native - Si se selecciona esta opción, la secuenciación se realiza por piezas. En cuanto se inician las fases de trabajo en una pieza, se procederá con otra pieza solo después de finalizar todas las fases de trabajo sobre esa pieza. La secuencia dentro de una pieza y todas sus instancias será la misma que el equipamiento nativo de la pieza.

Inner first - Todo el utillaje en contornos interiores se secuencia primero seguido de los contornos exteriores.

Sort by contour size - Todo el equipamiento en los contornos interiores se mecaniza antes que los contornos exteriores y se clasifica en base al tamaño de contorno.

Do pre-pierce part-by-part - Las operaciones de penetración se harán por piezas. Una vez que se ha hecho toda la penetración en una pieza, finalizará el corte antes de proceder a la siguiente pieza.

Route Traverse

Move pierce points to reduce traverse - Esta casilla de verificación es para elegir una ubicación de punto de aproximación diferente para reducir la distancia transversal.

Move pierce points to prevent tilting - Esta casilla de verificación es para elegir una ubicación de punto de aproximación diferente para evitar el volcado de los recortes.

Microjoint nested holes if tilting/falling through - Esta casilla de verificación es para hacer una microjoint de un contorno con piezas con nesting dentro de las mismas, únicamente si el contorno se evalúa como caído o inclinado.

Ignore holes smaller than this - Las perforaciones inferiores al valor dado no se consideran lo suficientemente grandes como para que el sensor de altura sumerja el cabezal de la máquina en ellas, por lo que se ignoran.

Minimum cutting head height when traversing - Distancia de cabezal mínima desde la chapa al realizar un desplazamiento con el cabezal hacia arriba.

Max. head down traverse distance - El software calcula automáticamente un límite transversal con cabeza hacia abajo basado en distancia de boquilla desde la chapa. Aquí se utilizarán el menor valor calculado y el valor introducido aquí.

Route traverse lines around holes - Activar para enrutar el cabezal láser alrededor de recortes si fuera posible, en lugar de elevar el cabezal hacia arriba.

Allowance when routing around holes - Al enrutar líneas para evitar recortes, esta distancia se mantendrá desde el carril de desplazamiento transversal hasta el borde de un corte. Si este valor es demasiado elevado para trazar líneas transversales alrededor de recortes, entonces esas líneas transversales se mecanizarán con el cabezal de la máquina levantado.

Lift nozzle if routing penalty more than - Esta opción se utiliza para definir si la boquilla debería levantarse si la penalización por trazado es superior a este valor.

Allowance when routing around tilting holes - Esta opción es para definir la tolerancia de enrutamiento alrededor de los orificios de inclinación.

Always reposition with head-up between holes - Esta opción debería comprobarse si los subprogramas deberían estar seguros para postproducción.

Laser heads

Esta sección es aplicable cuando se selecciona la opción relevante.

Cut with single head - Esta opción permite a los cabezales láser duales cortar con un cabezal láser simple.
