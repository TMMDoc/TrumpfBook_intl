Importar/Exportar

Importar/Exportar

En esta sección, cubriremos la configuración de Import settings.

Haga clic en el icono Settings de la página de inicio.

Import settings

En esta sección, cubriremos la configuración de Import settings. Haga clic en el icono Settings. Haga clic en Import/Export y navegue a Import settings.

Units for DXF files - Seleccione aquí entre milímetros o pulgadas.

Stitch together lines/arcs closer than this - Establezca este valor (0>1mm). Al importar una pieza que tiene líneas/arcos más cerca que el valor preestablecido, el programa los unirá automáticamente al importarlos.

Maximum thickness for sheet-metal part - Para reconocer una pieza de chapa grande, el umbral de reconocimiento de chapa debería incrementarse. (10>40 mm) Se ajusta automáticamente de forma predeterminada en 25 mm o una pulgada, dependiendo de la unidad utilizada.

Point entities - En función de la opción seleccionada se determinará cómo se van a importar los puntos.

Import all - Se importarán y mostrarán todos los puntos.

Skip points on polylines - Esto omite todos los puntos que se detectan en las polilíneas.

Skip all - Esto ignora todos los puntos de la importación y no se muestra ninguno.

Ignore layers in DXF/DWG files - Los dibujos DXF y DWG se crean, por lo general, en diferentes niveles. Con este ajuste, el programa ignora estos niveles y desplaza los objetos al nivel predeterminado.

Explode blocks in 2D drawing - Divide las agrupaciones durante la importación

Convert white entities to black - Active este ajuste para convertir objetos blancos en negros durante la importación.

Darken colors during DXF import- Active este ajuste para oscurecer objetos de color durante la importación.

Remove duplicate segments - Active este ajuste para eliminar cualquier geometría duplicada hallada en la pieza durante la importación.

DXF settings

En esta sección, cubriremos la configuración de DXF settings. Haga clic en el icono Settings. Haga clic en Import/Export y navegue a DXF settings.

Angles in DXF are interior angles - Active este ajuste para que los ángulos en un DXF se traten como ángulos de apertura.

Export settings

En esta sección, cubriremos la configuración de Export settings. Haga clic en el icono Settings. Haga clic en Import/Export y navegue a Export settings.

No POLYLINE objects in DXF output - Habitualmente, los contornos cerrados se generan como polilíneas al exportar a formato DXF. Algunos sistemas CAD no pueden procesar estas salidas. Con este ajuste, el programa emite los DXF con líneas y arcos circulares. Los archivos se pueden leer en cualquier lugar, pero los archivos creados son más grandes y las conexiones entre las líneas y los arcos circulares se pierden

Output bend-info when saving DXF files - Active este ajuste para que un DXF exportado salga con información del plegado.

Bend-info in Starmatik format - Active este interruptor para extraer la información de plegado en formato Starmatik. Aquí es donde se posiciona una entidad de texto de forma exacta en el centro de cada línea que va a ser una línea de plegado.

Convert black to gray on output - Al exportar datos en 2D, los objetos del archivo DXF salen en negro por defecto. Para poder reconocer mejor los objetos en los programas CAD, los objetos salen en gris con este ajuste.

Start MetaCAM when PDG files are exported - Active este ajuste para que un archivo PDG se abra automáticamente en MetaCAM

Format for flat pattern - Al exportar un patrón plano, este se puede ajustar en formato de archivo GEO, DXF o PDG.

Spline conversion

En esta sección, cubriremos la configuración de ajustes Spline conversion. Haga clic en el icono Settings. Haga clic en Import/Export y navegue a los ajustes Spline conversion.

Convert splines on import - Determine aquí si la conversión de spline se desactiva o se activa para convertir las splines en líneas o arcos circulares. En cualquier caso, cada spline se convertirá en un objeto polilínea individual que contiene segmentos de línea o segmentos de arco circular.

Node-count computation - El número de líneas o arcos circulares que se generan se calcula utilizando uno de los dos mecanismos siguientes: paso de rosca o desviación.

Length of each line or arc segment - Si se escoge el paso de rosca en el cálculo del recuento de nodos,ajuste aquí la longitud para cada segmento de arco circular o de línea para descomponer la spline utilizando esta longitud de paso.

Maximum deviation during approximation - Si se escoge la desviación en el cálculo de recuento de nodos, ajuste aquí la desviación máxima que se permite entre la spline suavizada original y la aproximación a la línea o al arco circular. La polilínea se construye con la menor cantidad de segmentos posibles, mientras se mantiene el error máximo dentro de este límite.

Assembly Explode

En esta sección, cubriremos la configuración de los ajustes de Assembly Explode. Haga clic en el icono Settings. Haga clic en Import/Export y vaya a los ajustes de Assembly Explode

Skip sheet-metal parts with no bends - Al despiezar un grupo de módulos, si activa este interruptor visualizará solo las piezas de chapa con líneas de plegado. Si lo desactiva, visualizará todas las piezas del grupo de módulos.

Components (nuts/bolts etc) to display - Utilice este ajuste para seleccionar qué opciones se mostrarán al despiezar un grupo de módulos que contenga otros componentes.

None - Solo se muestran las piezas de chapa tras el despiece, pero no otras tuercas o pernos.

Unidentified - Solo se muestran componentes que todavía se deben identificar en el programa.

All - Se muestran todos los componentes.

Layer mapping

En esta sección, cubriremos la configuración de Layer mapping. Haga clic en el icono Settings . Haga clic en Import/Export y navegue a los ajustes de Layer mapping.En esta sección, los niveles que se utilizan en piezas que se importan al programa pueden mapearse automáticamente según su funcionalidad (uso).

Ejemplo

Si una pieza importada tiene un nivel MARCA, este se puede ajustar para que utilice automáticamente el Nivel marca en el programa.

Layer name - Este es el nombre del nivel que, en caso de importar una pieza con ese nivel, utilizará la funcionalidad ajustada en el panel "Uso".

Use - Esta es la funcionalidad de la capa. Las diferentes opciones disponibles son:

Standard - Se trata del nivel estándar que se utilizará para CAM.

Auxiliary - Un nivel auxiliar; no se utilizará para CAM.

Mark - Cualquier entidad en este nivel se marcará, no se cortará.

Approach marker - Entidades de punto que indican la posición de avance del láser.

Sequence marker - Marcadores de texto que indican la secuencia de los contornos.

Forming center - Marca de centro para conformado (un punto o una L pequeña).

Forming foot print - El borde (la huella) de un conformado.

Evaporate - Este nivel se utilizará para distinguir la erosión de la lámina.

Dot marking - Este nivel se utilizará para los códigos QR.

Info - Es un nivel solo informativo.
