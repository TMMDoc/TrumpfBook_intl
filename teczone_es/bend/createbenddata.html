Crear datos de plegado

Puede crear datos de tecnología de plegado para una pieza de chapa utilizando el módulo Plegado. Esto también se llama preparar la pieza para el plegado. Supongamos que ya ha instalado una máquina, y que ha configurado el inventario de útiles, y que ha importado una pieza CAD (2D o 3D):

Cambio a CAM de plegado

Ahora, la pieza se puede preparar para plegarse con solo pulsar la tecla B. La pieza se prepara con la máquina de plegado predeterminada, que es la prensa plegadora que ha utilizado más recientemente. Se calcula una secuencia preparación, configuración de plegado, posiciones de calibración, y se debería ver un resultado como este:

Si no hay errores, el programa CN (y posiblemente la hoja de configuración) para la pieza también se generan y se guardan en la carpeta de salida que está configurada para esta máquina.Aquí hay una sección de una configuración de pieza típica sheet[1] (también conocida como informe de plegado).

Consulte también el panel Flujo de trabajo, al que se puede acceder pulsando la tecla W. Esto proporciona un mayor control sobre la creación de datos de tecnología de plegado y opciones para el enrutamiento de la pieza a través de CAM láser o nesting.

Otras operaciones

A continuación se ofrece un breve resumen de algunas de las operaciones que se pueden realizar una vez que se ha preparado una pieza para el plegado.

  • Si hace clic en Spacebar, se inicia una simulación de plegado. También puede utilizar los controles de simulación en el Bend Navigator para iniciar, detener o rebobinar la simulación:

  • Si hay alguna advertencia o error, aparecen en el Bend Navigator, y puede verlos y resolverlos haciendo clic en las celdas correspondientes:

  • Puede editar los útiles de plegado (punzones y matrices), la configuración de la calibración o la medición del ángulo con tan solo hacer clic directamente en ese objeto en la vista de simulación:

  • Puede cambiar la máquina haciendo clic en el nombre de la máquina en la pestaña debajo de la pieza, y seleccionando una máquina diferente:

  • Haga clic en el icono Ajustes de la barra de herramientas a la izquierda para editar ajustes adicionales para cada plegado (o para editar la configuración predeterminada utilizada para esta máquina de plegado, o para toda la aplicación TecZone Bend).

  • Haga clic en el icono Visualización de la barra de herramientas izquierda para editar la vista de la máquina: puede activar o desactivar la visualización de componentes adicionales como el carril de la matriz, la viga de plegado, el sistema de soporte de dedo de tope. También puede convertir varios componentes en translúcidos.Cuando se crea una solución de plegado sin errores, el programa CN de la pieza y una hoja de configuración para el plegado se generan automáticamente. Puede utilizar la página de configuración Bend Outputs para configurar si el programa CN de la pieza y la hoja de configuración se generan automáticamente, y también para configurar el destino de estas salidas, y el formato para los informes. Si se ha generado código, el botón código cn en la barra de herramientas izquierda está atenuado:

A medida que realiza cualquier cambio en la pieza, este botón se habilita nuevamente y se puede volver a generar el programa CN haciendo clic en este icono de nuevo (o pulsando la tecla C).

1. Puede usar el Editor de informes para personalizar esta hoja de configuración
