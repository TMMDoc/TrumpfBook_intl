Prensa plegadora

Prensa plegadora

Una chapa plana se puede doblar a su forma final utilizando una prensa plegadora. Estas son algunas de las tareas que hace TecZone Bend cuando se utiliza para programar estas máquinas:

  • Calcular una secuencia de plegado factible y óptima que evite colisiones y minimice las rotaciones o volteos de la pieza.

  • Componer estaciones de preparación de plegado a partir del inventario disponible de útiles de plegado, utilizando punzones, matrices, adaptadores y extensores según sea necesario.

  • Calcular las ubicaciones de calibración donde se pueden colocar los dedos de tope para alinear la pieza para plegarla.

  • Configurar equipos especiales como medición de ángulos o ayudas de soporte de plegado con los que la prensa plegadora puede estar equipada.

  • Generar un programa CN que pueda transmitirse a la prensa plegadora para su procesamiento.

  • Generar informes (como hojas de configuración) para el proceso de plegado.

Además, TecZone Bend también tiene un módulo Plegado que admite máquinas de plegado basculante y un módulo Plegado robótico que admite prensas plegadoras robóticas (cuando se utiliza un robot para realizar la manipulación de piezas).

TecZone Bend incorpora algoritmos y heurística para calcular la solución de flexión automáticamente, y en la mayoría de los casos se debe requerir poca o ninguna intervención. Sin embargo, TecZone Bend también proporciona un rico conjunto de útiles y operaciones para modificar interactivamente la secuencia de plegado, las configuraciones de útiles, las posiciones de calibrado y todos los parámetros del proceso de plegado.

A medida que se realizan estas modificaciones, se efectúan varias comprobaciones (incluida una gama completa de comprobaciones de colisión y viabilidad) en tiempo real, y siempre puede ver los resultados de sus ediciones mientras trabaja, sin tener que solicitar un nuevo cálculo.

Primeros pasos

Estos son algunos recursos con los que puede empezarTecZone Bend:

  • Todos los módulos de TecZone Bend comparten un modelo de interacción con el usuario llamado interfaz Clarity. Lea la sección de la interfaz Clarity cuando comience con TecZone Bend.

  • Puede instalar máquinas plegado desde la Base de datos de máquinas, y puede configurarlas para que coincidan con las máquinas reales disponibles en su área de producción.

  • Puede utilizar la Base de datos de útiles de plegado para instalar catálogos de útiles, importar herramientas personalizadas y configurar el inventario de útiles de plegado para que coincida con lo que realmente tiene en su área de producción.

  • El punto de partida para todos los módulos CAM son los datos CAD, y usted puede obtener esos archivos CAD importando datos 2D o 3D, o incluso dibujándolos usando las herramientas Esquemas integradas TecZone Bend.

Una vez que tenga una solución de plegado calculada:

  • Puede utilizar los distintos paneles del editor para hacer clic directamente en y editar atributos como la configuración de útiles o los topes.

  • El Bend Navigator se puede utilizar como un panel de control para ver y editar el estado de toda la operación de plegado. Haciendo clic en una celda de advertencia o error aquí le lleva a la fuente del problema, que luego puede solucionar de forma interactiva.

  • La sección Avanzado describe operaciones adicionales en la tecnología de plegado, que incluye Recalcular la solución con ajustes modificados y el Procesamiento por lotes.
