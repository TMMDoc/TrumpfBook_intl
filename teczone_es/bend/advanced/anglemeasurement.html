Medición de ángulo

Medición de ángulo

Medición de ángulo sistemas como ACB™™, ACBLaser o LCB se™ pueden utilizar para mejorar la precisión del ángulo durante el plegado. Estos sistemas miden el ángulo real logrado durante el plegado y la recuperación elástica de la chapa y utilizan esta información para ajustar el punto muerto central inferior de la viga dinámicamente para lograr el ángulo de plegado objetivo.

Sistemas de discos y láser

Los sistemas de medición de ángulo se basan en discos sensores montados en los punzones, o en sistemas ópticos utilizando un rayo láser proyectado sobre la pieza de trabajo. TecZone Bend admite ambos, y cuando configura una máquina, hay algunos interruptores de opción que puede activar dependiendo de las opciones de máquina que tenga instaladas.

La imagen de arriba muestra un panel de configuración para la máquina Trumpf 5085 B23. Esta máquina admite dos tipos de medición de ángulo basada en disco (ACB-Classic y ACB-Wireless) así como la medición basada en láser (ACB-Laser). Esta máquina en particular está configurada con las tres opciones activadas.Si una prensa plegadora tiene ambos sistemas de medición de ángulos, es posible que desee priorizar uno sobre el otro, o es posible que desee desactivar la medición del ángulo para una pieza en particular. Puede hacerlo utilizando el panel Configuración.[1]

La sección Medición de ángulo de la página Configuración/BendCAM tiene una lista de opciones que le permite encender o apagar los diversos sistemas de medición de ángulo. Puedes priorizar uno sobre el otro seleccionándolo y moviéndolo hacia arriba o hacia abajo usando los botones de flecha arriba/abajo contiguos. En este ejemplo particular, está habilitada tanto la medición de ángulo basada en discos como en láser y priorizamos el láser primero.[2]

La opción Usar velocidad ACB decide si la estrategia ACB Speed (disponible en máquinas Touchpoint) se utilizarán de forma predeterminada.

Métodos ACB

Hay diferentes métodos de ACB que se pueden utilizar en TecZone Bend. La mayoría de estos son comunes entre la medición de ángulo basada en discos y la basada en láser, por lo que discutimos los diferentes métodos primero.

Identificar

El método Identificar o Identificación o Identificar resiliencia es el uso canónico del sistema ACB. La prensa plegadora utiliza el sistema de medición de ángulos para medir el ángulo que se forma cuando la pieza se pliega. Después de que la pieza esté completamente plegada, la máquina pasa al estado de descompresión para permitir que el metal se recupere. La recuperación elástica también se mide usando ACB, y se compensa con un sobreplegado preciso. Para evitar la posibilidad de sobrepasar el ángulo objetivo durante la medición, la máquina apunta primero a un ligero infraplegado y luego realiza un nuevo plegado para alcanzar el objetivo. Este es el método ACB más preciso, pero también el que más tiempo consume.

Aprender Y

En el método Aprender Y, el sistema ACB no se utiliza en absoluto. El objetivo del rayo Y (punto muerto inferior) simplemente se copia de un plegado anterior (conocido como Plegado de referencia) que ya se ha medido utilizando el método Identificar. Este plegado anterior debe ser idéntico en términos de ángulo, radio y uso de útiles.

Resiliencia aprendida

En el método Resiliencia aprendida o Aprender resiliencia, el sistema ACB utiliza el ángulo de resiliencia de un plegado anterior que se ha medido utilizando el método Identificar. El sistema ACB todavía se utiliza para regular el plegado, pero dado que no es necesaria una segunda medición después de la descompresión, el ciclo es más rápido que un ciclo completo Identificar.

Introducción de recuperación elástica

El método Introducción de recuperación elástica, o Introducir resiliencia o Corregido o ACB Inteligente utiliza el sistema ACB para medir el ángulo objetivo, pero emplea un valor de resiliencia que es introducido por el usuario. En otras palabras, el ACB simplemente plegará la pieza al ángulo objetivo + el valor de corrección introducido por el usuario. No se realiza ninguna segunda medición después de la descompresión.

Notas sobre la medición de ángulo

Nombres de sistemas láser

El sistema de medición de ángulo basado en láser se conoce con diferentes nombres como láser ACB o LCB.Todos están programados de forma idénticaTecZone Bend, aunque el hardware real y la implementación de la máquina difieren. La mayor parte de la interfaz de usuario TecZone Bend solo usa la palabra ACB o láser ACB, así que también usaremos este término en esta discusión.

Comportamiento de Auto-tooler

El TecZone Bend auto-tooler asigna métodos láser ACB a cada plegado de manera óptima.Inicialmente, comienza con el uso del método Identificar para el primer plegado, y luego intenta utilizar el método Aprender Y para curvas sucesivas cuando sea posible. Si ya existe un plegado con el mismo ángulo, radio, conjunto de útiles y con una orientación de grano similar, TecZone Bend utilizará el método Aprender Y, en lugar de hacer otra identificación.Algunos plegados se pueden omitir automáticamente porque:

  • El ángulo de plegado no está dentro del rango de medición del sensor (demasiado agudo / demasiado obtuso).

  • Este no es un plegado al aire (por ejemplo, plegado por estampado, plegado en Z, prensado o aplastado)

  • Este es un plegado anterior para otro plegado (no necesita un ángulo preciso)

La imagen de arriba muestra una pieza donde algunos plegados no tienen método ACB - estos se indican con los iconos de círculos huecos en la fila ACB, y el tooltip para cada celda indica un motivo por el cual no se realiza la medición de ACB.

Obtener una medición válida

Cuando se realiza una medición de ACB, TecZone Bend decidirá si se requieren una, dos o tres mediciones en función de la longitud de la línea de plegado. Las posiciones de medición reales se calculan automáticamente, pero también se pueden modificar utilizando el panel ACB (tratado a continuación). Por lo general, TecZone Bend moverá ambos sensores ACB agrupados y utilizará sensores delanteros y traseros para un Medición del láser ACB, pero esto también se puede editar.TecZone Bend puede evaluar la longitud del trazado del láser para un Medición del láser ACB. Esta es la longitud de la línea láser proyectada que es visible para la cámara. Algunas partes de la línea pueden estar obstruidas por la matriz o por los topes. Algunas piezas pueden pasar por encima de agujeros en la chapa o por encima de conformados. La evaluación tiene en cuenta todo esto y puede calcular la longitud de línea disponible para el sistema óptico. TecZone Bend compara esta longitud disponible con una longitud mínima e ideal definida por la máquina, y genera un error o una advertencia adecuados (como se muestra en la sección TecZone Bend Símbolos anterior).

TecZone Bend tiene algo de inteligencia para mover la posición de medición a la izquierda o a la derecha para evitar agujeros, conformados u otros factores que puedan limitar la longitud del trazado. Además, TecZone Bend también añadirá automáticamente una retracción a los topes si se interponen en el camino de una medición láser. Además, la evaluación de la longitud del trazado láser se realiza de forma continua, por lo que si se mueven las posiciones del sensor ACB o se ajustan los topes, los resultados se pueden ver inmediatamente en el navegador.

1. Al igual que con todos los ajustes, utilice Ajustes de la aplicación para configurarlo globalmente (para todas las máquinas), use Configuración de la máquina para establecer los valores predeterminados para una máquina en particular, y use *Ajustes de las piezas*para cambiar los valores predeterminados para esta pieza en particular.
2. La habilitación de un sistema de medición de ángulo aquí tiene efecto solo si la máquina tiene ese tipo particular de sistema de medición de ángulo instalado como opción. En otras palabras, solo se puede utilizar para impedir que se utilice un sistema concreto.
