Recalcular solución

Recalcular solución

Al pulsar o utilizar el panel de flujo de trabajo para calcular una solución, BTecZone Bend calcula una solución de plegado utilizando un conjunto de configuraciones y parámetros predeterminados.Puede hacer un nuevo cálculo de esta solución utilizando diferentes configuraciones haciendo clic en el menú Recalcular de la barra de comandos. A veces, esto también es útil para completar el cálculo después de haber editado interactivamente algunos útiles, o la secuencia.
Por ejemplo:

  • Puede cambiar la secuencia y pedir TecZone Bend que se calculen nuevos puestos para útiles a la vez que se mantiene la secuencia sin cambios.

  • Puede cambiar las estaciones de útiles y pedir TecZone Bend que se calcule una nueva secuencia o alinear la pieza con las nuevas estaciones creadas.

  • Puede probar la actividad de preparación con un conjunto diferente de útiles, o probar usando una Configuración ya creada.

Overrides

Esta sección contiene algunos interruptores que puede utilizar para anular el comportamiento predeterminado de TecZone Bend.

  • Encienda el interruptor No cambiar secuencia para forzar que TecZone Bend calcule una nueva solución sin cambiar la secuencia. Diferentes estaciones de útiles, posiciones de topes o direcciones de inserción de pieza serán probadas por TecZone Bend, mientras se mantiene la secuencia sin cambios.

  • Si el interruptor Sin medición de ángulo. se activa, TecZone Bend calcula una solución que no realiza mediciones de ángulo y no utilizará ningún sensor de tope (útiles ACB)

  • Ignorar útiles de los archivos GEO/DXF ordena a TecZone Bend que ignore las sugerencias de los tooltips que a menudo se encuentran en archivos GEO y, a veces, en archivos DXF.

Configuraciones

Esta sección establece la estrategia para las configuraciones de útiles; de forma predeterminada, TecZone Bend calculará la configuración desde cero, pero puede usar esta configuración para cambiar este comportamiento.

  • Calcular nueva configuración es el comportamiento predeterminado: TecZone Bend calculará una configuración utilizando todos los útiles permitidos (consulte la sección Útiles a continuación).

  • Mantener la configuración existente indica a TecZone Bend que no se deben realizar cambios en la configuración; intenta procesar todos los plegados utilizando la configuración actual de la pieza. Si algunos plegados tienen colisiones, o tienen una longitud de herramienta insuficiente para el procesamiento, esos errores permanecen sin corregir y pueden verse después del proceso de recálculo.

  • Si selecciona UsarSetup de archivo, se le pedirá que elija un archivo de configuración de plegado (bsetup).La configuración cargada desde este archivo se utiliza para el cálculo; esa configuración no se modifica, ni se crean estaciones adicionales. Se muestran las configuraciones que ha cargado recientemente desde el archivo en el selector que aparece debajo de esta sección, y luego puede hacer clic en el botón + cerca de ese selector para elegir un archivo de configuración diferente.

  • La opción Usar configuración de otra pieza solo está disponible si hay otras piezas actualmente abiertas, y han sido preparadas para la misma prensa plegadora. En este caso, puede solicitar a TecZone Bend que utilice la misma configuración que una de estas otras piezas; aparece un selector para que pueda elegir la pieza que desee.

Útiles

Si elige la opción Calcular nueva configuración en la sección anterior, puede usar la sección Útiles para elegir la lista de útiles de plegado que se pueden utilizar para calcular la nueva configuración. Si elige una de las otras opciones para la configuración, esta sección no está disponible.

  • Útiles preferidos para la máquina es la opción predeterminada. Para cada prensa plegadora, puede elegir un conjunto de tools[1] como los útiles preferidos, y TecZone Bend usará estos útiles al mecanizar la pieza.[2] (Si no ha configurado los útiles preferidos para una máquina, TecZone Bend utiliza todos los útiles del inventario como el conjunto de útiles preferidos para la máquina).

  • Si elige Útiles en Inventario, TecZone Bend utilizará todos los útiles que están disponibles en el inventario de la instalación. Si no ha configurado el inventario de útiles, TecZone Bend utiliza todos los útiles del Catálogo como su inventario.

  • Todos los útiles del catálogo es la selección más amplia de útiles; TecZone Bend utilizará todos los útiles disponibles en el catálogo de útiles del fabricante para intentar una solución. Aunque esto puede significar que la solución no sea realmente viable para la producción (utilizando el inventario disponible), resulta útil para explorar de forma especulativa qué útiles se necesitarían para producir una pieza especialmente compleja. (Además, este modo también indica a TecZone Bend que asuma un inventario infinito de piezas, es decir, que se produce el utillaje óptimo suponiendo que se dispone de un número infinito de piezas de cada longitud).

  • Puede crear conjuntos de útiles personalizados y almacenarlos en archivos de lista de útiles (btools). Estos archivos se pueden usar para mecanizar la pieza, cuando se selecciona la opción Usar conjunto de útiles desde Archivo.Al seleccionar esta opción, se le pedirá que seleccione un archivo de lista de útiles, y se mostrarán los archivos de lista de útiles que haya utilizado recientemente para facilitar la selección. Puede hacer clic en el signo + cerca de esta lista para elegir una nueva lista de herramientas.

  • Para un control más directo sobre los punzones y matrices, puede seleccionar la opción Indicar útil superior e inferior y luego elegir el punzón y la matriz exactos que desea usar.

Prioridad de secuencia

Al secuenciar los plegados en una pieza, hay algunos requisitos potencialmente conflictivos.Por ejemplo, ¿debería TecZone Bend intentar minimizar el número de rotaciones de piezas, o debería intentar minimizar los plegados donde el centro de gravedad de la pieza se encuentra dentro de la máquina?
Es muy probable que cada operador tenga un conjunto diferente de reglas de prioridad, y también es posible que diferentes piezas requieran diferentes ajustes de prioridad. TecZone Bend por lo tanto, le permite hacer un nuevo cálculo con un orden diferente de prioridades, utilizando la

sección Prioridad de secuencia de este cuadro de diálogo.

Utilice los botones de flecha y a la derecha para mezclar las reglas; los elementos cercanos a la parte superior tienen mayor prioridad. Puede desactivar algunos de estos elementos prioritarios por completo; esto indica a TecZone Bend que no debe considerar esto como un factor a la hora de decidir la secuencia. (A veces, desactivar algunas de estas opciones puede generar una secuencia que utilice pocos puestos de equipamiento).

  • Evitar colisiones durante el sobreplegado: para lograr la precisión del ángulo, la mayoría de los plegados necesitarán sobreplegarse ligeramente para compensar la resiliencia. Esta regla de secuencia indica a TecZone Bend que realice plegados en un orden tal que sea posible el sobreplegado. Por ejemplo, en la pieza de abajo, debemos hacer el plegado 1 primero y el plegado 2 después; el otro orden sería sobreplegado de bloque.

  • Evitar útiles con prioridad baja: Indica a TecZone Bend que evite el uso de útiles de baja prioridad, o útiles que no están en la lista preferida, en la medida de lo posible.

  • Evitar hacer tope en ángulos abiertos: Lo ideal es que las posiciones de tope estén en el mismo plano que el eje de plegado, y no debe haber ningún plegado ya procesado entre la línea de plegado y el punto de calibración. Este es un ejemplo de una calibración deficiente que esta regla intenta evitar:

  • Longitud de arista más larga predominantemente delante de línea de plegado: Especialmente con piezas pesadas, es útil evitar situaciones en las que el centro de gravedad de la pieza se encuentra detrás de la línea del útil. Esta regla intenta minimizar tales situaciones.[3]

  • Minimizar volteos de piezas: Los volteos de piezas se producen cuando la pieza debe sobregirarse (los sucesivos plegados se realizan en diferentes direcciones hacia arriba/abajo).

  • Minimizar giros de piezas: La rotación de una pieza es cuando una pieza se gira hacia la izquierda o hacia la derecha en el plano horizontal; esto se considera más fácil para el operador que voltear una pieza; el orden predeterminado de estas dos últimas reglas intenta favorecer las rotaciones de piezas frente a los volteos, siempre que sea posible.

1. Para hacer esto, use el botón Inventario en la barra de comandos, y elija Útiles preferidos en el menú que aparece.
2. TecZone Bend intentará usar solo los útiles preferidos para la pieza, pero si hay colisiones u otros problemas que no se puedan resolver, comenzará a utilizar otras herramientas del inventario disponible.
3. TecZone Bend utiliza cierta discreción sobre el peso de la pieza y la distancia a la que se encuentra el centro de gravedad detrás de la línea del útil para considerar esto como un problema. Técnicamente, considera que una parte debe inclinarse hacia atrás si hay un momento de vuelco de 0,15 Nm (Newton metros) hacia atrás.
