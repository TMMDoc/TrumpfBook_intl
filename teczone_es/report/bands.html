Bandas (subinformes)

Una tabla es útil para algunos tipos de datos repetidos. Sin embargo, en ocasiones, es posible que necesitemos mostrar no solo texto o valores numéricos en columnas, sino que requiramos una presentación más elaborada para cada elemento de la tabla (por ejemplo, es posible que necesitemos una imagen o incluso una subtabla). Para tales situaciones, es más útil crear una Banda (esto a veces se llama sub-informe). Cuando haga clic en alguna fuente de datos que pueda representarse como una tabla o como una banda, TecZone Bend le preguntará si desea insertar una banda o una tabla:

Si elige insertar una banda, se inserta una banda vacía (el área sombreada que se muestra a continuación) y, a la derecha, se muestra el conjunto de todos los campos, imágenes y tablas que se pueden añadir a la banda.

Ahora puede componer los datos de la banda añadiendo los campos, tablas e imágenes de la derecha. Además de estos, también puede añadir etiquetas, mapas de bits estáticos o formas (como líneas divisorias) seleccionándolos de la fuente de datos Sys. En este ejemplo, hemos creado una banda para los datos Pieza > Plegado > Sujeción > Soportes, por lo que cada banda mostrará campos, imágenes y tablas relacionados con un soporte de plegado. Después de añadir algunos elementos a la banda, esta podría quedar finalmente así:

Hemos compuesto esta banda seleccionando algunos campos (posición, longitud, volteado, etc.). También añadimos el campo Nombre en la parte superior y cambiamos la fuente por una más grande. A continuación, añadimos una imagen (la sección transversal del útil) y una tabla (el conjunto de segmentos de este soporte).A medida que añadimos estos elementos, la banda muestra datos en tiempo real para todos ellos, pero solo para el primer soporte de la sujeción.Más tarde, cuando se imprima el informe, toda la banda (con campos, imágenes y tablas) se repetirá para cada soporte en el informe.
