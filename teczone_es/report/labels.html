Etiquetas, mapas de bits y formas

Etiquetas, mapas de bits y formas

Además de los datos que provienen de la pieza, es posible que los informes necesiten algunos otros adornos: logotipos de empresa, mensajes estándar y elementos geométricos simples, como líneas divisorias. Se accede a ellos desde la fuente de datos Sys.

Insertar un mapa de bits

Puede añadir un mapa de bits (como el logotipo de una empresa, por ejemplo) eligiendo Sys > Mapa de bits.Elija cualquier archivo de imagen para insertarlo en el informe. A continuación, se puede redimensionar o posicionar según sea necesario. Si activa la opción Vincular al archivo, Flux solo almacenará un enlace al archivo de imagen en la plantilla del informe. De lo contrario, almacenará una copia de todo el mapa de bits en la plantilla del informe.

Insertar una forma

Utilice la opción Sys > Forma para insertar una forma geométrica simple en el informe.Puede establecer los colores de fondo, relleno y contorno por separado, y puede controlar el grosor del contorno de la forma.

Además de estas formas simples, también puede insertar formas mucho más complejas leyéndolas desde un archivo 2D (como GEO, DXF o DWG). Elija la opción Desde archivo en la forma y luego seleccione un archivo de dibujo 2D.[1]. Todos los contornos cerrados del archivo se seleccionan y se rellenan con el color de relleno seleccionado y se contornean con el lápiz seleccionado.

Estas formas insertadas desde DXF y mapas de bits suelen tener fines similares. Sin embargo, las formas insertadas desde un archivo DXF son imágenes vectoriales y se escalan muy bien, incluso si el informe se imprime en alta resolución. Por otro lado, los mapas de bits pueden no escalar bien a resoluciones altas.

Insertar una etiqueta

Las etiquetas se insertan automáticamente al añadir un campo o una tabla. Sin embargo, puede insertar una etiqueta independiente (o cualquier otro texto fijo) seleccionando Sys > Etiqueta.

Estos elementos (mapas de bits, etiquetas y formas) están disponibles para su inserción incluso dentro de un subinforme o una banda. Luego se repiten tantas veces como se representa la banda.

1. Una vez insertada la forma, el archivo DXF/de geometría ya no es necesario; los contornos cerrados de ese archivo se importan y se almacenan como parte de la plantilla del informe.
