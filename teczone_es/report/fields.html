Campos, imágenes y tablas

Al hacer clic en las fuentes de datos, se abren y se muestran niveles adicionales de botones habilitados. Estos se pueden utilizar para añadir campos, imágenes y tablas al informe.

Añadir un campo

Un campo es una simple visualización numérica o de texto del informe. Por ejemplo, al hacer clic en Pieza > Información > Nombre, puede insertar un campo en el informe que muestre el material de la Pieza:

Al hacerlo, TecZone Bend añadirá realmente dos elementos al informe: un Campo que muestra el valor y una Etiqueta a la izquierda de ese campo. Puede editar la configuración de la etiqueta o del campo con solo hacer clic en él.

Editar una etiqueta

Al hacer clic en la etiqueta que acaba de añadir, aparecerá un panel que le permitirá editar varios ajustes de dicha etiqueta. La etiqueta también se resalta y se añaden controles de tamaño alrededor de la etiqueta, lo que le permite cambiar su tamaño arrastrando cualquiera de ellos. También puede hacer clic con el botón derecho del ratón sobre la etiqueta y arrastrarla para cambiar su posición. Cuando arrastra un elemento para moverlo o cambiar su tamaño, se muestran automáticamente varios indicadores de ajuste para facilitar la alineación precisa de esta etiqueta con otros elementos que ya se encuentran en el informe.

El panel de edición de la etiqueta (y de la mayoría de los demás elementos del informe) contiene un gran número de ajustes para configurar todos los aspectos de la visualización. Puede verlos abriendo los distintos grupos expandibles:

La imagen de arriba a la derecha muestra un selector de color que se utiliza para establecer el color del texto de la etiqueta.

Insertar una imagen

Una imagen en un informe se indica mediante un icono de cámara (en lugar del icono de lápiz que se utiliza para los campos simples). Por ejemplo, al hacer clic en Pieza > Modelo > Imagen, pueden insertar una imagen del modelo 3D de la pieza:

La imagen de arriba muestra una foto insertada y luego editada al hacer clic sobre ella.El panel de la imagen muestra algunos ajustes utilizados para controlar la imagen. Los ajustes exactos pueden cambiar, dependiendo de la imagen real que se insertó.En este caso, el ajuste DPI se utiliza para controlar la resolución de renderizado de la imagen (los valores más altos se verán mejor, pero harán que el tamaño final del informe sea mayor). El ajuste Sin PDF 3D se utiliza para desactivar la generación de archivos PDF 3D. Nota al pie [TecZone Bend puede renderizar algunas de sus imágenes 3D como imágenes PDF 3D incrustadas; estas se pueden ampliar, girar o desplazar utilizando algunos visores de PDF. Sin embargo, no todos los visores de PDF los admiten, y algunos pueden mostrarlos como cuadros en blanco.].

Si inserta una imagen de un dibujo en 2D (plano), las opciones de imagen son diferentes, como se ve en la imagen siguiente:

Los ajustes aquí controlan el ancho de línea utilizado para los contornos y para las dimensiones.Hay ajustes para mostrar los números de secuencia de plegado y para controlar si se rellena el contorno cerrado de la pieza.

Insertar una tabla

Puede insertar una tabla en un informe, eligiendo uno de los elementos que muestra un icono de tabla. Por ejemplo, si la tabla Pieza > Plegado > Plegados se utiliza para mostrar información sobre todas las operaciones de plegado de una pieza:

La sección Seleccionar columnas se utiliza para seleccionar las columnas que se muestran en la tabla (de entre todas las columnas disponibles para estos datos tabulares). Puede editar más atributos de cada columna haciendo clic en el enlace de navegación Columna en la parte inferior del panel de la tabla:

Esto abre el Editor de columna, donde se pueden editar los atributos de la columna, como se muestra a continuación:

El campo Título se utiliza para establecer el título que se muestra para la columna, mientras que el campo Expresión es el dato TecZone Bend interno que se muestra en esta columna. Puede utilizar los botones Desplazar para desplazar esta columna a la izquierda o la derecha en el orden y utilizar los botones Insertar y Eliminar para insertar nuevas columnas o para eliminar la columna seleccionada.
