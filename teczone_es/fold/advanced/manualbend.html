Finalización del plegado manual

En ciertos casos, evitar colisiones requiere dividir una operación de plegado en dos pasos: un plegado previo seguido de plegados posteriores, después de lo cual se completa el plegado. El preplegado se realiza utilizando una operación estándar FZ1 o FZ3 hasta el ángulo de preplegado definido, mientras que el plegado posterior se completa manualmente con movimientos de eje manual FZ31.

Preplegado se puede habilitar para una operación de plegado salvo que sea la última o única operación en un lado. Habilitarlo divide la operación de plegado en dos plegados, un Plegado anterior y un Plegado posterior.- El Plegado anterior tiene el mismo número de secuencia que la operación de plegado original.- Se añade una operación de Plegado posterior después del preplegado.En las imágenes de ejemplo que se muestran a continuación, el preplegado está habilitado para el Plegado 1 y el Plegado 3 es el plegado posterior recién añadido.El ángulo de preplegado se puede ajustar utilizando el cuadro de texto Ángulo previo. El ángulo en el cuadro de texto es el ángulo externo del plegado.

Después de habilitar Preplegado, el plegado recién añadido debe completarse manualmente seleccionando el botón Pasos manuales y añadiendo movimientos de eje manual (FZ31).

  • Navegue hasta el panel de plegado posterior.

  • Utilice el cuadro combinado Insertar después para especificar el plegado después del cual se realiza el plegado posterior.

  • Seleccione el botón Pasos manuales para mostrar el panel Pasos manuales.

  • Añadir movimientos de eje manual (FZ31).

Panel Pasos manuales

Los pasos manuales se pueden gestionar en el panel de pasos manuales.

  • Utilice Añadir para insertar un nuevo paso manual.

  • Utilice Eliminar para eliminar un paso existente.

  • La lista Pasos muestra todos los pasos manuales del plegado posterior seleccionado, donde cada paso representa el movimiento relativo de un eje particular durante este paso.

En cada paso manual se pueden configurar los siguientes parámetros:

  • Eje de la máquina: Selecciona el eje a mover.

  • Posición: La posición inicial de este eje al inicio de este paso.

  • Posición final: especifica la posición final del eje seleccionado al final de este paso.

  • Tiempo de espera (ms): Establece una duración de pausa (en milisegundos) antes de que se ejecute el siguiente paso.

  • Ángulo final: muestra el ángulo de plegado al final del paso actual.
