Editar un plegado

Algunos ajustes básicos de una operación de plegado se pueden editar abriendo el panel de Plegado. Para abrir el panel de plegado, utilice el Fold Navigator:

  • Haga clic en uno de los números en el Fold Navigator para cambiar a un plegado.

  • Haga clic de nuevo en el mismo número para abrir el panel de plegado para ese plegado.

El panel de plegado se puede ver a lo largo. Este tiene algunos ajustes y operaciones para trabajar en el plegado.

  • El cuadro de entrada Holgura se utiliza para ajustar el espacio de aire para el plegado (este es el espacio entre la cuchilla y los pisadores). A medida que se ajusta el espacio de aire a partir de su valor preestablecido, el Fold Navigator mostrará una advertencia de espacio de aire no estándar. La comprobación de colisiones también se realizará inmediatamente con el espacio de aire actualizado y podrá ajustar el espacio de aire para evitar colisiones. También se calculará la fuerza de plegado, y si se establece un espacio de aire tan bajo que se supere la fuerza de plegado máxima, el Fold Navigator lo mostrará como un error.

  • El ajuste P86 se utiliza para controlar la aplicación de presión de sujeción adicional durante el ciclo de plegado.

  • Utilice el botón Saltar plegado para omitir este plegado; este plegado no será procesado por la máquina de plegado basculante y deberá procesarse más tarde (posiblemente con una prensa plegadora).

  • El botón Nuevo lado se utiliza para dividir un lado en dos lados; este aparece solo si no está editando el primer plegado en un lado. Hacer clic en este botón divide el lado en este plegado y comienza un nuevo lado. Hay varias razones por las que es posible que desee dividir un lado:

    • Es posible que el lado deba dividirse en dos y que la pieza deba voltearse entre los dos lados antes de que se pueda completar el procesamiento (solo en máquinas manuales).

    • Puede que sea necesario completar algunos plegados del otro lado antes de poder completar los plegados restantes de este lado. En este caso, puede dividir el lado y reordenar los lados para que los plegados del otro lado se inserten en medio.

  • Los botones Volver y Continuar se utilizan para navegar entre los diferentes plegados en la pieza. (También puede utilizar las teclas PageUp y PageDown para ello).

  • El botón Lado se utiliza para navegar desde esta operación de plegado al lado contenedor. Al hacer clic en este botón, se muestra el panel Lado de plegado (descrito más adelante), que se puede utilizar para realizar algunas operaciones en todo el lado de plegado (o sección).Si se ha omitido este plegado curva, estos ajustes no se muestran; solo hay un botón No superar plegado para incorporar el plegado al proceso.
