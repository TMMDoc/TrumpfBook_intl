Importar pinza desde DXF

Una pinza de succión al vacío para las máquinas BendMaster se puede importar desde un archivo DXF. El archivo DXF representa la vista superior de la pinza (vista superior significa desde las ventosas, NO desde el BendMaster). Las entidades POLYLINE en el archivo DXF proporcionan la forma de la placa de la pinza y las entidades TEXT proporcionan información adicional sobre la pinza (como su nombre, espesor y colocación de ventosas).

Los archivos en el formato requerido (DXF, ARV o GEO) se pueden importar en la base de datos de Bend.

Para importar una pinza:

  1. Abra la Database dentro de TecZone Bend.

  2. Seleccione la opción Bend Grippers de la lista.

  3. Haga clic en la opción Import…​ y seleccione una pinza como archivo DXF del directorio.

Así es como se ve esta pinza después de importarla (se muestra en dos vistas desde abajo y arriba).

Pinza básica

Consulte el dibujo simple que se muestra a continuación, que define una pinza y se puede guardar como un archivo DXF para importar de nuevo dentro del Gripper inventory.

Estos son algunos puntos importantes sobre cómo se dibuja el DXF:

  • El origen del dibujo (0,0) corresponde al centro de la muñeca del robot. Puede ver esto representado por un punto y un círculo en la figura. (Tanto el punto como el círculo no son realmente necesarios para la importación de la pinza, ya que el centro de la pinza está implícitamente siempre en el origen).

  • Este dibujo también tiene algunas entidades POINT que representan cada una de las ubicaciones de ventosas. De nuevo, estas son principalmente para hacer el dibujo más claro y no son realmente utilizadas por el importador de pinzas (las posiciones de las ventosas se especifican mediante entidades de texto).

  • La forma real de la placa se dibuja como entidades POLYLINE cerradas. Si tiene otras polilíneas dentro de la exterior, se convierten en agujeros en la placa.

Pinza de dos capas

Las pinzas ligeramente más complejas se pueden modelar usando una capa BASEPLATE separada. Cree una capa llamada BASEPLATE en el archivo DXF y dibuje algunas formas POLYLINE cerradas en esa capa. Luego, esta capa puede tener límites de extrusión Z separados especificados por la clave BASEPLATEEXTENT. Las herramientas necesarias para crear y editar capas ahora se agregan.

  1. Haga clic en Related cuando se está editando un dibujo.

  2. Seleccione el menú Layers para agregar nuevas capas, cambiarles el nombre, establecer colores y cambiar estilos de línea.

  3. En el dibujo, hacer clic en una entidad le permite cambiar la capa de esa entidad.

    La capa aparece en el panel de entidad solo si hay varias capas en el dibujo.

  4. Después de editar el dibujo usando estas herramientas, guarde el dibujo en formato DXF. Aquí hay un DXF dibujado usando dicha capa BASEPLATE (se muestra en verde en la imagen).

    Para esta pinza, la muñeca se extiende desde Z=0 hasta Z=23. Luego, la base plate (verde) se extiende desde Z=23 hasta Z=40, como lo indica la clave BASEPLATEEXTENT. Finalmente, la placa de pinza real (que sostiene las ventosas) se extiende desde Z=40 hasta Z=63, como lo indica la clave PLATEEXTENT.
    La placa de la pinza también tiene algunos agujeros hexagonales.

Pinza multicapa

Multi-Layered Grippers es una versión actualizada de Two Layered Gripper en la que más de dos capas se pueden crear e importar usando TecZone Bend. Para usar una pinza multicapa, el DXF tendrá que incluir el texto VERSION = 2. Se muestra una pinza multicapa de muestra a continuación:

Cada capa tiene que definirse con sus extensiones como LAYER_1 = 23 : 35. Las entidades en esta capa se agregarán a estas extensiones. De manera similar, las ventosas se definen en la forma, CupNo = X pos, Y pos, [Cup type, Mounting height, Cup group].

Las posiciones X e Y son obligatorias.

A continuación se explican las diferentes definiciones posibles para las ventosas:

CupNo = X pos, Y pos con el nombre de la ventosa especificado en CUPNAME como en la versión anterior
CupNo = X pos, Y pos, Cup type, donde cada tipo de ventosa se define por separado (para ventosas ovaladas, los nombres varían con el ángulo)
CupNo = X pos, Y pos, Cup type, Mounting Height
CupNo = X pos, Y pos, Cup type, Mounting Height, Cup Group

Ejemplo

CUP1 = 275, 314, SAOB60x30_90grad_ged, 72, 2

Si no se especifica la altura de montaje, la extensión máxima de la placa se considera de forma predeterminada. De manera similar, el grupo de ventosas se establecerá en 1, si no se define.

En este momento, TecZone Bend no admite diferentes alturas de montaje para las ventosas.

Pinza de múltiples circuitos

Puede crear una pinza de múltiples circuitos (una pinza que tiene varios circuitos de vacío independientes). Para dicha pinza, cada ventosa se asigna a un número de grupo distinto de cero.

Asigne grupos de succión usando un formato especial para CUP1POS, CUP2POS, etc., como se muestra a continuación:

CUP1POS = 340,120 #6

CUP2POS = 420,140 #8

El valor después de # es el número de grupo de succión para la ventosa. En este ejemplo, Cup 1 se asigna al grupo de succión 6, Cup 2 se asigna al grupo de succión 8, y así sucesivamente.

Para que una pinza de múltiples circuitos se configure correctamente, cada ventosa debe ser asignada a un número de grupo de succión de esta manera.

El número máximo de ventosas que se pueden usar en una pinza está limitado a 100. Si importa una pinza con más de 100 ventosas, TecZone mostrará un mensaje de error de importación fallida.
