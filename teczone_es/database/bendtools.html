Útiles de plegado

Útiles de plegado

Los términos utilizados en la creación de útiles de plegado se describen a continuación:

S.No Término Significado

1

Listas de herramientas

Las listas de útiles con los que se trabajará se pueden seleccionar aquí.Las nuevas listas de útiles se pueden crear con la opción Nuevo.

2

Clasificar

Las herramientas se pueden clasificar de acuerdo con criterios específicos aquí (altura, nombre, forma, radio, ángulo, anchura de matriz, prioridad, gestión de útiles).

3

Buscar

Los útiles se pueden buscar usando la barra de búsqueda. Aquí, por ejemplo, se puede hacer una búsqueda alfabética de nombres de útiles.Además, se pueden realizar solicitudes de búsqueda más complejas.Esta búsqueda encuentra todos los útiles con UT o EV en el nombre que también cumplen las siguientes condiciones: ángulo ≤ 30, altura ≥ 140, altura ≤ 220, radio = 1.

4

Escalar

El tamaño de la vista de los útiles se puede cambiar con el deslizador.

5

Acciones

Se pueden realizar las siguientes acciones:

- A Añadir catálogo: para añadir diferentes tipos de máquinas.

- B Nueva lista de útiles: para crear una nueva colección de útiles.

- C Clonar lista de útiles: para clonar la colección de útiles creada.

- D Exportar lista de útiles: la lista de útiles actual se puede exportar como archivo ".btools".

- E Importar lista de útiles: se puede importar una lista de útiles como ".btools".

- F Importar desde ARV: un útil se puede importar desde un archivo ARV, estos útiles se archivan de forma personalizada en el catálogo de útiles (se creará si no estaba disponible de antemano).

- G Importar desde DXF: se puede importar un útil desde un archivo DXF. Estos útiles se archivan de forma personalizada en el catálogo de útiles (este se creará si no estaba disponible de antemano).

- H Restablecer lista de útiles: restaura la lista de herramientas a su estado predeterminado.

- I Cancelar: cierra la ventana "Acciones…​" .

6

Gestión de útiles

Muestra una descripción general del útil seleccionado.Se pueden realizar las siguientes acciones:

- Restablecer a cero: El inventario total del útil seleccionado se elimina de la lista de útiles actual.-Añadir un conjunto: solo se añade un conjunto del útil seleccionado a la lista actual.-Añadir longitud: para añadir la longitud del útil

- Añadir todo: se carga el inventario completo del útil seleccionado en la lista de útiles actual. El inventario se define en la lista de útiles <Todos los útiles>.

- Guardar: para guardar los cambios aplicados.

7

Inventario de útiles

Muestra el inventario del útil seleccionado.Haga clic con el botón izquierdo del ratón para añadir un segmento.

8

Útil sin inventario

Un útil sin inventario no está incluido en la lista de útiles actual. Al hacer clic en el campo vacío, se añade un inventario estándar para el útil y éste se incorpora a la lista de útiles actual.

9

Prioridad

En función de la prioridad asignada por el usuario, se selecciona el útil.1 = Útil de alta prioridad

5 = Útil de baja prioridad

10

Área de visualización de útiles

En el área de visualización de útiles se muestran todos los útiles de la lista de útiles seleccionada. Los útiles sin inventario también se pueden mostrar con la casilla de verificación "Incluir útiles sin inventario".

11

Filtro de útiles

Aquí, los útiles de la lista actual están estructurados según su tipo y forma.Seleccionar un elemento en esta estructura permite filtrar.

12

Catálogo

Con esto se puede seleccionar uno de los muchos catálogos de útiles instalados (incluidos TRUMPF, personalizados, etc.).

13

Condición de filtro

La condición de filtro para la lista de útiles actual, que es utilizada automáticamente por el auto-tooler. Esta se puede utilizar para una lista de útiles con nombre, pero no para la lista especial <Todos los útiles>. Es posible filtrar según diversos aspectos, incluyendo máquina, material, grosor de la chapa, etc.

Crear una lista de útiles

  1. Para crear una nueva lista de útiles, haga clic en el botón Acciones y seleccione Nueva lista de útiles.

  1. Cambie el nombre de la lista de útiles haciendo clic en el icono de edición .

  1. De forma predeterminada, la nueva lista de útiles muestra todos los útiles, incluyendo punzones, matrices, portapunzones y portamatrices, pero no se añaden al inventario. Asegúrese de que la casilla de verificación Incluir útiles no en esta colección esté seleccionada.

  1. Para añadir un útil, haga clic en la casilla cuadrada encima del nombre del útil y en la tecla ctrl+enter para eliminar un útil.

  1. La opción Usar esto cuando permite al usuario filtrar y establecer reglas para una nueva lista dependiendo de varios criterios.

  2. Haga clic en guardar para añadir esta lista de útiles a la colección.

Añadir regla para una lista de útiles

Haga clic en la opción Filtrar más para aplicar una regla de los diferentes criterios de filtro disponibles.

  1. Máquina: se refiere a la máquina utilizada para las operaciones de plegado.

  2. Material: indica el tipo de sustancia que se está plegando.

  3. Espesor: indica el espesor del material que se está plegando.

  4. Radio: se refiere a la curvatura del plegado.

  5. Superficie: describe la textura o el acabado exterior del material.

  6. Tratamiento: describe cualquier proceso previo o posterior al plegado aplicado al material, como tratamiento térmico, recubrimiento o galvanización.

  7. Película: se refiere a una capa protectora o funcional aplicada al material.

  8. Etiqueta: etiqueta personalizada asignada para categorizar, filtrar y organizar elementos para una identificación más fácil.

La lista de útiles debe tener al menos un criterio de filtrado antes de poder guardarla (de lo contrario, esta lista de útiles se usaría para cada pieza).
