Flujo de trabajo de tarea

TecZone Laser tiene un motor de nesting que pueden tomar cantidades variables de diferentes tipos de piezas y empaquetarlas de manera eficiente en una o más chapas de diferentes tamaños. Para crear una nueva tarea de nesting, utilice el menú File / Nueva /Tarea de nesting o simplemente pulse la tecla J.

  • Escriba un Name para la tarea de nesting (un número de tarea se calcula automáticamente).

  • Seleccione una Machine de la lista de máquinas disponibles.

  • Los ajustes Material, Thickness, Treatment y Film se utilizan para seleccionar el material que se utilizará para todas las piezas en la tarea de nesting.

  • En función de las entradas anteriores, puede seleccionar una de las TTL (tablas de tecnología láser) disponibles que coincidan. El hipervínculo a 2D Laser Guide…​ puede ayudar en esta selección.Cuando introduce todos estos valores y hace clic en Enter, se crea una tarea de nesting y se muestra el Panel de tarea.La tarea de nesting se puede guardar utilizando el menú File / Guardar como un archivo .fxjob. Contiene toda la información utilizada para definir la tarea de nesting (incluidas las copias de las piezas utilizadas para el nesting), por lo que es autónomo y se puede abrir más tarde usando el menú*File / Abrir* para rehacer el nesting con ajustes modificados.
