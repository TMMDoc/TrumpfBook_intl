Pestaña Piezas

La pestaña Piezas se utiliza para añadir piezas para nesting y para configurar parámetros como la cantidad a anidar, ángulos de rotación aceptables, etc.

Botones de acción

Estos son los botones a lo largo del lado izquierdo del panel de piezas, utilizados para añadir o retirar piezas de la tarea y para realizar algunas operaciones en todas las piezas:

  • Utilice Añadir…​ para añadir una pieza para nesting. Puede ser un archivo .fx ya _preparado* *o cualquier archivo 2D o 3D que TecZone Laser pueda importar y convertir en una pieza de chapa. Estos archivos se prepararán luego como se explica en la sección Flujo de trabajo de pieza.

  • El botón Añadir montaje…​ se utiliza para añadir un grupo de módulos 3D completo. TecZone Laser seleccionará las piezas de chapa del grupo de módulos y las añadirá a la tarea de nesting. Las cantidades de los componentes individuales del grupo de módulos se establecen para producir una cantidad del grupo de módulos completo.

  • El botón Añadir diseño…​ se utiliza para importar un archivo DXF que contiene una chapa completamente anidada y la descompone de nuevo en una tarea de nesting.

  • El botón Importar CSV…​ se utiliza para cargar una definición de tarea de nesting desde un archivo CSV (valores separados por comas). El archivo debe tener un formato específico con 8 columnas, tal y como se explica en el tooltip que aparece a continuación:

  • Guardar pieza(s)…​ se utiliza para guardar las piezas seleccionadas como archivos .FX. Cuando se inicia la tarea de nesting con archivos de geometría (como DXF, GEO, datos 3D), estos se preparan automáticamente cuando se añaden a la lista de piezas. Este botón se puede utilizar para guardar las versiones preparadas de estas piezas para que puedan reutilizarse en otras tareas de nesting.

  • El Reequipar todo se utiliza para volver a calcular la preparación para todas las piezas en la tarea. Es útil después de realizar algunos cambios en la configuración de la máquina o en la configuración de esta tarea de nesting en particular, para asegurarse de que todas las piezas utilicen la nueva configuración.

Columnas de la tabla de piezas

Estas son algunas de las columnas de la tabla de piezas:

  • La Pieza es la cantidad requerida de la pieza. Puede introducir un número simple aquí, o un rango como 20 - 30. Esto significa que se anidarán al menos 20 copias de esta pieza, pero no más de 30. TecZone Laser podrá anidar libremente cualquier cantidad entre estos dos valores para maximizar la eficiencia del nesting, pero nunca comenzará una nueva chapa solo para anidar piezas que superen la cantidad mínima.

  • Las piezas se anidan en orden creciente de Prioridad, si se establecen diferentes valores de prioridad para diferentes piezas. No obstante, Prioridad = 0 se trata como un caso especial - estas piezas tienen la prioridad más baja.

  • Dirección de laminación de pieza pref. se utiliza para ajustar la dirección de laminación de la pieza si es importante. Si se establece en un valor distinto de Ninguno, esto limitará los ángulos en los que la pieza puede ser girada por el motor de nesting. (Para que esto funcione correctamente, también debe especificar la dirección de laminación de las chapas en la pestaña Chapas).

  • El valor Grupo es una indicación para el motor de nesting - TecZone Laser intentará mantener juntas en la misma chapa las piezas con el mismo valor de grupo, siempre que sea posible.

  • Si el interruptor TwinLine está activado para una pieza, el motor de nesting intentará utilizar el corte de líneas comunes para esta pieza (la pieza solo comparte líneas comunes con otras copias de sí misma).
