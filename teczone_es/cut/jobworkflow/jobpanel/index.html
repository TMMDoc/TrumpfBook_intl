Panel de tareas

Cuando se crea una nueva tarea de nesting o se abre una tarea de nesting existente desde un archivo, el tarea panel se muestra en la parte superior de la pantalla:

El panel de tareas tiene varias pestañas, y puede acceder a ellas con teclas de acceso rápido, tal y como se muestra en la tabla siguiente.

Pestaña Tarea

La pestaña tarea muestra algunos de los datos que se introdujeron en el cuadro de diálogo Nueva tarea y proporciona algunos ajustes más para que el motor de nesting los utilice al calcular el nesting.

  • Tecnología: Puede seleccionar una TTL diferente para usar (para la máquina, el material y el grosor especificados). Las TTL que se utilizan con mayor frecuencia aparecen en primer lugar y están marcadas con un icono de estrella.

  • Distancia de piezas: Este ajuste se utiliza para especificar la distancia mínima permitida entre piezas (también conocida como ancho de puente) durante el nesting.

  • Iniciar esquina: Especifica la esquina de la chapa desde donde comenzará el nesting.

  • Permitir nesting en _perforaciones: Si se activa, las piezas pequeñas se anidarán dentro de los orificios de las piezas más grandes, cuando sea posible.

  • Expandir las piezas en el último diseño sin recordatorio: Si esta opción está activada, TecZone Laser tomará la última chapa del nesting que no tenga suficiente espacio para ser utilizada como chapa restante y distribuirá las piezas en esa chapa para minimizar los problemas de disipación de calor (ya que, de todos modos, no hay reducción en la eficiencia del nesting).

  • Evitar colocar piezas largas finas a lo largo de lamas: Si se activa, las piezas largas y delgadas se colocarán de de manera transversal respecto a los listones de apoyo en lugar de a lo largo de ellos, para minimizar el riesgo de que las piezas se caigan entre los listones.

  • Minimizar el número de diseños diferentes: Si se activa, TecZone Laser intentará repetir los diseños varias veces (dentro de las limitaciones de la cantidad de piezas necesarias) para minimizar el número de diseños diferentes creados. //AVREVIEW : ¿Por qué?

Cambio de material / máquina

Puede hacer clic en el enlace Cambiar material en la pestaña Tarea para cambiar la máquina y el material utilizado para el nesting. Esto volverá a preparar todas las piezas con la nueva máquina y el nuevo material, y volverá a hacer el nesting.
