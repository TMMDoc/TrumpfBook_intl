Tratamiento de esquinas

El panel Tratamiento de esquina se utiliza para realizar diversos tipos de tratamientos especiales en las esquinas afiladas de un corte por láser. Se accede a este panel haciendo clic cerca de una esquina de un corte láser.

  • Autocalcular: Cuando la configuración de cálculo automático está activada (por defecto), TecZone Laser calcula un procesamiento de esquina adecuado basándose en la máquina, la TTL utilizada y el ángulo de la esquina. Si desea editar el tratamiento de las esquinas, debe desactivar esta opción.

  • Tratamiento de superficie: El tipo de tratamiento de esquinas (consulte la tabla siguiente para obtener más detalles). Algunos de los métodos de tratamiento tienen un parámetro (como radio o tiempo que se utiliza para controlar el proceso de esquina).

  • Radio o Tiempo (segundos): Algunos tratamientos (como Redondeo de esquinas y Bucles) necesitan un radio como entrada. Algunos como el Permanencia y Enfriamiento necesitan un retardo.

  • Los selectores en la parte inferior de este panel se pueden utilizar para extender el conjunto de esquinas seleccionadas para que puedan editarse todas juntas:

    • Similar en este contorno: Si haces clic aquí, todas las esquinas de este contorno que tengan un ángulo similar al seleccionado se resaltarán con pequeñas cruces azules. Entonces, todas las ediciones que realice se aplicarán a todas estas esquinas.[1]

    • Todo en este contorno: Igual que el selector anterior, pero selecciona todas las esquinas en este contorno, independientemente del ángulo.

    • Similar en todos los contornos: Selecciona todas las esquinas similares (mismo ángulo) en todos los contornos mecanizados de la pieza para que todas puedan editarse en paralelo.

Métodos de tratamiento

La siguiente tabla muestra los diferentes métodos del tratamiento de esquinas. El Redondeo de esquinas y Bucles alteran visiblemente la geometría del corte, mientras que los otros métodos añaden un pequeño icono en la esquina para indicar el tipo de procesamiento que se está realizando allí.

Tratamiento Explicación Ejemplo

No hacer nada

No se realiza el mecanizado de esquinas

Redondeo de esquinas

Se redondea en las esquinas

Bucles

Se efectúa un trazado curvo en las esquinas

Permanencia

Permanece en las esquinas

Enfriamiento

Refrigera el láser en las esquinas

Parada

Se para en la esquina

Desaceleración

Ralentiza el láser en las esquinas

1. Después de seleccionar varias esquinas de esta manera, recuerde hacer clic en el interruptor Autocalcular para desactivarlo en todas las esquinas antes de editar su configuración.
