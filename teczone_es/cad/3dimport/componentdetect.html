Detección de componentes

Los componentes como tuercas, tornillos, etc. normalmente no requieren procesamiento adicional en este software. Sin embargo, resulta útil recopilar, detectar y utilizar dichos componentes para generar una posible lista de materiales de los artículos adquiridos adicionalmente que son necesarios para componer el grupo de módulos.

Existen grandes diferencias en la práctica a la hora de dibujar incluso un componente estándar como un tornillo M8. El software utiliza un sistema de aprendizaje que aprende los nombres de los componentes basándose en el ejemplo. Al principio, TecZone Bend no conoce ningún componente estándar y estos se muestran en amarillo (no identificados).Para detectar el componente:

  1. Importar una pieza de montaje.

  1. Haga clic en el icono Workflow y seleccione el grupo de módulos Explode.

  1. Todos los componentes del grupo de módulos se abren como archivos 3D separados.

  1. Haga clic en cualquier componente amarillo en su superficie para acceder al Surface panel.

Figure 1. Panel de superficie

  1. Haz clic en Identificar…​ para acceder al Add Component panel.

Figure 2. Añadir componente

  1. Ajuste la configuración según sea necesario:

    1. Show: Todos los demás componentes que ya están definidos también se muestran aquí. Este componente se puede mostrar como referencia haciendo clic en un nombre Para volver a añadir el nuevo componente, debe hacer clic en NUEVO en la parte superior de la lista.

    2. Preview: Muestra una vista previa del componente seleccionado.

    3. Variants: Se puede utilizar el mismo nombre que el nombre de un componente existente. Es común que haya diferentes formas en que, por ejemplo, se modela el mismo tornillo M8. Si se añaden varios componentes con el mismo nombre, se convierten en variantes. Los componentes con varias variantes tienen su cantidad indicada después del nombre (por ejemplo: x4).Si se selecciona uno de estos componentes, se pueden mostrar todas las variantes diferentes.

    4. Name: Este nombre se utiliza para mostrar el componente.

    5. Identifier: Aquí se puede introducir un ID.Todas estas diferentes variantes tienen el mismo nombre y el mismo ID. Solo se representan mediante diferentes modelos 3D. Si el software detecta uno de estos modelos 3D en un grupo de módulos, este componente se marcará con este nombre y este ID.Es posible que dos componentes diferentes con el mismo nombre se hayan agrupado incorrectamente. En tal situación, el botón Separate se puede utilizar para separar las variantes específicas con un nuevo nombre.Por el contrario, si se desean agrupar dos componentes que originalmente tenían nombres diferentes, basta con seleccionar uno de ellos y cambiarle el nombre para que coincida con el del otro. Ambos se agrupan como variantes del mismo componente.

  2. Haga clic en el botón Add para guardar el componente en la base de datos para su uso posterior.
