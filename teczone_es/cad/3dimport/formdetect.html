Detección de forma

El software puede detectar formas de punzón que se encuentran en una capa de un modelo de chapa.Si un modelo de superficie se convierte en un modelo metálico, se detectan todas las áreas conformadas y se convierten en formas de punzón.

Importar una chapa (con áreas conformadas) en TecZone Bend. Seleccione Convertir en chapa del panel de flujo de trabajo.

Si la forma del punzón ya se ha identificado, se resalta en verde claro, de lo contrario es de color amarillo.

Añadir una nueva forma a la pieza

Para detectar un área formada en una pieza:

  1. Haga clic en cualquier área formada resaltada en amarillo para acceder al panel Conformado.

Figure 1. Panel de conformado

  1. Seleccione la opción Identificar… para acceder al panel Añadir conformado.

    1. Realizar perforación: la forma del punzón se convierte en un agujero.

    2. Borrar: Se elimina la forma del punzón.

    3. Similar: Esta función selecciona todas las formas que son similares en apariencia a la forma seleccionada.

  2. Ajuste la configuración según sea necesario en el panel de conformado.

Figure 2. Añadir conformado

  1. Mostrar: Todas las demás formas que ya están definidas también se muestran aquí. Esta forma se puede mostrar como referencia haciendo clic en un nombre. Para volver a añadir la nueva forma, debe hacer clic en NUEVO en la parte superior de la lista.

  2. Vista previa: Muestra una vista previa de la forma seleccionada.

  3. Variantes: Se puede usar el mismo nombre que el nombre de una forma existente. Esto es normal, ya que hay diferentes maneras de modelar una forma. Si se añaden varias formas con el mismo nombre, se convierten en variantes. Las formas con varias variantes tienen su cantidad indicada después de su nombre como Componentes del objeto (ej.: 3). Si se selecciona una de estas formas, se pueden mostrar diferentes variantes.Todas estas variantes diferentes tienen el mismo nombre y el mismo ID. Solo se representan mediante diferentes formas. Si el software detecta una de estas formas en un grupo de módulos, esta forma se marcará con este nombre y este ID.Es posible que dos formas diferentes con el mismo nombre se hayan agrupado incorrectamente. En tal situación, el botón Separar se puede utilizar para separar las variantes específicas con un nuevo nombre.Por el contrario, si se desean agrupar dos formas que originalmente tenían nombres diferentes, basta con seleccionar una de ellos y cambiarle el nombre para que coincida con el de la otra. Ambas se agrupan como variantes de la misma forma.

  4. Nombre: Este nombre se utiliza para mostrar la forma.

  5. Identificador: Aquí se puede introducir un ID.

  6. Salida nivel (DXF): Este ajuste se utiliza para especificar una capa específica en la que se emite esta forma. El sistema CAM de punzonado lo utiliza para asignar un útil de punzonado adecuado.

  1. Pulse el botón Añadir para guardar la forma en la base de datos para su uso posterior.
