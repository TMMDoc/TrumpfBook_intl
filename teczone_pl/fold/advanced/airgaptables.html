Niestandardowe tabele szczelin powietrznych

Tabela szczelinowa do giętarki krawędziowej odwzorowuje konkretną grubość blachy zgodnie ze standardową szczeliną powietrzną używaną podczas obróbki. Tabela jest wgrana w każdą z giętarek krawędziowych, ale można je zastąpić, wgrywając niestandardową tabelę szczelin powietrznych w postaci pliku JSON. Ten plik JSON nazywa się air-gap-tables.json i jest umieszczony pod następującą ścieżką dostępu C:\DaneProgramu\Trumpf{APPNAME}\Dane\System.

Poniżej znajduje się typowy plik air-gap-tables.json:

 {
  "version":1  "airGapTables": [
    {
      "machine": "112002,112003", "material": "DC01"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
     {
      "machine": "112003", "material": "*"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
    {
      "machine": "*", "material": "Alluminio"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    }
  ]
}

  • Plik zawiera zestaw wartości szczeliny powietrznej. Każdy tabela posiada przycisk pól wyboru maszyny i materiału.

  • Klawisze te są ustawione na * (gwiazdkę), aby wskazać symbol wieloznaczny (mogący oznaczać wszystko) lub są listą oddzielonych przecinkami wartości maszyny lub materiału.

  • W powyższym przykładzie znajdują się trzy tabele. Pierwsza jest używana do maszyn 112001 lub 112003 tylko dla materiału DC01. Druga jest używana do wszystkich materiałów do maszyny 112003, a trzecia jest używana do wszystkich maszyn do materiału Alluminio. Jeśli wiele tabel może pasować do kombinacji maszyny/materiału detalu, używana jest pierwsza pasująca tabela. Tak więc w przypadku maszyny 112003 stosuje się materiał DC01 i pierwszą tabelę (chociaż druga również pasuje).

  • Każda tabela składa się z prostego zestawu klawiszy grubości z odpowiednimi wartościami szczeliny powietrznej. Przykładowo pierwszy wpis powyżej należy interpretować następująco: jeżeli grubość jest mniejsza lub równa 0,75 mm, należy zastosować szczelinę 2,5 mm. Klawisze grubości muszą być w kolejności rosnącej. Jeśli żadna z tabel w tym pliku nie pasuje do materiału bieżącej detalu, zostanie użyta tabela domyślna dla maszyny.

Jeśli w pliku .json zostaną wprowadzone zmiany, należy ponownie uruchomić TecZone Fold, żeby zadziałały.
Jeśli plik JSON jest źle utworzony, zostanie wyświetlony błąd podczas próby obróbki detalu w giętarce krawędziowej, a następnie TecZone Fold powróci do korzystania z domyślnej tabeli szczelin powietrznych dla maszyny i praca będzie kontynuowana.
