Alur Kerja Komponen

Alur Kerja Komponen

Karena TecZone Bend memiliki banyak modul terintegrasi, sering kali ada beberapa jalur yang dapat dilalui oleh data komponen.

Contoh 1: saat Anda memuat data komponen datar (dari file GEO atau file DXF), Anda dapat memilih untuk

  • Menetapkan pengaturan alat laser ke komponen, sehingga dapat ditempatkan pada lembaran dan dipotong, bersama dengan komponen lainnya.

  • Melipat komponen sepanjang garis tekuk menjadi komponen 3-D, sehingga dapat diproses dan urutan tekukan dapat dihitung untuk press brake.

Contoh 2: saat mengimpor model permukaan 3-D (dari file IGES atau STEP), Anda dapat memilih untuk

  • Menganalisis permukaan dan menempatkan pengaturan alat CAM laser 5-sumbu di sepanjang bukaan yang perlu dipotong.

  • Melakukan pengenalan fitur, mengonversi model permukaan menjadi model lembaran logam, dan membentangkannya menjadi pola datar untuk dilakukan punching dengan mesin punch-press.

Panel Alur Kerja

Panel Alur kerja berfungsi sebagai pusat kendali dari mana Anda dapat mengarahkan semua gerakan ini. Saat Anda membuka komponen, Anda selalu dapat membuka panel alur kerja dengan menekan W, atau mengklik ikon alur kerja di baris perintah di sebelah kiri. Mari kita jelajahi panel alur kerja; mulailah dengan mengimpor komponen 2D dengan parameter tekukan (file GEO, misalnya). Menampilkan panel alur kerja pada tahap ini akan menampilkan hal berikut:

  • Kita mulai dengan Bend_28.geo, mengimpor dan membersihkan untuk membuat komponen lembaran logam 2D (dimensi komponen ditampilkan)

  • Kemudian ada cabang dalam alur kerja.

    1. Kita dapat menetapkan Laser Tech ke komponen (ini hanya berarti penetapan jalur pemotongan laser ke kontur komponen).

    2. Kita dapat melipat komponen yang datar menjadi 3D (terdeteksi 9 penekukan).

Memperluas node alur kerja: Tahap 1

Klik ikon Laser Tech untuk menetapkan pengaturan alat laser ke komponen. Anda akan melihat komponen tersebut segera dianalisis dan pengaturan alat laser ditambahkan padanya. Kemudian, klik ikon Lipat ke 3D untuk melipat komponen datar menjadi 3D. Setelah langkah-langkah ini, panel alur kerja akan terlihat seperti ini:

Seperti yang ditunjukkan oleh anotasi, ada berbagai jenis node dalam diagram alur kerja.

  • Ada node Tampilan Komponen yang mewakili berbagai jenis pemrosesan yang dapat dilakukan pada komponen. Dengan mengklik node ini, komponen akan beralih ke tampilan tersebut dan serangkaian aktivitas yang tersedia pada komponen tersebut mewakili tampilan tersebut. Misalnya, dalam tampilan Komponen Laser, Anda dapat melihat dan mengedit pengaturan alat laser yang ditetapkan untuk komponen tersebut.

  • Anda dapat beralih di antara tampilan ini dengan mengklik ikon-ikon ini. Semua tampilan komponen yang beragam ini juga memiliki tombol pintasan, yang dapat Anda lihat jika Anda mengarahkan mouse ke salah satu ikon tampilan. Pelajari tombol pintasan ini untuk menavigasi alur kerja dengan cepat. Jadi, setelah beberapa saat, Anda akan menggunakan urutan tombol seperti W B Esc untuk membuka panel alur kerja, beralih ke tampilan komponen ke penekukan, lalu menutup panel alur kerja.

  • Data komponen didorong di antara node-node ini melalui berbagai proses, dan proses-proses ini direpresentasikan dalam panel alur kerja menggunakan ikon bintang 13 sudut. Misalnya, Anda berpindah dari tampilan Komponen 2D ke tampilan Komponen Laser dengan proses Laser Tech (yang menganalisis komponen 2D dan menetapkan pengaturan alat laser untuknya).

Proses yang telah Anda selesaikan akan ditandai dengan tanda centang di dalamnya. Proses yang belum Anda selesaikan (tetapi masih tersedia) akan ditandai dengan tanda silang di dalamnya. Anda dapat mengklik node proses ini untuk menyelesaikan prosesnya.

Mari kita rangkum apa yang dapat kita lihat pada tahap alur kerja komponen ini:

  • Saat ini ada 3 tampilan komponen yang tersedia (Komponen 2D, Komponen Laser, dan Model 3D yang dapat kita pilih secara bergantian).

  • Ada empat proses lebih lanjut yang tersedia:

    • Kita dapat melakukan Pascaproses Komponen Datar (ini menghasilkan laporan komponen datar yang berguna bagi operator laser atau mesin punch-press; biasanya laporan ini mencakup waktu pemotongan laser, pengaturan alat untuk mesin punch press, dan prasyarat pengaturan alat khusus lainnya untuk komponen ini).

    • Kita dapat melakukan Nesting Cepat (nesting cepat adalah nesting yang hanya berisi satu jenis komponen) dan menghasilkan lembaran penuh yang dipenuhi dengan komponen ini. Ini dapat digunakan untuk memproduksi lembaran yang hanya berisi komponen ini, atau untuk membantu membuat perkiraan biaya atau waktu yang cepat untuk komponen ini.

    • Kami dapat menetapkan Bend Tech (pengaturan alat press brake) untuk komponen tersebut.

    • Kami dapat menetapkan Weld Tech (pengaturan alat robot pengelasan) untuk komponen tersebut.

Memperluas node alur kerja: Tahap 2

Mari kita lanjutkan: Klik semua node proses yang tersedia satu per satu, dan lihat bagaimana panel alur kerja meluas. Lanjutkan hingga tidak ada lagi node yang tersisa. Berikut tampilannya setelah proses ini selesai :

Dalam keadaan sepenuhnya meluas, panel alur kerja memudahkan Anda untuk beralih secara instan antara enam tampilan berbeda dari komponen dalam berbagai modul pemrosesan. Anda juga dapat melihat dan mengirimkan atau mencetak semua output yang dihasilkan oleh modul-modul ini. (Output dapat berupa laporan, kode NC, atau studi waktu).

Menavigasi panel alur kerja

Panel alur kerja menampilkan banyak informasi dan aktivitas dalam bentuk grafis yang ringkas. Sering kali, panel ini akan berfungsi sebagai pusat kendali saat Anda mengerjakan komponen. Mari kita lihat lebih dekat beberapa ikon di panel alur kerja untuk memahami cara menggunakannya.

Node proses yang tersedia

Bintang 13 sudut dengan tanda + di dalamnya mewakili langkah pemrosesan yang sekarang tersedia. Misalnya, bisa berupa hemming benda datar 2-D menjadi komponen 3-D, atau penetapan pengaturan alat teknologi laser. Arahkan mouse ke node tersebut untuk menampilkan tip alat yang menjelaskan fungsi node tersebut.

Ini mengikuti pola khas untuk banyak node proses yang tersedia. Dengan mengklik node, proses akan dijalankan dengan pengaturan default. Ctrl+Click pada node akan memunculkan halaman pengaturan terlebih dahulu, dan setelah Anda meninjau/mengedit pengaturan, proses akan dijalankan. Misalnya, berikut adalah yang muncul saat Anda Ctrl+Click pada node Nesting Cepat:

Pengaturan nesting cepat ditampilkan sehingga Anda dapat mengeditnya sebelum melakukan nesting.

Node proses yang yang telah selesai

Setelah proses selesai, node akan berubah dari node proses yang tersedia menjadi node proses yang telah selesai; simbolnya berubah menjadi bintang dengan tanda centang di dalamnya. Pada titik ini, opsi yang tersedia untuk node tersebut akan berubah.

Ini adalah sekumpulan opsi khas yang tersedia pada node proses yang telah selesai. Mengklik node akan memunculkan pengaturan proses kembali, sehingga Anda dapat menyesuaikannya dan mencoba pemrosesan kembali. Opsi Ctrl+Click juga biasanya tersedia untuk menghapus data proses. Jika Anda memilih opsi ini, Anda akan diminta untuk mengonfirmasi sebelum penghapusan dilakukan. Misalnya, berikut adalah yang terjadi jika Anda Ctrl+Click pada node Model 3D, untuk komponen yang telah diproses sepenuhnya:

Perintah tambahan

Banyak node memiliki ikon kecil di dekatnya, yang menyediakan perintah tambahan. Perintah ini menyediakan beberapa fungsi yang terkait dengan node tersebut. Berikut beberapa contohnya.

  • Ikon tambahan di dekat setiap node teknologi biasanya memungkinkan Anda memilih mesin yang berbeda dan menyiapkan alat untuk mesin tersebut.

  • Ikon di dekat node Nesting cepat memungkinkan Anda melakukan nesting dengan ukuran lembar yang berbeda.

  • Ikon di dekat node output memungkinkan Anda melihat berbagai output dari node pemrosesan (laporan, program NC, atau studi waktu).

Pelacakan file sumber

Sebagian besar pemrosesan TecZone Bend dimulai dengan mengimpor data CAD (baik 2-D maupun 3-D). Komponen TecZone Bend yang dibuat dari data CAD ini dapat terus melacak komponen sumber tersebut. Saat sebuah komponen dibuka, TecZone Bend dapat memeriksa apakah file CAD asli yang digunakan untuk membuatnya telah diubah selama waktu tersebut. Jika ya, komponen tersebut sekarang sudah kedaluwarsa dan hal ini dapat dilihat di panel alur kerja.

  • Anda dapat memilih untuk me-refresh komponen dengan mengklik ikon tambahan refresh komponen di dekat node komponen sumber. TecZone Bend akan mengimpor kembali geometri CAD dan membangun kembali komponen tersebut.

  • Anda juga dapat memilih untuk berhenti melacak geometri CAD asli. Ini mungkin berguna, misalnya, jika file CAD asli ada di media yang dapat dilepas atau di drive jarak jauh yang mungkin tidak dapat diakses di masa mendatang. Untuk melakukannya, klik ikon tambahan putus tautan di dekat node komponen sumber. Ini akan memunculkan perintah untuk menghentikan pelacakan file sumber:

Ringkasan

Berikut ini ringkasan singkat tentang prinsip-prinsip yang terdapat di panel alur kerja.

  • Panel alur kerja menampilkan node yang mewakili berbagai tampilan komponen (seperti komponen laser, komponen yang ditekuk) dan node yang mewakili berbagai proses (seperti lipat ke 3D, tetapkan pengaturan alat laser).

  • Node proses yang tersedia (belum dilakukan) direpresentasikan sebagai bintang 13 sudut dengan tanda di dalamnya. Node proses yang sudah selesai direpresentasikan oleh bintang dengan tanda di dalamnya.

  • Mengklik node proses yang tersedia akan memanggil proses tersebut dengan pengaturan default. Ctrl+Click pada node proses yang tersedia akan memunculkan editor untuk mengedit pengaturan proses terlebih dahulu, kemudian memanggil proses tersebut.

  • Mengklik node proses yang sudah selesai memungkinkan Anda menyesuaikan pengaturan proses dan menerapkan pemrosesan kembali. Ctrl+Click pada node proses yang telah selesai akan menghapus data pemrosesan.

  • Ikon tambahan kecil di dekat node proses, atau node tampilan komponen menyediakan kontrol untuk mengubah beberapa parameter penting dari node proses tersebut (seperti mesin target, atau ukuran lembar nesting).
