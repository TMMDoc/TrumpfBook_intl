Menginstal Perangkat Lunak

Menginstal Perangkat Lunak

Bersiap untuk Menginstal

Prasyarat sistem

  • Salah satu sistem operasi berikut. Perhatikan bahwa Windows 7 memerlukan Service Pack 1 untuk diinstal, jika tidak, TecZone tidak dapat berfungsi pada Windows 7.

    • Windows 7 SP1, 64-bit (Windows 7 SP1 didukung dengan Pembaruan Keamanan Ekstensi yang diinstal)

    • Windows 8.1, 64-bit

    • Windows 10, 64-bit

    • Windows 11, 64-bit

  • RAM 8 GB (disarankan 16 GB)

  • Resolusi layar 1024 x 768 (disarankan 1920 x 1080)

  • Kartu grafis yang mendukung OpenGL 4.0 (disarankan GPU diskret Nvidia)

  • Mouse 3 tombol atau trackball

Perangkat lunak prasyarat

Perangkat lunak prasyarat berikut ini diperlukan, dan akan diinstal oleh penginstal TecZone secara otomatis, jika tidak ditemukan:

  • Visual C++ 2005 SP1 Redistributable (x86)

  • Visual C++ 2010 SP1 Redistributable (x86)

  • Visual C++ 2013 Redistributable (x64) – hanya pada platform 64-bit

  • Visual C++ 2019 Redistributable (x64) – hanya pada platform 64-bit

  • Microsoft .Net Framework 4.7.2

  • Microsoft .Net 6.0.1

Menginstal Perangkat Lunak

Jalankan TecZoneSetup.exe (nama file yang tepat mungkin berbeda, dan mungkin termasuk nomor revisi atau tanggal). Jika prasyarat (seperti .Net Framework) belum diinstal, penginstalan mungkin memerlukan waktu hingga 15 menit hingga selesai. Jika prasyarat sudah terinstal, proses penginstalan akan memakan waktu sekitar 1 atau 2 menit.

Ikuti petunjuk untuk menginstal TecZone. Proses penginstalan ini akan menempatkan pintasan ke aplikasi TecZone di desktop Anda, dan di menu mulai. Anda juga dapat memulai aplikasi dengan menjalankan program TecZone.exe secara langsung.

Menjalankan TecZone

Saat Anda menjalankan TecZone untuk pertama kalinya, selektor bahasa akan muncul dan Anda dapat memilih bahasa yang akan digunakan untuk TecZone. (Tergantung pada instalasi Anda, Anda mungkin melihat set bahasa yang berbeda di sini).

Bahasa yang Anda pilih di sini digunakan untuk TecZone, dan ini dapat diubah kemudian dengan mengklik Bahasa pada baris perintah. (Ikon ini muncul saat Anda memulai TecZone, dan ketika tidak ada komponen yang terbuka).
