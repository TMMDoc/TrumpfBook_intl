Inventaris alat

Ada banyak penahan lembaran kosong ENW, pisau ZBW, dan manipulator komponen (gripper) yang dapat dipasang dan digunakan dengan penekuk panel.

Figure 1. Gambaran umum pengaturan alat

Jendela Inventaris Alat di TecZone Fold digunakan untuk mengonfigurasi alat tambahan dan alat untuk penekuk panel.

Untuk mengakses jendela Inventaris Alat, buka komponen, dan atur alat untuk penekuk panel tertentu. Kemudian, klik tombol inventaris di baris perintah untuk membuka jendela Inventaris Alat.

Mengedit pisau ZBW atau alat ENW

Gunakan selektor Jenis untuk memilih jenis alat atau gripper yang ingin Anda lihat dan edit. Saat alat dipilih, panel akan terbuka di sebelah kanan, tempat Anda dapat mengontrol apakah alat ini tersedia untuk digunakan pada mesin ini, serta mengatur jumlah inventaris masing-masing segmen.

  • Klik Tersedia untuk mengaktifkan atau menonaktifkan alat ini jika ingin menggunakannya di mesin ini. Alat yang tidak tersedia akan berwarna abu-abu (seperti alat ENW-S3 pada gambar di sebelah kanan). Jika suatu alat ditandai sebagai tersedia, pemilih alat otomatis TecZone Fold akan mencoba menggunakannya secara otomatis jika dapat membantu menghindari benturan.

  • Gunakan kolom Hitung pada tabel inventaris untuk mengatur inventaris potongan agar sesuai dengan yang Anda miliki di lantai produksi. Ini digunakan oleh TecZone Fold untuk menyusun berbagai panjang dengan menggabungkan potongan-potongan ini.

Mengimpor data alat khusus

TecZone dilengkapi dengan banyak alat standar ENW dan ZBW yang sudah terpasang dan siap digunakan dalam database-nya. Anda hanya perlu memilih alat yang diinginkan dan menandainya sebagai Tersedia agar alat tersebut dapat digunakan oleh pemilih alat otomatis. Terkadang, Anda akan membeli alat khusus yang tidak standar. Dalam hal ini, alat tersebut akan disertai dengan file data alat (atau file .tdata). Anda dapat mengimpor file ini dengan mengklik tombol Tambahkan Baru dan memilihnya.

Alat khusus tersebut kemudian akan ditambahkan ke katalog. File .tdata berisi catatan yang memungkinkan TecZone Fold mengetahui geometri dan data teknis alat, sehingga alat tersebut dapat digunakan seperti tool standar pada umumnya.

Mengedit konfigurasi mesin

Beberapa jenis alat seperti pisau atas dan bawah, penahan lembaran kosong atas dan bawah, atau bahkan rangka-C mesin sangat jarang diganti. Untuk mengakses jenis alat ini dalam dialog Inventaris Alat, Anda harus menjalankan TecZone dalam mode admin[1] baris perintah -admin.

Dalam mode ini, ada lebih banyak jenis alat yang ditampilkan dalam daftar tarik-turun Jenis dan Anda dapat memilih dan mengubah salah satunya. Tidak seperti alat lainnya (seperti ZBW, ENW, Grippers), hanya satu komponen ini yang dapat dipasang pada mesin dalam satu waktu. Jadi, Anda cukup memilih satu saja. 1. Anda dapat memulai TecZone dalam mode admin dengan menggunakan mode parameter
