Buat data lipat

Anda dapat membuat Data Teknologi Lipat (pengaturan alat penekuk panel) untuk komponen lembaran logam menggunakan modul Lipat. Anggaplah Anda telah memasang mesin dan mengimpor komponen CAD (2D atau 3D):

Beralih ke CAM Lipat

Sekarang, komponen tersebut dapat diatur alatnya untuk penekuk panel hanya dengan menekan tombol F. TecZone Fold akan memunculkan menu tempat Anda dapat memilih salah satu penekuk panel yang telah dipasang. Komponen tersebut kemudian diatur pengaturannya untuk mesin ini. Urutan tekukan, komposisi penahan lembaran kosong, stopper, dan alat ZBW atau ENW opsional semuanya dihitung, dan Anda akan melihat hasil seperti ini:

Jika tidak ada kondisi kegagalan, program NC akan dibuat untuk komponen tersebut dan disimpan ke folder output yang telah diatur untuk mesin ini.

Lihat juga panel Alur Kerja, yang dapat diakses dengan menekan tombol W. Ini memberikan kontrol lebih besar atas pembuatan data teknologi penekukan, dan opsi untuk merutekan komponen melalui CAM laser, nesting, dll.

Aktivitas lebih lanjut

Berikut adalah ringkasan singkat dari beberapa aktivitas yang dapat Anda lakukan setelah komponen telah diatur untuk penekukan. Halaman-halaman berikut dalam panduan ini memberikan penjelasan lebih rinci tentang masing-masing aktivitas tersebut.

  • Jika Anda mengklik Spacebar, simulasi penekuk panel akan dimulai. Anda juga dapat menggunakan kontrol simulasi pada Navigator Lipat untuk memulai, menghentikan atau memundurkan simulasi:

  • Jika ada petunjuk peringatan atau kondisi kegagalan, hal tersebut akan muncul di Navigator Lipat, dan Anda dapat melihatnya serta menyelesaikannya dengan mengklik sel yang sesuai.

  • Anda dapat mengedit pengaturan alat lipat (penahan lembaran kosong, stopper, alat ENW dan ZBW, gripper).

  • Anda dapat mengganti mesin dengan mengklik nama mesin di tab di bawah komponen, lalu memilih mesin lain:

Saat solusi lipat dibuat, jika tidak ada kondisi kegagalan, program NC komponen akan dibuat. Anda dapat mengontrol perilaku ini (apakah program NC dibuat secara otomatis) dari halaman Output Lipat pada dialog Pengaturan. Jika kode telah dibuat, tombol kode nc di baris alat kiri akan menjadi abu-abu:

Saat Anda melakukan perubahan pada komponen, tombol ini akan diaktifkan kembali, dan Anda dapat membuat ulang program NC dengan mengklik ikon ini lagi (atau dengan menekan tombol C).
