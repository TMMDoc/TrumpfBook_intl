Tabel celah udara khusus

Tabel celah udara khusus

Tabel celah udara untuk penekuk panel memetakan ketebalan lembaran tertentu ke celah udara standar yang digunakan selama pemrosesan. Ini sudah terintegrasi dalam setiap penekuk panel, tetapi dapat diganti dengan menyediakan tabel celah udara khusus dalam bentuk file JSON. File JSON ini bernama air-gap-tables.json dan terletak di C:\ProgramData\Trumpf{APPNAME}\Data\System.

Di bawah ini adalah file air-gap-tables.json yang khas:

 {
  "version":1  "airGapTables": [
    {
      "machine": "112002,112003", "material": "DC01"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
     {
      "machine": "112003", "material": "*"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    },
    {
      "machine": "*", "material": "Alluminio"
      "values": [
        { "t": 0.75, "gap": 2.5 },
        { "t": 0.95, "gap": 3 },
        { "t": 1.25, "gap": 3.5 },
        { "t": 1.45, "gap": 4.5 },
        { "t": 1.65, "gap": 5.5 },
        { "t": 1.75, "gap": 6 },
        { "t": 2.05, "gap": 6.5 },
        { "t": 2.15, "gap": 7.5 },
        { "t": 2.55, "gap": 8 },
        { "t": 9.99, "gap": 8.5 }
      ]
    }
  ]
}

  • File ini berisi kumpulan nilai celah udara. Setiap tabel memiliki tombol mesin dan bahan yang berfungsi sebagai selektor.

  • Tombol-tombol ini diatur ke '*' (tanda bintang) untuk menunjukkan wildcard (cocok dengan semua), atau berupa daftar nilai mesin atau bahan yang dipisahkan dengan koma.

  • Pada contoh di atas, ada tiga tabel. Tabel pertama digunakan untuk mesin 112001 atau 112003, hanya untuk bahan DC01. Tabel kedua digunakan untuk semua bahan, untuk mesin 112003, dan tabel ketiga digunakan untuk semua mesin, untuk bahan Alluminio. Jika beberapa tabel cocok dengan kombinasi mesin/bahan komponen, tabel pertama yang cocok akan digunakan. Jadi, untuk mesin 112003, bahan DC01 tabel pertama digunakan (meskipun tabel kedua juga cocok).

  • Setiap tabel terdiri dari susunan sederhana tombol ketebalan, dengan nilai celah udara yang sesuai. Misalnya, entri pertama di atas harus diartikan sebagai: jika ketebalan kurang dari atau sama dengan 0,75 mm, gunakan celah 2,5 mm. Tombol ketebalan harus dalam urutan naik. Jika tidak ada tabel dalam file ini yang cocok dengan kombinasi bahan dan ketebalan komponen saat ini, tabel celah udara default untuk mesin akan digunakan.

Jika Anda membuat perubahan apa pun pada file air-gap-tables.json, Anda harus memulai ulang TecZone Fold sebelum perubahan ini berlaku.
Jika file JSON tidak terbentuk dengan benar, kondisi kegagalan akan ditampilkan saat Anda mencoba memproses komponen penekuk panel, tetapi TecZone Fold akan kembali menggunakan tabel celah udara default untuk mesin, dan akan melanjutkan.
