Solusi Perhitungan Ulang

Saat Anda menekan B, atau menggunakan panel alur kerja untuk menghitung solusi, TecZone Bend menghitung program tekuk menggunakan serangkaian pengaturan dan parameter default. Anda dapat melakukan penghitungan ulang solusi ini menggunakan pengaturan yang berbeda dengan mengklik menu Hitung Ulang dari baris perintah. Terkadang, ini juga berguna untuk menyelesaikan perhitungan setelah Anda mengedit beberapa pengaturan alat atau urutan secara interaktif.
Misalnya:

  • Anda dapat mengubah urutan, dan meminta TecZone Bend untuk menghitung stasiun alat baru sambil menjaga urutan tidak berubah.

  • Anda dapat mengubah stasiun alat, dan meminta TecZone Bend untuk menghitung urutan baru, atau untuk menyejajarkan komponen ke stasiun yang baru dibuat.

  • Anda dapat mencoba aktivitas pengaturan alat dengan set tool yang berbeda, atau mencoba menggunakan Pengaturan yang sudah dibuat.

Dialog Perhitungan Ulang

Dialog Perhitungan Ulang terlihat seperti gambar di samping.

Pengesampingan

Area ini berisi beberapa sakelar yang dapat Anda gunakan untuk mengesampingkan perilaku default TecZone Bend.

  • Aktifkan sakelar Jangan ubah urutan untuk memaksa TecZone Bend menghitung solusi baru tanpa mengubah urutan. Berbagai stasiun alat, posisi stopper, atau arah penyisipan komponen akan dicoba oleh TecZone Bend, sambil mempertahankan urutan yang tidak berubah.

  • Jika sakelar Tidak ada pengukuran sudut. diaktifkan, TecZone Bend menghitung solusi yang tidak melakukan pengukuran sudut, dan tidak akan menggunakan sensor stopper (tool sensor)

  • Abaikan alat dari file GEO/DXF menginstruksikan TecZone Bend untuk mengabaikan catatan pengaturan alat yang sering ditemukan dalam file GEO, dan terkadang dalam file DXF.

Pengaturan

Bagian ini mengatur strategi untuk pengaturan alat; secara default, TecZone Bend akan menghitung Pengaturan dari awal, tetapi Anda dapat menggunakan pengaturan ini untuk mengubah perilaku tersebut.

  • Menghitung Pengaturan baru adalah perilaku default - TecZone Bend akan menghitung pengaturan menggunakan semua alat yang diizinkan (lihat bagian Alat di bawah).

  • Pertahankan Pengaturan yang ada memberi tahu TecZone Bend bahwa tidak ada perubahan Pengaturan yang harus dilakukan; ini mencoba memproses semua penekukan menggunakan pengaturan saat ini untuk komponen tersebut. Jika beberapa penekukan mengalami benturan, atau panjang alat tidak cukup untuk pemrosesan, kondisi kegagalan tersebut tetap tidak diperbaiki, dan dapat dilihat setelah proses penghitungan ulang.

  • Jika Anda memilih GunakanSetup dari file, Anda diminta untuk memilih file pengaturan penekukan (bsetup). Pengaturan yang dimuat dari file ini kemudian digunakan untuk perhitungan; pengaturan tersebut tidak diubah, dan tidak ada stasiun tambahan yang dibuat. Pengaturan yang baru saja Anda muat dari file akan ditampilkan di selektor yang muncul di bawah bagian ini, dan Anda dapat mengklik tombol + di dekat selektor tersebut untuk memilih file pengaturan yang berbeda.

  • Opsi Gunakan Pengaturan dari Komponen lain hanya tersedia jika ada beberapa komponen lain yang sedang terbuka, dan telah dipasangi alat untuk press brake yang sama. Dalam hal ini, Anda dapat meminta TecZone Bend untuk menggunakan pengaturan yang sama seperti salah satu komponen lain tersebut; sebuah selektor akan muncul untuk memungkinkan Anda memilih komponen yang diinginkan.

Alat

Jika Anda memilih opsi Hitung Pengaturan baru di area atas, Anda dapat menggunakan area Alat untuk memilih daftar alat tekuk yang dapat digunakan untuk menghitung pengaturan baru. Jika Anda memilih salah satu opsi lain untuk Pengaturan, area ini tidak tersedia.

  • Alat pilihan untuk Mesin adalah opsi default. Untuk setiap press brake, Anda dapat memilih satu set tools[1] sebagai alat pilihan, dan TecZone Bend akan menggunakan alat tersebut saat melakukan pengaturan alat komponen.[2] (Jika Anda belum benar-benar mengatur alat-alat pilihan untuk sebuah mesin, TecZone Bend akan menggunakan semua alat yang ada dalam inventaris sebagai set tool pilihan untuk mesin tersebut).

  • Jika Anda memilih Alat di Inventaris, TecZone Bend akan menggunakan semua alat yang tersedia di inventaris pemasangan. Jika Anda belum benar-benar membuat daftar inventaris alat, TecZone Bend akan menggunakan semua alat yang ada di Katalog sebagai inventaris Anda.

  • Semua Alat dalam Katalog adalah pilihan alat yang paling lengkap; TecZone Bend akan menggunakan semua alat yang tersedia dalam katalog alat produsen untuk mencoba solusi. Meskipun hal ini mungkin berarti solusi tersebut tidak dapat diterapkan dalam produksi (menggunakan inventaris yang Anda miliki), hal ini berguna untuk eksplorasi spekulatif tentang alat apa saja yang diperlukan untuk memproduksi komponen yang sangat kompleks. (Selain itu, mode ini juga menginstruksikan TecZone Bend untuk mengasumsikan inventaris tak terbatas untuk setiap bagian - artinya, pengaturan alat optimal dihasilkan dengan mengasumsikan Anda memiliki jumlah tak terbatas dari setiap panjang bagian).

  • Anda dapat membuat set tool khusus dan menyimpannya dalam file daftar alat (btools). File-file ini kemudian dapat digunakan untuk menyiapkan alat pada komponen, saat Anda memilih opsi Gunakan set tool dari File. Saat Anda memilih opsi ini, Anda akan diminta untuk memilih file daftar alat, dan file daftar alat yang baru saja Anda gunakan akan ditampilkan untuk memudahkan pemilihan. Anda dapat mengklik tanda + di dekat daftar ini untuk memilih daftar alat baru.

  • Untuk kontrol paling langsung atas punch dan die, Anda dapat memilih opsi Pilih pisau atas dan die bawah, lalu memilih punch dan die yang ingin digunakan.

Prioritas urutan

Saat mengurutkan penekukan di suatu komponen, ada beberapa prasyarat yang berpotensi bertentangan. Misalnya, apakah TecZone Bend harus mencoba meminimalkan jumlah putaran komponen, atau meminimalkan penekukan di bagian titik gravitasi komponen yang berada di dalam mesin?
Sangat mungkin bahwa setiap operator mungkin memiliki seperangkat aturan prioritas yang berbeda, dan juga kemungkinan bahwa komponen yang berbeda mungkin memerlukan pengaturan prioritas yang berbeda. Dengan demikian, TecZone Bend memungkinkan Anda untuk melakukan perhitungan ulang dengan urutan prioritas yang berbeda, menggunakan area Prioritas Urutan dalam dialog ini.

Gunakan tombol panah dan di sebelah kanan untuk mengacak aturan; item yang berada di bagian atas memiliki prioritas lebih tinggi. Anda dapat menonaktifkan beberapa item prioritas ini sepenuhnya; hal ini menginstruksikan TecZone Bend untuk tidak mempertimbangkan item tersebut sebagai faktor saat menentukan urutan. (Terkadang, menonaktifkan beberapa item ini dapat menghasilkan urutan yang menggunakan sedikit stasiun alat).

  • Hindari benturan selama penekukan berlebih: Untuk mencapai akurasi sudut, sebagian besar penekukan harus sedikit ditekuk berlebih untuk mengimbangi lenting balik. Aturan urutan ini menginstruksikan TecZone Bend untuk melakukan penekukan dalam urutan yang memungkinkan terjadinya penekukan berlebih. Misalnya, pada komponen di bawah ini, kita harus melakukan penekukan 1 terlebih dahulu, baru kemudian penekukan 2; urutan sebaliknya akan menghalangi penekukan berlebih.

  • Hindari alat dengan prioritas rendah: Menginstruksikan TecZone Bend untuk menghindari penggunaan alat-alat dengan prioritas rendah, atau alat-alat yang tidak tercantum dalam daftar pilihan, sejauh mungkin.

  • Hindari pengencangan di sudut terbuka: Posisi pengukuran idealnya berada pada bidang yang sama dengan sumbu penekukan, dan tidak boleh ada penekukan yang telah diproses antara garis tekuk dan titik pengukuran. Berikut adalah contoh penyetelan stopper yang buruk yang ingin dihindari oleh aturan ini:

  • Panjang sisi yang lebih panjang sebagian besar sebelum garis tekuk: Terutama pada komponen yang berat, penting untuk menghindari situasi di mana titik gravitasi komponen berada di belakang garis alat. Aturan ini mencoba meminimalkan situasi seperti itu.[3]

  • Minimalkan penggunaan komponen: Komponen terbalik terjadi ketika komponen harus dibalik (penekukan berturut-turut berada pada arah atas/bawah yang berbeda).

  • Minimalkan putaran komponen: Putaran komponen adalah ketika sebuah komponen diputar ke kiri atau ke kanan pada bidang horizontal; hal ini dianggap lebih mudah bagi operator daripada membalik komponen; urutan default dari dua aturan terakhir ini mencoba untuk mengutamakan putaran komponen daripada membalik komponen, jika memungkinkan.

1. Untuk melakukannya, gunakan tombol inventaris di baris perintah, lalu pilih Alat Pilihan dari menu yang muncul.
2. TecZone Bend akan mencoba menggunakan hanya alat pilihan untuk komponen tersebut, tetapi jika terjadi benturan atau masalah lain yang tidak dapat diselesaikan, ia akan mulai menggunakan alat lain dari inventaris yang tersedia.
3. TecZone Bend menggunakan kebijaksanaan dalam menentukan bobot komponen dan sejauh mana titik gravitasi berada di belakang garis alat untuk mempertimbangkan hal ini sebagai masalah. Secara teknis, aturan ini menganggap suatu komponen akan terjatuh ke belakang jika ada momen terjatuh sebesar 0,15 Nm (Newton meter) ke belakang.
