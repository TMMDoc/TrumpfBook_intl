Bending Aids

Bending Aids

Bending Aids adalah komponen mesin press brake yang membantu menekuk logam lembaran yang lebih tebal. Mereka dirancang untuk memberikan dukungan dan kontrol tambahan selama proses pembengkokan, memastikan hasil yang akurat dan konsisten.

Press brake dapat memiliki beberapa bending aids atau satu bending aid yang diinstal, tergantung pada persyaratan spesifik operasi pembengkokan.

Bending aids hanya berlaku untuk seri TruBend 5000 dan 8000.

Mengonfigurasi Bending Aids

Bending Aids dapat dikonfigurasi di Database / Machines / Settings.

Ketika bagian diimpor dan mesin dengan bending aids dipilih, TecZone Bend secara otomatis mengonfigurasi bending aids berdasarkan persyaratan bagian.

Menggunakan Bending Aids

Setelah solusi bending yang layak dihasilkan, bending aids dapat disesuaikan lebih lanjut untuk mengoptimalkan proses pembengkokan.

Klik langsung pada bending aid di TecZone Bend untuk membuka panel Bend Aid. Nomor di panel bend aid mewakili urutan pembengkokan.

  • Index - Menunjukkan bending aid yang saat ini dipilih.

  • Position (Z,R) - Digunakan untuk menyesuaikan posisi bending aid di sepanjang sumbu Z dan R.

  • Enabled - Pilih kotak centang ini untuk mengaktifkan bending aid.

  • Auto-Place - Digunakan untuk secara otomatis memposisikan bending aid berdasarkan geometri bagian.

Speed:

  • Down - Digunakan untuk menyesuaikan kecepatan retraksi bending aid setelah operasi bend.

  • Set All Bends - Digunakan untuk menerapkan pengaturan kecepatan saat ini ke semua lekukan dalam urutan.

Advanced:

  • Method - Digunakan untuk mendefinisikan mode operasional bending aid. Opsi meliputi:

    • Default - Mengatur ulang pengaturan bending aid ke nilai default.

    • Return at UDP - Mengonfigurasi bending aid untuk kembali ke posisi yang ditentukan pengguna setelah setiap lekukan.

    • Tilt product - Memungkinkan bending aid untuk memiringkan bagian selama proses pembengkokan untuk keselarasan yang lebih baik.

    • Static angle - Memungkinkan pengaturan sudut tetap untuk bending aid selama operasi.

  • Stop angle - Digunakan untuk mengatur sudut di mana bending aid akan berhenti selama operasi.

Posisi parkir Bending Aids

Bending Aids dapat diparkir di posisi yang ditentukan ketika tidak digunakan. Ini membantu mengoptimalkan ruang kerja dan memastikan keamanan selama operasi. Ini juga mencegah tabrakan dengan komponen mesin lainnya.

Bending Aids dapat diparkir di posisi berikut:

  1. Default - Bending aids dapat diparkir hingga panjang meja.

  2. Small - Bending aids dapat ditempatkan 700 mm di luar panjang meja.

  3. Large - Bending aids dapat ditempatkan 1200 mm di luar panjang meja.

Untuk mesin tandem, ada empat bending aids yang tersedia. Untuk mesin tandem kiri, hanya parkir sisi kiri yang diizinkan. Untuk mesin tandem kanan, hanya parkir sisi kanan yang diizinkan.

Opsi posisi parkir hanya akan ditampilkan untuk seri TruBend 8000 dan mesin VarioPress.

Untuk mesin tandem B36, posisi parkir small tidak berlaku.

Membuat custom bending aids

Untuk membuat custom bending aids di TecZone Bend, Anda pertama-tama perlu membuat file ini yang mendefinisikan parameter bending aid dan file mesh untuk komponen bending aid (Carriage, Table, dan Plate).

Contoh file baid.ini
[BAData]
; The user defined name of the BendAid. Only used for custom bend aids
Name=BATest
; The range of V-widths the bend aid can support
DieV=16,250
; The range of die-heights the bend aid can support
DieHeight=-30,250
; Maximum angle, in degrees
MaxAngle=55
; Lift capacity, in kg
Payload=300
; Maximum lift speed, in degrees/second
Speed=45
; When the bending aid is at X=0, this is the bound of the support table in top view
Table=-159,-1032,159,-245.6
; The X-Span of the bending aid (when positioned at 0)
XExtent=-213,213
; Tells whether the user can change the forward speed
CanChangeForwardSpeed=False
; Tells whether safety stop angle is possible in the machine
CanStopAtReturn=True
; Can the bending aid movement be varied using different methods?
AdvancedControl=True
; What type of existing bending aid can this be mapped to? Used only by custom bend aids
EType=TB8000_2000N_B36

[Carriage]
Model=Carriage.mesh
Stroke=-1000,1000
Color=White
Shift=0,0,100

[Table]
Model=Table.Mesh
Stroke=50,200
Color=White

[Plate]
Model=Plate.mesh
Stroke=0,56
Color=#4A4A4A

Tempatkan file ini dan file mesh yang sesuai dalam folder yang kemudian dapat diimpor ke TecZone Bend.

Mengimpor custom bending aids

Untuk mengimpor custom bending aids:

  1. Navigasi ke File / Import / Bending Aids…​ dan pilih file ini yang sudah dibuat (pastikan file mesh yang sesuai berada di folder yang sama).

  2. Bending aids yang diimpor kemudian akan tersedia dalam dropdown pilihan bending aid di pengaturan mesin.

Opsi ini berguna ketika bending aid perlu disesuaikan untuk tugas pembengkokan spesifik atau ketika menggunakan bending aids khusus yang tidak termasuk dalam database TecZone Bend default.
