Taivutusapuvälineet

Taivutusapuvälineet ovat särmäyspuristimen konekomponentteja, jotka auttavat taivuttamaan paksumpia levymetalleja. Ne on suunniteltu tarjoamaan lisätukea ja hallintaa taivutusprosessin aikana, varmistaen tarkat ja johdonmukaiset tulokset.

Särmäyspuristimessa voi olla useita taivutusapuvälineitä tai yksi taivutusapuväline asennettuna riippuen taivutustoiminnon erityisvaatimuksista.

Taivutusapuvälineet ovat soveltuvia vain TruBend 5000- ja 8000-sarjoille.

Taivutusapuvälineiden konfigurointi

Taivutusapuvälineet voidaan konfiguroida kohdassa Database / Machines / Settings.

Kun osa tuodaan ja kone, jossa on taivutusapuvälineet, valitaan, TecZone Bend konfiguroi automaattisesti taivutusapuvälineet osan vaatimusten perusteella.

Taivutusapuvälineiden käyttö

Kun toteutettavissa oleva taivutusratkaisu on luotu, taivutusapuvälineitä voidaan edelleen säätää taivutusprosessin optimoimiseksi.

Napsauta suoraan taivutusapuvälinettä TecZone Bend:ssa avataksesi Bend Aid -paneelin. Taivutusapuvälinepaneelissa oleva numero edustaa taivutusjärjestystä.

  • Index - Ilmaisee valittuna olevan taivutusapuvälineen.

  • Position (Z,R) - Käytetään taivutusapuvälineen sijainnin säätämiseen Z- ja R-akselien suunnassa.

  • Enabled - Valitse tämä valintaruutu aktivoidaksesi taivutusapuvälineen.

  • Auto-Place - Käytetään taivutusapuvälineen automaattiseen sijoittamiseen osan geometrian perusteella.

Speed:

  • Down - Käytetään taivutusapuvälineen vetäytymisnopeuden säätämiseen taivutusoperaation jälkeen.

  • Set All Bends - Käytetään nykyisen nopeusasetuksen soveltamiseen kaikkiin taivutuksiin järjestyksessä.

Advanced:

  • Method - Käytetään taivutusapuvälineen toimintatilan määrittämiseen. Vaihtoehtoja ovat:

    • Default - Nollaa taivutusapuvälineen asetukset oletusarvoihin.

    • Return at UDP - Konfiguroi taivutusapuvälineen palaamaan käyttäjän määrittämään sijaintiin jokaisen taivutuksen jälkeen.

    • Tilt product - Mahdollistaa taivutusapuvälineen kallistaa osaa taivutusprosessin aikana parempaa kohdistusta varten.

    • Static angle - Mahdollistaa kiinteän kulman asettamisen taivutusapuvälineelle toiminnan aikana.

  • Stop angle - Käytetään kulman asettamiseen, jossa taivutusapuväline pysähtyy toiminnan aikana.

Taivutusapuvälineiden pysäköintipaikka

Taivutusapuvälineet voidaan pysäköidä määritettyyn paikkaan, kun niitä ei käytetä. Tämä auttaa optimoimaan työtilan ja varmistaa turvallisuuden toiminnan aikana. Tämä myös estää törmäykset muiden konekomponenttien kanssa.

Taivutusapuvälineet voidaan pysäköidä seuraaviin paikkoihin:

  1. Default - Taivutusapuvälineet voidaan pysäköidä pöydän pituuteen asti.

  2. Small - Taivutusapuvälineet voidaan sijoittaa 700 mm pöydän pituuden yli.

  3. Large - Taivutusapuvälineet voidaan sijoittaa 1200 mm pöydän pituuden yli.

Tandem-koneissa on käytettävissä neljä taivutusapuvälinettä. Vasemman tandem-koneen osalta sallitaan vain vasen puolen pysäköinti. Oikean tandem-koneen osalta sallitaan vain oikean puolen pysäköinti.

Pysäköintipaikka-vaihtoehto näytetään vain TruBend 8000 -sarjan ja VarioPress-koneille.

B36 tandem -koneille pysäköintipaikka small ei ole käytettävissä.

Mukautettujen taivutusapuvälineiden luominen

Jotta voit luoda mukautettuja taivutusapuvälineitä TecZone Bend:ssa, sinun on ensin luotava ini-tiedosto, joka määrittää taivutusapuvälineen parametrit, ja mesh-tiedostot taivutusapuvälineen komponenteille (Carriage, Table ja Plate).

Esimerkki baid.ini-tiedostosta
[BAData]
; The user defined name of the BendAid. Only used for custom bend aids
Name=BATest
; The range of V-widths the bend aid can support
DieV=16,250
; The range of die-heights the bend aid can support
DieHeight=-30,250
; Maximum angle, in degrees
MaxAngle=55
; Lift capacity, in kg
Payload=300
; Maximum lift speed, in degrees/second
Speed=45
; When the bending aid is at X=0, this is the bound of the support table in top view
Table=-159,-1032,159,-245.6
; The X-Span of the bending aid (when positioned at 0)
XExtent=-213,213
; Tells whether the user can change the forward speed
CanChangeForwardSpeed=False
; Tells whether safety stop angle is possible in the machine
CanStopAtReturn=True
; Can the bending aid movement be varied using different methods?
AdvancedControl=True
; What type of existing bending aid can this be mapped to? Used only by custom bend aids
EType=TB8000_2000N_B36

[Carriage]
Model=Carriage.mesh
Stroke=-1000,1000
Color=White
Shift=0,0,100

[Table]
Model=Table.Mesh
Stroke=50,200
Color=White

[Plate]
Model=Plate.mesh
Stroke=0,56
Color=#4A4A4A

Sijoita ini-tiedosto ja vastaavat mesh-tiedostot kansioon, joka voidaan sitten tuoda TecZone Bend:iin.

Mukautettujen taivutusapuvälineiden tuonti

Mukautettujen taivutusapuvälineiden tuominen:

  1. Siirry kohtaan File / Import / Bending Aids…​ ja valitse jo luotu ini-tiedosto (varmista, että vastaavat mesh-tiedostot ovat samassa kansiossa).

  2. Tuodut taivutusapuvälineet ovat sitten käytettävissä taivutusapuvälineen valintapudotusvalikossa koneasetuksissa.

Tämä vaihtoehto on hyödyllinen, kun taivutusapuväline on mukautettava erityisiin taivutustehtäviin tai kun käytetään erikoistuneita taivutusapuvälineitä, joita ei sisälly TecZone Bend:n oletustietokantaan.
