Tuonti/vienti

Tässä osiossa käsittelemme kohdan Import settings asetuksia.

Napsauta aloitussivulta kuvaketta Settings .

Import settings

Tässä osiossa käsittelemme kohdan Import settings asetuksia. Napsauta kuvaketta Settings. Napsauta kohtaa Import/Export ja siirry kohtaan Import settings.

Units for DXF files - Aseta tähän yksiköksi joko millimetrit tai tuumat.

Stitch together lines/arcs closer than this - Aseta tämä arvo (0>1 mm). Jos tuotavan osan viivat/kaaret ovat asetettua arvoa lähempänä, ohjelmisto yhdistää ne automaattisesti tuonnin yhteydessä.

Maximum thickness for sheet-metal part -Jotta suuren ohutlevyosan tunnistaminen olisi mahdollista ohutlevyn tunnistuskynnystä on nostettava. (10>40 mm) Sovellus asettaa tämän kohdan oletusasetukseksi automaattisesti 25 mm tai vastaavan tuuma-arvon riippuen käytettävästä yksiköstä.

Point entities - Määrittelee valitun vaihtoehdon perusteella tavan, jolla pisteet on määrä tuoda.

Import all - Tuo ja näyttää kaikki pisteet.

Skip points on polylines - Tämä asetus ohittaa kaikki polyline-objekteissa tunnistetut pisteet.

Skip all - Tämä asetus ohittaa kaikki pisteet tuonnin yhteydessä, eikä yksikään piste ilmesty tällöin näyttöön.

Ignore layers in DXF/DWG files - DXF- ja DWG-piirustukset laaditaan yleensä eri kerroksille. Tätä asetusta käytettäessä ohjelmisto ohittaa nämä kerrokset ja siirtää kaikki objektit oletuskerrokseen.

Explode blocks in 2D drawing - Purkaa ryhmät tuonnin aikana.

Convert white entities to black - Valittaessa tämä asetus sovellus muuntaa valkoiset objektit mustiksi tuonnin aikana.

Darken colors during DXF import - Valittaessa tämä asetus sovellus tummentaa värilliset objektit tuonnin aikana.

Remove duplicate segments - Valittaessa tämä asetus sovellus poistaa kaikki geometrioiden identtiset kopiot, jotka ovat löytyneet osasta tuonnin yhteydessä.

DXF settings

Tässä osiossa käsittelemme kohdan DXF settings asetuksia. Napsauta kuvaketta Settings. Napsauta kohtaa Import/Export ja siirry kohtaan DXF settings.

Angles in DXF are interior angles - Valitessasi tämän asetuksen sovellus käsittelee DXF-tiedoston kulmia avautumiskulmina.

Export settings

Tässä osiossa käsittelemme kohdan Export settings asetuksia. Napsauta kuvaketta Settings. Napsauta kohtaa Import/Export ja siirry kohtaan Export settings.

No POLYLINE objects in DXF output - Yleensä suljetut muodot tulostuvat polyline-objekteina DXF-tiedostojen viennin yhteydessä. Eräät CAD-järjestelmät eivät pysty käsittelemään tällaisia tulostetietoja. Tätä asetusta käytettäessä ohjelmisto tulostaa DXF-tiedoston viivoja ja kaaria käyttäen. Nämä tiedostot ovat luettavissa kaikkialla, mutta luotavat tiedostot ovat tällöin alkuperäisiä suurempia ja myös linjojen ja kaarien väliset yhteydet katoavat.

Output bend-info when saving DXF files - Valittaessa tämä asetus sovellus tulostaa viedyn DXF-tiedoston yhdessä taivutustietojen kanssa.

Bend-info in Starmatik format - Valittaessa tämä asetus sovellus tulostaa taivutustiedot Starmatik-muodossa. Sovellus asemoi tällöin tekstielementin täsmälleen kaikkien sellaisten viivojen keskelle, joiden on määrä olla taivutuslinjoja.

Convert black to gray on output - 2D-tietojen viennin yhteydessä DXF-tiedoston objektit tulostuvat oletusasetuksen mukaan mustina. Jotta objektit olisivat helpommin tunnistettavissa CAD-ohjelmissa, sovellus tulostaa objektit tätä asetusta käytettäessä harmaina.

Start MetaCAM when PDG files are exported - Valittaessa tämä asetus sovellus konfiguroi PDG-tiedoston automaattisesti siten, että se avautuu MetaCAM-sovelluksessa.

Format for flat pattern - Tässä kohdassa levitysmallille voi valita vientiä varten tiedostomuodon GEO, DXF tai PDG.

Spline conversion

Tässä osiossa käsittelemme kohdan Spline conversion asetuksia. Napsauta kuvaketta Settings. Napsauta kohtaa Import/Export ja siirry asetuksiin Spline conversion.

Convert splines on import - Tässä kohdassa voi valita, onko splinikäyrän muuntaminen pois käytöstä vai muuntaako sovellus splinikäyrät viivoiksi tai kaariksi. Kummassakin tapauksessa sovellus muuntaa jokaisen splinikäyrän yksittäiseksi polyline-objektiksi, joka sisältää viivasegmenttejä tai kaarisegmenttejä.

Node-count computation - Luotavien viivojen tai kaarien lukumäärän laskenta tapahtuu joko solmujen välin tai poikkeaman perusteella.

Length of each line or arc segment - Jos solmujen lukumäärän laskentaa koskevan valinnan kohdalla laskennan on valittu tapahtuvan solmujen välin perusteella, tässä kohdassa voi määritellä yksittäisen kaari- tai viivasegmentin pituuden. Splinikäyrän purkaminen tapahtuu tällöin tämän käyrän osan pituuden perusteella.

Maximum deviation during approximation - Jos solmujen lukumäärän laskentaa koskevan valinnan kohdalla laskennan on valittu tapahtuvan poikkeaman perusteella, tässä kohdassa voi määritellä suurimman sallitun poikkeaman alkuperäisen sileän splinikäyrän sekä viivojen tai kaarien avulla suoritettavan approksimoinnin välillä. Muodostettava polyline-objekti sisältää mahdollisimman vähän segmenttejä, samalla kun enimmäisvirhe pysyy tämän rajan sisällä.

Assembly Explode

Tässä osiossa käsittelemme kohdan Assembly Explode asetuksia. Napsauta kuvaketta Settings. Napsauta kohtaa Import/Export ja siirry asetuksiin Assembly Explode.

Skip sheet-metal parts with no bends - Jos tämä kytkin kytketään päälle rakenneryhmän purkamisen yhteydessä, sovellus näyttää vain ohutlevyosat, joissa on taivutuslinjoja. Jos kytkin kytketään pois päältä, sovellus näyttää kaikki kokoonpanon osat.

Components (nuts/bolts etc) to display - Tämän asetuksen avulla voit valita, mitkä lisäkomponentit sovellus näyttää lisäkomponentteja sisältävän rakenneryhmän purkamisen yhteydessä.

None - Sovellus näyttää purkamisen jälkeen vain ohutlevyosat. Sovellus ei näytä muita muttereita/pultteja.

Unidentified - Sovellus näyttää vain komponentit, jotka ohjelmiston on vielä tunnistettava.

All - Sovellus näyttää kaikki komponentit.

Layer mapping

Tässä osiossa käsittelemme kohdan Layer mapping asetuksia. Napsauta kuvaketta Settings. Napsauta kohtaa Import/Export ja siirry asetuksiin Layer mapping.

Tässä osiossa osissa käytettävät ja ohjelmistoon tuotavat kerrokset on mahdollista kohdentaa automaattisesti kerrosten toimintojen (käyttötarkoituksen) perusteella.

Esimerkki

Jos tuodussa osassa on MERKINTÖJEN kerros, tämän asetuksen perusteella sovellus voi sijoittaa kerroksen automaattisesti merkintöjen kerrokseen ohjelmistossa.

Layer name - Tuotuun osaan sisältyvä kerros käyttää tätä kerroksen nimeä vastaavaa toimintoa, joka on asetettu paneelissa 'Käyttö'.

Use - Tämä on kerrokseen liittyvä toiminto. Valittavissa ovat seuraavat vaihtoehdot:

Standard - Tämä on CAM-toimintojen yhteydessä käytettävä oletuskerros.

Auxiliary - Lisäkerros, jota ei ole tarkoitettu käytettäväksi CAM-toimintojen yhteydessä.

Mark - Sovellus käyttää kaikkia tämän kerroksen elementtejä merkitsemiseen, ei leikkaamiseen.

Approach marker - Laserin lähestymisaseman ilmaisevia piste-elementtejä.

Sequence marker - Muotojen järjestyksen ilmaisevia tekstimerkkejä.

Forming center - Muovauksen keskikohdan merkki (piste tai pieni L-kirjain).

Forming foot print - Muovauksen ääriviiva (jalanjälki).

Evaporate - Tätä kerrosta sovellus käyttää kalvon palamisen tunnistukseen.

Dot marking - Tämä kerros on varattu QR-koodeille.

Info - Tämä on varattu pelkästään informaatiolle.
