曲げ加工 CAM

曲げ加工 CAM

Back-gauges

Min.thickness to always pull-back (mm) -この設定で、入力した板厚に達した時、バックゲージが自動的に戻る距離をミリメートル単位で入力できます。

Clamping strategy -これで、バックゲージが加工品上でどう停止するかを設定します。次の選択オプションが利用可能です：

Default -バックゲージは、各曲げ加工でソフトウェアが最適と判断する方法で停止します。

Stop + Stop -両方のバックゲージの直線面が加工品に当たって止まります。

Clamp + Stop - 11つのバックゲージが加工品をクランプし、もう 1 つのバックゲージが直線面で加工品を止めます。

Angle Measurement

Angle measurementトルンプ（社） マシン専用です。自動制御曲げシステムの角度測定システムを使用すると、シリーズの最初のパーツも完璧に仕上がります。センサーがスプリングバックとともに実際の角度を検出し、プレスビームを誘導して目的の角度を実現します。このセクションで、マシンのACB設定を入力できます。

Angle Measurement Systems

Laser -マシンにACB Laserが搭載されている場合は、このオプションにチェックを入れてください。ACB Laserは、レーザが材料にに線を投影し、カメラが角度を検出するところです。

Wireless -　 -マシンにACB Laserワイヤレスを使用している場合、このオプションにチェックを入れてください。ACBワイヤレスは、上部ツールに統合されたセンサーディスクです。

Classic -マシンがACB Classicを使用している場合は、このオプションをオンにします。

Min. bend length to use 2 angle measures -このオプションは、折り曲げ加工時に正しい角度を確保するために、折り曲げの最小長さを 2 回測定します。50 > 4000 ミリメートルの範囲になります。

Min. bend length to use 3 angle measures -このオプションは、折り曲げ加工時に正しい角度を確保するために、折り曲げの最小長さを 3回測定します。50 > 4000 ミリメートルの範囲になります。

ACB laser sensor park position -これはレーザ センサーの休止位置です。レーザーセンサーの位置。

ACB Quality limit - warning -これは、パーツの測定時に警告を出す限界値で、ファッジファクターのようなものです。それが指定の範囲にある場合、警告が出ます。

ACB Quality limit - error - -これは、パーツの測定時にエラーを出す限界値で、ファッジファクターのようなものです。それが指定の範囲にある場合、エラーが出ます。

Geometry Adaptation Strategy

ソフトウェアで2Dまたは3Dファイルを編集すると、曲げ半径および縮み量係数がチェックされます。Easy Programmingで加工品調整のために選択したストラテジーに応じて、様々な加工計画が実行されます。決定がインタラクティブに設定されている場合は、Adapt Geometryダイアログが表示されます。

When starting with 2D flat

Ask each time - 2Dパーツから曲げ加工に移行する際、この設定を使用するとAdapt Geometryダイアログが自動的に表示されます。ここには、フラット サイズを保持するか、3D ディメンションを保持するか、何も変更しないかのオプションが表示され、これが非常にインタラクティブでユーザー主導の機能なことがわかります。

Keep 3D dimensions -このオプションは展開を変更します。それを手動で保存して下さい。ブランク材がまだ切断されていない場合、このストラテジーを使用します。

Keep 2D dimensions -このオプションでは、曲げ半径と縮み量係数が既存の展開に調整されます；曲げ線は同じ位置に保持されます。これでパーツの辺の長さを変更します。展開は変更されません。このストラテジーは、ブランク材がすでに切断されている場合に使用されます。

No adjustments -このオプションでは、曲げ半径と縮み量係数は変更されません。パーツは正しい寸法で3D表示されますが、表示された曲げ半径は実際の曲げ半径と一致しません。各加工品の曲げ寸法は変えられます。

When starting with 3D model

Ask each time - 3Dパーツから曲げ加工に移行する際、この設定を使用するとAdapt Geometryダイアログが自動的に表示されます。ここには、フラット サイズを保持するか、3D ディメンションを保持するか、何も変更しないかのオプションが表示され、これが非常にインタラクティブでユーザー主導の機能なことがわかります。

Keep 3D dimensions -このオプションは展開を変更します。それを手動で保存して下さい。ブランク材がまだ切断されていない場合、このストラテジーを使用します。

Keep 2D dimensions -このオプションでは、曲げ半径と縮み量係数が既存の展開に調整されます；曲げ線は同じ位置に保持されます。これでパーツの辺の長さを変更します。このストラテジーは、ブランク材がすでに切断されている場合に使用されます。

No adjustments -このオプションでは、曲げ半径と縮み量係数は変更されません。パーツは正しい寸法で3D表示されますが、表示された曲げ半径は実際の曲げ半径と一致しません。各加工品の曲げ寸法は変えられます。

Bend deduction calculation -このオプションを使用すると、補正係数 kの使用が義務になります。この設定では、アプリケーション初期設定にすることも、マシンごとに個別に調整することもできます。

トルンプ（社）以外のマシンの場合、補正係数 kは常に使用されます；トルンプ（社） 曲げ計算はトルンプ（社）プレスブレーキ専用です。

BG-Left variant -使用するBGの種類に応じて、NCプログラムがそれに合わせて修正されます。

BG-Right variant -使用するBGの種類に応じて、NCプログラムがそれに合わせて修正されます。

Other Settings

Hemming pre-bend angle -このオプションでは、折り曲げ加工予備曲げ角度を10度から60度の間で編集できます。初期設定は40度です。これは、ほとんどのメーカーが推奨する折り曲げ加工予備曲げ角度です。トルンプ（社）のマシンのすべてのエディションでは、これが30度に設定されています。

Bend-guard mode override -これを自動計算（初期設定）に設定すると、Bendは各曲げ加工の曲げ加工安全装置モードを計算します。ただし、適切かどうかに関係なく、ベンドガードモードを特定のオプションに強制的に設定して、そのモードのみを使用することも可能です。

Application Settingsの代わりに、マシンの初期設定にして、各マシンのこの設定を上書き出来ます。

Machine table position -この設定では、テーブル位置に関する 3 つのオプションがあります：

  1. 取り出し済み

  2. 下

  3. 上

Tool inventory check -このオプションが有効だと、加工品を曲げるのに必要なツールセグメントが表示されます。在庫不足マーカーが表示されます。

Avoid tool overload if possible -このオプションをオンにすると、ツールの過負荷を回避できます。優先度の高いツールがパーツに適していても、過負荷を回避するため、代わりに優先度の低いツールが使用されます。

Ignore tool hints from GEO/DXF files -このオプションを有効にすると、DXF/GEOファイルの使用するツーリングに関する情報は無視されます。
