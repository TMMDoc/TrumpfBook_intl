マシンのデフォルト設定

マシンのデフォルト設定

曲げパーツがツール化されると、設定メニューから Machine Defaultsオプションにアクセスできます。

Bend-CAM Settings、Bend Outputs、 Bend View、およびBend Cellのほと んどのパラメータは、Application Settingsと同様です。これらのパラメータ の説明については、Application Settingsセクションを参照し てください。

Machine Defaultsエディタで使用可能な設定は、選択したマシンタイ プと構成によって異なります。

Bend Defaults設定では、曲げプロセスに関連するさまざまなパラメータを構 成できます。これには、Beam cycle、Back-gauge movement、およ びOther settingsが含まれます。

Beam cycle

Approach speed to Mute point - ミュートポイントまでのビー ムの速度。

Mute point (distance above sheet surface) - ミュートポイン トで、最高速度（急速下降）が自動的に減速速度（クランピング）に切り替わります。

Hemming distance (X-axis) - この距離は、ヘムエッジとツール表面の間の ギャップを指します。ヘムエッジがヘミングツールに突き当たるのを防ぐために提供され ています。

この設定はTruBendCellマシンにのみ適用されます。

Mute point for hemming - ヘムツールのミュートポイント距 離。

Slow speed to pinch point - ミュートポイントとクランピング ポイント間のビームの速度。

Pinch-point correction - 標準ピンチポイントに追加され る補正値。

Pinch-point delay - 曲げが開始される前のピンチポイント の遅延時間。

Bending speed - 曲げ中のビームの速度。

Bend hold - ビームはこの期間、下死点に留まります。

Decompress speed - 減圧経路沿いのビームの速度（減 圧）。

Auto-compute decompress distance - 曲げ角度と材料 特性に基づいて減圧距離を自動的に計算します。

Decompress distance - 減圧経路により、曲げ後にマシン のフレームの張力が制御された方法で解放されます。これは、下死点（LDP）で最大張力 下にあるためです。減圧経路は、ビームをゆっくりした速度で上方に移動させます。

Minimum ram opening - ビーム開放は、曲げ操作後にビームが 上方に移動する量です。
正の値： プレスビームはミュートポイントの上に配置されます。
負の値： プレスビームはミュートポイントの下に配置されます。

Back-gauge movement

Z axis speed - Z軸移動のバックゲージ速度。

X axis speed - X軸移動のバックゲージ速度。

R axis Speed - R軸移動のバックゲージ速度。

Movement trigger - バックゲージ移動のトリガー。

Movement delay - バックゲージ移動開始前の遅延。

Movement path - バックゲージ移動の経路を定義します。

Safe velocity：バックゲージは、プログラムされた速度で 危険ゾーンを迂回します（ツールの50 mm後方および上方）。バックゲージは最終ステッ プで前進します（X軸）。ゲージフィンガーとツールの間の距離が50 mm未満の場合、バッ クゲージは減速します。

Safe path：バックゲージは、プログラムされた速度で危険 ゾーンを迂回します（ツールの50 mm後方および上方）。バックゲージは最終ステップで下 降します（R軸）。すべての軸はプログラムされた速度で移動します。

Parallel retraction (along with beam opening) - バッ クゲージはビーム開放と並行して後退します。

Other settings

Min. overhang of bend beyond tooling - ツーリング を超える最小曲げ長さ。

Minimum punch-die gap - マウント後のパンチとダイの最小 ギャップ。Minimum punch-die gapは、オープンハイト（パンチレール とダイレール間の距離）からダイ高さを引いた値です。

Bending aid UP speed - ベンディングエイドの上昇速度。

Bending aid DOWN speed - ベンディングエイドの下降速度。

Operator console/foot switch - ベンディングエイドを制御す るために使用されるオペレーターコンソールまたはフットスイッチ。オペレーターは、マ シンを操作するために1つまたは両方のフットスイッチのいずれかを選択できます。
