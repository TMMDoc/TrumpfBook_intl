パネルベンディング

Geometry Adaptation Strategy

2 Dまたは3 Dファイルがソフトウェアで編集されると、曲げ半径と縮み量係数がチェックされます。Easy Programming 設定で加工品調整のために選択したストラテジーに応じて、様々な加工計画が実行されます。決定がインタラクティブに設定されている場合は、。 Adapt Geometry ダイアログが表示されます。

2 Dフラットから始める場合：

Ask each time - 2D パーツから曲げ加工へ移動する際に、この設定を使うと自動的にAdapt Geometryダイアログが表示されます。ここには、フラット サイズを保持するか、3D 寸法を保持するか、何も変更しないかのオプションが表示され、これが非常にインタラクティブでユーザー主導の機能なことがわかります。

Keep 3D dimensions - このオプションで展開を変更します。それは手動で保存して下さい。ブランク材がまだ切断されていない場合、このストラテジーを使用します。

Keep 2D dimensions - このオプションでは、曲げ半径と縮み量係数が既存の展開に調整されます；曲げ線は同じ位置に保持されます。これでパーツの辺の長さを変更します。展開は変更されません。このストラテジーは、ブランク材がすでに切断されている場合に使用されます。

No adjustments - このオプションでは、曲げ半径と縮み量係数は変更しません。パーツは正しい寸法で3D表示されますが、表示された曲げ半径は実際の曲げ半径と一致しません。各加工品の曲げ寸法は変えられます。

3 Dモデルから始める場合：

Ask each time - 3Dパーツから曲げ加工に移行する際、この設定を使用するとダイアログが自動的に表示されます。 Adapt Geometry ダイアログが自動的に表示されます。ここには、フラット サイズを保持するか、3D 寸法を保持するか、何も変更しないかのオプションが表示され、これが非常にインタラクティブでユーザー主導の機能なことがわかります。

Keep 3D dimensions - このオプションで展開を変更します。それは手動で保存して下さい。ブランク材がまだ切断されていない場合、このストラテジーを使用します。

Keep 2D dimensions - このオプションでは、曲げ半径と縮み量係数が既存の展開に調整されます；曲げ線は同じ位置に保持されます。これでパーツの辺の長さを変更します。このストラテジーは、ブランク材がすでに切断されている場合に使用されます。

No adjustments - このオプションでは、曲げ半径と縮み量係数は変更しません。パーツは正しい寸法で3D表示されますが、表示された曲げ半径は実際の曲げ半径と一致しません。各加工品の曲げ寸法は変えられます。

Machine options

KB40 version -マシンにKB40ローダーが搭載されている場合、このオプションを有効にしてください。これで、KB40パレットの作成および変更機能が有効になります。

KB40 frame - KB40 frameのタイプ（ STDまたはXL ）

Place part at center of the KB40 pallet -このオプションを有効にして、パーツをKB 40パレットの中央に配置します。

Additional pad - loading head -このオプションを有効にして、中央真空パッドを追加します。

Modular central vacuum - loading head -モジュール式中央真空パッド（真空のオン/オフ切り替えが可能）

KB44 part-flipper installed -ターンステーション（KB44）が装備されている場合は、このオプションを有効にしてください。
