ベンディングエイド

ベンディングエイド

ベンディングエイドは、より厚い板金の曲げを支援するプレスブレーキマシンの コンポーネントです。曲げプロセス中に追加のサポートと制御を提供するように設計されており、 正確で一貫した結果を保証します。

プレスブレーキには、曲げ作業の特定の要件に応じて、 複数のベンディングエイドまたは単一のベンディングエイドを取り付けることができます。

ベンディングエイドは TruBend 5000 および 8000 シリーズにのみ適用されます。

ベンディングエイドの設定

ベンディングエイドは Database / Machines / Settings で設定できます。

パーツがインポートされ、ベンディングエイド付きのマシンが選択されると、 TecZone Bend はパーツの要件に基づいてベンディングエイドを自動的に設定します。

ベンディングエイドの使用

実行可能な曲げソリューションが生成されると、ベンディングエイドをさらに 調整して曲げプロセスを最適化できます。

TecZone Bend でベンディングエイドを直接クリックして Bend Aid パネルを開きます。 ベンドエイドパネルの番号は曲げシーケンスを表します。

  • Index - 現在選択されているベンディングエイドを示します。

  • Position (Z,R) - Z 軸と R 軸に沿ったベンディングエイドの位置を調整するために使用されます。

  • Enabled - このチェックボックスを選択してベンディングエイドをアクティブにします。

  • Auto-Place - パーツのジオメトリに基づいてベンディングエイドを自動的に配置するために使用されます。

Speed:

  • Down - 曲げ操作後のベンディングエイドの格納速度を調整するために使用されます。

  • Set All Bends - 現在の速度設定をシーケンス内のすべての曲げに適用するために使用されます。

Advanced:

  • Method - ベンディングエイドの動作モードを定義するために使用されます。オプションには次のものが含まれます:

    • Default - ベンディングエイドの設定をデフォルト値にリセットします。

    • Return at UDP - 各曲げ後にユーザー定義の位置に戻るようにベンディングエイドを設定します。

    • Tilt product - より良い位置合わせのために曲げプロセス中にパーツを傾けることができるようにします。

    • Static angle - 操作中にベンディングエイドの固定角度を設定できるようにします。

  • Stop angle - 操作中にベンディングエイドが停止する角度を設定するために使用されます。

ベンディングエイドの待機位置

ベンディングエイドは、使用していないときに指定された位置に待機させることができます。これにより、 ワークスペースが最適化され、操作中の安全性が確保されます。また、 他のマシンコンポーネントとの衝突を防ぎます。

ベンディングエイドは次の位置に待機させることができます:

  1. Default - ベンディングエイドはテーブルの長さまで待機できます。

  2. Small - ベンディングエイドはテーブルの長さを超えて 700 mm 配置できます。

  3. Large - ベンディングエイドはテーブルの長さを超えて 1200 mm 配置できます。

*タンデム*マシンの場合、4 つのベンディングエイドが利用可能です。左タンデム マシンの場合、左側の待機のみが許可されます。右タンデムマシンの場合、 右側の待機のみが許可されます。

待機位置オプションは TruBend 8000 シリーズおよび VarioPress マシンに対してのみ表示されます。

B36 タンデムマシンの場合、待機位置 small は適用されません。

カスタムベンディングエイドの作成

TecZone Bend でカスタムベンディングエイドを作成するには、まずベンディングエイドの パラメータを定義する ini ファイルと、ベンディングエイドコンポーネント (キャリッジ、テーブル、プレート) の mesh ファイルを作成する必要があります。

サンプル baid.ini ファイル
[BAData]
; The user defined name of the BendAid. Only used for custom bend aids
Name=BATest
; The range of V-widths the bend aid can support
DieV=16,250
; The range of die-heights the bend aid can support
DieHeight=-30,250
; Maximum angle, in degrees
MaxAngle=55
; Lift capacity, in kg
Payload=300
; Maximum lift speed, in degrees/second
Speed=45
; When the bending aid is at X=0, this is the bound of the support table in top view
Table=-159,-1032,159,-245.6
; The X-Span of the bending aid (when positioned at 0)
XExtent=-213,213
; Tells whether the user can change the forward speed
CanChangeForwardSpeed=False
; Tells whether safety stop angle is possible in the machine
CanStopAtReturn=True
; Can the bending aid movement be varied using different methods?
AdvancedControl=True
; What type of existing bending aid can this be mapped to? Used only by custom bend aids
EType=TB8000_2000N_B36

[Carriage]
Model=Carriage.mesh
Stroke=-1000,1000
Color=White
Shift=0,0,100

[Table]
Model=Table.Mesh
Stroke=50,200
Color=White

[Plate]
Model=Plate.mesh
Stroke=0,56
Color=#4A4A4A

ini ファイルと対応する mesh ファイルをフォルダに配置し、 TecZone Bend にインポートできます。

カスタムベンディングエイドのインポート

カスタムベンディングエイドをインポートするには:

  1. File / Import / Bending Aids…​ に移動し、すでに作成された ini ファイルを選択します (対応する mesh ファイルが同じフォルダにあることを確認してください)。

  2. インポートされたベンディングエイドは、マシン設定の ベンディングエイド選択ドロップダウンで利用可能になります。

このオプションは、特定の曲げタスクに合わせてベンディングエイドをカスタマイズする 必要がある場合、またはデフォルトの TecZone Bend データベースに含まれていない特殊な ベンディングエイドを使用する場合に便利です。
