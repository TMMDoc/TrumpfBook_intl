ジオメトリの修正

TecZone Bend では、強力な 2D スケッチエディターを使用して、図形ﾃﾞｰﾀを修正したり、クリーンアップ/付け加えができます。ショートカットキーSでスケッチモードに切り替えます。エディタは、展開ビューにパーツを表示します。

展開で、Sketch アイコンをクリックするか、ショートカットキーを押します S。

メニューが開き、展開の加工ようにさまざまなアイコンが表示されます：

スケッチパネル

アイコン アイコン 意味

Select

オブジェクト、線、入力などの選択

Line

線のスケッチ

Connected line

任意の数の線をスケッチします

Parallel

直線に平行な線をスケッチします

Normal

円弧線の接線をスケッチします

Bendline

線の法線をスケッチします

Center-Point Arc

曲げ線をスケッチします

2-Point Arc

中心点、スタート位置、終点からの円弧をスケッチします

3-Point Arc

規定した2つの点（スタート位置と終点）を通る円弧をスケッチします

Tangent Arc

描画エレメントに接する円弧をスケッチします

Rectangle

長方形をスケッチします

Center-Point Rectangle

中央から長角をスケッチします

Circle

円をスケッチします 円の中心点を選択し、カーソルをドラッグして、 半径を規定したり、半径の値を入力します

2-Point Circle

円周に応じて円をスケッチします 円周上の１点を選択して、次いで第２および第３の点を選択します

3-Point Circle

円周に応じて円をスケッチします 円周上の１点を選択して、次いで第２および第３の点を選択します。

2-Tangent Circle

2つの接線のある円をスケッチします。円の直径を入力し、次に第１の接線、そして第２の接線を選択します。

3-Tangent Circle

3つの接線のある円をスケッチします。円の直径を入力し、次に 第１の接線、それから第２、第３の接線を選択します。

Inscribed Polygon

3つの接線のある円をスケッチします。円の直径を入力し、次に 第１の接線、それから第２、第３の接線を選択します。

Circumscribed Polygon

多角形をスケッチします。辺の数を指定し、中心点と辺の中央を選択します

Edge Polygon

多角形をスケッチします。側面の数を指定し、辺のスタート位置と終点を規定します。

Fillet

2つの描画エレメントの交点に、指定した半径で角を丸め、接線円弧を作成します。

Chamfer

2つの描画エレメントの交差コーナーに斜面を作成します

In-Fillet

入力した半径で、2つの描画エレメント交差コーナーをトリミングします。

Corner Step

2つの描画エレメントの交差コーナーを長角でトリミングします。この 長角のサイズは事前に入力できます。

Edge Recess

矩形のフリーカットを作成します。角の間隔、フリーカットの深さを入力してから、 コーナーを選択します。

Edge U-Cut

長丸開口部のフリーカットを作成します。角の間隔、フリーカットの幅と深さを入力してから、 コーナーを選択します。

Edge V-Cut

三角形のフリーカットを作成します。角の間隔、フリーカットの幅と深さを入力してから、 コーナーを選択します。

Keyslot

入力した値で、円に鍵穴を作成します。

Fillet 3-Seg

3つの接続した描画エレメントを丸めます

Extend

延長する描画エレメントを選択します
ctrl境界線の選択
Shift接続
Alt ドラッグ

Trim

トリミングする描画エレメントの選択
Ctrl境界線の選択
Shift エレメントの削除
Altポリラインの削除

Join

複数の個別のポリラインをトリミングして1つに結合するのに使用します

Offset

1つ以上の描画エレメント、スケッチしたモデルのエッジ、またはモデルの面を、指定した間隔でオフセットします

Move

描画エレメントをctrlで選択し、原点を指定して描画エレメントを移動します

Rotate

描画エレメントをctrlで選択し、回転の中心点、それからスタート位置と終点を選択して、描画エレメントを回転します

Scale

描画エレメントをctrlで選択し、原点を指定した後、スタート位置と終原点を選択して、描画エレメントを拡大縮小します。

Mirror

描画エレメントをctrlで選択し、その後、ミラーラインのスタート位置と終点を選択して、描画エレメントを鏡映（ミラー）します。

Rectangle Array

リニアパターンを使用して、1つ以上の描画エレメントの参照コピーを複数作成し、それらを1本または2本の直線パスに沿って等間隔に配置します。 リニアパターンをクリックし、希望の値を入力します

Polar Array

リニアパターンを使用して、1つ以上の描画エレメントの参照コピーを複数作成し、軸の周りに等間隔で配置します。 円パターンをクリックし、希望の値を入力します

Union

2つ以上の閉じた描画エレメントを選択して、互いのサーフェスを結合します。

Intersection

2つ以上の閉じた描画エレメントを選択して、選択したエレメントの切断領域を生成します。

Subtraction

2つ以上の閉じた描画エレメントを選択して、サーフェイスをトリミングします

Copy Notch

このツールを使用して、エッジに沿って切り欠き部の複数のコピーを作成できます。まず、 コピー間のスペースを入力し、作成したい切り欠き部のコピー数を入力します。次に、切り欠き部に隣接する2つの線セグメントをクリックして、切り欠き部を選択します

Delete Notch

このツールで、角または線セグメントに沿って切り欠き部を削除します。切り欠き部に隣接する2つの線セグメントをクリックして、切り欠き部を削除します

Mirror Notch

このツールで、角または線セグメントに沿って切り欠き部を鏡映（ミラー）します。切り欠き部に隣接する2つの線セグメントをクリックして、切り欠き部をミラーします

Spline

新規スプラインを開始するには、スタート位置をクリックし、次の点をクリックすると、スプラインを作成できます。スプラインを閉じるには、ALTキーを押して、 次にクリックします

Profile

ベースの長さ、フランジの高さ、厚さ、フランジの角度、内側半径をタイプして、 Enter キーを押し、プロフィールを作成します

Text

レーザ加工機がパーツにマーキングするテキストの描画にに使用します。このツールボタンをクリックすると、入力バーにテキスト、サイズ、および回転角度の入力ボックスが表示されます。

Truetype Text

任意のTrueTypeフォントの文字の形状からポリラインを作成するために使用されます。その後、レーザツーリングをこれらのポリラインに適用し、切断できます。このボタンをクリックすると、フォント ダイアログが表示され、テキストに使用するフォントを選択出来ます。

Common Shape

いくつかの一般的な形状を作成し、図面に挿入するために使用されます。このボタンをクリックすると、形状作成ダイアログが表示され、リストの一般的な図形のパレットから選択出来ます。

Simple Dimension

最初の寸法点を選択し、次に2番目の寸法点を選択して、寸法線を配置します

Baseline Dimension

最初の寸法点を選択し、次に2番目の寸法点を選択して、寸法線を配置します

Continue Dimension

最初の寸法点を選択し、次に2番目の寸法点を選択して、寸法線を配置して、次の寸法点を選択します

Horizontal Ordinate Dimension

座標寸法は、図面の座標ゼロから測定する寸法セットです。原点を選択し、寸法を配置します

Vertical Ordinate Dimension

座標寸法は、図面の座標ゼロから測定する寸法セットです。原点を選択し、寸法を配置します

Angular Dimension

角度の寸法を作成します。最初の線を選択し、次に角度を寸法付記入したい2本目の線を選択します。

Radius Dimension

半径の寸法を作成します。半径を寸法記入したい円を選択します。 ctrlで直径の寸法記入します

Center Point Radius Dimension

半径の連続引出線付き寸法記入を作成します。半径の寸法記入をする円を選択します。 ctrlで直径の寸法記入します

Callout Dimension

吹き出し形式で図面に注釈を追加するために使用します。吹き出しを作成するには、 表示するテキストをタイプし、クリックして矢印のポイントを指定します。 もう一度クリックして、テキストを配置する場所を指定します。

Segment Dimension

直線と曲線セグメントの寸法記入を追加するために使用します。寸法記入するセグメントをクリックして、 もう一度クリックして寸法を配置します。またを長押し、セグメントをクリックして、寸法を自動的に配置します。
