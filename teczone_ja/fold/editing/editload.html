ローディングの編集

自動パネルベンダー（ TBC -7030やTBC -7020など）のローディングプロセスを編集するには、上端のナビゲーターのローディングセルを2回クリックします。これはLというラベルの付いたセルです。最初のクリックでローディングプロセスを選択し、2回目のクリックでロードパネルが表示されます。

7030 （フロントローディング）

以下は、フロントローディング7030マシンのローディングパネルです：

  • Centering angle 設定で、心合わせオペレーション中のパーツ回転方向を指定します。これはマシンの内側に面しているサイドで、実際にフロントゲージの基準になるサイドとは正反対です。この例では、サイドC 2はマシンに面しています（そしてサイドC 1は+ 90度ではゲージングに使用されます）。この設定では、初期の心合わせに適した回転方向のリストが表示されます。

  • Part Pickup セレクターを使用して、パーツをサクションカップかリアプライヤーでピックアップするかを選択します。（両方の選択肢がある場合に限り表示。サイズにより使用不可のケースあり）

  • 下端の Simulate スライダーを使用して、初期心合わせのさまざまなステップをシミュレートできます。

Handling セクション

このセクションで、心合わせ中の回転位置とその他ハンドリングオプションを設定します。

  • Rotor (Z, X) 設定でローター位置（回転中心）を制御し、心合わせ時のパーツ中心を基準に測定されます。

  • Propagate をクリックして、すべてのサイドのプロセスに同じ回転中心を使用します。

  • Speed % 設定は、回転速度を制御します（最大可能速度のパーセンテージ）。

Gauging セクション

この折りたたみ可能なセクションでは、さまざまな位置ストップ配置を切り替えられます。

  • Auto-Place 設定を使用して、異なる位置ストップ配置を循環します。

  • Slack 設定で、位置ストップ間の追加開き量（Z方向）を追加します。

Fine Position セクション

この折りたたみ可能なセクションで、ゲージ位置を手動編集できます。通常は不要です。

  • Left (Z, X) 設定で左位置ストップを制御します。

  • Right (Z, X) 設定で右位置ストップを制御します。

  • トリミング済み 表示は、トリミング済み / 未トリミング / 斜め の間で切り替わり、適用位置ストップタイプを示します。

7030 + KB 40 （フロント/背面ローディング）

KB 40搭載7030の場合、ローディング中に他オプションも利用可能：

  • Stack angle 設定で、KB 40パレット上パーツ回転方向を選択します。ゲージングセルを指すとスタック角度が視認できます：

  • Loading mode 設定で、背面心合わせ / フロント心合わせモードを選択。

  • Centering angle 設定で最初の心合わせ用角度を選択します（モードにより処理が異なります）。

Handling および Fine Position セクションは7030相当ですが、設定に応じ突き当てまたはフロントゲージ基準になります。

7020 （エルゴ心合わせ）

7020パネルベンダーにはフロントゲージがありません。最初の心合わせは突き当て基準です：

Centering Angle 設定で、最初の心合わせ中の回転方向を選択します。

Ergo-Centering セクション

  • Ergonomic load 設定が＜No＞ならエルゴ心合わせ無効。左/右ならサイクル有効。

  • Gripping plier で使用ペンチ選択。

  • Z-Right / Z-Left でグリップペンチ位置設定。

  • Regrip with both スイッチで搬送安定化用再グリップ有効化。

  • Z-Regrip 設定は再グリップ位置（再グリップ有効時のみ表示）。

Handling セクション

  • Grip with 設定で真空マニピュレータ (X3) / 回転クランプ (X2/X4) を選択。

  • Rotor (Z, X) 設定で回転クランプ位置を指定。

Gauging と Fine Position 設定は7030と同様ですが突き当てを制御します。
