ユーザーインターフェース

TecZone Bendは、クラリティ というコードネームの革新的なユーザーインターフェイスデザインを使用しています。TecZone Bendでの作業中は、すっきりと整頓されたミニマリスト ビューが表示されます。

TecZone Bendインターフェースでは、以下はありません：

  • 現在のコンテキストに関連するのはほとんどないのに、多くのアイコン行が積み重ねられています。

  • 何十ものメニューがあるメニュー構造。

  • ユーザーインターフェースのリボンが画面の多くのスペースを取ります。

代わりに、クラリティインターフェースは Select の原則に基づいて 操作 します。

  • 選択にはクリックします。

  • クラリティパネル がポップアップし、ここは選択したもの用の様々な Settings 対 操作 があります。

  • 多くの場合、パネルには、他の同様のものを選択して選択範囲を 拡張 するためのコントローラーがあります。

  • また多くの場合、パネルには、使っている様々なものを ナビゲート するコントローラーがあります。

これらユーザー インターフェイスの原則は、 TecZone Bendの全作動モードで一貫しています。 3 Dモデルのクリーンアップ、ネストティング済み材料のパーツ配置、またはプレスブレーキ ツーリングの編集にも、同じキーコンセプトのセットが使用されています。 これらの抽象的なアイデアを具体化するため、実際にクラリティ インターフェースが作動しているのを見てみましょう (曲げ加工TecZone Bendのレポート エディタTecZoneの例を取り上げます)。

ガイド付きツアー

これは、クラリティインターフェースの仕組みの簡単なツアーです。いくつかの簡単な概念がTecZone Bendのどこにでも繰り返し使われているので、いったんこれら概念を理解すれば、大抵のTecZone Bendを使うのに資料やトレーニングは必要ありません。つまり、このセクションを読めば、TecZone Bend ユーザーガイドの残りの部分はスキップできます！

サンプルレポート

レポートテンプレートの作成と編集の詳細については、後のセクションを参照してください。ここでは、クラリティ インターフェースの概念を説明するため、レポートで使われている用語をいくつか紹介します。

上の画像は、レポートに配置できるいくつかのエンティティのタイプです。

  • Label は単純なテキストです；フォント、テキストの配置、色などのさまざまなプロパティを設定できます。

  • Field は、基礎となるパーツからの実際のデータの一部です。

  • Shape は、レポートに背景、ハイライト、区切り線を追加するための単純な図形データ (楕円、長角、線など) です。

  • Table は、基になっているパーツのデータを表形式で表示します。この例では、このパーツの曲げプロセスのリストが表示され、それぞれのプロパティが表示されます。

原則：編集するにはクリック

フィールドをクリックすると、クラリティ パネル が表示されます（ここでは単にパネルと呼びます）。以下がその注釈付き図です：

フィールドをクリックして選択すると、編集用のパネルが表示されます。このパネルには以下のエレメントがあります：

  • キャプションバーでパネル位置を変更できます（パネル背景をドラッグ）。プッシュピン を設定すると、パネルは常に画面上のその場所で開きます。そうしなければ、パネルはマウスカーソルの近くに開きます。

  • 編集ボックス、数値入力、選択リスト、フォントおよび配置セレクターなどを使ってフィールドを編集します。

  • アクションボタンで選択したオブジェクトの操作を行います。ここでは Delete ボタンでフィールドを削除できます。

  • パネルには多くのエレメントがあることがあります。簡素化のため エクスパンダー でセクションに分割します。上の画像では Visual は閉、フォントは開状態です。エクスパンダーをクリックして開閉します。

原則：複数を選択しまとめて編集

Shift を押しながらクリックで追加選択できます。複数選択すると共通設定を同時編集するパネルが表示されます。

原則：選択サブパネルで選択拡張

多くのパネル下部に Select サブパネルがあります。類似オブジェクトの追加選択を支援します。

原則：ナビゲーションサブパネル

エンティティは階層（ツリー）構造。例：レポート → 項目 → テーブル列。

テーブル選択時：

Prev / Next で同レベル移動。 Report で親へ。

根では下方向のみ： 項目。テーブルから列編集へは Columns。

列レベルでは Prev/Next が列間を移動。 Table で戻る。

概要

  • クリックして編集。

  • 複数選択で一括編集。

  • 異なるタイプも共通項目を編集。

  • 複数選択で追加操作出現。

  • 選択サブパネルで選択拡張。

  • ナビゲーションサブパネルで階層移動。
