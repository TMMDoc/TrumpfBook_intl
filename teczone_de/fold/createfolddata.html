Falzdaten erstellen

Sie können Falztechnologiedaten (Rüsten von Schwenkbiegemaschinen) für ein Blechteil mit dem Falzmodul erstellen. Nehmen wir an, Sie haben bereits eine Maschine installiert und ein CAD-Teil (2D oder 3D) importiert:

Umschalten auf Falz-CAM

Jetzt kann das Teil durch einfaches Drücken der Taste F für eine Schwenkbiegemaschine gerüstet werden. TecZone Fold öffnet ein Menü, in dem Sie eine der installierten Schwenkbiegemaschinen wählen können. Das Teil wird dann für diese Maschine gerüstet. Es werden die Biegefolge, die Zusammensetzung der Niederhalter, die Anschläge und optionale ZBW- oder ENW-Werkzeuge berechnet, und Sie sollten ein Ergebnis wie dieses sehen:

Liegen keine Fehler vor, wird das NC-Programm für das Teil generiert und in dem Ausgabeordner gespeichert, der für diese Maschine eingerichtet wurde.

Siehe auch den Fensterbereich Workflow, auf den Sie durch Drücken der Taste W zugreifen können. Dies bietet mehr Kontrolle über die Erstellung von Biegetechnologiedaten und Optionen zum Routen des Teils durch Laser-Cam, Schachteln usw.

Sonstige Vorgänge

Hier finden Sie eine kurze Zusammenfassung einiger Vorgänge, die Sie durchführen können, sobald ein Teil zum Biegen gerüstet wurde. Die folgenden Seiten dieses Handbuchs enthalten weitere Einzelheiten zu jedem dieser Vorgänge.

  • Wenn Sie auf die Spacebar klicken, wird eine Schwenkbiegemaschinensimulation gestartet. Sie können auch die Simulationssteuerelemente im Falznavigator verwenden, um die Simulation zu starten, zu stoppen oder zurückzuspulen:

  • Wenn Warnungen oder Fehler auftreten, erscheinen sie im Falznavigator, und Sie können sie durch Anklicken der entsprechenden Zellen anzeigen und beheben.

  • Sie können die Falzwerkzeuge (Niederhalter, Anschläge, ENW- und ZBW-Werkzeuge, Greifer) bearbeiten.

  • Sie können die Maschine ändern, indem Sie auf den Maschinennamen auf dem Tab unter dem Teil klicken und eine andere Maschine wählen:

Wenn ein Falzprogramm fehlerfrei erstellt wurde, wird das NC-Programm des Teils generiert. Sie können dieses Verhalten (ob das NC-Programm automatisch generiert wird) über die Seite Falzausgaben des Dialogs „Einstellungen“ steuern. Wenn ein Programm generiert wurde, ist die Schaltfläche NC-Programm in der linken Symbolleiste ausgegraut:

Wenn Sie Änderungen am Teil vornehmen, wird diese Schaltfläche wieder aktiviert, und Sie können das NC-Programm neu generieren, indem Sie erneut auf dieses Icon klicken (oder die Taste C drücken).
