Ändern der Geometrie

In TecZone Bend steht Ihnen ein leistungsstarker 2D-Editor für Skizzen zum Ändern, Bereinigen oder Ergänzen Ihrer Geometrie zur Verfügung. Verwenden Sie den Shortcut S, um in den Skizzenmodus zu gelangen. Der Editor zeigt das Teil in abgewickelter Ansicht an.

Klicken Sie in der Abwicklung auf das Icon Bereinigen oder verwenden Sie den Shortcut S.

Es öffnet sich ein Menü mit verschiedenen Icons zum Bearbeiten der Abwicklung:

Fensterbereich „Skizzieren“

Icon Symbol Bedeutung

Auswählen

Objekte, Linien, Einträge, usw. auswählen

Linie

Eine Linie skizzieren

Verbundene Linien

Skizziert beliebig viele Linien

Parallele

Skizziert eine Parallele zu einer Linie

Normal

Skizziert eine Tangente zu einer Kurve

Biegelinie

Skizziert eine normale Linie zu einer Linie

Mittelpunktbogen

Eine Biegelinie skizzieren

2-Punkt-Kreisbogen

Skizziert einen Kreisbogen aus einem Mittel-, Anfangs- und Endpunkt

3-Punkt-Kreisbogen

Skizziert einen Kreisbogen durch Angabe von zwei Punkten (Anfangs- und Endpunkt)

Tangentialer Kreisbogen

Skizziert einen Kreisbogen, der zu Skizzenelementen tangential ist

Rechteck

Skizziert ein Rechteck

Mittelpunkt-Rechteck

Skizziert ein Rechteck von der Mitte

Kreis

Skizziert einen Kreis. Wählen Sie den Mittelpunkt eines Kreises aus und ziehen Sie den Cursor, um den Radius festzulegen, oder geben Sie einen Wert für den Radius ein

Umkreis mit 2 Punkten

Skizziert einen Kreis entsprechend des Umfangs. Wählen Sie einen Punkt auf dem Umfang aus, dann einen zweiten und einen dritten Punkt.

Umkreis mit 3 Punkten

Skizziert einen Kreis entsprechend des Umfangs. Wählen Sie einen Punkt auf dem Umfang aus, dann einen zweiten und einen dritten Punkt.

Kreis mit 2 Tangenten

Skizziert einen Kreis mit zwei Tangenten. Geben Sie den Kreisdurchmesser ein und wählen Sie danach die erste und dann die zweite Tangente.

Kreis mit 3 Tangenten

Skizziert einen Kreis mit drei Tangenten. Geben Sie den Kreisdurchmesser ein und wählen Sie danach die erste, dann die zweite und dann die dritte Tangente.

Polygon Mittelpunkt - Eckpunkt

Skizziert einen Kreis mit drei Tangenten. Geben Sie den Kreisdurchmesser ein und wählen Sie danach die erste, dann die zweite und dann die dritte Tangente.

Polygon Mittelpunkt - Mitte Seitenkante

Skizziert ein Polygon. Geben Sie die Anzahl der Seiten an und wählen Sie einen Mittelpunkt und die Mitte einer Seitenkante

Polygon mit Seitenkante

Skizziert ein Polygon. Geben Sie die Anzahl der Seiten an und definieren Sie den Anfangs- und den Endpunkt einer Seite.

Verrundung

Verrundet die Ecke am Schnittpunkt von zwei Skizzenelementen mit einem eingegebenen Radius, wodurch ein tangentialer Bogen erstellt wird

Fase

Erstellt eine Fase an der Ecke am Schnittpunkt von zwei Skizzenelementen

Eckentrimmung rund

Trimmt die Ecke am Schnittpunkt von zwei Skizzenelementen mit einem eingegebenen Radius

Eckentrimmung eckig

Trimmt die Ecke am Schnittpunkt von zwei Skizzenelementen mit einem Rechteck. Die Größe des Rechtecks kann vorab eingegeben werden.

Freischnitt Quadratisch

Erstellt einen rechteckigen Freischnitt. Sie müssen Eckabstand und Freischnitttiefe eingeben und anschließend eine Ecke wählen.

Freischnitt Langloch

Erstellt einen langlochförmigen Freischnitt. Sie müssen Eckabstand, Freischnittbreite und -tiefe eingeben und anschließend eine Ecke wählen.

Freischnitt Dreieckig

Erstellt einen dreieckigen Freischnitt. Sie müssen Eckabstand, Freischnittbreite und -tiefe eingeben und anschließend eine Ecke wählen.

Schlüsselloch

Erstellt mit den eingegebenen Werten ein Schlüsselloch in einem Kreis.

Volle Verrundung 3 Segmente

Verrundet drei miteinander verbundene Skizzenelemente

Element verlängern

Wählen Sie das zu verlängerte Skizzenelement
ctrl Grenze auswählen
Shift Verbinden
Alt Ziehen

Element trimmen

Wählen Sie das zu trimmende Skizzenelement
Ctrl Grenze auswählen
Shift Element Löschen
Alt Polylinie löschen

Verschmelzen

Wird zum Trimmen und Vereinigen mehrerer einzelner Polylinien in eine verwendet

Offset Elemente

Versetzen Sie ein oder mehrere Skizzenelemente, skizzierte Modellkanten oder Modellflächen um einen festgelegten Abstand

Verschieben

Wählen Sie ein Skizzenelement mit ctrl, wählen Sie einen Bezugspunkt und verschieben Sie das Skizzenelement

Rotieren

Wählen Sie ein Skizzenelement mit ctrl, wählen Sie einen Rotationsmittelpunkt, dann einen Anfangspunkt und einen Endpunkt, um das Skizzenelement zu drehen

Skalieren

Wählen Sie ein Skizzenelement mit ctrl, wählen Sie einen Basispunkt, dann einen Anfangsreferenzpunkt und einen Endreferenzpunkt, um das Skizzenelement zu skalieren

Spiegeln

Wählen Sie ein Skizzenelement mit ctrl, dann den Anfang der Spiegellinie und danach das Ende der Spiegellinie, um das Skizzenelement zu spiegeln

Lineares Muster

Verwenden Sie lineare Muster zum Erstellen mehrerer referenzierter Kopien von einem oder mehreren Skizzenelementen, die Sie in gleichen Abständen entlang eines oder zweier linearer Pfade anordnen können. Klicken Sie auf das lineare Muster und geben Sie die gewünschten Werte ein

Kreismuster

Verwenden Sie Kreismuster zum Erstellen mehrerer referenzierter Kopien von einem oder mehreren Skizzenelementen, die Sie in gleichmäßigen Abständen um eine Achse herum anordnen können. Klicken Sie auf Kreismuster und geben Sie die gewünschten Werte ein

Flächen kombinieren

Wählen Sie zwei oder mehrere geschlossene Skizzenelemente, um Flächen miteinander zu vereinen

Schnittfläche

Wählen Sie zwei oder mehrere geschlossene Skizzenelemente, um eine Schnittfläche der ausgewählten Elemente zu erzeugen

Fläche trimmen

Wählen Sie zwei oder mehrere geschlossene Skizzenelemente, um die Fläche zu trimmen

Freischnitt kopieren

Sie können mehrere Kopien einer Klinkung an der Kante mithilfe dieses Werkzeugs herstellen. Geben Sie zuerst den Abstand zwischen den Kopien und die gewünschte Anzahl der Kopien der Klinkung ein. Wählen Sie dann die Klinkung durch Klicken auf die Zwei-Linien-Segmente, die an der Klinkung anliegen

Freischnitt löschen

Sie können eine Klinkung in einer Ecke oder entlang eines Liniensegments mit diesem Werkzeug löschen. Klicken Sie auf die Zwei-Linien-Segmente, die an der Klinkung anliegen, und die Klinkung wird entfernt

Freischnitt spiegeln

Sie können eine Klinkung in einer Ecke oder entlang eines Liniensegments mit diesem Werkzeug spiegeln. Klicken Sie auf die Zwei-Linien-Segmente, die an der Klinkung anliegen, und die Klinkung wird gespiegelt

Kurvenzug

Um einen neuen Spline zu starten, klicken Sie auf den Startpunkt. Wenn Sie auf die nachfolgenden Punkte klicken, wird der Spline erstellt. Wenn Sie den Spline schließen möchten, drücken Sie die Alt-Taste und klicken Sie dann auf

Profil

Geben Sie Basislänge, Bordhöhe, Dicke, Schenkelwinkel und Innenradius ein und drücken Sie Enter, um ein Profil zu erstellen

Text

Wird zum Zeichnen von Text verwendet, der auf dem Teil durch die Lasermaschine markiert wird. Wenn Sie auf diese Werkzeugschaltfläche klicken, zeigt die Eingabeschaltfläche Eingabefelder für den Text, die Größe und den Drehwinkel an

Text

Wird verwendet, um die Formen der Zeichen in jedem TrueType-Schriftfont in Polylinien zu konvertieren. Dann kann die Laserbearbeitung auf diese Polylinien angewandt werden und sie können geschnitten werden. Sobald die Schaltfläche das erste Mal angeklickt wird, erscheint der Schriftfontdialog, sodass Sie den Schriftfont auswählen können, der für den Text verwendet wird

Standardformen

Wird verwendet, um verschiedene übliche Formen zu erstellen und diese in die Zeichnung einzufügen. Wenn Sie auf diese Schaltfläche klicken, erscheint der Dialog „Form erstellen“, in dem Sie aus einer Palette an üblichen aufgelisteten Formen wählen können

Einfache Bemaßung

Wählen Sie den ersten Bemaßungspunkt, dann den zweiten Bemaßungspunkt, und positionieren Sie die Bemaßungslinie

Bezugsbemaßung

Wählen Sie den ersten Bemaßungspunkt, dann den zweiten Bemaßungspunkt, und positionieren Sie die Bemaßungslinie

Maßkette

Wählen Sie den ersten Bemaßungspunkt, dann den zweiten Bemaßungspunkt, positionieren Sie die Bemaßungslinie und wählen Sie den nächsten Bemaßungspunkt

Horizontales Ordinatenmaß

Ordinatenbemaßungen sind ein Satz von Bemaßungen, die von der Ordinate Null in der Zeichnung gemessen werden. Wählen Sie einen Referenzpunkt und positionieren Sie die Bemaßung

Vertikales Ordinatenmaß

Ordinatenbemaßungen sind ein Satz von Bemaßungen, die von der Ordinate Null in der Zeichnung gemessen werden. Wählen Sie einen Referenzpunkt und positionieren Sie die Bemaßung

Winkel Bemaßung

Erstellt eine Bemaßung für einen Winkel. Wählen Sie die erste und dann die zweite Linie, an der Sie den Winkel bemaßen möchten

Radius Bemaßung

Erstellt eine Bemaßung für einen Radius. Wählen Sie den Kreis aus, an dem Sie den Radius bemaßen möchten. Bemaßen Sie den Durchmesser mit ctrl

Radius Bemaßung mit durchgezogener Hinweislinie

Erstellt eine Bemaßung mit durchgezogener Hinweislinie für einen Radius. Wählen Sie den Kreis aus, an dem Sie den Radius bemaßen möchten. Bemaßen Sie den Durchmesser mit ctrl

Beschriftung

Wird zum Hinzufügen von Hinweisen an zur Zeichnung in Form von Beschreibungen verwendet. Um eine Beschreibung zu erstellen, geben Sie den anzuzeigenden Text ein, klicken Sie dorthin, wo der Pfeil hinzeigen soll, und klicken Sie noch einmal, um zu zeigen, wo der Text platziert werden soll.

Segment Bemaßung

Wird zum Hinzufügen von Bemaßung zu einer geraden Linie und gebogenen Segmenten verwendet. Klicken Sie auf das zu bemaßende Segment und klicken Sie noch einmal, um die Bemaßung zu positionieren. Oder halten Sie gedrückt und klicken Sie auf ein Segment, um die Bemaßung automatisch zu positionieren.
