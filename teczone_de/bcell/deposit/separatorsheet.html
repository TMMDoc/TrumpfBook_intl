Separator-Bleche hinzufügen

Separator-Bleche hinzufügen

Einige Ablagemuster können Separator-Bleche zwischen den Schichten von Teilen verwenden, um die Ablage stabiler zu machen. Durch Aktivieren des Kontrollkästchens Add separator beim Bearbeiten einer Ablage wird eine weitere Palette mit den Separator-Blechen hinzugefügt.

Separator-Panel

Sie können auf das Separator-Blech in der Simulation klicken, um das oben gezeigte Panel Separator aufzurufen.

  • Der Name-Auswahlbereich kann verwendet werden, um ein anderes Separator-Blech auszuwählen. Sie können den Link Import…​ in dieser Auswahl verwenden.

  • Der Abschnitt Pick (Z,X) wird verwendet, um die Position und Ausrichtung des Separator-Blechs auf der Aufnahmepalette einzurichten. Im obigen Bild wurde das Separator-Blech um 90° gedreht und zu einer Seite der Palette verschoben.

  • Der Abschnitt Drop (Z,X) wird verwendet, um die Position und Ausrichtung des Separator-Blechs auf dem Ablagestapel einzurichten.

  • Der Abschnitt Gripper (Z,X) wird verwendet, um die Position und Ausrichtung des Greifers auf dem Separator-Blech einzurichten. Dies könnte verwendet werden, um den Greifer von Schlitzen oder Löchern im Separator-Blech fernzuhalten. (Das obige Bild zeigt den Greifer um 135° von der Standardausrichtung gedreht).

  • Der Simulieren-Schieberegler kann verwendet werden, um nur den Separator-Blech-Zyklus zu simulieren, sodass Änderungen, die Sie an den obigen Einstellungen vornehmen, schnell getestet werden können.

  • Der Link Stack wird verwendet, um zur Ablagebearbeitung zu wechseln, und der Link Palette wird verwendet, um die Position und Ausrichtung der Palette zu bearbeiten, auf der das Separator-Blech platziert wird.

Benutzerdefinierte Separator-Bleche

Sie können auch ein benutzerdefiniertes Separator-Blech aus dem Menü Datei importieren:

Wenn dies korrekt funktioniert, wird eine Importmeldung wie diese angezeigt (andernfalls wird eine entsprechende Fehlermeldung angezeigt). Das Format der DXF- oder GEO-Datei zur Definition eines benutzerdefinierten Separator-Blechs wird auf dieser Seite erläutert.

Gewichtsprüfung

Das Separator-Blech kann für den Greifer zu schwer sein. In diesem Fall können Sie eine Fehleranzeige im Navigator-Panel sehen, und wenn Sie auf den Fehler zeigen, werden weitere Details über die Überlastung angezeigt.

Saugprüfung

Wenn die Saugnäpfe auf einem Loch des Separator-Blechs liegen, wird der Saugverlust erkannt und als Fehler in der Zeile Greifer der Spalte D1 im Biegenavigator angezeigt.
