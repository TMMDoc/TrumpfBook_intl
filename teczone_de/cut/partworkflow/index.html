Teileworkflow

Mit dem Schneidemodul können Sie Schneidtechnologiedaten für ein Blechteil erstellen. Nehmen wir an, Sie haben bereits eine Maschine installiert, ein CAD-Teil (2D oder 3D) importiert und es bereinigt, um ein verwendbares Blechteil zu erhalten. Sie können auf den Materialnamen und die Blechdicke im Tab „Dokument“ unten klicken, um diese Werte zu bearbeiten, wie auf der Seite 2D-Cleanup erläutert.

Umschalten auf Schneide-CAM

Jetzt kann das Teil durch einfaches Drücken der Taste L für das Laserschneiden gerüstet werden. Das Teil wird mit der Standardlasermaschine gerüstet, bei der es sich um die Lasermaschine handelt, die Sie zuletzt verwendet haben. Schnittbahnen, Anfahrsegmente, Eckenbearbeitung und Werkzeugreihenfolge sind vollständig eingerichtet, und Sie sollten ein Ergebnis wie dieses sehen:

Der Tab unten zeigt an, dass das Teil für die Maschine FX-LaserTUS gerüstet wurde und die LTT (Lasertechnologietabelle) ST030MD0-N2S0-30-2 verwendet.

  • Sie können das Teil für eine andere Maschine rüsten, indem Sie auf den Maschinennamen klicken und eine neue Maschine wählen.

  • Sie können eine andere LTT aus den für die gewählte Maschine, das Material und die Dicke verfügbaren wählen, indem Sie auf den LTT-Namen klicken und eine andere wählen:

Siehe auch den Fensterbereich Workflow, auf den Sie durch Drücken der Taste W zugreifen können. Dies bietet mehr Kontrolle über die Erstellung von Schneidtechnologiedaten und Optionen zum Routen des Teils durch Biegen und Schachteln.

Reihenfolgennavigator

Der Auto-Tooler berechnet auch eine Reihenfolge, in der die Laserschnitte für das Teil abgearbeitet werden sollen. Dieselbe teilinterne Reihenfolge kann auch verwendet werden, wenn das Teil zusammen mit anderen Teilen in ein Layout eingefügt wird. Sie können die Verfahrlinien zwischen den Schnitten sehen, wenn Sie die Einstellung Verfahrbewegungen in der Symbolleiste links umschalten (oder die Taste X verwenden).

Sie können die Reihenfolge auch bearbeiten, indem Sie den Reihenfolgennavigator am rechten Fensterrand öffnen. Klicken Sie dazu auf den Chevron unter dem Tab oder drücken Sie die Taste Z.

Jedes Rüstelement im Teil hat einen Eintrag im Reihenfolgennavigator, und Sie können Sie das Rüsten neu anordnen, indem Sie diese Elemente durch Ziehen und Ablegen verschieben. Sie können hier auch einige weitere Vorgänge durchführen. Beispielsweise können Sie steuern, ob der Kopf vor dem Verfahren zur nächsten Kontur angehoben wird, usw.

Der Reihenfolgennavigator wird auf der Seite Layout-Reihenfolge besprochen. Dort verwenden wir den Navigator, um die Reihenfolge für ein komplettes Layout (Tafel mit mehreren Teilen) anzuzeigen und zu bearbeiten.
