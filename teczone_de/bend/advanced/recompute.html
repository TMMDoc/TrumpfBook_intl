Programm neu berechnen

Programm neu berechnen

Wenn Sie B drücken oder den Fensterbereich „Workflow“ verwenden, um ein Programm zu berechnen, berechnet TecZone Bend ein Biegeprogramm anhand eines Satzes von Standardeinstellungen und Parametern. Sie können dieses Programms mit verschiedenen Einstellungen neu berechnen lassen, indem Sie von der Befehlsleiste aus auf Neu berechnen im Menü klicken. Manchmal ist dies auch hilfreich, um die Berechnung abzuschließen, nachdem Sie das Rüsten oder die Reihenfolge interaktiv bearbeitet haben.
Zum Beispiel:

  • Sie können die Reihenfolge ändern und TecZone Bend auffordern, neue Werkzeugstationen zu berechnen, die Reihenfolge aber beizubehalten.

  • Sie könnten die Werkzeugstationen ändern und TecZone Bend auffordern, eine neue Reihenfolge zu berechnen oder das Teil auf die neu erstellten Stationen auszurichten.

  • Sie können den Rüstvorgang mit einem anderen Werkzeugsatz ausprobieren oder versuchen, ein bereits erstelltes Setup anzuwenden.

Der Dialog „Neu berechnen“

Der Dialog Neu berechnen sieht aus wie das nebenstehende Bild.

Überschreibungen

Dieser Abschnitt enthält einige Schalter, mit denen Sie das Standardverhalten von TecZone Bend überschreiben können.

  • Schalten Sie den Schalter Reihenfolge nicht ändern ein, um TecZone Bend zu zwingen, eine neues Programm zu berechnen, ohne die Reihenfolge zu ändern. TecZone Bend probiert dann verschiedene Werkzeugstationen, Anschlagpositionen oder Einlegerichtungen für Teile bei unveränderter Reihenfolge aus.

  • Wenn der Schalter Keine Winkelmessung. eingeschaltet ist, berechnet TecZone Bend ein Programm, das keine Winkelmessungen durchführt und keine Messsensoren (Sensorwerkzeuge) verwendet

  • Werkzeuge aus GEO-/DXF-Dateien ignorieren weist TecZone Bend an, die Rüsthinweise zu ignorieren, die häufig in GEO-Dateien und manchmal auch in DXF-Dateien zu finden sind.

Aufspannungen

In diesem Abschnitt wird die Strategie für Aufspannungen festgelegt. Standardmäßig berechnet TecZone Bend die Aufspannung von Grund auf neu, Sie können dieses Verhalten aber mit diesen Einstellungen ändern.

  • Neue Aufspannung berechnen ist das Standardverhalten - TecZone Bend berechnet ein eine Aufspannung mit allen Werkzeugen, die erlaubt sind (siehe Abschnitt „Werkzeuge“ unten).

  • Vorhandene Aufspannung beibehalten weist TecZone Bend an, keine Änderungen an der Aufspannung vorzunehmen; es wird versucht, alle Biegungen mit der aktuellen Aufspannung für das Teil abzuarbeiten. Wenn es bei einigen Biegungen zu Kollisionen kommt oder die Werkzeuglänge zum Bearbeiten nicht ausreicht, bleiben diese Fehler unkorrigiert und können angezeigt werden, wenn die Neuberechnung abgeschlossen ist.

  • Wenn Sie Setup aus Datei verwenden wählen, werden Sie aufgefordert, eine Biegeaufspannungsdatei (bsetup) auszuwählen. Die aus dieser Datei geladene Aufspannung wird dann zur Berechnung verwendet; weder wird diese Aufspannung geändert, noch werden zusätzliche Stationen angelegt. Aufspannungen, die Sie kürzlich aus einer Datei geladen haben, werden in der Auswahl angezeigt, die unter diesem Abschnitt erscheint, und Sie können dann auf die Schaltfläche + in der Nähe dieser Auswahl klicken, um eine andere Aufspannungsdatei auszuwählen.

  • Die Option Aufspannung von einem anderen Teil verwenden ist nur verfügbar, wenn gerade andere Teile geöffnet sind, die für dieselbe Abkantpresse gerüstet wurden. In diesem Fall können Sie TecZone Bend auffordern, dieselbe Aufspannung wie für eines dieser anderen Teile zu verwenden; es erscheint ein Auswahlfeld, damit Sie das gewünschte Teil auswählen können.

Werkzeuge

Wenn Sie im obigen Abschnitt die Option Neue Aufspannung berechnen auswählen, können Sie den Abschnitt Werkzeuge verwenden, um die Liste der Biegewerkzeuge auszuwählen, die zur Berechnung der neuen Aufspannung verwendet werden können. Wenn Sie sich für eine der anderen Optionen für die Aufspannung entscheiden, ist dieser Abschnitt nicht verfügbar.

  • Bevorzugte Werkzeuge für die Maschine ist die Standardoption. Für jede Abkantpresse können Sie eine Reihe von Werkzeugen tools[1] als die bevorzugten Werkzeuge auswählen. TecZone Bend verwendet diese Werkzeuge beim Rüsten des Teils.[2] (Wenn Sie für eine Maschine keine ausdrücklich bevorzugten Werkzeuge eingerichtet haben, verwendet TecZone Bend alle Werkzeuge im Bestand als bevorzugten Werkzeugsatz für die Maschine).

  • Wenn Sie Werkzeuge im Bestand wählen, verwendet TecZone Bend alle Werkzeuge im Bestand der Anlage. Wenn Sie den Werkzeugbestand nicht ausdrücklich eingerichtet haben, verwendet TecZone Bend alle Werkzeuge im Katalog als Ihren Bestand.

  • Alle Werkzeuge im Katalog ist die umfangreichste Werkzeugauswahl; TecZone Bend wird jedes Werkzeug im Werkzeugkatalog des Herstellers bei der Suche nach einer Lösung verwenden. Das kann zwar dazu führen, dass die Lösung für die Produktion möglicherweise nicht realisierbar ist (unter Verwendung des Bestands, den Sie zur Verfügung haben), es ist aber hilfreich, um spekulativ zu erkunden, welche Werkzeuge erforderlich wären, um ein besonders komplexes Teil herzustellen. (Darüber hinaus weist dieser Modus TecZone Bend auch an, dass es einen unbegrenzten Bestand an Werkzeugteilen annehmen soll - das heißt, unter der Annahme, dass Ihnen eine unbegrenzte Anzahl an Werkzeugteilen aller Längen zur Verfügung steht, wird das Teil optimal gerüstet.)

  • Sie können benutzerdefinierte Werkzeugsätze erstellen und in Werkzeuglistendateien (btools) speichern. Diese Dateien können dann verwendet werden, um das Teil zu rüsten, wenn Sie die Option Werkzeugsatz aus Datei verwenden wählen. Wenn Sie diese Option auswählen, werden Sie aufgefordert, eine Werkzeuglistendatei anzugeben, und kürzlich verwendete Werkzeuglistendateien werden Ihnen zur einfachen Auswahl angezeigt. Sie können auf das +-Zeichen in der Nähe dieser Liste klicken, um eine neue Werkzeugliste auszuwählen.

  • Um Stempel und Matrizen möglichst direkt zu kontrollieren, können Sie die Option Ober- und Unterwerkzeug vorgeben wählen. Dann haben Sie die Möglichkeit, genau die Stempel und Matrizen auszuwählen, die Sie verwenden möchten.

Reihenfolgenpriorität

Beim Festlegen der Reihenfolge der Biegungen in einem Teil gibt es einige potenziell widersprüchliche Anforderungen. Sollte TecZone Bend zum Beispiel versuchen, die Anzahl der Teiledrehungen zu minimieren, oder sollte es versuchen, die Biegungen zu minimieren, bei denen der Teileschwerpunkt innerhalb der Maschine liegt?
Es ist sehr wahrscheinlich, dass jeder Bediener einen anderen Satz von Prioritätsregeln hat, und es ist durchaus möglich, dass verschiedene Teile unterschiedliche Prioritätseinstellungen erfordern. Deshalb gibt Ihnen TecZone Bend die Möglichkeit, eine Neuberechnung mit einer anderen Prioritätenreihenfolge durchzuführen, indem Sie den

Abschnitt Reihenfolgenpriorität dieses Dialogs verwenden.

Verwenden Sie die Pfeilschaltflächen und auf der rechten Seite, um die Regeln in eine andere Reihenfolge zu bringen; je weiter oben ein Element ist, umso höher ist seine Priorität. Sie können einige dieser prioritären Elemente vollständig deaktivieren. Das weist TecZone Bend an, diese Elemente bei der Entscheidung über die Reihenfolge nicht als Faktor zu berücksichtigen. (Manchmal kann das Deaktivieren einiger dieser Elemente eine Reihenfolge ergeben, die wenige Werkzeugstationen verwendet).

  • Kollisionen beim Überbiegen vermeiden: Um Winkelgenauigkeit zu erreichen, müssen die meisten Biegungen leicht überbogen werden, um die Rückfederung auszugleichen. Diese Reihenfolgenregel weist TecZone Bend an, Biegungen in einer Reihenfolge durchzuführen, in der das Überbiegen möglich ist. In dem Teil unten sollten wir zum Beispiel zuerst Biegung 1 und dann Biegung 2 durchführen; die umgekehrte Reihenfolge würde das Überbiegen blockieren.

  • Werkzeuge mit niedriger Priorität vermeiden: Weist TecZone Bend an, die Verwendung von Werkzeugen mit niedriger Priorität oder von Werkzeugen, die nicht in der Liste der bevorzugten Werkzeuge stehen, nach Möglichkeit zu vermeiden.

  • Anschlagen an offenen Winkeln vermeiden: Anschlagpositionen sollten sich idealerweise auf derselben Ebene befinden wie die Biegeachse, und es sollte keine bereits bearbeiteten Biegungen zwischen der Biegelinie und dem Anschlagpunkt geben. Hier ist ein Beispiel für einen schlechten Anschlag, den diese Regel zu vermeiden versucht:

  • Längere Schenkellänge überwiegend vor Biegelinie: Besonders bei schweren Teilen ist es sinnvoll, Situationen zu vermeiden, in denen der Teileschwerpunkt hinter der Werkzeuglinie liegt. Diese Regel versucht, solche Situationen zu minimieren.[3]

  • Teilewendungen minimieren: Von Teilewendungen spricht man, wenn das Teil komplett umgedreht werden muss (aufeinanderfolgende Biegungen in verschiedenen Aufwärts- und Abwärtsrichtungen).

  • Teiledrehungen minimieren: Eine Teiledrehung liegt dann vor, wenn ein Teil in der horizontalen Ebene nach links oder rechts gedreht wird. Das gilt für den Bediener als einfacher als eine Teilewendung. Die Standardreihenfolge dieser letzten beiden Regeln versucht, Teiledrehungen gegenüber Teilewendungen zu bevorzugen.

1. Verwenden Sie dazu die Schaltfläche Bestand in der Befehlsleiste und wählen Sie im angezeigten Menü Bevorzugte Werkzüge.
2. TecZone Bend wird versuchen, nur die bevorzugten Werkzeuge für das Teil zu verwenden, wenn es aber zu Kollisionen oder anderen Problemen kommt, die nicht gelöst werden können, wird es auch andere Werkzeuge aus dem verfügbaren Bestand verwenden.
3. TecZone Bend hat einen gewissen Spielraum bei der Beurteilung, ob das Teilegewicht und der Abstand des Schwerpunkts von der Werkzeuglinie ein Problem darstellen. Technisch betrachtet handelt es sich um ein Teil, das bei einem nach hinten gerichteten Kippmoment von 0,15 Nm (Newtonmeter) nach hinten kippt.
