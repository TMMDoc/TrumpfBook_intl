Import/Export

In diesem Abschnitt behandeln wir das Konfigurieren von Importeinstellungen.

Klicken Sie auf der Startseite auf das Icon Einstellungen .

Importeinstellungen

In diesem Abschnitt behandeln wir das Konfigurieren von Importeinstellungen. Klicken Sie auf das Icon Einstellungen. Klicken Sie auf Import/Export und navigieren Sie zu Importeinstellungen.

Einheiten für DXF-Dateien - Wählen Sie hier entweder Millimeter oder Zoll aus.

Maximaler Abstand zwischen nicht Zusammengeführten + - Stellen Sie diesen Wert ein (0 > 1 mm). Beim Import eines Teils, dessen Linien/Bögen einen geringeren Abstand als den eingestellten Wert haben, führt die Software diese Linien/Bögen beim Import automatisch zusammen.

Maximale Blechdicke - Um ein großes Blechteil zu erkennen, muss der Schwellenwert für die Blecherkennung erhöht werden. (10 > 40 mm) Dieser Wert ist je nach verwendeter Einheit standardmäßig auf 25 mm oder einen Zoll gesetzt.

Punkteinheiten - Die ausgewählte Option bestimmt, wie Punkte importiert werden.

Alle importieren - Alle Punkte werden importiert und angezeigt.

Punkte auf Polylinien überspringen - Damit werden alle Punkte übersprungen, die auf Polylinien gefunden werden.

Alle überspringen - Damit werden beim Import alle Punkte übersprungen, und keiner wird angezeigt.

Ebenen in DXF-/DWG-Dateien ignorieren - DXF und DWG Zeichnung werden normalerweise in verschiedenen Schichten erstellt. Mit dieser Einstellung ignoriert die Software diese Schichten und verschiebt alle Objekte auf die Standardschicht.

Blöcke in 2D Zeichnung zerlegen - Zerlegt Gruppierungen beim Importieren

Weiße Objekte in Schwarz umwandeln - Mit dieser Einstellungen können Sie weiße Objekte beim Import in schwarze Objekte umwandeln.

Dunklere Farben während DXF-Import - Mit dieser Einstellung können Sie farbige Objekte beim Import verdunkeln.

Duplizierte Segmente entfernen - Mit dieser Einstellung können Sie jede erkannte doppelte Geometrie beim Import entfernen.

DXF Einstellungen

In diesem Abschnitt behandeln wir das Konfigurieren von DXF Einstellungen. Klicken Sie auf das Icon Einstellungen. Klicken Sie auf Import/Export und navigieren Sie zu DXF Einstellungen.

Winkel in dxf Datei sind Öffnungswinkel. - Aktivieren Sie diese Einstellung, wenn die Winkel in einer DXF als Öffnungswinkel gehandhabt werden sollen.

Exporteinstellungen

In diesem Abschnitt behandeln wir das Konfigurieren von Exporteinstellungen. Klicken Sie auf das Icon Einstellungen. Klicken Sie auf Import/Export und navigieren Sie zu Exporteinstellungen.

Keine POLYLINE-Objekte in DXF-Ausgabe - Normalerweise werden geschlossene Konturen beim Exportieren von DXFs als Polylinien ausgegeben. Einige CAD-Systeme können diese Ausgabe nicht verarbeiten. Durch diese Einstellung gibt die Software die DXF mit Linien und Bögen aus. Diese Dateien sind überall lesbar, jedoch sind die erzeugten Dateien größer, und die Verbindungen zwischen Linien und Bögen gehen verloren.

Beim Exportieren von DXF-Dateien Biegeinformationen ausgeben. - Aktivieren Sie diese Einstellung, wenn eine exportierte DXF-Datei eine Ausgabe mit Biegeparametern sein soll.

Bend-Infos im Starmatik-Format - Aktivieren Sie diesen Schalter, um die Biegeparameter in Starmatik-Format auszugeben. Hierbei wird ein Textelement genau in der Mitte jeder Linie positioniert, die eine Biegelinie sein soll.

Schwarz bei Ausgabe in Grau umwandeln - Beim Exportieren von 2D-Daten werden standardmäßig die Objekte in der DXF-Datei in Schwarz ausgegeben. Um in den CAD-Programmen die Objekte besser erkennen zu können, werden mit dieser Einstellung die Objekte in Grau ausgegeben.

MetaCAM starten mit dem Export von PDG Dateien. - Aktivieren Sie diese Einstellung, wenn eine PDG-Datei automatisch zum Öffnen in MetaCAM konfiguriert werden soll.

Ausgabeformat der Abwicklung - Zum Exportieren eines Flachmusters kann hier das Dateiformat GEO, DXF oder PDG eingestellt werden.

Spline Konvertierung

In diesem Abschnitt werden wir das Konfigurieren von Einstellungen für Spline Konvertierung. Klicken Sie auf das Icon Einstellungen. Klicken Sie auf Import/Export und navigieren Sie zu den Einstellungen für Spline Konvertierung.

Splines während Import konvertieren - Stellen Sie hier ein, ob die Spline-Konvertierung ausgeschaltet ist oder Splines in Linien oder Kreisbögen konvertiert werden sollen. In jedem Fall wird jedes Spline in ein einzelnes Polylinienobjekt konvertiert, das Liniensegmente oder Kreisbogensegmente enthält.

Stützstellen-Berechnung - Die Anzahl der Linien und Kreisbögen, die erstellt werden, wird mit einem von zwei Mechanismen berechnet: Lochabstand oder Abweichung.

Länge pro Linien- oder Bogensegment - Wenn für die Berechnung der Knotenanzahl Lochabstand gewählt wurde, stellen Sie hier die Länge jedes Kreisbogen- oder Liniensegments ein, um das Spline mit dieser Schrittlänge zu untergliedern.

Maximale Abweichung während Approximation - Wenn für die Berechnung der Knotenzahl Abweichung gewählt wurde, stellen Sie hier die maximale Abweichung ein, die zwischen dem ursprünglichen glatten Spline und der Linien- oder Kreisbogenapproximation zulässig ist. Die Polylinie wird mit so wenig Segmenten wie möglich konstruiert, während der maximale Fehler innerhalb dieser Grenze gehalten wird.

Baugruppe auflösen

In diesem Abschnitt behandeln wir das Konfigurieren der Einstellungen für Baugruppe auflösen. Klicken Sie auf das Icon Einstellungen. Klicken Sie auf Import/Export und navigieren Sie zu den Einstellungen für Baugruppe auflösen.

Schneidteile ignorieren - Wenn Sie das beim Auflösen einer Baugruppe aktivieren, werden nur die Blechteile mit Biegelinien angezeigt. Ist die Funktion ausgeschaltet, werden alle Teile der Baugruppe angezeigt.

Anzeige von Komponenten (Einpressmuttern, Bolzen, + - Mit dieser Einstellung können Sie auswählen, welche Optionen angezeigt werden sollen, wenn Sie eine Baugruppe auflösen, die andere Komponenten enthält.

Aus - Nach dem Auflösen werden nur die Blechteile angezeigt, nicht aber andere Komponenten wie Einpressmutter oder Bolzen.

Unbekannt - Es werden nur Komponenten angezeigt, die in der Software noch identifiziert werden müssen.

Alle - Alle Komponenten werden angezeigt.

Layer Zuordnung

In diesem Abschnitt behandeln wir das Konfigurieren von Layer Zuordnung. Klicken Sie auf das Icon Einstellungen. Klicken Sie auf Import/Export und navigieren Sie zu den Einstellungen für Layer Zuordnung.

In diesem Abschnitt können Schichten, die auf in die Software importierte Teile angewendet werden, automatisch ihrer Funktion (Verwendung) zugeordnet werden.

Beispiel

Wenn ein importiertes Teil eine Schicht MARKIEREN hat, kann dies mit der Funktion Markierungsschicht in der Software automatisch eingestellt werden.

Layer Name - Das ist die Benennung der Schicht. Wenn ein Teil damit importiert wird, wird die Funktionalität verwendet, die im Fensterbereich „Verwendung“ eingestellt wurde.

verwenden - Das ist die Funktionalität der Schicht. Diese Optionen stehen zur Auswahl:

Standard - Das ist die Standardschicht, die für CAM verwendet werden soll.

Hilfsmittel - Eine Hilfsschicht, die nicht für CAM verwendet werden soll.

Markieren - Alle Elemente in dieser Schicht werden markiert, nicht geschnitten.

Annäherungsmarker - Punktelemente, die die Anfahrposition des Lasers angeben.

Sequenzmarker - Textmarker, die die Reihenfolge von Konturen angeben.

Umformungszentrum - Mittelmarkierung für die Umformung (Punkt oder kleines L).

Fußabdruck umformen - Die Umrandung (der Umriss) einer Umformung.

Verdampfen - Diese Schicht wird verwendet, um Folieneinbrand zu unterscheiden.

Punktmarkierung - Diese Schicht wird für QR-Codes verwendet.

Information - Diese Schicht dient nur zur Information.
