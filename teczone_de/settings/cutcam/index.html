Schneide-CAM

Schneide-CAM

Die Seite CAM schneiden des Dialogs „Einstellungen“ liefert den grundlegenden Satz von Einstellungen für Laser-Cam.

Finishing rules

Einzelheiten zu diesem Abschnitt finden Sie auf der Seite Abschließende Regeln.

Microjointeinstellungen

  • Länge - Geben Sie die Größe des Microjoints ein.

  • Max. zulässiges Eintauchen in die Auflageleisten (bestimmt die Anzahl der Microjoints) - Das ist die maximale Distanz, die der Microjoint in die Auflageleisten auf einer Kontur kippen darf.

  • Abstand (bestimmt auch die Anzahl der Microjoints) - Diese Einstellung bestimmt zusammen mit der obigen Einstellung die Anzahl der Microjoints. Eine der beiden Methoden, die mit der höheren Microjointanzahl, wird verwendet.

  • Obergrenze für die Anzahl der Microjoints auf der Kontur - Die maximal erlaubte Anzahl an Microjoints auf der Kontur.

  • Bevorzugter Microjointtyp:

    • Hart - Fest programmierter Microjoint, der nicht an der Maschine modifiziert werden kann.

    • Soft - Microjoint, der an der Maschine modifiziert werden kann.

    • Nano - Eine Alternative zum üblichen Microjoint, bei der kein Anfahren benötigt wird. Nur auf großen Konturen möglich. Üblicherweise bei der TwinLine-Bearbeitung verwendet.

  • Stabilität des Nanojoint - Kontrolliert die Stabilität der Nanojoints.

Nanojoints sind eine bessere Alternative zu üblichen Microjoints. Im Fall der TwinLine-Bearbeitung sind sie eine Innovation. TecZone Laser kann statt festen oder verstellbaren Microjoints direkt Nanojoints verwenden, wenn diese fortschrittliche Funktion auf der Maschine des Anwenders verfügbar ist. Im Einstellungsfenster für TruLaser-Maschinen können Anwender ihre bevorzugte Art von Microjoints auswählen.

Anordnung der Elemente

  • Start Ecke ist die Startecke der Bearbeitung. Die Starteckenmöglichkeiten sind „unten links“, „unten rechts“, „oben links“ und „oben rechts“.

  • Sotierrichtung ist die Reihenfolge der Teile auf der Tafel.

    • Auto - Sortiert die Teile automatisch, indem die Richtung mit dem kürzesten Abstand ausgewählt wird.

    • X Erste - Sortiert die Teile zuerst in X-Richtung.

    • Y Erste - Sortiert die Teile zuerst in Y-Richtung.

    • Gestaffelt - Die Reihenfolge wird so festgelegt, dass das nächste Teil einen maximalen Abstand zum zuletzt geschnittenen Teil hat, um die Wärmeableitung zu berücksichtigen.

Weitere Einstellungen

Diese Option bestimmt, ob die Punkte beim Importieren bearbeitet oder übersprungen werden sollten.

Keine - Alle Punkte importieren.

Punkte auf Polylinien - Punkte auf Polylinien überspringen.

Alle - Alle Punkte beim Import überspringen.

Vorschneiden

  • Blank ist bereits auf Größe geschoren - Diese Option wird aktiviert, wenn die Tafel bereits auf die richtige Größe geschnitten wurde.

  • Abstand zum Rand - Diese Option wird zum Einfügen eines Randbereichs verwendet, der an die Kante des Rohlings geschnitten werden soll.

  • Messabstand von der Blechkante - Abstand von der Tafelkante, in dem das Vorschneiden beginnen soll.

  • Overtravel nach Blechkante - Das ist der gesamte erlaubte Abstand des Nachlaufwegs zur Tafelkante.
